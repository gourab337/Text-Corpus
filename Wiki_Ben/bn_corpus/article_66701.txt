'''রাজামেহার''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == রাজামেহার ইউনিয়ন পূর্বে ১১নং রাজামেহার উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার রাজামেহার ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পূর্বে গুনাইঘর দক্ষিণ ইউনিয়ন, পূর্বে ধামতী ইউনিয়ন, দক্ষিণে ভানী ইউনিয়ন, পশ্চিমে মুরাদনগর উপজেলার বাবুটিপাড়া ইউনিয়ন পাহাড়পুর ইউনিয়ন এবং উত্তরে মুরাদনগর উপজেলার ধামঘর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রাজামেহার ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
