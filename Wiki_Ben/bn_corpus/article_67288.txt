'''জিনোদপুর''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == জিনোদপুর ইউনিয়নের আয়তন ৩৩৬৪ একর (১৩.৬১ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে জিনোদপুর ইউনিয়নের মোট জনসংখ্যা ২০,৯৩৯ জন। মোট পরিবার ৪,১২২টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৫৩৮ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার দক্ষিণাংশে জিনোদপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে সাতমোড়া ইউনিয়ন রসুল্লাবাদ ইউনিয়ন, উত্তরে ইব্রাহিমপুর ইউনিয়ন, পূর্বে লাউর ফতেপুর ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার মুরাদনগর উপজেলার আকুবপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জিনোদপুর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == শামসুল হক –– বীর বিক্রম খেতাবপ্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
