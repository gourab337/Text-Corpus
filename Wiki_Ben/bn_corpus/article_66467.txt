'''শ্যামপুর ইউনিয়ন''' বাংলাদেশের ঢাকা জেলার তেজগাঁও উন্নয়ন সার্কেলের একটি প্রশাসনিক ইউনিয়ন। যা ঢাকা সিটি করপোরেশনের সাথে সঙ্গে একীভূত করা হয়। স্থানীয় সরকার (সিটি করপোরেশন) সীমানা পরিবর্তন (সম্প্রসারণ সংকোচন) বিধি ৫(২) বিধান অনুসারে ডিসিসির অন্তর্ভুক্ত করা হয়। == অবস্থান সীমানা == উত্তরে ঢাকা সিটি করপোরেশনের ৯০,৮৯নং ওয়ার্ড, পূর্বে দনিয়া ইউনিয়ন, পশ্চিমে বুড়িগঙ্গা নদী, দক্ষিণে কুতুবপুর ইউনিয়ন। == ইতিহাস == == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' চেয়ারম্যানগণের তালিকা == আরও দেখুন == দক্ষিণখান ইউনিয়ন ওয়ারী থানা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার ইউনিয়ন বিলুপ্ত ইউনিয়ন
