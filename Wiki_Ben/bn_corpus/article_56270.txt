'''দরবারপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফুলগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফুলগাজী উপজেলার মধ্যাংশে দরবারপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ফুলগাজী ইউনিয়ন, পূর্বে আমজাদহাট ইউনিয়ন, দক্ষিণে জিএমহাট ইউনিয়ন এবং পশ্চিমে মুন্সিরহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দরবারপুর ইউনিয়ন ফুলগাজী উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফুলগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফুলগাজী উপজেলা ফুলগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
