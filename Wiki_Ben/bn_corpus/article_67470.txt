পৌরসভা''' বাংলাদেশের জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == পৌরসভার আয়তন ৪৩৪৩ একর (১৭.৫৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে পৌরসভার মোট জনসংখ্যা ১,৭২,০১৭ জন। মোট পরিবার ৩৩,৫১৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৯৭৮৫ জন। == অবস্থান সীমানা == সদর উপজেলার মধ্যাংশে পৌরসভার অবস্থান। পৌরসভার পশ্চিমে নাটাই দক্ষিণ ইউনিয়ন নাটাই উত্তর ইউনিয়ন, উত্তরে নাটাই উত্তর ইউনিয়ন সুহিলপুর ইউনিয়ন, পূর্বে সুহিলপুর ইউনিয়ন, দক্ষিণ-পূর্বে মাছিহাতা ইউনিয়ন, দক্ষিণে রামরাইল ইউনিয়ন এবং দক্ষিণ-পশ্চিমে তিতাস নদী নবীনগর উপজেলার নাটঘর ইউনিয়ন অবস্থিত। == ইতিহাস == == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == পৌরসভা একটি 'ক' শ্রেণীর পৌরসভা। পৌরসভার প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। পৌরসভায় ১২টি ওয়ার্ড রয়েছে। ওয়ার্ডভিত্তিক পৌরসভার এলাকাসমূহ হল: ওয়ার্ড নং আওতাধীন এলাকা ১নং ওয়ার্ড মধ্য মেড্ডা, পুলিশ লাইন, পশ্চিম মেড্ডা, শরীফপুর ২নং ওয়ার্ড পাইকপাড়া (অংশ), পূর্ব মেড্ডা (মেরুড়া অংশ) ৩নং ওয়ার্ড খৈয়াসার, পশ্চিম পাইকপাড়া, ফুলবাড়িয়া, শেরপুর, জেলখানা ৪নং ওয়ার্ড বাজার এলাকা, পূর্ব পাইকপাড়া ৫নং ওয়ার্ড দক্ষিণ মধ্যপাড়া, উত্তর মধ্যপাড়া ৬নং ওয়ার্ড ছয়ঘড়িয়া, দাড়িয়াপুর, দক্ষিণ পৈরতলা, উত্তর পৈরতলা ৭নং ওয়ার্ড আমিনপুর, চণ্ডালখিল, ছোট গোকর্ণ, গোকর্ণ, ছয়বাড়িয়া ৮নং ওয়ার্ড কাজীপাড়া ৯নং ওয়ার্ড দক্ষিণ মৌড়াইল, দাতিয়ারা, নয়নপুর, পুনিয়াউট, উত্তর মৌড়াইল (অংশ) ১০নং ওয়ার্ড কান্দিপাড়া, কাউতলি, উত্তর মৌড়াইল (অংশ) ১১নং ওয়ার্ড উত্তর ভাদুঘর, শিমরাইলকান্দি ১২নং ওয়ার্ড দক্ষিণ ভাদুঘর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == আবদুর রহমান খান –– প্রাক্তন সমবায় পাট মন্ত্রী (পাকিস্তান শাসনামল)। == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা সদর উপজেলা
