'''ঝুলনপোল বেণী মাধব উচ্চ বিদ্যালয়''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার ইছাখালী ইউনিয়নে বিদ্যালয়টি অবস্থিত। == ইতিহাস == == পরিচালনা ব্যবস্থা == == == == অবকাঠামো == == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == কৃতিত্ব ফলাফল == == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
