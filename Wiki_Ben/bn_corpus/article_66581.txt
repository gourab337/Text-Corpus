'''দুর্গাপুর উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা আদর্শ সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == কুমিল্লা আদর্শ সদর উপজেলার উত্তর-মধ্যাংশে দুর্গাপুর উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কুমিল্লা সেনানিবাস; দক্ষিণে দুর্গাপুর দক্ষিণ ইউনিয়ন; পূর্বে কুমিল্লা সিটি কর্পোরেশন; উত্তরে গোমতী নদী, আমড়াতলী ইউনিয়ন বুড়িচং উপজেলার ষোলনল ইউনিয়ন এবং উত্তর-পশ্চিমে বুড়িচং উপজেলার ময়নামতি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দুর্গাপুর উত্তর ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা কোতোয়ালী থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৪নং নির্বাচনী এলাকা কুমিল্লা-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কুমিল্লা আদর্শ সদর উপজেলা কোতোয়ালী থানা, কুমিল্লা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == আদর্শ সদর উপজেলার ইউনিয়ন
