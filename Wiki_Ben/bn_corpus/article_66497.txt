বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == লক্ষ্মীপুর ইউনিয়ন পূর্বে পয়ালগাছা দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার সর্ব-দক্ষিণে লক্ষ্মীপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পয়ালগাছা ইউনিয়ন, পূর্বে লাকসাম উপজেলার মুদাফফরগঞ্জ ইউনিয়ন, দক্ষিণ-পূর্বে লাকসাম উপজেলার মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন, দক্ষিণ-পশ্চিমে চাঁদপুর জেলার শাহরাস্তি উপজেলার রায়শ্রী উত্তর ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার আশরাফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == লক্ষ্মীপুর ইউনিয়ন বরুড়া উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। এই ইউনিয়নটি নিম্নলিখিত গ্রাম নিয়ে গঠিত: পেরুল খাজুরিয়া নোয়াদ্দা অষ্টদোনা দুর্গাপুর কালিকাপুর ধুলিয়ামুড়ি মইশাইর ডাবুরিয়া মুড়াকোটা দৌলতপুর লক্ষ্মীপুর বদরপুর সিংগুরিয়া পাচপাড়া মনুয়া ডুমুরিয়া নলুয়া চাঁদপুর নোয়াপাড়া নলুয়া প্রসাদপুর পাঁচথুবী শ্বশুরকাটা রতনপুর নিশ্চন্তপুর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
