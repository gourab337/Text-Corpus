বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == শাহজাদাপুর ইউনিয়নের আয়তন ৭৭৭৩ একর (৩১.৪৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে শাহজাদাপুর ইউনিয়নের মোট জনসংখ্যা ২৪,২৮১ জন। এর মধ্যে পুরুষ ১১,৭৮৩ জন এবং মহিলা ১২,৪৯৮ জন। মোট পরিবার ৪,৯১০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ৭৭২ জন। == ইতিহাস == == অবস্থান সীমানা == সরাইল উপজেলার পূর্বাংশে শাহজাদাপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে নোয়াগাঁও ইউনিয়ন, দক্ষিণে শাহবাজপুর ইউনিয়ন, দক্ষিণ-পূর্বে বিজয়নগর উপজেলার বুধন্তি ইউনিয়ন, পূর্বে নাসিরনগর উপজেলার হরিপুর ইউনিয়ন, উত্তরে নাসিরনগর উপজেলার পূর্বভাগ ইউনিয়ন এবং উত্তর-পূর্বে নাসিরনগর উপজেলার গোকর্ণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শাহজাদাপুর ইউনিয়ন সরাইল উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে শাহজাদাপুর ইউনিয়নের সাক্ষরতার হার ৩৬.২%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
