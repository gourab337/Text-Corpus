'''বালিগাঁও''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার বালিগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে ধলিয়া ইউনিয়ন, পূর্বে কালিদহ ইউনিয়ন, উত্তরে ফেনী পৌরসভা, উত্তর-পশ্চিমে পাঁচগাছিয়া ইউনিয়ন, পশ্চিমে দাগনভূঁইয়া উপজেলার জায়লস্কর ইউনিয়ন মাতুভূঁইয়া ইউনিয়ন এবং দক্ষিণে সোনাগাজী উপজেলার চর মজলিশপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বালিগাঁও ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। ইউনিয়নের গ্রামগুলো হল: মরুয়ার চর কাতালিয়া মধুয়াই চর মধুয়াই বালিগাঁও আকরামপুর সুন্দরপুর বেতাগাঁও হকদি চর হকদি চর হুজুরি বারাহিগুণী বক্স বাজার ধুমসাহাদ্দা == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == বালিগাঁও উচ্চ বিদ্যালয় মধুয়াই উচ্চ বিদ্যালয় ধুমসাহাদ্দা উচ্চ বিদ্যালয় সুন্দরপুর উচ্চ বিদ্যালয় হকদি উচ্চ বিদ্যালয় আফতাব বিবি সিনিয়র ফাজিল (ডিগ্রী) মাদ্রাসা *ফকির হাট উচ্ছ বিদ্যালয় *সুন্দরপুর সফিকিয়া দাখিল মাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ১.সুন্দরপুর বাজার ২.আপতাব বিবির হাট ৩.মিয়ার বাজার ৪.ফকির হাট ৫.কাতালিয়া ৬.বক্স বাজার ৭.ইক্কার দোকান ৮.দঃ মরূয়ারচর == দর্শনীয় স্থান == বালিগাঁও স্কুল গায়েবী মসজিদ সুন্দরপুর সুন্দরপুর গ্যাস ফিল্ড == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
