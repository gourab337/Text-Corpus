'''কুটি''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == কুটি ইউনিয়নের আয়তন ৫০২২ একর (২০.৩২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে কুটি ইউনিয়নের মোট জনসংখ্যা ৩৯,৬৫১ জন। মোট পরিবার ৭,৬৩৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৯৫১ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার পশ্চিমাংশে কুটি ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে কায়েমপুর ইউনিয়ন, পূর্বে কসবা পশ্চিম ইউনিয়ন, উত্তরে মেহারী ইউনিয়ন, পশ্চিমে কুমিল্লা জেলার মুরাদনগর উপজেলার আন্দিকোট ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার মাধবপুর ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার ব্রাহ্মণপাড়া উপজেলার মাধবপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কুটি ইউনিয়ন কসবা উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
