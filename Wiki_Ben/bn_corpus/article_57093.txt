'''কাদরা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সেনবাগ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সেনবাগ উপজেলার উত্তর-পূর্বাংশে কাদরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ডুমুরুয়া ইউনিয়ন, পশ্চিমে ডুমুরুয়া ইউনিয়ন সেনবাগ পৌরসভা, দক্ষিণে সেনবাগ পৌরসভা এবং পূর্বে ফেনী জেলার দাগনভূঁইয়া উপজেলার রাজাপুর ইউনিয়ন সিন্দুরপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কাদরা ইউনিয়ন সেনবাগ উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
