বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মানিকছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মানিকছড়ি উপজেলার যোগ্যাছোলা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বাটনাতলী ইউনিয়ন, পূর্বে মানিকছড়ি ইউনিয়ন তিনটহরী ইউনিয়ন, দক্ষিণে চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার পাইন্দং ইউনিয়ন এবং পশ্চিমে চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার ভূজপুর ইউনিয়ন নারায়ণহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == যোগ্যাছোলা ইউনিয়ন মানিকছড়ি উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মানিকছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মানিকছড়ি উপজেলা মানিকছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
