'''দিঘিরপাড় ইউনিয়ন''' বাংলাদেশের মুন্সিগঞ্জ জেলার অন্তর্গত টঙ্গিবাড়ী উপজেলার একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == দিঘিরপাড় ইউনিয়নের মোট আয়তন ৯২৬৮ একর। গ্রাম ১৬ টি, মৌজা টি == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী দিঘিরপাড় ইউনিয়নের মোট জনসংখ্যা ৬২০৫ জন। এদের মধ্যে ৩০২১ জন পুরূষ এবং ৩১৮৪ জন মহিলা। == ইতিহাস == == শিক্ষা == শিক্ষার হার ৪৮.০৩%। প্রাথমিক বিদ্যালয় টি মাধ্যমিক বিদ্যালয় টি মাদরাসা টি == অর্থনীতি == == কৃতী ব্যক্তিত্ব == যোগেন্দ্রনাথ গুপ্ত ইতিহাস বেত্তা গবেষক মোঃ সামসুদ্দোহা পাকিস্তান আমলে খাদ্য কৃষি মন্ত্রী ছিলেন। দীনেশ চৌধুরী বৃটিশ বিরোধী আন্দোলনের নেতা == দশনীয় স্থান == পদ্মা নদীর চর বাহেরক গ্রামে আশ্রায়ণ প্রকল্প == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র == বিভাগের ইউনিয়ন বিষয়শ্রেণী: টংগিবাড়ী উপজেলার ইউনিয়ন বিষয়শ্রেণী: মুন্সিগঞ্জ
