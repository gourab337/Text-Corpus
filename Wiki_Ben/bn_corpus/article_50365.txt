'''চৌমুহনী পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == চৌমুহনী পৌরসভার আয়তন ২০.৭০ বর্গ কিলোমিটার। == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার মধ্যাংশে চৌমুহনী পৌরসভার অবস্থান। পৌরসভার উত্তর-পশ্চিমে মিরওয়ারিশপুর ইউনিয়ন, উত্তরে নরোত্তমপুর ইউনিয়ন, উত্তর-পূর্বে দুর্গাপুর ইউনিয়ন, পূর্বে হাজীপুর ইউনিয়ন, দক্ষিণে শরীফপুর ইউনিয়ন একলাশপুর ইউনিয়ন এবং দক্ষিণ-পশ্চিমে পশ্চিমে বেগমগঞ্জ ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯৭৩ সালে চৌমুহনী পৌরসভা প্রতিষ্ঠিত হয়। == প্রশাসনিক এলাকা == চৌমুহনী পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। এটি একটি 'ক' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == চৌমুহনী পৌরসভার জনসংখ্যা প্রায় লক্ষ ৬৫ হাজার। == শিক্ষা == চৌমুহনী সরকারি এস কলেজ চৌমুহনী মদন মোহন উচ্চ বিদ্যালয় বেগমগঞ্জ সরকারি উচ্চ বিদ্যালয় বেগমগঞ্জ কারিগরি উচ্চ বিদ্যালয় গনিপুর পাইলট বালিকা উচ্চ বিদ্যালয় মদন মোহন উচ্চ বিদ্যালয় আব্দুল মালেক উকিল মেডিকেল কলেজ == অর্থনীতি == == জনপ্রতিনিধি == পৌর মেয়র: আক্তার হোসেন ফয়সাল == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
