'''অর্জুনতলা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সেনবাগ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সেনবাগ উপজেলার উত্তর-মধ্যাংশে অর্জুনতলা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কেশারপাড় ইউনিয়ন ডুমুরুয়া ইউনিয়ন, পূর্বে সেনবাগ পৌরসভা, দক্ষিণে কাবিলপুর ইউনিয়ন এবং পশ্চিমে সোনাইমুড়ি উপজেলার অম্বরনগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == অর্জুনতলা ইউনিয়ন সেনবাগ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == অর্জুনতলা ইউনিয়নের প্রধান ৩টি হাট-বাজার হল ছিলোনিয়া বাজার, দরবেশ হাট, হাজির হাট। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
