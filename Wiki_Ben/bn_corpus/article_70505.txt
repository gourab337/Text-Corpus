''' ----------- ইউনিয়ন''' বাংলাদেশের মুন্সিগঞ্জ জেলার অন্তর্গত টঙ্গিবাড়ী উপজেলার একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == ---------- ইউনিয়নের মোট আয়তন ------- একর। গ্রাম টি, মৌজা টি == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী -------- ইউনিয়নের মোট জনসংখ্যা ---------- জন। এদের মধ্যে --------- জন পুরূষ এবং ----------- জন মহিলা। == ইতিহাস == == শিক্ষা == শিক্ষার হার প্রাথমিক বিদ্যালয় টি মাধ্যমিক বিদ্যালয় টি মাদরাসা টি == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == দশনীয় স্থান == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র == বিভাগের ইউনিয়ন বিষয়শ্রেণী: টংগিবাড়ী উপজেলার ইউনিয়ন বিষয়শ্রেণী: মুন্সিগঞ্জ
