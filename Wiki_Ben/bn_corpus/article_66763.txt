বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার ছালিয়াকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে জাহাপুর ইউনিয়ন, পূর্বে দারোরা ইউনিয়ন পাহাড়পুর ইউনিয়ন, দক্ষিণে পাহাড়পুর ইউনিয়ন, পশ্চিমে তিতাস উপজেলার ভিটিকান্দি ইউনিয়ন এবং উত্তর-পশ্চিমে তিতাস উপজেলার কলাকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছালিয়াকান্দি ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ১৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মুরাদনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা মুরাদনগর থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
