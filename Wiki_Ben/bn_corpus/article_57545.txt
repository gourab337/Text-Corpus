'''চর কিং''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত হাতিয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাতিয়া উপজেলার মূল দ্বীপের উত্তর-পশ্চিমের উপদ্বীপে চর কিং ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে হাতিয়া চ্যানেল জাহাজমারা ইউনিয়ন; পূর্বে হাতিয়া চ্যানেল, হাতিয়া পৌরসভা সুখচর ইউনিয়ন; উত্তরে মেঘনা নদী, সুবর্ণচর উপজেলার চর ওয়াপদা ইউনিয়ন লক্ষ্মীপুর জেলার রামগতি উপজেলার চর গাজী ইউনিয়ন এবং পশ্চিমে হাতিয়া চ্যানেল ভোলা জেলার মনপুরা উপজেলার মনপুরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর কিং ইউনিয়ন হাতিয়া উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাতিয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৩নং নির্বাচনী এলাকা নোয়াখালী-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাতিয়া উপজেলা হাতিয়া থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
