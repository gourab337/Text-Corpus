'''আলীনগর ইউনিয়ন''' বাংলাদেশের সিলেট বিভাগের মৌলভীবাজার জেলার কমলগঞ্জ উপজেলার একটি ইউনিয়ন। == অবস্থান == কমলগঞ্জ উপজেলা হতে ০৬ নং আলীনগর ইউনিয়ন পরিষদের দুরত্ব ৩.১৭ কি:মি:। == ইতিহাস == পাহাড়ের অরন্যের প্রাকৃতিক নয়নাভিরাম চা বাগানের ঘেরা আলীনগর কমলগঞ্জ উপজেলার ঐতিহ্য সমৃদ্ধ ইউনিয়ন। মনিপুরি তাঁত শিল্প, পান পুঞ্জি, কুটির শিল্প, মৃত কারু শিল্প, বাস বেতের তৈরী আসবাবপত্র, বিভিন্ন রকমের পাটি তৈরী, ইত্যাদি স্ব-স্ব ঐতিহ্যে রয়েছে। == আয়তন জনসংখ্যা == '''আয়তন-''' ৩৭.৩৪ বর্গ কি:মি:। '''জনসংখ্যা-''' ২৯,১১১জন (২০১১ সালের আদমশুমারী অনুসারে)। == গ্রাম সমূহ == কামারছড়া পুনজি, কামারছড়া চা বাগান, জাঙ্গালীয়া, আলীনগর চা বাগান, আলীনগর বস্থী, মথুরাপুর, গকুলনগর, ঘোষপুর(কালীপুর) কামুদপুর, গবিন্দপুর, জালালীয়া, নছরতপুর, রাজকান্দি, রামেশ্বরপুর, মংগলপুর, শ্রীনাথপুর, বারামপুর, যোগীবিল, তিলকপুর, ফুলতলী, সুনছড়া চা বাগান, চিতলীয়া। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার:''' ৯০.১% '''শিক্ষা প্রতিষ্ঠাণ:''' সরকারী প্রাথমিক বিদ্যালয়ের সংখ্যা- ১৮টি উচ্চবিদ্যালয়- ২টি (জুনিয়রস্কুল ১টি) মাদ্রাসা- ১টি == ভাষা সংস্কৃতি == মনিপুরীদের ভাষা মনিপুরী চা বাগানের ভাষা হিন্দি, বাংলা অন্যান্য ভাষা বাংলা, ইংরেজী খাসিয়া পুঞ্জির লোকদের ভাষা খাসিয়া মনিপুরী সংস্কৃতি। ধর্মিয় পুজা, রাস উৎসব, অন্যান্য আচার- অনুষ্টানে তাদের গুরুত্বপুর্ন অবদান রয়েছে == দর্শনীয় স্থান == আলীনগর চা বাগান সুনছড়া চা বাগান কামারছড়া চা বাগান হরিনারায়ণ দিঘী == হাট-বাজার == সংখ্যা- ৪টি == খাল নদী == নদী নেই, তবে দুইটি- সুনছড়া, লাগাটাছড়া। == আরও দেখুন == আলীনগর ইউনিয়ন, বিয়ানীবাজার আলীনগর ইউনিয়ন -চাঁপাইনবাবগঞ্জ আলীনগর ইউনিয়ন, ভোলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাপিডিয়ায় ''কমলগঞ্জ উপজেলা'' আলীনগর ইউনিয়ন- বাংলাদেশ জাতীয় তথ্য বাতায়ন উপজেলার ইউনিয়ন
