'''পেরুল দক্ষিণ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লালমাই উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লালমাই উপজেলার পেরুল দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পেরুল উত্তর ইউনিয়ন, পূর্বে বেলঘর উত্তর ইউনিয়ন, দক্ষিণে লাকসাম উপজেলার লাকসাম পূর্ব ইউনিয়ন, দক্ষিণ-পশ্চিমে লাকসাম পৌরসভা এবং পশ্চিমে লাকসাম উপজেলার কান্দিরপাড় ইউনিয়ন বাকই ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পেরুল দক্ষিণ ইউনিয়ন লালমাই উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লালমাই থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লালমাই উপজেলা লালমাই থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
