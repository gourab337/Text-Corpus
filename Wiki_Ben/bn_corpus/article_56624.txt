'''রাজাপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার উত্তরাংশে রাজাপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে পূর্ব চন্দ্রপুর ইউনিয়ন, পূর্বে পূর্ব চন্দ্রপুর ইউনিয়ন জায়লস্কর ইউনিয়ন, উত্তর-পূর্বে ফেনী সদর উপজেলার পাঁচগাছিয়া ইউনিয়ন, উত্তরে সিন্দুরপুর ইউনিয়ন এবং পশ্চিমে নোয়াখালী জেলার সেনবাগ উপজেলার কাদরা ইউনিয়ন সেনবাগ পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == রাজাপুর ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
