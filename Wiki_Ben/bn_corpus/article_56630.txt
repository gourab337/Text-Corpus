বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == দাগনভূঁইয়া ইউনিয়নের আয়তন ১০.০৯ বর্গ কিলোমিটার। এটি আয়তনের দিক থেকে ফেনী জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার দক্ষিণাংশে দাগনভূঁইয়া ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ইয়াকুবপুর ইউনিয়ন দাগনভূঁইয়া পৌরসভা; উত্তরে রামনগর ইউনিয়ন, দাগনভূঁইয়া পৌরসভা মাতুভূঁইয়া ইউনিয়ন; পূর্বে সোনাগাজী উপজেলার চর মজলিশপুর ইউনিয়ন এবং দক্ষিণে নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার চর পার্বতী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দাগনভূঁইয়া ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == সরকারি ইকবাল মেমোরিয়াল কলেজ ফাজিলেরঘাট উচ্চ বিদ্যালয় আতাতুর্ক সরকারি মডেল প্রাথমিক বিদ্যালয় দাগনভুইয়া অ্যাকাডেমি দাগনভূঞা বালিকা উচ্চ বিদ্যালয়। == যোগাযোগ ব্যবস্থা == দাগনভূঁইয়া ইউনিয়নের উপর দিয়ে ফেনী-নোয়াখালী জাতীয় মহাসড়ক(এন-১০৪) গিয়েছে। এছাড়া দাগনভূঁইয়া জিরো পয়েন্ট থেকে দুইটি জিলা সড়ক সড়ক গিয়েছে। ইউনিয়নের প্রায় সকল গ্রামীণ সড়কই পাকা। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
