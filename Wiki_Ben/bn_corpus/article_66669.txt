'''শিদলাই''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত ব্রাহ্মণপাড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == ব্রাহ্মণপাড়া উপজেলার পশ্চিমাংশে শিদলাই ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে দুলালপুর ইউনিয়ন ব্রাহ্মণপাড়া সদর ইউনিয়ন, পূর্বে চান্দলা ইউনিয়ন, উত্তরে মাধবপুর ইউনিয়ন দেবিদ্বার উপজেলার বড়শালঘর ইউনিয়ন, পশ্চিমে দেবিদ্বার উপজেলার ইউসুফপুর ইউনিয়ন এবং দক্ষিণ-পশ্চিমে দেবিদ্বার উপজেলার সুবিল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শিদলাই ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ব্রাহ্মণপাড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: গিয়াসউদ্দিন মুহাম্মদ == আরও দেখুন == ব্রাহ্মণপাড়া উপজেলা ব্রাহ্মণপাড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
