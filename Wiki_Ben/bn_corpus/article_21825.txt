কোন সরকারের '''দপ্তরবিহীন মন্ত্রী''' হচ্ছেন এমন একজন মন্ত্রী যিনি কোন নির্দিষ্ট মন্ত্রণালয়ের দায়িত্বপ্রাপ্ত নন। অর্থাৎ এদের কোন দপ্তর থাকে না। মণ্ত্রীদের মধ্যে দায়িত্ব বণ্টন করে থাকেন। কোন মন্ত্রণালয়ের দায়িত্ব না দিয়েও প্রধানমন্ত্রী কোন কোন সময় ধরনের মন্ত্রী নিয়োগ করতে পারেন। তবে প্রধানমন্ত্রীর পক্ষ থেকে এদের বিশেষ ধরনের দায়িত্ব দেয়া হয়। দপ্তরবিহীন মন্ত্রী অন্যান্য ক্যাবিনেট মন্ত্রীদের সমান আর্থিক অন্যান্য সুবিধা, মর্যাদা রাষ্ট্রীয় সুরক্ষা পেয়ে থাকেন যেমন তারা গাড়ীতে জাতীয় পতাকা লাগাতে পারেন। তবে তারা মন্ত্রী সভার সভায় যোগ দিতে পারবেন কি না তা নির্ভর করে প্রধানমন্ত্রীর মর্জ্জির ওপর। == অস্ট্রেলিয়া == স্ট্যানলি ব্রুস প্রাক্তন প্রধানমন্ত্রী == বাংলাদেশ == বাংলাদেশে দপ্তরবিহীন মন্ত্রী নিয়োগ দেওয়া হয় মন্ত্রীসভা বণ্টন বা নতুন নিয়োগের সময়। সমসাময়িক কালে একজন দপ্তরবিহীন মন্ত্রীকে নিয়োগ দেওয়া হয় ২৮ নভেম্বর ২০১১ সালে। তিনি হলেন সুরঞ্জিত সেনগুপ্ত == বুলগেরিয়া == বোজিদার দিমিত্রোভ == কানাডা == অনেকক্ষেত্রে দপ্তরবিহীন মন্ত্রী নিছক নিয়োগ হিসেবে দেখা হলেও কানাডায় এই ভূমিকাটি অনেক গুরুত্বপূর্ণ মানুষ পালন করেছেন যাদের মধ্যে আছেন প্রাক্তন প্রধানমন্ত্রী। == ডেনমার্ক == তিনজন নিয়ন্ত্রক মন্ত্রী দ্বিতীয় বিশ্বযুদ্ধের সময় দপ্তরবিহীন মন্ত্রী হিসেবে কাজ করেছেন। == জার্মানি == ইউলহেম ফ্রিক (১৯৩৬–১৯৪৩) আর্থার সেইস ইনকোয়ার্থ (১৯৩৯–১৯৪৫) হিজালমার সাথ (১৯৩৯–১৯৪৩) == হাঙ্গেরি == ==আইসল্যান্ড== == ভারত == == আয়ারল্যাণ্ড == == ঈসরাইল == দপ্তরবিহীন মন্ত্রী নিয়োগ দেওয়ার ঘটনা ঈসরাইলে প্রায় নিয়মিত। এর প্রধান কারণ শরিক দলগুলোর প্রধানদের নিয়ে মন্ত্রীপরিষদ গঠন করা। বর্তমান সময়ের সকল মন্ত্রিসভায় এধরনের নিয়োগ ছিল। ১৯৪৯ সালের পর থেকে নিয়োগ দেওয়া দপ্তরবিহীন মন্ত্রীদের একটি তালিকা নিচে দেওয়া হলোঃ is: ইউসেফ আলমোগি (১৯৬১-৬২) সুলামিত আলনি (১৯৭৪, ১৯৯৩) ইহুদা আমিতাল (১৯৯৫–৯৬) সাউল আমর (১৯৯৯) জালমান আরান (১৯৫৪–৫৫) মোসে অ্যারেন্স (১৯৮৪–৮৬, ১৯৮৭–৮৮) রুহামা আভ্রাহাম (২০০৭-০৮) আমি আয়ালন (২০০৭-০৮) ঈসরায়েল বারজিলাই (১৯৬৯-৭০) == ইতালি == ==মেসেডোনিয়া== == নেপাল == == নেদারল্যাণ্ড == == নরওয়ে == == ফিলিপাইন == == সার্বিয়া == == তাইওয়ান
