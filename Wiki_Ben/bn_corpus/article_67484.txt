'''নাটাই উত্তর''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == নাটাই উত্তর ইউনিয়নের আয়তন ২৯৪৬ একর (১১.৯২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নাটাই উত্তর ইউনিয়নের মোট জনসংখ্যা ৩৪,৪০৬ জন। মোট পরিবার ৬,১২৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ২৮৮৬ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার নাটাই উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে তালশহর পূর্ব ইউনিয়ন, পূর্বে সুহিলপুর ইউনিয়ন পৌরসভা, দক্ষিণে পৌরসভা নাটাই দক্ষিণ ইউনিয়ন, পশ্চিমে আশুগঞ্জ উপজেলার আশুগঞ্জ সদর ইউনিয়ন এবং উত্তর-পশ্চিমে আশুগঞ্জ উপজেলার তালশহর পশ্চিম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নাটাই উত্তর ইউনিয়ন সদর উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
