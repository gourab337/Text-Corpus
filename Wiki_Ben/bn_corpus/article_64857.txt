'''দেওয়ানবাজার ইউনিয়ন''' বাংলাদেশের সিলেট জেলার বালাগঞ্জ উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == দেওয়ান বাজার ইউনিয়ন পরিষদ থেকে বালাগঞ্জ উপজেলার দূরত্ব প্রায় ৩০ কি: মি। == ইতিহাস == == আয়তন জনসংখ্যা == == গ্রাম সমূহ == == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == == প্রখ্যাত ব্যক্তিত্ব == *আল্লামা নুর উদ্দীন আহমদ উপমহাদেশের প্রখ্যাত আলীম শায়খুল হাদিস। == হাট-বাজার == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান- নাজমুল আলম নজম''' ====== পুরাতন চেয়ারম্যান বৃন্দ ====== '''ক্রম নং''' চেয়ারম্যানের নাম '''সময়কাল''' ০১ নাজমুল আলম নজম ০২ শামীম ০৩ নেফুর মিয়া ০৪ সৈদুর রহমান ০৫ '''নাজমুল আলম নজম''' -''পদাধিকারী'' == আরও দেখুন == বালাগঞ্জ ইউনিয়ন পশ্চিম পৈলনপুর ইউনিয়ন বালাগঞ্জ উপজেলা == তথ্যসূত্র == == বহিঃসংযোগ == বালাগঞ্জ উপজেলা ''-'' বাংলাপিডিয়ায় প্রাতিষ্ঠানিক ওয়েবসাইট বাংলাদেশ জাতীয় তথ্য বাতায়ন। ''বাংলাদেশের ইউনিয়ন বিষয়ক এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' উপজেলার ইউনিয়ন
