'''ধর্মপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত নোয়াখালী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নোয়াখালী সদর উপজেলার দক্ষিণাংশে ধর্মপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে উত্তরে নোয়াখালী ইউনিয়ন, পশ্চিমে এওজবালিয়া ইউনিয়ন এবং দক্ষিণে সুবর্ণচর উপজেলার চর জব্বর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ধর্মপুর ইউনিয়ন নোয়াখালী সদর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুধারাম মডেল থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নোয়াখালী সদর উপজেলা সুধারাম মডেল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
