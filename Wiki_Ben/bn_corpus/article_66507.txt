'''সুহিলপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == সুহিলপুর ইউনিয়ন পূর্বে সুহিলপুর পশ্চিম ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার সুহিলপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে গল্লাই ইউনিয়ন, পূর্বে বাতাঘাসী ইউনিয়ন, উত্তরে দাউদকান্দি উপজেলার ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন, পশ্চিমে দাউদকান্দি উপজেলার মালিগাঁও ইউনিয়ন এবং দক্ষিণ-পশ্চিমে দক্ষিণে চাঁদপুর জেলার কচুয়া উপজেলার সাচার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সুহিলপুর ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ১নং সুহিলপুর সরকারি প্রাথমিক বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
