'''ভানী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == ভানী ইউনিয়ন পূর্বে ১২নং রাজামেহার দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার ভানী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে রাজামেহার ইউনিয়ন, পূর্বে ধামতী ইউনিয়ন সুলতানপুর ইউনিয়ন, দক্ষিণে চান্দিনা উপজেলার কেরণখাল ইউনিয়ন মাধাইয়া ইউনিয়ন, দক্ষিণ-পশ্চিমে চান্দিনা উপজেলার বাতাঘাসী ইউনিয়ন এবং পশ্চিমে মুরাদনগর উপজেলার বাবুটিপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভানী ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
