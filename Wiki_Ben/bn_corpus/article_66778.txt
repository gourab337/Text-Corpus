'''আকুবপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার উত্তরাংশে আকুবপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে আন্দিকোট ইউনিয়ন, দক্ষিণ-পূর্বে পূর্বধইর পূর্ব ইউনিয়ন, দক্ষিণে পূর্বধইর পশ্চিম ইউনিয়ন বাঙ্গরা পূর্ব ইউনিয়ন, পশ্চিমে শ্রীকাইল ইউনিয়ন, উত্তর-পশ্চিমে জেলার নবীনগর উপজেলার সাতমোড়া ইউনিয়ন, উত্তরে জেলার নবীনগর উপজেলার জিনোদপুর ইউনিয়ন এবং উত্তর-পূর্বে জেলার নবীনগর উপজেলার লাউর ফতেপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আকুবপুর ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঙ্গরাবাজার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা বাঙ্গরাবাজার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
