'''চান্দুরা''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == চান্দুরা ইউনিয়নের আয়তন ৪৬৪৪ একর (১৮.৭৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে চান্দুরা ইউনিয়নের মোট জনসংখ্যা ৩০,৭৭২ জন। মোট পরিবার ৫,৮১১টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৬৩৭ জন। == ইতিহাস == == অবস্থান সীমানা == বিজয়নগর উপজেলার উত্তরাংশে চান্দুরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বুধন্তি ইউনিয়ন, পূর্বে হরষপুর ইউনিয়ন, দক্ষিণে ইছাপুরা ইউনিয়ন, পশ্চিমে চর ইসলামপুর ইউনিয়ন এবং উত্তর-পশ্চিমে সরাইল উপজেলার শাহবাজপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চান্দুরা ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
