কয়া ইউনিয়ন পরিষদ অস্থায়ী কার্যালয়। '''কয়া ইউনিয়ন''' বাংলাদেশের খুলনা বিভাগের কুষ্টিয়া জেলার কুমারখালী উপজেলা অন্তর্গত একটি ইউনিয়ন। == আয়তন == কয়া ইউনিয়নের আয়তন ৪৬০৮ একর (১৯.৬১ বর্গ কিলোমিটার)। এই ইউনিয়নে মোট ৫৫০৯টি পরিবার আছে।। == অবস্থান == == প্রশাসনিক এলাকা == == ইতিহাস == == জনসংখ্যার উপাত্ত == বাংলাদেশর ১৯৯১ সনের আদমশুমারি অনুসারে হরিপুর ইউনিয়নর জনসংখ্যা ২৯,৯০৯ জন। == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == ==নদীসমূহ== কয়া ইউনিয়নে পদ্মা নদী। কয়া ইউনিয়নে রয়েছে পদ্মা নদী। == চিত্তাকর্ষক স্থান == == আরও দেখুন == ==তথ্যসূত্র== == বহিঃসংযোগ == কয়া ইউনিয়নের সরকারি ওয়েবসাইট জেলার ইউনিয়ন উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
