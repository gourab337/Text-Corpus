'''বাংলাদেশের হোমিওপ্যাথিক মেডিকেল কলেজের তালিকা''' বাংলাদেশ হোমিওপ্যাথি বোর্ড কর্তৃক স্বীকৃতিপ্রাপ্ত হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সমূহের নাম, ঠিকানা কোড নম্বর নিন্মে দেওয়া হলঃ == ঢাকা == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ বাংলাদেশ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ৪৬/২, টয়েনবী সার্কুলার রোড, ঢাকা-১০০০ ০১ ঢাকা ১৯৪২ ফেডারেল হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ৮৭-৯২/বি গ্রীন রোড (৫ম তলা), ফার্মগেট, ঢাকা ০৯ ঢাকা তানজিম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল মাসদাইর, নারায়ণগঞ্জ ০২ ঢাকা গাজীপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল তেলীপাড়া, চান্দনা, গাজীপুর ২৩ ঢাকা ময়মনসিংহ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সেহড়া, ময়মনসিংহ ১১ ঢাকা টাঙ্গাইল হোমিও মেডিকেল কলেজ হাসপাতাল পৌরসভা স্ট্যান্ড রোড, আকুর টাকুর পাড়া, টাঙ্গাইল ৩২ ঢাকা আলহাজ্ব আঃ কদ্দুছ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল তারাপাশা, (মনিপুরঘাট) কিশোরগঞ্জ ৩৫ ঢাকা জামালপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল জামালপুর সদর ২৪ ঢাকা সরিষাবাড়ী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সরিষাবাড়ী, জামালপুর ১৪ ঢাকা ফরিদপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল চৌরঙ্গী ঝিলটুলী, জেলা ফরিদপুর ২০ ঢাকা রাজবাড়ী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল প্রধান সড়ক, পোঃ জেলা-রাজবাড়ী ২১ ঢাকা গোপালগঞ্জ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল গোপালগঞ্জ ৩১ ঢাকা কাজী সিরাজুল ইসলাম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল আলফাডাঙ্গা, ফরিদপুর ৩৪ ঢাকা নরসিংদী বেগম ফটিকুন্নেছা ইদ্রিস হোমিওপ্যাথিক মেডিকেলকলেজ হাসপাতাল বাসাইল, নরসিংদী ৩৮ ঢাকা শরীয়তপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল কলুকাঠি, নড়িয়া শরীয়তপুর ৪২ ঢাকা মানিকগঞ্জ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বেওথা, মানিকগঞ্জ ৪৯ ঢাকা টুংগীপাড়া হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ডিমাডাঙ্গা, টুংগীপাড়া ৫০ ঢাকা উত্তরা হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল কামারপাড়া, উত্তরা, ঢাকা ৫১ ঢাকা শেখ কামাল হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল হামিদ আকন্দ রোড, বাদাম তলা, সদর, মাদারীপুর ৫২ ঢাকা সাভার হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বি-৭/৩, জাহাঙ্গীরনগর কো অপারেটিভ হাউজিং সোসাইটি, ভাটপাড়া, সাভার, ঢাকা ৫৬ ঢাকা ডাঃ দিলীপ রায় হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বোয়ালমারী (বাসস্ট্যান্ড সংলগ্ন), ফরিদপুর ৫৮ ঢাকা বীর মুক্তিযোদ্ধা আতিউর রহমান মডেল হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সাপমারী, ভাতশালা, শেরপুর সদর, শেরপুর ৫৯ ঢাকা == চট্টগ্রাম == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ আজিজুর রহমান হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল কে, বি, আমান আলী রোড, বাকলিয়া, চকবাজার, চট্টগ্রাম ২৫ চট্টগ্রাম ডাঃ জাকির হোসেন সিটি কর্পোরেশন হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ১৮০, কবি নজরুল ইসলাম রোড, ফিরিঙ্গি বাজার, চট্টগ্রাম ০৪ চট্টগ্রাম চট্টগ্রাম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ৯, এ.সি. দত্ত লেইন, পাথরঘাটা, কোতয়ালী, চট্টগ্রাম ০৫ চট্টগ্রাম হ্যানিম্যান হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল পুরাতন চৌধুরী পাড়া, কুমিল্লা ১৭ চট্টগ্রাম নোয়াখালী মৌলবী আব্দুর রহমান হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল চৌমুহনী, নোয়াখালী ৩৬ চট্টগ্রাম আদর্শ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল খলিশাডুলী, চাঁদপুর ৩৩ চট্টগ্রাম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল কাজীপাড়া, (পুরাতন বাসষ্ট্যান্ড) ২২ চট্টগ্রাম লক্ষীপুর রোকেয়া এন ইসলাম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল মেঘনা রোড, সদর লক্ষীপুর ৪৭ চট্টগ্রাম এইচ এম পার্বত্য হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল পশ্চিম শান্তিনগর সদর, খাগড়াছড়ি ৪৫ চট্টগ্রাম ফেনী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল পাঠান বাড়ী রোড, ফেনী সদর, ফেনী ৫৭ চট্টগ্রাম চাঁদপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল শাহরাস্তি, চাঁদপুর ৬১ চট্টগ্রাম == রাজশাহী == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ রাজশাহী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল মিয়াপাড়া, পো ঘোড়ামারা, রাজশাহী ০৬ রাজশাহী বগুড়া হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ফুলদিঘী, বগুড়া ২৬ রাজশাহী পাবনা হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল লস্করপুর, পাবনা ১০ রাজশাহী রহনপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল গোমাস্তাপুর, চাঁপাইনবাবগঞ্জ ৪১ রাজশাহী মোতাহার হোসেন তালুকদার হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল গজারিয়া, সদর, সিরাজগঞ্জ ৫৩ রাজশাহী নওগাঁ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ঢাকা রোড, সদর, নওগাঁ ৫৪ রাজশাহী == খুলনা == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ খুলনা হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল শের-ই-বাংলা রোড, খুলনা ০৩ খুলনা ডিবিখান হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল পলাশ পোল, সাতক্ষীরা ২৭ খুলনা বাগেরহাট হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বাগেরহাট ৩০ খুলনা যশোর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল আর, এন, রোড, যশোর ০৮ খুলনা কুষ্টিয়া হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল রাজবাড়ী মহাসড়ক, কুষ্টিয়া ২৮ খুলনা নড়াইল হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল রতনগঞ্জ, নড়াইল ২৯ খুলনা চুয়াডাঙ্গা হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল মুসলিম পাড়া, বাইপাস সড়ক, চুয়াডাঙ্গা ৪৩ খুলনা কলারোয়া হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ৬০ খুলনা == বরিশাল == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ এ্যাপেক্স হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল গোড়াচাঁদ দাস রোড, (বটতলা) বরিশাল ১৫ বরিশাল লায়ন্স হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সদর রোড, ভোলা ১৯ বরিশাল বরগুনা হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বরগুনা ৪০ বরিশাল পটুয়াখালী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল আদালতপাড়া, পটুয়াখালী ৩৯ বরিশাল স্বরূপকাঠি হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল নেছারাবাদ, পিরোজপুর ৪৮ বরিশাল ফিরোজা আমু হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল ২০, ডাক্তার পট্টি রোড, ঝালকাঠী বরিশাল == সিলেট == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ জালালাবাদ হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল মির্জাজাঙ্গাল, সিলেট ১৬ সিলেট == রংপুর == কলেজের নাম ঠিকানা কলেজ কোড বিভাগ স্থাপিত তারিখ -১৯৫৮ রংপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল জে, এন, সি রোড, নূরপুর, আলমনগর, রংপুর ০৭ রংপুর ১৯৫৮ দিনাজপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল সুইহারী, দিনাজপুর ১২ রংপুর সৈয়দপুর হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল বানীচাঁদ সড়ক, নয়াবাজার, সৈয়দপুর ১৮ রংপুর নীলফামারী হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল নীলফামারী ১৩ রংপুর পঞ্চগড় হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল পঞ্চগড় সদর, পঞ্চগড় ৩৭ রংপুর কুড়িগ্রাম হোমিওপ্যাথিক মেডিকেল কলেজ হাসপাতাল
