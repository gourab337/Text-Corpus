'''নবীনগর পশ্চিম''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == নবীনগর পশ্চিম ইউনিয়নের আয়তন ২৫৭৭ একর (১০.৪৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নবীনগর পশ্চিম ইউনিয়নের মোট জনসংখ্যা ১৮,৩৭৪ জন। মোট পরিবার ৩,৩৬৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৭৬২ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার পশ্চিমাংশে নবীনগর পশ্চিম ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পশ্চিমে শ্যামগ্রাম ইউনিয়ন; দক্ষিণে শ্রীরামপুর ইউনিয়ন নবীনগর পৌরসভা; পূর্বে নবীনগর পৌরসভা; উত্তরে তিতাস নদী, কৃষ্ণনগর ইউনিয়ন বীরগাঁও ইউনিয়ন এবং পশ্চিমে মেঘনা নদী, নরসিংদী জেলার রায়পুরা উপজেলার চানপুর ইউনিয়ন, পাড়াতলী ইউনিয়ন, বাঁশগাড়ী ইউনিয়ন মির্জারচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নবীনগর পশ্চিম ইউনিয়ন নবীনগর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
