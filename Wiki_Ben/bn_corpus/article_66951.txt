'''বড়কান্দা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মেঘনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মেঘনা উপজেলার পশ্চিমাংশে বড়কান্দা ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে ভাওরখোলা ইউনিয়ন; পূর্বে গোবিন্দপুর ইউনিয়ন মানিকারচর ইউনিয়ন; উত্তরে চন্দনপুর ইউনিয়ন এবং পশ্চিমে মেঘনা নদী, চালিভাঙ্গা ইউনিয়ন মুন্সিগঞ্জ জেলার গজারিয়া উপজেলার বালুয়াকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বড়কান্দা ইউনিয়ন মেঘনা উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মেঘনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মেঘনা উপজেলা মেঘনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
