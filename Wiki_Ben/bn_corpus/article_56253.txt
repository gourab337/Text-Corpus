'''চিথলিয়া''' বাংলাদেশের ফেনী জেলার অন্তর্গত পরশুরাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == পরশুরাম উপজেলার সর্ব-পশ্চিমে চিথলিয়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মির্জানগর ইউনিয়ন, পূর্বে পরশুরাম পৌরসভা বক্স মাহমুদ ইউনিয়ন, দক্ষিণে ফুলগাজী উপজেলার ফুলগাজী ইউনিয়ন এবং পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == চিথলিয়া ইউনিয়ন পরশুরাম উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম পরশুরাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == পরশুরাম উপজেলা পরশুরাম থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
