'''দেবিদ্বার পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দেবিদ্বার উপজেলার পশ্চিম-মধ্যাংশে দেবিদ্বার পৌরসভার অবস্থান। পৌরসভার উত্তর-পূর্বে রসুলপুর ইউনিয়ন, পূর্বে সুবিল ইউনিয়ন ফতেহাবাদ ইউনিয়ন, দক্ষিণ-পূর্বে জাফরগঞ্জ ইউনিয়ন, দক্ষিণে এলাহাবাদ ইউনিয়ন গুনাইঘর দক্ষিণ ইউনিয়ন, দক্ষিণ-পশ্চিমে গুনাইঘর উত্তর ইউনিয়ন, উত্তর-পশ্চিমে মুরাদনগর উপজেলার ধামঘর ইউনিয়ন নবীপুর পশ্চিম ইউনিয়ন এবং উত্তরে মুরাদনগর উপজেলার নবীপুর পূর্ব ইউনিয়ন অবস্থিত। == ইতিহাস == দেবিদ্বার উপজেলার ৫নং দেবিদ্বার ইউনিয়ন এর সম্পূর্ণ অংশ নিয়ে দেবিদ্বার পৌরসভা প্রতিষ্ঠিত হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == দেবিদ্বার পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
