'''চুণ্টা''' বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == চুণ্টা ইউনিয়নের আয়তন ৬২২১ একর (২৫.১৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে চুণ্টা ইউনিয়নের মোট জনসংখ্যা ৩১,৬৩৯ জন। এর মধ্যে পুরুষ ১৪,৬৩১ জন এবং মহিলা ১৭,০০৮ জন। মোট পরিবার ৫,৮৯৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১২৫৭ জন। == ইতিহাস == == অবস্থান সীমানা == সরাইল উপজেলার মধ্যাংশে চুণ্টা ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে পানিশ্বর ইউনিয়ন কালিকচ্ছ ইউনিয়ন; পূর্বে কালিকচ্ছ ইউনিয়ন; উত্তরে পাকশিমুল ইউনিয়ন এবং পশ্চিমে পাকশিমুল ইউনিয়ন, মেঘনা নদী কিশোরগঞ্জ জেলার ভৈরব উপজেলার আগানগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চুণ্টা ইউনিয়ন সরাইল উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে চুণ্টা ইউনিয়নের সাক্ষরতার হার ৩৪%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == প্রবোধচন্দ্র সেন –– ছন্দবিশারদ। == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
