'''মেহের দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == শাহরাস্তি উপজেলার মধ্যাংশে মেহের দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে সূচীপাড়া উত্তর ইউনিয়ন, পূর্বে রায়শ্রী দক্ষিণ ইউনিয়ন রায়শ্রী উত্তর ইউনিয়ন, উত্তরে মেহের উত্তর ইউনিয়ন শাহরাস্তি পৌরসভা এবং পশ্চিমে টামটা দক্ষিণ ইউনিয়ন হাজীগঞ্জ উপজেলার গন্ধর্ব্যপুর উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মেহের দক্ষিণ ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
