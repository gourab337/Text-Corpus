'''মানিকারচর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মেঘনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মেঘনা উপজেলার মধ্যাংশে মানিকারচর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে রাধানগর ইউনিয়ন, উত্তরে চন্দনপুর ইউনিয়ন, পশ্চিমে বড়কান্দা ইউনিয়ন এবং দক্ষিণে গোবিন্দপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মানিকারচর ইউনিয়ন মেঘনা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মেঘনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মেঘনা উপজেলা মেঘনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
