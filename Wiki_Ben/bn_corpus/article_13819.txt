'''বাবুছড়া''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত দীঘিনালা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দীঘিনালা উপজেলার উত্তরাংশ জুড়ে বাবুছড়া ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে রাঙ্গামাটি জেলার বাঘাইছড়ি উপজেলার সাজেক ইউনিয়ন; দক্ষিণে দীঘিনালা ইউনিয়ন বোয়ালখালী ইউনিয়ন; পশ্চিমে পানছড়ি উপজেলার লতিবান ইউনিয়ন, পানছড়ি ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ এবং উত্তরে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বাবুছড়া ইউনিয়ন দীঘিনালা উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দীঘিনালা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == বাবুছড়া ইউনিয়নের মধ্য দিয়ে প্রবাহিত হচ্ছে মাইনী নদী। == হাট-বাজার == বাবুছড়া ইউনিয়নের প্রধান হাট-বাজার হল বাবুছড়া বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দীঘিনালা উপজেলা দীঘিনালা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
