'''দুর্গাপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত বেগমগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার উত্তর-পূর্বাংশে দুর্গাপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে নরোত্তমপুর ইউনিয়ন চৌমুহনী পৌরসভা, দক্ষিণে চৌমুহনী পৌরসভা হাজীপুর ইউনিয়ন, পূর্বে রসুলপুর ইউনিয়ন কুতুবপুর ইউনিয়ন এবং উত্তরে সোনাইমুড়ি উপজেলার অম্বরনগর ইউনিয়ন নাটেশ্বর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দুর্গাপুর ইউনিয়ন বেগমগঞ্জ উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == এই ইউনিয়নের প্রধান শিক্ষা প্রতিষ্ঠান গুলোর মধ্যে আছে '''দুর্গাপুর উচ্চ বিদ্যালয় ,লক্ষ্মীনারায়ন পুর উচ্চ বিদ্যালয়'''। এদের মধ্যে সর্বাপেক্ষা প্রাচীন দুর্গাপুর উচ্চ বিদ্যালয় যা '''১৯৩৪''' সালে প্রতিষ্ঠিত হয়। এছাড়াও এই ইউনিয়নে প্রাথমিক বিদ্যালয়, মাদ্রাসা, কিন্ডার গার্টেন বিদ্যালয় রয়েছে। == যোগাযোগ ব্যবস্থা == প্রধান যোগাযোগ ব্যবস্থা সড়ক পথ। এই ইউনিয়নে বেশির ভাগ সড়ক বর্তমানে পাকা সড়ক ,যেগুলো সাধারন মানুষের চলাচল পন্য পরিবহন এর কাজে ব্যবহার হয়। == খাল নদী == == হাট-বাজার == উল্লেখযোগ্য বাজারের মধ্যে আছে বালী মসজিদ বাজার, দাওয়াত খোলা বাজার, নতুন বাজার, বেপারী পাড়া বাজার, পশার পুকুর পাড় ইত্যাদি। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
