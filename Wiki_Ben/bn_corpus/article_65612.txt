'''উত্তর ফেঞ্চুগঞ্জ ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত ফেঞ্চুগঞ্জ উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == ২০১১ সালে প্রতিষ্টিত হয় উত্তর ফেঞ্চুগঞ্জ ইউনিয়নটি। এই ইউনিয়নের নং ফেঞ্চুগঞ্জ নং ঘিলাছড়া ইউনিয়নের অন্তর্ভুক্ত ছিল। ১৪ টি গ্রাম নিয়ে বর্তমান ইউনিয়ন পরিষদটি গঠিত। == প্রশাসনিক এলাকা == গ্রামের সংখ্যা: ১৪টি, মৌজার সংখ্যা: ৮টি। '''গ্রাম সমূহের নাম''' মল্লিকপুর কুতুবপুর লামাগঙ্গাপুর গঙ্গাপুর উজান গঙ্গাপুর সুলতানপুর মানিককোনা দক্ষিণ মানিককোনা পশ্চিম-১ মানিককোনা পশ্চিম-২ খাটখাল( ভেলকোনা) মানিককোনা পূব সুড়িকান্দি সাইলকান্দি == আয়তন জনসংখ্যা == আয়তন-২০.৪০ বর্গ কি:মি:। লোকসংখ্যা: ২০০০০জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার:''' ৭০% ''(২০১১ সালের আদমশুমারী অনুসারে)'' '''শিক্ষা প্রতিষ্ঠাণ''' সরকারী প্রাথমিক বিদ্যালয়ের সংখ্যা- ৫টি বে-সরকারী প্রাথমিক বিদ্যালয়ের সংখ্যা-২ টি উচ্চ বিদ্যালয়- ১টি মাধ্যমিক মাদ্রাসা-৩টি মহিলা মাদ্রাসা-১টি == দর্শনীয় স্থান == হাকালুকি হাওর == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান- ''মোঃ এমরান উদ্দিন''''' তালিকা '''ক্রম নং''' নাম''' '''সময়কাল''' ০১ ০২ ০৩ == আরও দেখুন == ফেঞ্চুগঞ্জ ইউনিয়ন *উত্তর কুশিয়ারা ইউনিয়ন, ফেঞ্চুগঞ্জ মাইজগাঁও ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == ফেঞ্চুগঞ্জ উপজেলা -বাংলাপিডিয়া উত্তর ফেঞ্চুগঞ্জ ইউনিয়ন -বাংলাদেশ জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
