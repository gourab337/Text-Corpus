'''পূর্ব চন্দ্রপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার মধ্য-পশ্চিমাংশে পূর্ব চন্দ্রপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে রামনগর ইউনিয়ন দাগনভূঁইয়া পৌরসভা; পূর্বে মাতুভূঁইয়া ইউনিয়ন জায়লস্কর ইউনিয়ন; উত্তরে রাজাপুর ইউনিয়ন এবং পশ্চিমে রাজাপুর ইউনিয়ন, নোয়াখালী জেলার সেনবাগ পৌরসভা সেনবাগ উপজেলার মোহাম্মদপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পূর্ব চন্দ্রপুর ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
