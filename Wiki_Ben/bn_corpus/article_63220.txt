'''হাটিলা পশ্চিম''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার উত্তর-পূর্বাংশে হাটিলা পশ্চিম ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কালচোঁ উত্তর ইউনিয়ন, কালচোঁ দক্ষিণ ইউনিয়ন হাজীগঞ্জ সদর ইউনিয়ন; দক্ষিণে হাজীগঞ্জ পৌরসভা; পূর্বে হাটিলা পূর্ব ইউনিয়ন কচুয়া উপজেলার কড়ইয়া ইউনিয়ন এবং উত্তরে কচুয়া উপজেলার কাদলা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাটিলা পশ্চিম ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
