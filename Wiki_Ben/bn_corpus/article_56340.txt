'''ধলিয়া''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার দক্ষিণাংশে ধলিয়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পূর্বে লেমুয়া ইউনিয়ন কালিদহ ইউনিয়ন, উত্তর-পশ্চিমে বালিগাঁও ইউনিয়ন, দক্ষিণ-পশ্চিমে সোনাগাজী উপজেলার চর মজলিশপুর ইউনিয়ন বগাদানা ইউনিয়ন, দক্ষিণে সোনাগাজী উপজেলার মঙ্গলকান্দি ইউনিয়ন আমিরাবাদ ইউনিয়ন এবং দক্ষিণ-পূর্বে সোনাগাজী উপজেলার নবাবপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ধলিয়া ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
