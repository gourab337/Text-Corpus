'''বদলকোট''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার সর্ব-উত্তরে বদলকোট ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পশ্চিমে পরকোট ইউনিয়ন, দক্ষিণে চাটখিল পৌরসভা, পূর্বে মোহাম্মদপুর ইউনিয়ন কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার হাসনাবাদ ইউনিয়ন, উত্তরে চাঁদপুর জেলার শাহরাস্তি উপজেলার চিতোষী পূর্ব ইউনিয়ন এবং পশ্চিমে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার ভাটরা ইউনিয়ন ভোলাকোট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বদলকোট ইউনিয়ন চাটখিল উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
