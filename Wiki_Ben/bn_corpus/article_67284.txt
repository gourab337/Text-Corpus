বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == ইব্রাহিমপুর ইউনিয়নের আয়তন ৩১৭২ একর (১২.৮৪ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ইব্রাহিমপুর ইউনিয়নের মোট জনসংখ্যা ১৩,৫৯৭ জন। মোট পরিবার ২,৮৩৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১০৫৯ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার মধ্যাংশে ইব্রাহিমপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে শ্রীরামপুর ইউনিয়ন, উত্তরে নবীনগর পৌরসভা, উত্তর-পূর্বে শিবপুর ইউনিয়ন, পূর্বে বিটঘর ইউনিয়ন, দক্ষিণে লাউর ফতেপুর ইউনিয়ন জিনোদপুর ইউনিয়ন এবং পশ্চিমে রসুল্লাবাদ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইব্রাহিমপুর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
