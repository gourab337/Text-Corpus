'''গোমতি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মাটিরাঙ্গা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মাটিরাঙ্গা উপজেলার মধ্যাংশে গোমতি ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে বেলছড়ি ইউনিয়ন, পূর্বে খাগড়াছড়ি সদর উপজেলার পেরাছড়া ইউনিয়ন ভাইবোনছড়া ইউনিয়ন, উত্তরে আমতলী ইউনিয়ন এবং পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == গোমতি ইউনিয়ন মাটিরাঙ্গা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মাটিরাঙ্গা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মাটিরাঙ্গা উপজেলা মাটিরাঙ্গা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
