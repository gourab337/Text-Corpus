'''চিতোষী পূর্ব''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == শাহরাস্তি উপজেলার চিতোষী পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে চিতোষী পশ্চিম ইউনিয়ন, উত্তরে রায়শ্রী দক্ষিণ ইউনিয়ন, পূর্বে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার ঝলম দক্ষিণ ইউনিয়ন বাইশগাঁও ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার হাসনাবাদ ইউনিয়ন নোয়াখালী জেলার চাটখিল উপজেলার বদলকোট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চিতোষী পূর্ব ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ১.চিতোষী সরকারি প্রাথমিক বিদ্যালয় ২.চিতোষী আর এন্ড এম উচ্চ বিদ্যালয়। ৩. মনিরা আজিম একাডেমী। ৪. চিতোষী ডিগ্রী কলেজ। ৫. আল ক্বলম একাডেমী। ৬. চিতোষী সুলতানিয়া ফাজিল ডিগ্রি মাদরাসা। ৬. ঘড়িমন্ডল সরকারি প্রাথমিক বিদ্যালয়। ৭. পীর শাহশরীফ সরকারি প্রাথমিক বিদ্যালয়। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
