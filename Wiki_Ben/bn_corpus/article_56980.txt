'''বিনাউটি''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == বিনাউটি ইউনিয়নের আয়তন ৫৯২৬ একর (২৩.৯৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বিনাউটি ইউনিয়নের মোট জনসংখ্যা ৩৩,৫২৫ জন। মোট পরিবার ৬,৩০৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৩৯৮ জন। == ইতিহাস == ১৯৪০ সালে বিনাউটি ইউনিয়ন প্রতিষ্টিত হয়। বিনাউটি গ্রামের নামে ইউনিয়নের নামকরণ করা হয়। == অবস্থান সীমানা == কসবা উপজেলার উত্তরাংশে বিনাউটি ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে গোপীনাথপুর ইউনিয়ন, দক্ষিণে কসবা পৌরসভা কসবা পশ্চিম ইউনিয়ন, পশ্চিমে খাড়েরা ইউনিয়ন বাদৈর ইউনিয়ন এবং উত্তরে আখাউড়া উপজেলার ধরখাড় ইউনিয়ন মনিয়ন্দ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বিনাউটি ইউনিয়ন কসবা উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। ইউনিয়নের গ্রামগুলো হল: সৈয়দাবাদ হাজীপুর মনিচং চান্দাইসার রাউৎহাট খিদিরপুর আন্দিরপাড় বিনাউটি নেমতাবাদ গাববাড়ী ধামসার নোয়াপাড়া (১) ভরাজাঙ্গাল চন্দ্রপুর মজলিশপুর ব্রাহ্মণগ্রাম অনন্তপুর আদ্রা চাপিয়া টিঘরিয়া নোয়াপাড়া (২) চকচন্দ্রপুর দুরুইল == শিক্ষা ব্যবস্থা == বিনাউটি ইউনিয়নের সাক্ষরতার হার ৪৬.৩৫%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == বিনাউটি ইউনিয়নে যোগাযোগের প্রধান সড়ক কুমিল্লা-সিলেট মহাসড়ক। যে কোন যানবাহনে যাতায়াত করা যায়। == খাল নদী == == হাট-বাজার == বিনাউটি ইউনিয়নের প্রধান হাট-বাজার হল রাউৎহাট বাজার। == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: আনিসুর রহমান == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
