'''টামটা উত্তর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == টামটা উত্তর ইউনিয়নের আয়তন ১৮.৫০ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের আদমশুমারী অনুযায়ী টামটা উত্তর ইউনিয়নের জনসংখ্যা ৬০,০০০ জন। এর মধ্যে পুরুষ ৩২,৫০০ জন এবং মহিলা ২৭,৫০০ জন। == অবস্থান সীমানা == শাহরাস্তি উপজেলার সর্ব-উত্তরে টামটা উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে টামটা দক্ষিণ ইউনিয়ন শাহরাস্তি পৌরসভা, পূর্বে মেহের উত্তর ইউনিয়ন কচুয়া উপজেলার আশরাফপুর ইউনিয়ন, উত্তরে কচুয়া উপজেলার গোহট দক্ষিণ ইউনিয়ন এবং পশ্চিমে হাজীগঞ্জ উপজেলার হাটিলা পূর্ব ইউনিয়ন অবস্থিত। == ইতিহাস == == প্রশাসনিক কাঠামো == টামটা উত্তর ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। ওয়ার্ডভিত্তিক ইউনিয়নের গ্রামগুলো হল: ওয়ার্ড নং গ্রামের নাম ১নং ওয়ার্ড গোলপুরা, মুড়াগাঁও ২নং ওয়ার্ড ইছাপুরা রাজাপুরা, বোস্তা, শিমলা, ইছাপুরা উত্তর ৩নং ওয়ার্ড তারালিয়া, যুগীনগর, বলশীদ উত্তর ৪নং ওয়ার্ড পরাণপুর, দৈলবাড়ী ৫নং ওয়ার্ড হোসেনপুর ৬নং ওয়ার্ড সেনগাঁও, করবা ৭নং ওয়ার্ড চেঙ্গাচাল, বলশীদ দক্ষিণ ৮নং ওয়ার্ড ইছাপুরা দক্ষিণ, পাঁচরুখী, ডুশুয়া, সুরসুই ৯নং ওয়ার্ড রাড়া, সেতরা == শিক্ষা ব্যবস্থা == টামটা উত্তর ইউনিয়নের সাক্ষরতার হার ৯২%। ইউনিয়নে ৭টি ফাজিল মাদ্রাসা, ৪টি মাধ্যমিক বিদ্যালয়, ১টি দাখিল মাদ্রাসা, ১টি নিম্ন মাধ্যমিক বিদ্যালয় ১২টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == ওয়ারুক রহমানিয়া উচ্চ বিদ্যালয় রহমানিয়া নূরিয়া সুন্নিয়া দাখিল মাদ্রাসা == যোগাযোগ ব্যবস্থা == টামটা উত্তর ইউনিয়নে যোগাযোগের প্রধান সড়ক সড়ক। সব ধরনের যানবাহনে যোগাযোগ করা যায়। == খাল নদী == == হাট-বাজার == টামটা উত্তর ইউনিয়নের প্রধান হাট-বাজার হল ওয়ারুক মধ্য বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
