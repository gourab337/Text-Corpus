'''চর ওয়াপদা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সুবর্ণচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সুবর্ণচর উপজেলার সর্ব-পশ্চিমে চর ওয়াপদা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চর জুবলী ইউনিয়ন চর জব্বর ইউনিয়ন; উত্তরে নোয়াখালী সদর উপজেলার কালাদরপ ইউনিয়ন; পশ্চিমে লক্ষ্মীপুর জেলার রামগতি উপজেলার চর পোড়াগাছা ইউনিয়ন, চর রমিজ ইউনিয়ন চর গাজী ইউনিয়ন এবং দক্ষিণে মেঘনা নদী, হাতিয়া উপজেলার চর কিং ইউনিয়ন সুখচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর ওয়াপদা ইউনিয়ন সুবর্ণচর উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুবর্ণচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সুবর্ণচর উপজেলা সুবর্ণচর থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
