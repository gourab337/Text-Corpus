'''চর ক্লার্ক''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সুবর্ণচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সুবর্ণচর উপজেলার সর্ব-পূর্বে চর ক্লার্ক ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মোহাম্মদপুর ইউনিয়ন; পশ্চিমে পূর্ব চর বাটা ইউনিয়ন, চর বাটা ইউনিয়ন চর আমানউল্যা ইউনিয়ন; উত্তরে কোম্পানীগঞ্জ উপজেলার চর এলাহী ইউনিয়ন এবং পূর্বে সন্দ্বীপ চ্যানেল, চট্টগ্রাম জেলার সন্দ্বীপ উপজেলার উড়িরচর ইউনিয়ন, সন্তোষপুর ইউনিয়ন কালাপানিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর ক্লার্ক ইউনিয়ন সুবর্ণচর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুবর্ণচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সুবর্ণচর উপজেলা সুবর্ণচর থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
