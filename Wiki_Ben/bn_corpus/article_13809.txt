'''হাফছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত গুইমারা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == গুইমারা উপজেলার সর্ব-দক্ষিণে হাফছড়ি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে গুইমারা ইউনিয়ন, পূর্বে মহালছড়ি উপজেলার মাইসছড়ি ইউনিয়ন, দক্ষিণ-পূর্বে মহালছড়ি উপজেলার সিন্দুকছড়ি ইউনিয়ন, দক্ষিণে লক্ষ্মীছড়ি উপজেলার দুল্যাতলী ইউনিয়ন, দক্ষিণ-পশ্চিমে মানিকছড়ি উপজেলার মানিকছড়ি ইউনিয়ন এবং উত্তর-পশ্চিমে রামগড় উপজেলার পাতাছড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাফছড়ি ইউনিয়ন ছিল রামগড় উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। বর্তমানে এটি গুইমারা উপজেলার আওতাধীন। ইউনিয়নের প্রশাসনিক কার্যক্রম গুইমারা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == গুইমারা উপজেলা গুইমারা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
