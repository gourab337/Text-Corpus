'''চিতড্ডা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == চিতড্ডা ইউনিয়ন পূর্বে ঝলম উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার চিতড্ডা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে খোশবাস দক্ষিণ ইউনিয়ন, দক্ষিণে ঝলম ইউনিয়ন, পশ্চিমে চান্দিনা উপজেলার জোয়াগ ইউনিয়ন বরকরই ইউনিয়ন এবং উত্তরে চান্দিনা উপজেলার মাইজখার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চিতড্ডা ইউনিয়ন বরুড়া উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
