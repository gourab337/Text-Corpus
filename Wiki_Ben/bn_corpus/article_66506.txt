'''চান্দিনা পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চান্দিনা উপজেলার পূর্বাংশে চান্দিনা পৌরসভার অবস্থান। পৌরসভার পশ্চিমে কেরণখাল ইউনিয়ন, দক্ষিণ-পশ্চিমে বাড়েরা ইউনিয়ন, দক্ষিণে এতবারপুর ইউনিয়ন বরকইট ইউনিয়ন, পূর্বে বুড়িচং উপজেলার মোকাম ইউনিয়ন এবং উত্তরে দেবিদ্বার উপজেলার বরকামতা ইউনিয়ন সুলতানপুর ইউনিয়ন অবস্থিত। == ইতিহাস == চান্দিনা উপজেলার তৎকালীন ৭নং চান্দিনা পশ্চিম ৮নং চান্দিনা পূর্ব ইউনিয়নদ্বয়ের উত্তরাংশের কিছু অংশ নিয়ে চান্দিনা পৌরসভা গঠন করা হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == চান্দিনা পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
