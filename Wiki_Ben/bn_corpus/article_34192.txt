'''আমজানখোরর ইউনিয়ন''' বাংলাদেশের রংপুর বিভাগের ঠাকুরগাঁও জেলার বালিয়াডাঙ্গী উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == আমজানখোর ইউনিয়নের আয়তন ৯২১৯ একর। == অবস্থান == আমজানখোর ইউনিয়নের পরিষদ বালিয়াডাঙ্গী থেকে ১৪ কিমি দূরে অবস্থিত। == প্রশাসনিক এলাকা == == ইতিহাস == == জনসংখ্যার উপাত্ত == আমজানখোর ইউনিয়নের মোট লোকসংখ্যাঃ ২৩,১৫২ জন। পুরুষঃ ১১৮৪৫ জন মহিলাঃ ১১৩০৭ জন। বাংলাদেশর ১৯৯১ সনের আদমশুমারি অনুসারে আমজানখোর ইউনিয়নের জনসংখ্যা ১৭,৪১৪ জন। == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == ==দর্শনীয় স্থান== শত বছরের হরিণমারীর সূর্যপূরী ঐতিহাসিক আম গাছ। == আরও দেখুন == == বহিঃসংযোগ == আমজানখোর ইউনিয়নের সরকারি ওয়েবসাইট ==তথ্যসূত্র== জেলার ইউনিয়ন উপজেলার ইউনিয়ন
