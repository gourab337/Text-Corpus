'''লক্ষ্মীপুর ইউনিয়ন''' বাংলাদেশের গাইবান্ধা সদর উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == এই ইউনিয়নের আয়তন ৫৫৭৪ একর। == অবস্থান == গাইবান্ধা সদর উপজেলা হতে ১৪ কিলোমিটার উত্তরে গাইবান্ধা সনু্দরগঞ্জ সড়কের উভয় পার্শে হাট লক্ষ্মীপুর নামক স্থানে লক্ষ্মীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক এলাকা == লক্ষ্মীপুর ইউনিয়নের দক্ষিণে কুপতলা খোলাহাটি ইউনিয়ন, পূর্বে মালিবাড়ি ইউনিয়ন, পশ্চিমে সাদুল্লাপুর উপজেলার কামারপাড়া ইউনিয়ন এবং উত্তরে সুন্দরগঞ্জ উপজেলার ধোপাডাংগা ছাপড়হাটি ইউনিয়ন অবস্থিত। == ইতিহাস == == নামকরণ == হিন্দুদের সম্পদের দেবী লক্ষ্মীরনামে তৎকালীন হিন্দু জমিদারেরা এই এলাকার নাম লক্ষ্মীপুর রাখেন। তখন থেকে এই ইউনিয়ন লক্ষ্মীপুর ইউনিয়ন নামে পরিচিত লাভ করে। == জনসংখ্যার উপাত্ত == ২০১১ সালের আদম শুমারি অনুযায়ী নারী ১৬,১৯১ জন পুরুষ ১৫,৫০৩ জন। == শিক্ষা == == অর্থনীতি == ইউনিয়নের বেশির ভাগ লোক কৃষি কাজ করে। == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র ==
