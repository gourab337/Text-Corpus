'''গোলকেরহাট পানজুবের নেছা বালিকা উচ্চ বিদ্যালয়''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার ধুম ইউনিয়নের গোলকেরহাট এলাকায় বিদ্যালয়টি অবস্থিত। == ইতিহাস == == পরিচালনা ব্যবস্থা == == == বিদ্যালয়ের প্রধান শিক্ষক জনাব কামাল উদ্দীন। == অবকাঠামো == == কার্যক্রম == বিদ্যালয়টি বালিকা মাধ্যমিক বিদ্যালয়। == কৃতিত্ব ফলাফল == == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
