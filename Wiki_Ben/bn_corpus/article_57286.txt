'''চর জুবলী''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সুবর্ণচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সুবর্ণচর উপজেলার মধ্যাংশে চর জুবলী ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চর বাটা ইউনিয়ন, চর আমানউল্যা ইউনিয়ন নোয়াখালী সদর উপজেলার নোয়াখালী ইউনিয়ন; উত্তরে নোয়াখালী সদর উপজেলার নোয়াখালী ইউনিয়ন; পশ্চিমে চর জব্বর ইউনিয়ন চর ওয়াপদা ইউনিয়ন এবং দক্ষিণে মেঘনা নদী হাতিয়া উপজেলার সুখচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর জুবলী ইউনিয়ন সুবর্ণচর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুবর্ণচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সুবর্ণচর উপজেলা সুবর্ণচর থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
