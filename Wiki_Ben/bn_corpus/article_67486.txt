'''সাদেকপুর''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == সাদেকপুর ইউনিয়নের আয়তন ২৯৮৩ একর (১২.০৭ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে সাদেকপুর ইউনিয়নের মোট জনসংখ্যা ২২,৭২২ জন। মোট পরিবার ৩,৬৯৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৮৮২ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার সর্ব-পশ্চিমে সাদেকপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে নাটাই দক্ষিণ ইউনিয়ন; উত্তরে আশুগঞ্জ উপজেলার আশুগঞ্জ সদর ইউনিয়ন তারুয়া ইউনিয়ন; পশ্চিমে আশুগঞ্জ উপজেলার তারুয়া ইউনিয়ন নবীনগর উপজেলার বড়াইল ইউনিয়ন এবং দক্ষিণে তিতাস নদী, নবীনগর উপজেলার বিদ্যাকুট ইউনিয়ন নাটঘর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সাদেকপুর ইউনিয়ন সদর উপজেলার আওতাধীন ২৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
