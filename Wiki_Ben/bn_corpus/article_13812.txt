'''বর্মাছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত লক্ষ্মীছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীছড়ি উপজেলার সর্ব-দক্ষিণে বর্মাছড়ি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে লক্ষ্মীছড়ি ইউনিয়ন, পূর্বে রাঙ্গামাটি জেলার নানিয়ারচর উপজেলার ঘিলাছড়ি ইউনিয়ন, দক্ষিণে রাঙ্গামাটি জেলার কাউখালী উপজেলার ফটিকছড়ি ইউনিয়ন চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার খিরাম ইউনিয়ন এবং চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার লেলাং ইউনিয়ন কাঞ্চননগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বর্মাছড়ি ইউনিয়ন লক্ষ্মীছড়ি উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লক্ষ্মীছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীছড়ি উপজেলা লক্ষ্মীছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
