'''লতিবান''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত পানছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == পানছড়ি উপজেলার সম্পূর্ণ দক্ষিণাংশ জুড়ে লতিবান ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পানছড়ি ইউনিয়ন উল্টাছড়ি ইউনিয়ন, পশ্চিমে মাটিরাঙ্গা উপজেলার বড়নাল ইউনিয়ন আমতলী ইউনিয়ন, দক্ষিণে খাগড়াছড়ি সদর উপজেলার ভাইবোনছড়া ইউনিয়ন এবং পূর্বে দীঘিনালা উপজেলার বাবুছড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == লতিবান ইউনিয়ন পানছড়ি উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম পানছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == সড়ক।দুর্বল নাজুক অবস্থা। == খাল নদী == লতিবান ইউনিয়নের মধ্য দিয়ে প্রবাহিত হচ্ছে চেঙ্গী নদী। লতিবান ছড়া নামে এলাকার মধ্যে খ্যাত আছে।এ ছড়ায় জন্মানো "লতি"বাঁশ থেকে এই নামের উৎপত্তি। ছাড়াও নালকাটা খাল,কুড়াদিয়া ছড়া উল্লেখযোগ্য। == হাট-বাজার == উপজেলার পানছড়ি ইউনিয়নে অবস্থিত পানছড়ি বাজার উপজেলা তথা জেলার ২য় বৃহত্তম বাজার।এছাড়াও লোগাং হাট উল্লেখযোগ্য। == দর্শনীয় স্থান == *লতিবান আদর্শ বৌদ্ধ বিহার *শীলাচার অরণ্য কুটির *সৈকত তৈরি করা হয়। *রাবার ড্যাম(ব্রীজ সহ) *চেঙ্গী নদী == জনপ্রতিনিধি == == আরও দেখুন == পানছড়ি উপজেলা পানছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
