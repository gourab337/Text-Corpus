'''মজলিশপুর''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == মজলিশপুর ইউনিয়নের আয়তন ৭৫৪৭ একর (৩০.৫৪ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে মজলিশপুর ইউনিয়নের মোট জনসংখ্যা ২৮,৮৪৭ জন। মোট পরিবার ৫,০৫৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৯৪৫ জন। == ইতিহাস == মজলিশপুর ইউনিয়ন পূর্বে সুহিলপুর উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == সদর উপজেলার উত্তর-পূর্বাংশে মজলিশপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বুধল ইউনিয়ন সুহিলপুর ইউনিয়ন, দক্ষিণে সুহিলপুর ইউনিয়ন বিজয়নগর উপজেলার পত্তন ইউনিয়ন, পূর্বে সরাইল উপজেলার শাহবাজপুর ইউনিয়ন, উত্তরে সরাইল উপজেলার নোয়াগাঁও ইউনিয়ন এবং উত্তর-পশ্চিমে সরাইল উপজেলার সরাইল সদর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মজলিশপুর ইউনিয়ন সদর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == #মৈন্দ উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == হাবিবুর রহমান (বীর উত্তম) == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
