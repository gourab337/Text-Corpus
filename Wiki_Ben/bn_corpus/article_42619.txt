'''বিতারা''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত কচুয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কচুয়া উপজেলার উত্তরাংশে বিতারা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে সাচার ইউনিয়ন পাথৈর ইউনিয়ন; পশ্চিমে মতলব দক্ষিণ উপজেলার নায়েরগাঁও উত্তর ইউনিয়ন নায়েরগাঁও দক্ষিণ ইউনিয়ন; দক্ষিণে সহদেবপুর পশ্চিম ইউনিয়ন, পালাখাল মডেল ইউনিয়ন কচুয়া উত্তর ইউনিয়ন এবং পূর্বে কুমিল্লা জেলার চান্দিনা উপজেলার গল্লাই ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯১৮ সালে বিতারা ইউনিয়ন প্রতিষ্ঠা লাভ করে। == প্রশাসনিক কাঠামো == বিতারা ইউনিয়ন কচুয়া উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। এটি ১১টি মৌজায় বিভক্ত এবং ইউনিয়নে ২৯টি গ্রাম রয়েছে। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == জনাব ইসহাক সিকদার == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (চাঁদপুর) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
