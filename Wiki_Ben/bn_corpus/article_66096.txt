'''চৌদ্দগ্রাম পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চৌদ্দগ্রাম উপজেলার পূর্ব-মধ্যাংশে চৌদ্দগ্রাম পৌরসভার অবস্থান। পৌরসভার উত্তরে ঘোলপাশা ইউনিয়ন, পশ্চিমে মুন্সিরহাট ইউনিয়ন, দক্ষিণে বাতিসা ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == ইতিহাস == চৌদ্দগ্রাম উপজেলার ৭নং চৌদ্দগ্রাম ইউনিয়ন এর সম্পূর্ণ অংশ নিয়ে চৌদ্দগ্রাম পৌরসভা প্রতিষ্ঠিত হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == চৌদ্দগ্রাম পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম চৌদ্দগ্রাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৯নং নির্বাচনী এলাকা কুমিল্লা-১১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == চৌদ্দগ্রাম উপজেলা চৌদ্দগ্রাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
