'''চর চারতলা''' বাংলাদেশের জেলার অন্তর্গত আশুগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == চর চারতলা ইউনিয়নের আয়তন ১৫৭২ একর (৬.৩৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে চর চারতলা ইউনিয়নের মোট জনসংখ্যা ২৫,৭৮৯ জন। মোট পরিবার ৫,০৩৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৪০৫৪ জন। == ইতিহাস == == অবস্থান সীমানা == আশুগঞ্জ উপজেলার পশ্চিমাংশে চর চারতলা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে আশুগঞ্জ সদর ইউনিয়ন আড়াইসিধা ইউনিয়ন; দক্ষিণে লালপুর ইউনিয়ন; পশ্চিমে মেঘনা নদী, নরসিংদী জেলার রায়পুরা উপজেলার চানপুর ইউনিয়ন মুছাপুর ইউনিয়ন এবং উত্তরে মেঘনা নদী কিশোরগঞ্জ জেলার ভৈরব পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == চর চারতলা ইউনিয়ন আশুগঞ্জ উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আশুগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == আশুগঞ্জ উপজেলা আশুগঞ্জ থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
