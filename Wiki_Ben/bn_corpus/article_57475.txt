'''রামপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কোম্পানীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কোম্পানীগঞ্জ উপজেলার মধ্যাংশে রামপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর হাজারী ইউনিয়ন, পূর্বে মুছাপুর ইউনিয়ন, দক্ষিণে চর ফকিরা ইউনিয়ন এবং পশ্চিমে চর ফকিরা ইউনিয়ন চর কাঁকড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রামপুর ইউনিয়ন কোম্পানীগঞ্জ উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কোম্পানীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কোম্পানীগঞ্জ উপজেলা, নোয়াখালী কোম্পানীগঞ্জ থানা, নোয়াখালী নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (নোয়াখালী) উপজেলার ইউনিয়ন
