'''লালদিয়া বন''' বরগুনা জেলার পাথরঘাটা উপজেলায় অবস্থিত একটি ছোট ম্যানগ্রোভ বন। এটি মূলত সুন্দরবনের অংশবিশেষ। এই বনের পূ্র্বে বিষখালি নদী এবং পশ্চিমে বলেশ্বর নদী। এই দুই নদীর মোহনা লালদিয়ার বনকে ঘিরে রেখেছে। এর পূর্ব প্রান্তে লালদিয়া সমুদ্রসৈকত। == নামকরণ == == ইতিহাস == == ভৌগোলিক গঠন == == জীবমন্ডল == == উদ্ভিদ বৈচিত্র == == প্রাণী বৈচিত্র == == অর্থনীতি == == গ্যালারি == == যোগাযোগ == বরগুনা হতে বাসযোগে পাথরঘাটা যাওয়ার পর মোটর সাইকেলে অতি সহজে লালদিয়া বনে যাওয়া যাবে। এছাড়াও পিরোজপুর হয়ে পাথরঘাটায় যাওয়া যায়। == আরও দেখুন == লালদিয়া সমুদ্রসৈকত == তথ্যসূত্র ==
