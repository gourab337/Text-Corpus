'''তারুয়া''' বাংলাদেশের জেলার অন্তর্গত আশুগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == তারুয়া ইউনিয়নের আয়তন ১৬০৩ একর (৬.৪৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে তারুয়া ইউনিয়নের মোট জনসংখ্যা ৮,৭৪৯ জন। মোট পরিবার ১,৭৮৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৩৪৯ জন। == ইতিহাস == == অবস্থান সীমানা == আশুগঞ্জ উপজেলার তারুয়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আশুগঞ্জ সদর ইউনিয়ন, পশ্চিমে আড়াইসিধা ইউনিয়ন শরীফপুর ইউনিয়ন, দক্ষিণে নবীনগর উপজেলার বড়াইল ইউনিয়ন সদর উপজেলার সাদেকপুর ইউনিয়ন এবং পূর্বে সদর উপজেলার সাদেকপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == তারুয়া ইউনিয়ন আশুগঞ্জ উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আশুগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == আশুগঞ্জ উপজেলা আশুগঞ্জ থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
