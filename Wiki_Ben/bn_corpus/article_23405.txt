'''কুপতলা ইউনিয়ন''' বাংলাদেশের গাইবান্ধা সদর উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == এই ইউনিয়নের আয়তন ৫১২৫ একর (২০.৭৫ বর্গ কিলোমিটার) == অবস্থান == গাইবান্ধা জেলা শহর থেকে কিমি উত্তর পশ্চিমে গাইবান্ধা হাট লক্ষীপুর সড়ক ঘেষে গোডাউনবাজার নামক স্থানে সদর উপজেলার কুপতলা ইউনিয়নের অবস্থান। == প্রশাসনিক এলাকা == এর পূর্বে খোলাহাটি ঘাগোয়া এবং লক্ষীপুর ইউপি দক্ষিণে খোলাহাটি, বল্লমঝাড় ইউপি, পশ্চিমে কামারপাড়া এবং উত্তরে লক্ষীপুর ইউপি। == ইতিহাস == == নামকরণ == কথিত আছে অনেক আগে গোডাউন বাজারে বিরাট বড় কুপ ছিল সে থেকেই এলাকার নামকরণ হয় কুপতলা। == জনসংখ্যার উপাত্ত == ইউনিয়নের জনসংখ্যা পুরুষ ১৩,৯৪০ জন মহিলা ১৪,৮৯৩ জন। == শিক্ষা == শিক্ষার হার ৬৮% == অর্থনীতি == ইউনিয়নের বেশির ভাগ লোক কৃষি কাজ করে। == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র ==
