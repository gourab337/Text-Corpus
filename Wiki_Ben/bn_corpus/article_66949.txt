বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মেঘনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মেঘনা উপজেলার দক্ষিণাংশে গোবিন্দপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পশ্চিমে লুটেরচর ইউনিয়ন, পশ্চিমে ভাওরখোলা ইউনিয়ন বড়কান্দা ইউনিয়ন, উত্তরে মানিকারচর ইউনিয়ন, উত্তর-পূর্বে কাঁঠালিয়া নদী রাধানগর ইউনিয়ন, পূর্বে কাঁঠালিয়া নদী তিতাস উপজেলার সাতানী ইউনিয়ন জগতপুর ইউনিয়ন, দক্ষিণ-পূর্বে কাঁঠালিয়া নদী তিতাস উপজেলার মজিদপুর ইউনিয়ন এবং দক্ষিণে দাউদকান্দি উপজেলার দাউদকান্দি উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গোবিন্দপুর ইউনিয়ন মেঘনা উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মেঘনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মেঘনা উপজেলা মেঘনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
