'''ছনুয়া''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার পূর্বাংশে ছনুয়া ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে ফাজিলপুর ইউনিয়ন, দক্ষিণে লেমুয়া ইউনিয়ন, পশ্চিমে কালিদহ ইউনিয়ন এবং উত্তরে মোটবী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছনুয়া ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
