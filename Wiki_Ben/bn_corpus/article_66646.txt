'''ভারেল্লা দক্ষিণ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার পশ্চিমাংশে ভারেল্লা দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ভারেল্লা ইউনিয়ন, পূর্বে ময়নামতি ইউনিয়ন, দক্ষিণে ময়নামতি ইউনিয়ন মোকাম ইউনিয়ন এবং পশ্চিমে মোকাম ইউনিয়ন দেবিদ্বার উপজেলার মোহনপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভারেল্লা দক্ষিণ ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
