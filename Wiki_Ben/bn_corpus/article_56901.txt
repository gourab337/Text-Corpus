'''চাটখিল পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == চাটখিল উপজেলার মধ্যাংশে চাটখিল পৌরসভার অবস্থান। নোয়াখালী জেলা সদর থেকে পৌরসভার দূরত্ব প্রায় ৩৮ কিলোমিটার। পৌরসভার উত্তরে বদলকোট ইউনিয়ন, পশ্চিমে পরকোট ইউনিয়ন, দক্ষিণ-পশ্চিমে হাটপুকুরিয়া ঘাটলাবাগ ইউনিয়ন, দক্ষিণে নোয়াখলা ইউনিয়ন এবং পূর্বে পাঁচগাঁও ইউনিয়ন মোহাম্মদপুর ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯৯৫ সালের জানুয়ারি চাটখিল পৌরসভার কার্যক্রম শুরু হয়। == প্রশাসনিক এলাকা == চাটখিল পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। এটি একটি ''ক'' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা প্রতিষ্ঠান == দারুস সালাম মাদরাসা চাটখিল == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: মোহাম্মদ উল্যাহ == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
