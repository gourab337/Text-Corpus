'''গল্লাই''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == গল্লাই ইউনিয়ন পূর্বে গল্লাই উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার পশ্চিমাংশে গল্লাই ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে সুহিলপুর ইউনিয়ন, উত্তরে বাতাঘাসী ইউনিয়ন, উত্তর-পূর্বে মহিচাইল ইউনিয়ন, দক্ষিণ-পূর্বে দোল্লাই নবাবপুর ইউনিয়ন, দক্ষিণ-পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার কচুয়া উত্তর ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার বিতারা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গল্লাই ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
