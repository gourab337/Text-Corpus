বাংলাদেশের পটুয়াখালী জেলার অন্তর্গত দশমিনা উপজেলার একটি ইউনিয়ন পরিষদ। == অবস্থান == উত্তরে কালাইয়া বন্দর দক্ষিণে দশমিনা পূর্বে তেতুলিয়া নদী এবং পশ্চিমে বহরমপুর ইউনিয়ান == প্রশাসনিক এলাকা গ্রাম (মৌজা) সমূহ == বাঁশবাড়িয়া ইউনিয়ন পরিষদ টি গ্রাম টি মৌজা নিয়ে গঠিতঃ চর-হোসনাবাদ। দক্ষিণ দাসপাড়া। #গছানী। #ঢনঢনিয়া। #বাঁশবাড়িয়া == ইতিহাস == == জনসংখ্যার উপাত্ত == বাঁশবাড়িয়া ইউনিয়নের জনসংখ্যা ২৪৮৮৬ জন পুরুষ ১২৯৯৭ জন এবং মহিলা ১১৮৮৯ জন == শিক্ষা == বাঁশবাড়িয়া ইউনিয়নের শিক্ষার হার শতকরা ৫৬% বাঁশবাড়িয়া ইউনিয়নে টি কলেজ, টি মাদ্যমিক বিদ্যালয় ,৩ টি মাদ্রাসা সহ সরকারী বেসরকারী ১৯ টি প্রাথমিক বিদ্যালয় এবং ১টি কিন্ডার গার্ডেন বিদ্যালয় রয়েছে == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == দশমিনা উপজেলা == তর্থসংযোগ == দেনিক সংবাদ তথ্য বাতায়ন == বহিঃসংযোগ == জেলার ইউনিয়ন
