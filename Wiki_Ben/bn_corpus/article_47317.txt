'''জাহানারা মমতাজ বালিকা উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার জাফতনগর ইউনিয়নে অবস্থিত। == ইতিহাস == ১৯৭০ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == পরিচালনা পরিষদ == == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাবা পল্লবী খাস্তগীর। এছাড়া আরো জন শিক্ষক-শিক্ষিকা বিদ্যালয়ে কর্মরত আছেন। == অবকাঠামো == বিদ্যালয় ভবনটি একটি একতলা ভবন। == কার্যক্রম == এটি একটি বালিকা মাধ্যমিক বিদ্যালয়। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৮৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
