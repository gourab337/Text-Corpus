বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == দরিয়াদৌলত ইউনিয়নের আয়তন ৪৯৫৯ একর (২০.০৭ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে দরিয়াদৌলত ইউনিয়নের মোট জনসংখ্যা ২৫,৩৮৪ জন। মোট পরিবার ৫,২৯০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১২৬৫ জন। == ইতিহাস == == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার উত্তরাংশে দরিয়াদৌলত ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে সোনারামপুর ইউনিয়ন বাঞ্ছারামপুর পৌরসভা, দক্ষিণে ছলিমাবাদ ইউনিয়ন, পূর্বে ছয়ফুল্লাকান্দি ইউনিয়ন ইউনিয়ন এবং উত্তরে মেঘনা নদী নরসিংদী জেলার নরসিংদী সদর উপজেলার আলোকবালী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দরিয়াদৌলত ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
