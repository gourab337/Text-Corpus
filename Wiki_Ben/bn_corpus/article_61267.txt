'''খাদেরগাঁও''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত মতলব দক্ষিণ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মতলব দক্ষিণ উপজেলার মধ্যাংশে খাদেরগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে নারায়ণপুর ইউনিয়ন; উত্তরে নায়েরগাঁও দক্ষিণ ইউনিয়ন, ধনাগোদা নদী মতলব উত্তর উপজেলার সুলতানাবাদ ইউনিয়ন; পশ্চিমে ধনাগোদা নদী, মতলব উত্তর উপজেলার সুলতানাবাদ ইউনিয়ন মতলব পৌরসভা এবং দক্ষিণে উপাদী উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == খাদেরগাঁও ইউনিয়ন মতলব দক্ষিণ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মতলব দক্ষিণ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬১নং নির্বাচনী এলাকা চাঁদপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == মতলব দক্ষিণ উপজেলা মতলব দক্ষিণ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == দক্ষিণ উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
