'''নবীনগর পূর্ব''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == নবীনগর পূর্ব ইউনিয়নের আয়তন ৩২৫২ একর (১৩.১৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নবীনগর পূর্ব ইউনিয়নের মোট জনসংখ্যা ১২,৩৩৭ জন। মোট পরিবার ২,১৫৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ৯৩৭ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার মধ্যাংশে নবীনগর পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে নবীনগর পৌরসভা, দক্ষিণে পূর্বে শিবপুর ইউনিয়ন, উত্তর-পূর্বে বিদ্যাকুট ইউনিয়ন এবং উত্তরে তিতাস নদী কৃষ্ণনগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নবীনগর পূর্ব ইউনিয়ন নবীনগর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
