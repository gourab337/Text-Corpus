'''ছলিমাবাদ''' বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == ছলিমাবাদ ইউনিয়নের আয়তন ৬২৩০ একর (২৫.২১ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ছলিমাবাদ ইউনিয়নের মোট জনসংখ্যা ৩২,৩১৬ জন। মোট পরিবার ৬,৩২৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১২৮২ জন। == ইতিহাস == == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার দক্ষিণাংশে ছলিমাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বাঞ্ছারামপুর পৌরসভা; উত্তরে দরিয়াদৌলত ইউনিয়ন, ছয়ফুল্লাকান্দি ইউনিয়ন রূপসদী ইউনিয়ন; পূর্বে ফরদাবাদ ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার হোমনা উপজেলার চান্দেরচর ইউনিয়ন দুলালপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছলিমাবাদ ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == ছলিমাবাদ ইউনিয়নের উপর দিয়ে বয়ে চলেছে মিরপুর খাল। == হাট-বাজার == ছলিমাবাদ ইউনিয়নের প্রধান প্রধান হাট-বাজারগুলো হল মিরপুর বাজার, ছলিমাবাদ বাজার, বটতলা বাজার এবং হোগলাকান্দা বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: আবদুল মতিন == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
