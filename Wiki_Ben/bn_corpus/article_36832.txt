'''অরুণাচল প্রদেশ'''-এ প্ৰায় ২০ টি মুখ্য জনগোষ্ঠী রয়েছে; এদের মধ্যে অনেকগুলো উপ-জনগোষ্ঠীও আছে। এখানকার জনগোষ্ঠীসমূহের মধ্যে সামাজিক এবং সংস্কৃতিক ভিন্নতা দেখা যায়। জনগোষ্ঠীসমূহের নিজস্ব কিছু ঐতিহ্য রয়েছে। সমূহের মধ্যে শিল্পনৈপুণ্য দক্ষতা দেখা যায়। চিত্ৰকলা, মৃৎ-শিল্প (pottery), দারু-শিল্প (wood carving) এবং বাঁশ-বেতের কার্যে জনগোষ্ঠীদের কলাকৌশল দেখতে পাওয়া যায়। কিছু জনগোষ্ঠীর লোকেরা বয়ন-শিল্প এবং দারু-শিল্পকে জীবিকা হিসেবে গ্ৰহণ করেছে। এসব জনগোষ্ঠী সমূহের অপর একটি বৈশিষ্ট্য হচ্ছে একতা-বদ্ধতা। == জনগোষ্ঠী == আদি আপাতানি বুগুন হ্ৰোছ’ ছিংফ’(জনগোষ্ঠী) খাম্বা এবং মেম্বা মিচিমি মনপা (জনগোষ্ঠী) নিচি (জনগোষ্ঠী) ছেৰ্ডুকপেন তাগিন খামতি (জনগোষ্ঠী) য়ানছো (জনগোষ্ঠী) নোক্তে য়োবিন চাকমা == আদি == '''আদি জনগোষ্ঠী'''র লোকেদের দুটা বিভাগ আছে: ''ব’গোম'' (Bogum) এবং ''ব’মি'' (Bomi)। == আপাতানি == আপাতানি মহিলা == বুগুন == == হ্ৰোছ’ == == ছিংফ’ == == খাম্বা এবং মেম্বা == == মিচিমি == মিচিমি জনগোষ্ঠীর লোক == ম’নপা == == নিচি == নিচি জনগোষ্ঠী == খামতি == খামতি দম্পতি == য়ানছ’ == == নোক্তে == == তাগিন == ==য়োবিন== == ছেৰ্ডুকপেন == == চিত্ৰশালা == জনগোষ্ঠীর লোক মেম্বা জনগোষ্ঠীর যুবতী য়ানছ’ জনগোষ্ঠীর একজন যুবক == আরও দেখুন == অরুণাচল প্রদেশ। == তথ্যসূত্ৰ == == বহিঃসংযোগ == প্রদেশ প্রদেশের জনগোষ্ঠী জাতিগোষ্ঠী
