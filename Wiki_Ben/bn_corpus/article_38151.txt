'''অমরাবতী জেলা''' ভারতীয় অধিরাজ্যের পশ্চিমাঞ্চলীয় রাজ্য মহারাষ্ট্রের জেলা। জেলার সদর বড় শহর অমরাবতী। অমরাবতী জেলা ২০°৩২' ২১°৪৬' উত্তর অক্ষাংশ এবং ৭৬°৩৭' ৭৮°২৭' পূর্ব দ্রাঘিমাংশের মধ্যে অবস্থিত। জেলার মোট আয়তন ১২,২৩৫ বর্গকিলোমিটার। জেলাটির উত্তরদিকে মধ্যপ্রদেশের, বেতুল জেলা, এবং উত্তরপূর্বে মহারাষ্ট্রের নাগপুর জেলা, পূর্বে ওয়ার্ধা জেলা দক্ষিণদিকে যাবাৎমাল জেলা, এবং দক্ষিণ-পশ্চিমে ওয়াশিম জেলা, অকোলা বুলধানা জেলা পশ্চিমে অবস্থিত। == ইতিহাস == == ভৌগোলিক উপাত্ত == == অর্থনৈতিক উপাত্ত == == কৃষি == == প্রশাসনিক বিভাগ == == জনতাত্ত্বিক == === ভাষা === == পরিবহন == == উল্লেখ্য ব্যক্তি == == দর্শনীয় স্থান == == শিক্ষাব্যবস্থা == == তথ্যসূত্র == জেলা
