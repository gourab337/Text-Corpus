'''মোটবী''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার পূর্বাংশে মোটবী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে ফাজিলপুর ইউনিয়ন, দক্ষিণে ছনুয়া ইউনিয়ন, পশ্চিমে কালিদহ ইউনিয়ন, উত্তরে ছাগলনাইয়া উপজেলার পাঠাননগর ইউনিয়ন এবং পূর্বে ছাগলনাইয়া উপজেলার রাধানগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মোটবী ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। ইউনিয়নের গ্রামগুলো হল: মোটবী মধ্যম মোটবী লস্কর হাট শাহপুর ইজ্জতপুর কচুয়া গঙ্গানগর সাতসতী বাঘাইয়া উত্তর লক্ষ্মীপুর দক্ষিণ লক্ষ্মীপুর মধ্যম লক্ষ্মীপুর স্বর্ণসেতু == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;কলেজ বেগম সামসুর নাহার স্কুল এন্ড কলেজ ;মাধ্যমিক বিদ্যালয় লস্করহাট এস সি লাহা উচ্চ বিদ্যালয় মোটবী আদর্শ উচ্চ বিদ্যালয় ইজ্জতপুর উচ্চ বিদ্যালয় ;মাদ্রাসা লস্করহাট মাদ্রাসা গোলাহাট মাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == মোটবী ইউনিয়নের প্রধান ২টি হাট-বাজার হল লস্করহাট বাজার এবং মোটবী ভোর বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
