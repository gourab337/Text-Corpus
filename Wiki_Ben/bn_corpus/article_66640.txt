'''ষোলনল''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার দক্ষিণাংশে ষোলনল ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে গোমতী নদী ময়নামতি ইউনিয়ন, উত্তর-পশ্চিমে পীরযাত্রাপুর ইউনিয়ন, উত্তরে বুড়িচং সদর ইউনিয়ন, উত্তর-পূর্বে বাকশীমূল ইউনিয়ন, পূর্বে কুমিল্লা আদর্শ সদর উপজেলার আমড়াতলী ইউনিয়ন, এবং দক্ষিণে গোমতী নদী কুমিল্লা আদর্শ সদর উপজেলার দুর্গাপুর উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ষোলনল ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
