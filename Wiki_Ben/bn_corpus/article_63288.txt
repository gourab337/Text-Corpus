'''কালচোঁ দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার মধ্যাংশে কালচোঁ দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দ্বাদশ গ্রাম ইউনিয়ন বাকিলা ইউনিয়ন, দক্ষিণে হাজীগঞ্জ সদর ইউনিয়ন, পূর্বে হাজীগঞ্জ সদর ইউনিয়ন হাটিলা পশ্চিম ইউনিয়ন এবং উত্তরে কালচোঁ উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কালচোঁ দক্ষিণ ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
