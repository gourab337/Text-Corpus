'''পরকোট''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার পশ্চিমাংশে পরকোট ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে সাহাপুর ইউনিয়ন, দক্ষিণ-পূর্বে হাটপুকুরিয়া ঘাটলাবাগ ইউনিয়ন, পূর্বে চাটখিল পৌরসভা, উত্তর-পূর্বে বদলকোট ইউনিয়ন, উত্তরে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার ভোলাকোট ইউনিয়ন এবং পশ্চিমে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার দরবেশপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পরকোট ইউনিয়ন চাটখিল উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == এই ইউনিয়নের প্রসিদ্ধ বাজার দশঘরিয়া, এটি পূর্বে যুগীর হাট নামে অত্র অঞ্চলে পরিচিত ছিল। ছাড়াও এই ইউনিয়নে বাইশসিন্ধুর নামে আরো একটি প্রসিদ্ধ বাজার রয়েছে। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
