'''মহামায়া''' বাংলাদেশের ফেনী জেলার অন্তর্গত ছাগলনাইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ছাগলনাইয়া উপজেলার সর্ব-উত্তরে মহামায়া ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে ছাগলনাইয়া পৌরসভা, পশ্চিমে পাঠাননগর ইউনিয়ন ফুলগাজী উপজেলার জিএমহাট ইউনিয়ন, উত্তরে ফুলগাজী উপজেলার আমজাদহাট ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == মহামায়া ইউনিয়ন ছাগলনাইয়া উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ছাগলনাইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ছাগলনাইয়া উপজেলা ছাগলনাইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
