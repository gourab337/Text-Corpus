বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == কড়িকান্দি ইউনিয়ন পূর্বে বলরামপুর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == তিতাস উপজেলার মধ্যাংশে কড়িকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে জগতপুর ইউনিয়ন বলরামপুর ইউনিয়ন, পূর্বে নারান্দিয়া ইউনিয়ন, দক্ষিণে জিয়ারকান্দি ইউনিয়ন এবং পশ্চিমে মজিদপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কড়িকান্দি ইউনিয়ন তিতাস উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == জনাব,মুহাম্মদ মহসীন ভূঁইয়া চেয়ারম্যান, নং কড়িকান্দি সদর ইউনিয়ন == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
