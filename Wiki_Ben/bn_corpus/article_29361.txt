'''আদিপর্ব''' হলো আঠারো পর্বেরর মহাভারতের প্রথম পর্ব। "আদি" (आदि) একটি সংস্কৃত শব্দ যার অর্থ হলো "প্রথম"। আদিপর্ব সর্বমোট ১৯টি উপপর্ব এবং ২৩৬টি অধ্যায় রয়েছে। তবে সমালোচনামূলক সংস্করণে রয়েছে ১৯টি উপপর্বের পাশাপাশি ২৫৫টি অধ্যায়। আদিপর্বে বলা হয়েছে কীভাবে সৌতি নৈমিষারণ্যে সমবেত ঋষিগণের কাছে মহাকাব্যটি আবৃত্তি করেছেন যা বৈশম্পায়ন প্রথম তক্ষশীলায় জনমেজয়কে তার সর্পসত্রে শুনিয়েছিলেন এই পর্বে সমগ্র মহাভারতের বিষয়বস্তু এবং কুরুবংশের আদি কাহিনী বর্ণিত হয়েছে। == গঠনশৈলী এবং অধ্যায়সমূহ == চিত্র:Raja Ravi Varma Mahabharata শকুন্তলা দোষ্মন্তের পুত্র, দুস্মন্তর সাথে তার প্রেমের বিয়ের পর। তাদের পূর্বরাগ এবং প্রণয়লীলা আদিপর্বের সম্ভবপর্ব অধ্যায়ে বর্ণনা করা হয়েছে। রাজা রবি বর্মা কর্তৃক আঁকা চিত্রকর্ম। আদিপর্ব ১৯টি উপপর্ব বা উপ-গ্রন্থ রয়েছে (এছাড়াও ছোট বই হিসেবেও অভিহিত)। প্রতিটি উপ-গ্রন্থকে আলাদাভাবে এক-একটি ''পর্ব'' বলা হয়, এবং অতিরিক্ত উপবিভাজিত অধ্যায়গুলোর মধ্যে, সর্বমোট ২৩৬টি আদিপর্ব অন্তর্ভুক্ত রয়েছে। ১৯টি উপ-গ্রন্থের সংক্ষিপ্ত বর্ণনাসহ নামগুলো হলো: :১। অণুক্রমণিকা পর্ব ::শৌনকের আশ্রমে সৌতি :২। সংগ্রহপর্ব :৩। পৌষ্যপর্ব :: জনমেজয়ের শাপ আরুণি, উপমন্যু বেদের উপাখ্যান :: উতঙ্ক, পৌষ্য তক্ষক এর উপাখ্যান :৪। পৌলমপর্ব :: ভৃগু পুলমা চ্যবন এর উপাখ্যান অগ্নির শাপমোচন :: রুরু, প্রমদবরা ডুণ্ডুভের উপাখ্যান :৫। আস্তিকপর্ব :: জরৎকারু মুনি কদ্রু বিনতা সমুদ্রমন্থন :: কদ্রু-বিনতার পণ গরুড় গজকচ্ছপ অমৃতহরণ :: আস্তিকের জন্ম পরীক্ষিতের মৃত্যুবিবরণ :: জন্মেজয়ের সর্পসত্র :৬। আদিবংশাবতরণপর্ব :: উপরিচর বসু পরাশর সত্যবতী বেদব্যাস :৭। সম্ভবপর্ব :: কচ দেবযানী :: দেবযানী, শর্মিষ্ঠা যযাতি :: যযাতির জরা :: দোষ্মন্ত Ravi Varma Mahabharata রবি বর্মা অঙ্কিত শকুন্তলা তাঁর দুই শখী প্রিয়ম্বদা এবং অনসুয়া :: মহাভিষ অষ্টবসু প্রতীপ শান্তনু গঙ্গা :: দেবব্রত ভীষ্ম সত্যবতী :: চিত্রাঙ্গদ বিচিত্রবীর্য কাশীরাজের তিন কন্যা :: দীর্ঘতমা ধৃতরাষ্ট্র, পাণ্ডু বিদুরের জন্ম অণীমাণ্ডব্য :: গান্ধারী, কুন্তী, মাদ্রী কর্ণ দুর্যোধনাদির জন্ম :: যুধিষ্ঠিরাদির জন্ম পাণ্ডু মাদ্রীর মৃত্যু :: হস্তিনাপুরে পঞ্চপাণ্ডব ভীমের নাগলোক দর্শন :: কৃপ দ্রোণ অশ্বত্থামা একলব্য অর্জুনের পটুতা :: অস্ত্রশিক্ষা প্রদরশন :: দ্রুপদের পরাজয় দ্রোণের প্রতিশোধ :: ধৃতরাষ্ট্রের ঈর্ষা :৮। জতুগৃহপর্ব :: বারণাবত জতুগৃহদাহ চিত্র:Escape of Pandavas from Fire মন্দ রাজা ধৃতরাষ্ট্র কর্তৃক অগ্নিদাহ্য গৃহ থেকে ধর্মচারী পাণ্ডবদের পলায়নের ঘটনার বর্ণনা রয়েছে। ভীম, পাণ্ডব ভ্রাতা, ধোঁয়ার মধ্যে তার কনিষ্ঠ ভ্রাতাকে বহন করেন। :৯। হিড়িম্ববধপর্ব :: হিড়িম্ব হিড়িম্বা ঘটোৎকচের জন্ম :১০। বকবধপর্বা :: একচক্রা বক রাক্ষস :১১। চৈত্ররথপর্ব :: ধৃষ্টদ্যুম্ন দ্রৌপদীর জন্মবৃত্তান্ত গন্ধর্বরাজ অঙ্গারপর্ণ :: তপতী সংবরণ :: বশিষ্ঠ, বিশ্বামিত্র, শক্ত্রি কল্মাষপাদ ঔর্ব ধৌম্য :১২। স্বয়ম্বরপর্ব :: দৌপদীর স্বয়ম্বর অর্জুনের লক্ষ্যভেদ :: কর্ণ শল্য ভীমার্জুনের যুদ্ধ কুন্তী সকাশে দ্রৌপদী :১৩। বৈবাহিকপর্ব :: দ্রুপদ যুধিষ্ঠিরের বিতর্ক :: ব্যাসের বিধান দ্রৌপদীর বিবাহ :১৪। বিদুরাগমনপর্ব :: হস্তিনাপুরে বিতর্ক :১৫। রাজ্যলাভপর্ব :: খাণ্ডবপ্রস্থ সুন্দ, উপসুন্দ তিলোত্তমা :১৬। অর্জুনবনবাসপর্ব :: অর্জুনের বনবাস উলূপী, চিত্রাঙ্গদা বর্গা বভ্রুবাহন :১৭। সুভদ্রাহরণপর্ব :: রৈবতক সুভদ্রাহরণ অভিমন্যু দ্রৌপদীর পঞ্চপুত্র :১৮। হরণাহরণকাপর্ব
