অন্য ব্যবহারের জন্য, দেখুন রঘুনাথপুর ইউনিয়ন''' বাংলাদেশের খুলনা বিভাগের ঝিনাইদহ জেলার হরিণাকুণ্ড উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন-''' ৭২৩৩ একর। '''জনসংখ্যা-''' পুরুষ- ১১০৪২, মহিলা- ১০৪২১। == গ্রাম সমূহ == == হাট-বাজার == চরপাড়া বাজার মান্দিয়া বাজার চুলকানী বাজার রঘুনাথপুর বাজার চেঙ্গেরমোড় বাজার == দর্শনীয় স্থান == সংযোগানন্দ গিরি স্মৃতি মন্দির পঞ্চবটী মহাশশ্মান, ওয়াহেদপুর। হযরত কালাই শাহ (রঃ) এর মাজার, ওয়াহেদপুর। == জনপ্রতিনিধি == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == রঘুনাথপুর ইউনিয়ন, হরিণাকুণ্ড -বাংলাদেশ জাতীয় তথ্য বাতায়ন। হরিণাকুন্ড উপজেলা -বাংলাপিডিয়া। ''বাংলাদেশের ইউনিয়ন বিষয়ক এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' উপজেলার ইউনিয়ন
