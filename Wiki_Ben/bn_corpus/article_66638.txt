'''বাকশীমূল''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার সর্ব-পূর্বে বাকশীমূল ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে উত্তর-পশ্চিমে রাজাপুর ইউনিয়ন, পশ্চিমে বুড়িচং সদর ইউনিয়ন ষোলনল ইউনিয়ন, দক্ষিণে কুমিল্লা আদর্শ সদর উপজেলার আমড়াতলী ইউনিয়ন পাঁচথুবী ইউনিয়ন এবং পূর্বে উত্তর-পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বাকশীমূল ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
