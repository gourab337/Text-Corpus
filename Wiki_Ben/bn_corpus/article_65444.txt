'''দেবহাটা ইউনিয়ন''' বাংলাদেশের খুলনা বিভাগের সাতক্ষীরা জেলার দেবহাটা উপজেলার একটি ইউনিয়ন, প্রশাসনিক এলাকা। == অবস্থান সীমানা == == ইতিহাস == == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' চেয়ারম্যানগণের তালিকা == আরও দেখুন == কুলিয়া ইউনিয়ন, দেবহাটা পারুলিয়া ইউনিয়ন সখিপুর ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
