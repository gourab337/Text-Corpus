'''মৌকরা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার মধ্যাংশে মৌকরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে রায়কোট দক্ষিণ ইউনিয়ন, পশ্চিমে নাঙ্গলকোট পৌরসভা হেসাখাল ইউনিয়ন, দক্ষিণে জোড্ডা পূর্ব ইউনিয়ন ঢালুয়া ইউনিয়ন এবং পূর্বে চৌদ্দগ্রাম উপজেলার চিওড়া ইউনিয়ন কনকাপৈত ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মৌকরা ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
