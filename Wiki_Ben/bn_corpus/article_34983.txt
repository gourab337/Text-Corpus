'''আদাঐর ইউনিয়ন''' বাংলাদেশের হবিগঞ্জ জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই ইউনিয়নের আয়তন ১৪.৯০ বর্গ কি.মি.। এটি মাধবপুর উপজেলা পরিষদ হইতে ০.৫০ কি:মি: দূরে দক্ষিণ পূর্ব দিকে এবং ঢাকা-সিলেট মহাসড়ক হইতে মাধবপুর-মনতলা রাস্তা দিয়ে প্রায় ০.২৫ কি:মি: দূরত্বে অবস্থিত। এখানকার প্রধান নদীর নাম নদী। == ইতিহাস == == ভৌগোণিক উপাত্ত == পাহাড়ী মৃত্তিকায় গঠিত এই অঞ্চলটি। == প্রশাসনিক উপাত্ত == এই ইউনিয়টি টি মৌজার ১৭টি গ্রামের সমন্বয়ে গঠিত। গ্রাম কবিলপুর, আদাঐর উত্তর, আদাঐর দক্ষিণ, গোয়ালনগর, আতকাপাড়া, হালুয়াপাড়া, গোপালপুর, মিঠাপুকুর, ঘিলাতলী, হোসেনপুর, রাজনগর, মৌজপুর, মেহেরপুর, সুলতানপুর, নজরপুর, খিলগাঁও এবং সম্বতপুর। == জনসংখ্যা উপাত্ত == ২০১১ সালের আদম শুমারি অণুযায়ী এই ইউনিয়নের লোক সংখ্যা ১৯,০৬২ জন; যাদের মধ্যে পুরুষ ৯,২৬৫ জন এবং মহিলা ৯,৭৯৭ জন। == নির্বাচিত জন-প্রতিনিধি == বর্তমান ইউনিয়ন পরিষদের সদস্যগণ শপথ গ্রহণ করে ২০১১ সালের ১২ আগষ্ট তারিখে এবং প্রথম সভায় মিলিত হয় একই বছর ২৫ আগষ্ট তারিখে; ফলে এর মেয়াদ উর্ত্তীন হবে ২০১৬ সালের ২৭ জুলাই তারিখ। == শিক্ষাব্যবস্থা == এখানকার শিক্ষার হার ৫৫%। এখানে মোট শিক্ষা প্রতিষ্ঠানের সংখ্যা নিম্নরূপ: সরকারী প্রাথমিক বিদ্যালয় ৭টি, মক্তব ২৮টি, উচ্চ বিদ্যালয় ১টি, মাদ্রাসা ৫টি। == স্বাস্থ্যসেবা == == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ত্ব == == দর্শনীয় গুরুত্বপূর্ণ স্থাপনা == আদাঐর জমিদার বাড়ী, শাহ শরিয়ত উল্লাহ ফকিরের মাজার হালুয়াপাড়া। == বিবিধ == == আরও দেখুন == মাধবপুর পৌরসভা; মাধবপুর উপজেলা; হবিগঞ্জ জেলা; সিলেট বিভাগ; বাংলাদেশের ইউনিয়নসমূহের তালিকা। == তথ্যসূত্র == == বহিঃসংযোগ == আদাঐর ইউনিয়ন জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
