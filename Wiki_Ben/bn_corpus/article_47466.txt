বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == জিয়ারকান্দি ইউনিয়নের আয়তন ১৪ বর্গ কিলোমিটার। == জনসংখ্যা == জিয়ারকান্দি ইউনিয়নের জনসংখ্যা ২৯,৫০০ জন। == ইতিহাস == == অবস্থান সীমানা == তিতাস উপজেলার দক্ষিণাংশে জিয়ারকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মজিদপুর ইউনিয়ন কড়িকান্দি ইউনিয়ন, পূর্বে নারান্দিয়া ইউনিয়ন, দক্ষিণে দাউদকান্দি উপজেলার জিংলাতলী ইউনিয়ন গৌরীপুর ইউনিয়ন এবং পশ্চিমে দাউদকান্দি পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == জিয়ারকান্দি ইউনিয়ন তিতাস উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। গোমতি নদীর তীরে ৭টি গ্রাম নিয়ে ইউনিয়ন গঠিত। গ্রামগুলো হল: জিয়ারকান্দি নোওয়াগাঁও শোলাকান্দি দড়িকান্দি গোপালপুর বাঘাইরামপুর চেংগাতুলী == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোহাম্মদ আলী আশরাফ == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
