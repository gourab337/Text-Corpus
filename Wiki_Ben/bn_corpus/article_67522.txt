'''নোয়াগাঁও''' বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == নোয়াগাঁও ইউনিয়নের আয়তন ৬৭৬৭ একর (২৭.৩৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নোয়াগাঁও ইউনিয়নের মোট জনসংখ্যা ৩৭,৮৬৩ জন। এর মধ্যে পুরুষ ১৮,৪৬১ জন এবং মহিলা ১৯,৪০২ জন। মোট পরিবার ৬,৮৫৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৩৮৩ জন। == ইতিহাস == == অবস্থান সীমানা == সরাইল উপজেলার মধ্যাংশে নোয়াগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কালিকচ্ছ ইউনিয়ন সরাইল সদর ইউনিয়ন, দক্ষিণে সদর উপজেলার মজলিশপুর ইউনিয়ন, দক্ষিণ-পূর্বে শাহবাজপুর ইউনিয়ন, পূর্বে শাহজাদাপুর ইউনিয়ন নাসিরনগর উপজেলার গোকর্ণ ইউনিয়ন এবং উত্তরে নাসিরনগর উপজেলার কুণ্ডা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নোয়াগাঁও ইউনিয়ন সরাইল উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে নোয়াগাঁও ইউনিয়নের সাক্ষরতার হার ৩৬.৯%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
