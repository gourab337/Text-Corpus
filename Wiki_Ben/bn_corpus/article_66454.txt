'''শাকপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == শাকপুর ইউনিয়ন পূর্বে দেওড়া দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার মধ্যাংশে শাকপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে গালিমপুর ইউনিয়ন, দক্ষিণে ভাউকসার ইউনিয়ন, দক্ষিণ-পশ্চিমে আড্ডা ইউনিয়ন, পশ্চিমে ঝলম ইউনিয়ন এবং উত্তরে বরুড়া পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == শাকপুর ইউনিয়ন বরুড়া উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
