'''নাটঘর''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == নাটঘর ইউনিয়নের আয়তন ৬৪৭৬ একর (২৬.২১ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নাটঘর ইউনিয়নের মোট জনসংখ্যা ২৬,০৯০ জন। মোট পরিবার ৪,৫২০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ৯৯৬ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার পূর্বাংশে নাটঘর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বিদ্যাকুট ইউনিয়ন; দক্ষিণে শিবপুর ইউনিয়ন কাইতলা উত্তর ইউনিয়ন; পূর্বে তিতাস নদী, সদর উপজেলার সুলতানপুর ইউনিয়ন, রামরাইল ইউনিয়ন পৌরসভা এবং উত্তরে তিতাস নদী সদর উপজেলার নাটাই দক্ষিণ ইউনিয়ন সাদেকপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নাটঘর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
