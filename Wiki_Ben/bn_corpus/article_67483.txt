'''তালশহর পূর্ব''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == তালশহর পূর্ব ইউনিয়নের আয়তন ৩১৫৩ একর (১২.৭৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে তালশহর পূর্ব ইউনিয়নের মোট জনসংখ্যা ২৫,৩৪৬ জন। মোট পরিবার ৪,৬৯০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৯৮৬ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার তালশহর পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে নাটাই উত্তর ইউনিয়ন, পূর্বে সুহিলপুর ইউনিয়ন বুধল ইউনিয়ন, উত্তরে সরাইল উপজেলার পানিশ্বর ইউনিয়ন এবং পশ্চিমে আশুগঞ্জ উপজেলার দুর্গাপুর ইউনিয়ন তালশহর পশ্চিম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == তালশহর পূর্ব ইউনিয়ন সদর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == ''''== হাট-বাজার ৭নং তালশহর পুর্ব ইউনিয়ন এর মধ্যে মোহনপুর বাজার অনেক পুরাতন বাজার, যা অএ ইউনিয়নে কেন্দ্রস্থলে অবস্থিত,এখানে সাপ্তাহিক ২দিন মঙ্গলবার এবং শুক্রবারে হাট জমে। আর পার্শবর্তি গ্রাম তেলিনগর অস্টগ্রামেও সাপ্তাহিক হাট বসে। == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
