ভাষাবিজ্ঞানীদের নিবন্ধের তালিকা। == == অটো ইয়েসপার্সেন == == আউগুস্ট ফ্রিড্‌রিশ পট আউগুস্ট লেসকিন আউগুস্ট শ্লাইখার আবুল কালাম মনজুর মোরশেদ আর্নল্ড জউইকি == == ইভান স্যাগ ইয়াকপ গ্রিম ইয়োহান ক্রিস্টফ আডেলুং ইয়োহানেস শ্মি‌ট‌ (ভাষাবিজ্ঞানী) == == উইলিয়াম লাবভ == == এডওয়ার্ড সাপির এমিল বঁভনিস্ত == == কার্ল পোলার্ড কার্ল ব্রুগ্‌মান কার্ল ভার্নার == == চার্লস হকেট == == জন রবার্ট রস জন লায়ন্স (ভাষাবিজ্ঞানী) জর্জ কিংস্‌লি জিফ জর্জ লেকফ জিন বার্কো গ্লীসন জে আর ফার্থ জেমস ম্যাকলি জেরল্ড ক্যাট্‌জ জেরাল্ড গাজডার == == ডেভিড ক্রিস্টাল ডেভিড ডিরিঙ্গার == == নিকোলাই ত্রুবেৎস্‌কোয় নোম চম্‌স্কি == == পল পোস্টাল পিটার ল্যাডিফোগিড == == ফ্রান্ৎস বপ ফ্রিড্‌রিশ ক্লুগে ফ্রিড্‌রিশ ফন শ্লেগেল ফের্দিনঁ দ্য সোস্যুর == == বের্টোল্ড ডেলব্রুক বার্নার্ড কম্‌রি বেনজামিন হোর্ফ == == ভিক্টর ইংভে ভিল্‌হেল্ম ফন হুম্বোল্ট ভিলহেল্ম ব্রাউন == == মুনীর চৌধুরী মুহম্মদ আবদুল হাই মুহম্মদ শহীদুল্লাহ মাইকেল হ্যালিডে মার্ক অ্যারনফ মোফাজ্জল হায়দার চৌধুরী == == রুডল্‌ফ ফন রাউমার রফিকুল ইসলাম (ভাষাবিজ্ঞানী) র‌্যান্‌ডল্ফ কোয়ার্ক রাস্‌মুস রাস্ক রে জ্যাকেন্ডফ রোমান ইয়াকবসন == == লুই ইয়েল্ম্‌স্লেভ লুইস হার্বার্ট গ্রে লুডভিগ উইটগেনস্টাইন লিওনার্ড ব্লুমফিল্ড == == সুকুমার সেন সুনীতিকুমার চট্টোপাধ্যায় স্টিভেন পিংকার == == হুমায়ুন আজাদ হের্মান অস্ট্‌হফ হের্মান গ্রাসমান হের্মান পাউল
