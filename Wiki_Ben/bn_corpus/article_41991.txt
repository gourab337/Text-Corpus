'''চান্দহর ইউনিয়ন''' ঢাকা বিভাগের মানিকগঞ্জ জেলার অন্তর্গত সিঙ্গাইর উপজেলার একটি ইউনিয়ন। == অবস্থান আয়তন == এই ইউনিয়নের আয়তন ১০ বর্গ কিলোমিটার। ইউনিয়নটিতে গ্রামের সংখ্যা ৩২ টি। == ইতিহাস == == ভৌগোলিক উপাত্ত == == প্রশাসনিক উপাত্ত == == জনসংখ্যার উপাত্ত == == শিক্ষা == == কৃষি == == অর্থনীতি == == যোগাযোগ == == কৃতী ব্যক্তিত্ব == মরহুম শাহাদাত হোসেন মাষ্টার। শাহজাহান মাষ্টার। কোনাল। সাইদুর রহমান সাইদ। == তথ্যসূত্র == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
