'''মিলিমিটার''' (ইন্টারন্যাশনাল বুরো অব ওয়েট এন্ড মিজার্স দ্বারা স্বীকৃত, এসআই একক '''এমএম''') হল দৈর্ঘ্য পরিমাপের একক, যা মেট্রিক পদ্ধতিতে ব্যবহৃত হয়। এক মিটারের এক হাজার ভাগের এক ভাগকে এক মিলিমিটার বলে। এটি ১০০০ মাইক্রোমিটার এবং ১০০০০০০ ন্যানোমিটারের সমান। মিলিমিটার (নিখুতভাবে ০.০৩৯৩৭০) ইঞ্চির সমান হয়। == সংজ্ঞা == ১৯৮৩ সাল থেকে দৈর্ঘ্য পরিমাপের একক হিসেবে আন্তর্জাতিক ভাবে মিটার ব্যবহৃত হয়ে আসছে আলো শূন্যস্থানে ১/২৯৯৭৯২৪৫৮ সেকেন্ডে যে দুরুত্ব অতিক্রম করে সে দুরত্ব হলো এক মিটার আর এক মিটারের ১/১০০০ ভাগ হলো এক মিলিমিটার ।অতএব ১/২৯৯৭৯২৪৫৮০০০ সেকেন্ডে আলো যে দুরুত্ব অতিক্রম করে তাকে এক মিলিমিটার বলে। == ইউনিকোড প্রতীক == Chinese, Japanese এবং Korean (CJK) অক্ষরের সাথে সঙ্গতি রাখার জন্য নিম্নোক্ত ইউনিকোড প্রতীক ব্যবহার করা হয়ঃ ১) মিলিমিটার (㎜) code U+339C ২) বর্গ মিলিমিটার (㎟) code U+339F ৩)ঘন মিলিমিটার (㎣) code U+33A3 == পরিমাপ == মেট্রিক পদ্ধতিতে সবচেয়ে ক্ষুদ্রতম পরিমাপ হল মিলিমিটার তবে ভার্ণিয়ার ক্যালিপার্স দিয়ে এখন ০.০১ মিলিমিটার পর্যন্তও পরিমাপ করা যায় == পরিমাপক == মিলিমিটার পরিমাপ নির্ণয় করার জন্য সবচেয়ে জনপ্রিয় পরিমাপক হলো মিটার স্কেল। এছাড়াও অন্যান্য যন্ত্র দিয়ে নিখুত পরিমাপ করা যায়। == আরো দেখুন == == তথ্যসূত্র == একক
