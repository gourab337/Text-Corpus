'''সাহাপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার সর্ব-পশ্চিমে সাহাপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে রামনারায়ণপুর ইউনিয়ন হাটপুকুরিয়া ঘাটলাবাগ ইউনিয়ন, উত্তরে পরকোট ইউনিয়ন, পশ্চিমে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার করপাড়া ইউনিয়ন এবং দক্ষিণে লক্ষ্মীপুর জেলার লক্ষ্মীপুর সদর উপজেলার বশিকপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সাহাপুর ইউনিয়ন চাটখিল উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
