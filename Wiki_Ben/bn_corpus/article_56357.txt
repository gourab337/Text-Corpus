'''শর্শদি''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার শর্শদি ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে ধর্মপুর ইউনিয়ন, দক্ষিণ-পূর্বে ফেনী পৌরসভা, দক্ষিণে পাঁচগাছিয়া ইউনিয়ন, পশ্চিমে দাগনভূঁইয়া উপজেলার সিন্দুরপুর ইউনিয়ন এবং উত্তরে কুমিল্লা জেলার চৌদ্দগ্রাম উপজেলার আলকরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শর্শদি ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == শর্শদি ইউনিয়নের প্রধান হাট-বাজার শর্শদি বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
