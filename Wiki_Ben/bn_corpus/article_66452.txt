'''ঝলম''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == ঝলম ইউনিয়ন পূর্বে ঝলম দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার পশ্চিমাংশে ঝলম ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চিতড্ডা ইউনিয়ন, উত্তর-পূর্বে খোশবাস দক্ষিণ ইউনিয়ন, পূর্বে বরুড়া পৌরসভা শাকপুর ইউনিয়ন, দক্ষিণে আড্ডা ইউনিয়ন এবং পশ্চিমে চান্দিনা উপজেলার জোয়াগ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ঝলম ইউনিয়ন বরুড়া উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
