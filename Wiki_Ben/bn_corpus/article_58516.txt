'''উত্তর জয়পুর''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার পূর্বাংশে উত্তর জয়পুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দত্তপাড়া ইউনিয়ন, দক্ষিণে হাজিরপাড়া ইউনিয়ন, পূর্বে চন্দ্রগঞ্জ ইউনিয়ন এবং উত্তরে নোয়াখালী জেলার চাটখিল উপজেলার খিলপাড়া ইউনিয়ন রামনারায়ণপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উত্তর জয়পুর ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
