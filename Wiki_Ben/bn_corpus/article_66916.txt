'''বারপাড়া''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বারপাড়া ইউনিয়ন পূর্বে সুন্দলপুর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দাউদকান্দি উপজেলার উত্তর-মধ্যাংশে বারপাড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে গৌরীপুর ইউনিয়ন, পশ্চিমে সুন্দলপুর ইউনিয়ন, দক্ষিণে বিটেশ্বর ইউনিয়ন এবং পূর্বে মোহাম্মদপুর পশ্চিম ইউনিয়ন জিংলাতলী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বারপাড়া ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == ঢাকা থেকে ৪৪ কিলোমিটার দূরেঅবস্থিত == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == বিখ্যাত আইনজীবী রিয়াজ খাঁন এর বাড়ি == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
