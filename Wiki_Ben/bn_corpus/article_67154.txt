'''তেজখালী''' বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == তেজখালী ইউনিয়নের আয়তন ৩৩৩৬ একর (১৩.৫০ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে তেজখালী ইউনিয়নের মোট জনসংখ্যা ২১,৮৯৩ জন। মোট পরিবার ৪,৪৩৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৬২২ জন। == ইতিহাস == == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার উত্তর-পূর্বাংশে তেজখালী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে দড়িকান্দি ইউনিয়ন ছয়ফুল্লাকান্দি ইউনিয়ন; পশ্চিমে ইউনিয়ন; উত্তরে ইউনিয়ন নবীনগর উপজেলার ছলিমগঞ্জ ইউনিয়ন এবং পূর্বে নবীনগর উপজেলার বড়িকান্দি ইউনিয়ন, ছলিমগঞ্জ ইউনিয়ন শ্যামগ্রাম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == তেজখালী ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
