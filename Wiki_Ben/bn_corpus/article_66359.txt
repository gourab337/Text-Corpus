'''হাসনাবাদ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মনোহরগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মনোহরগঞ্জ উপজেলার সর্ব-পশ্চিমে হাসনাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বাইশগাঁও ইউনিয়ন, দক্ষিণে নোয়াখালী জেলার চাটখিল উপজেলার মোহাম্মদপুর ইউনিয়ন, পশ্চিমে নোয়াখালী জেলার চাটখিল উপজেলার বদলকোট ইউনিয়ন এবং উত্তরে চাঁদপুর জেলার শাহরাস্তি উপজেলার চিতোষী পূর্ব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাসনাবাদ ইউনিয়ন মনোহরগঞ্জ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মনোহরগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মনোহরগঞ্জ উপজেলা মনোহরগঞ্জ থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
