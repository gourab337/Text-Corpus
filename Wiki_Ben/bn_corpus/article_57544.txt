'''চর ঈশ্বর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত হাতিয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাতিয়া উপজেলার পূর্বাংশে চর ঈশ্বর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে নলচিরা ইউনিয়ন; পশ্চিমে হাতিয়া পৌরসভা তমরদ্দি ইউনিয়ন; দক্ষিণে বুড়িরচর ইউনিয়ন এবং পূর্বে সন্দ্বীপ চ্যানেল চট্টগ্রাম জেলার সন্দ্বীপ উপজেলার সারিকাইত ইউনিয়ন, মাইটভাঙ্গা ইউনিয়ন আজিমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর ঈশ্বর ইউনিয়ন হাতিয়া উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। হাতিয়া মূলদ্বীপ থেকে বিচ্ছিন্ন ভাসানচর ব্যতীত ইউনিয়নের প্রশাসনিক কার্যক্রম হাতিয়া থানার আওতাধীন এবং ভাসানচরের প্রশাসনিক কার্যক্রম ভাসানচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৩নং নির্বাচনী এলাকা নোয়াখালী-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাতিয়া উপজেলা হাতিয়া থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
