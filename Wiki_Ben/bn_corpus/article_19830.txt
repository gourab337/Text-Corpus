'''ডোমার পৌরসভা''' বাংলাদেশের নীলফামারী জেলার অন্তর্গত একটি পৌরসভা। == অবস্থান == ডিবি রোড, ডোমার, নীলফামারী। == প্রশাসনিক এলাকা == মেয়রঃ জনাব মনছুরুল ইসলাম দানু মোট ওয়াডঃ ০৯ টি == ইতিহাস == == জনসংখ্যার উপাত্ত == মোট জনসংখ্যাঃ জন পুরুষ মহিলা (২০১২ আদম শুমারী == শিক্ষা == প্রাথমিক বিদ্যালয়ঃ টি মাধ্যমিক বিদ্যালয়ঃ ০৪টি মোট মহাবিদ্যালয়ঃ ০৩ টি == অর্থনীতি == ডোমারের প্রধান অর্থনীতি কৃষি৷ এছাড়া মত্‍স্য চাষ, হাঁস মুরগ খামার, গবাদী খামার, ক্ষুদ্র কুঠির শিল্প, ব্যবসা বাণিজ্য এলাকার অর্থনীতিকে সচল রেখেছে == কৃতী ব্যক্তিত্ব == অন্যান্য == বিবিধ == ডোমার পৌরসভায় বাস স্ট্যান্ডের নিকট ডিবি রোডের একটি দৃশ্য == আরও দেখুন == == বহিঃসংযোগ == বিভাগের পৌরসভা‎ জেলার পৌরসভা
