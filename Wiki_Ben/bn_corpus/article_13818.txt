'''পানছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত পানছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == পানছড়ি উপজেলার মধ্যাংশ থেকে উত্তর-পূর্বাংশ জুড়ে পানছড়ি ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে চেঙ্গী ইউনিয়ন উল্টাছড়ি ইউনিয়ন, দক্ষিণে লতিবান ইউনিয়ন, পূর্বে দীঘিনালা উপজেলার বাবুছড়া ইউনিয়ন এবং উত্তরে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == পানছড়ি ইউনিয়ন পানছড়ি উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম পানছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == পানছড়ি ইউনিয়নের পশ্চিম সীমান্ত দিয়ে প্রবাহিত হচ্ছে চেঙ্গী নদী। == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == পানছড়ি উপজেলা পানছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
