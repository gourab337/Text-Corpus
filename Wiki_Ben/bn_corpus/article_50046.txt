'''ফরিদগঞ্জ পৌরসভা''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == ফরিদগঞ্জ পৌরসভার আয়তন ১৯.৭৫ বর্গ কিলোমিটার। আয়তনের দিক থেকে এটি চাঁদপুর জেলার দ্বিতীয় বৃহত্তম পৌরসভা। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী ফরিদগঞ্জ পৌরসভার জনসংখ্যা ৩৪,৬১১ জন। এর মধ্যে পুরুষ ১৬,৪৮৫ জন এবং মহিলা ১৮,১২৬ জন। == অবস্থান সীমানা == ফরিদগঞ্জ উপজেলার মধ্যাংশে ফরিদগঞ্জ পৌরসভার অবস্থান। পৌরসভার পূর্বে রূপসা উত্তর ইউনিয়ন, উত্তরে পাইকপাড়া দক্ষিণ ইউনিয়ন, পশ্চিমে গোবিন্দপুর উত্তর ইউনিয়ন গোবিন্দপুর দক্ষিণ ইউনিয়ন এবং দক্ষিণে ফরিদগঞ্জ দক্ষিণ ইউনিয়ন অবস্থিত। == ইতিহাস == == নামকরণ == == প্রতিষ্ঠাকাল == ২০০৫ সালের সেপ্টেম্বর ফরিদগঞ্জ উপজেলার ১৩নং ফরিদগঞ্জ উত্তর ইউনিয়ন পরিষদের আওতাধীন সম্পূর্ণ এলাকা নিয়ে ফরিদগঞ্জ পৌরসভা প্রতিষ্ঠিত হয়। প্রতিষ্ঠাকালে ''গ'' শ্রেণীর পৌরসভা হিসেবে প্রতিষ্ঠিত হলেও ২০১৬ সালের ২০ জুন পৌরসভাকে ''খ'' শ্রেণীতে উন্নীত করা হয়। == প্রশাসনিক এলাকা == ফরিদগঞ্জ পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম ফরিদগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৩নং নির্বাচনী এলাকা চাঁদপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == ফরিদগঞ্জ পৌরসভার সাক্ষরতার হার ৬২.৪৬%। পৌরসভায় ১টি কলেজ, ২টি মাদ্রাসা, ২টি মাধ্যমিক বিদ্যালয় এবং ১৫টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == ফরিদগঞ্জ বঙ্গবন্ধু সরকারি ডিগ্রী কলেজ ফরিদগঞ্জ আর পাইলট উচ্চ বিদ্যালয় আর্দশ একাডেমি ফরিদগঞ্জ ফরিদগঞ্জ বালিকা উচ্চ বিদ্যালয় ফরিদগঞ্জ মাজিদিয়া কামিল মাদ্রাসা দক্ষিণ কাছিয়াড়া মহিলা সিনিয়র আলিম মাদ্রাসা == যোগাযোগ ব্যবস্থা == শুধু মাত্র সড়ক যোগাযোগ ব্যবস্থা মোটরসাইকেল, সিএনজি চালিত অটোরিক্সা, বাস, রিক্সা বিদ্যমান। == অর্থনীতি == মূলত কৃষি, ব্যবসা এবং বৈদেশিক রেমিটেন্স থেকে পৌরসভার অর্থনীতি নির্ভরশীল। == দর্শনীয় স্থান == ওনুয়া স্মৃতি ভাস্কর্য ফরিদগঞ্জ ব্রীজ গুধারা ঘাট রাজা পার্ক == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == পৌর মেয়র: মোহাম্মদ মাহফুজুল হক == আরও দেখুন == ফরিদগঞ্জ উপজেলা ফরিদগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
