'''বসুরহাট পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == কোম্পানীগঞ্জ উপজেলার উত্তরাংশে বসুরহাট পৌরসভার অবস্থান। পৌরসভার উত্তরে পশ্চিমে সিরাজপুর ইউনিয়ন, দক্ষিণে চর কাঁকড়া ইউনিয়ন, পূর্বে চর হাজারী ইউনিয়ন এবং উত্তর-পূর্বে চর পার্বতী ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == বসুরহাট পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম কোম্পানীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। এটি একটি ''ক'' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == চট্টগ্রাম শহর থেকে বসুরহাট যাওয়ার জন্য ''বসুরহাট এক্সপ্রেস'' নামে একটি বাস সার্ভিস রয়েছে। == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: আব্দুল কাদের মির্জা == আরও দেখুন == কোম্পানীগঞ্জ উপজেলা, নোয়াখালী কোম্পানীগঞ্জ থানা, নোয়াখালী নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা, নোয়াখালী
