'''ডক্টর মুহাম্মদ এনামুল হক একাডেমী বহুমুখী উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার বখতপুর ইউনিয়নে অবস্থিত। == ইতিহাস == ১৯৭০ সালে এটি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == == অবকাঠামো == == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান প্রতিষ্ঠিত শিক্ষা প্রতিষ্ঠান
