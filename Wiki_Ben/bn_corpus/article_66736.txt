'''মুরাদনগর সদর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার পশ্চিমাংশে মুরাদনগর সদর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে রামচন্দ্রপুর দক্ষিণ ইউনিয়ন কামাল্লা ইউনিয়ন, পূর্বে নবীপুর পশ্চিম ইউনিয়ন, দক্ষিণ-পূর্বে ধামঘর ইউনিয়ন, দক্ষিণে জাহাপুর ইউনিয়ন এবং পশ্চিমে হোমনা উপজেলার ভাসানিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মুরাদনগর সদর ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মুরাদনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা মুরাদনগর থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
