'''দক্ষিণ বাণীগ্রাম ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত কানাইঘাট উপজেলার একটি ইউনিয়ন। == অবস্থান == সিলেট শহর হতে সড়ক যোগে কানাইঘাট হতে ১৫ কি: মি: পশ্চিমে মাছুখাল বাজর থেকে বাম দিকে কি.মি পায়ে হেটে মানিকগঞ্জ বাজরের পাশে ৭নং দক্ষিন বানীগ্রাম ইউনিয়। সীমানা: পূর্বে- কানাইঘাট ইউনিয়ন, পশ্চিমে- ঝিংগাবাড়ী ইউনিয়ন, উত্তরে- জৈন্তাপুর উপজেলা, দক্ষিনে- বিয়ানীবাজার উপজেলা। == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন-''' '''জনসংখ্যা-''' মোট- ভোটারসংখ্যা- ১৬,৫০০ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার-''' ২৫% '''শিক্ষা প্রতিষ্ঠাণ-''' সরকারী- ১২টি। নাই। বেসরকারী ২টি। কলেজেরসংখ্যা- বেসরকারী ১টি। আলীয়া- ১টি, কওমি- ৫টি, অন্যান্য ৫টি। মসজিদ-৫৮ টি, মন্দির- ১টি। == দর্শনীয় স্থান == == হাট-বাজার == গাছবাড়ী বাজার বড়দেশ বাজার মানিকগঞ্জ বাজার সীমা বাজার মাছুখাল বাজার == খাল নদী == পাশ দিয়ে বয়ে চলেছে সৌন্দর্যপূর্ন সুরমা নদী। ইউনিয়নে মোট টি ছোট বড় খাল রযেছে। প্রত্যেকটি খালই সুরমা নদীর সাথে সংযুক্ত হয়েছে। == কৃতী ব্যক্তিত্ব == মরহুম ক্বারী আতহার আলি; প্রতিষ্ঠাতা গাছবাড়ী জামিউল উলুম কামিল মাদ্রাসা (১৮৩০-১৯০৯)। মরহুম মাওলানা কামিল আহমেদ; অধ্যক্ষ গাছবাড়ী জামিউল উলুম কামিল মাদ্রাসা (১৮৫০-১৯১৩)। মরহুম মাওলানা আবু ইউসুফ ইয়াকুব; অধ্যক্ষ গাছবাড়ী জামিউল উলুম কামিল মাদ্রাসা (১৯১৯-১৯৬১)। == আরও দেখুন == পূর্ব দিঘীরপার ইউনিয়ন সাতবাঁক ইউনিয়ন বড়চতুল ইউনিয়ন কানাইঘাট ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাপিডিয়ায় ''কানাইঘাট উপজেলা'' কানাইঘাট উপজেলা জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
