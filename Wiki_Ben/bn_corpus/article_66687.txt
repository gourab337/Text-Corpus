'''বড়শালঘর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বড়শালঘর ইউনিয়ন পূর্বে ১নং বড়শালঘর উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার সর্ব-উত্তরে বড়শালঘর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে ইউসুফপুর ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার শিদলাই ইউনিয়ন, পূর্বে ব্রাহ্মণপাড়া উপজেলার শিদলাই ইউনিয়ন মাধবপুর ইউনিয়ন, উত্তরে মুরাদনগর উপজেলার পূর্বধইর পূর্ব ইউনিয়ন এবং পশ্চিমে মুরাদনগর উপজেলার পূর্বধইর পশ্চিম ইউনিয়ন টনকী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বড়শালঘর ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
