'''কসবা পশ্চিম''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == কসবা পশ্চিম ইউনিয়নের আয়তন ১৩৩৫ একর (৫.৪০ বর্গ কিলোমিটার)। এটি আয়তনের দিক থেকে জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে কসবা পশ্চিম ইউনিয়নের মোট জনসংখ্যা ১০,০৪২ জন। মোট পরিবার ১,৮৭১টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৮৫৯ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার মধ্যাংশে কসবা পশ্চিম ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে কসবা পৌরসভা, উত্তরে বিনাউটি ইউনিয়ন খাড়েরা ইউনিয়ন, পশ্চিমে মেহারী ইউনিয়ন কুটি ইউনিয়ন এবং দক্ষিণে কায়েমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কসবা পশ্চিম ইউনিয়ন কসবা উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
