'''মাধবপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত ব্রাহ্মণপাড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == ব্রাহ্মণপাড়া উপজেলার সর্ব-উত্তরে মাধবপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে চান্দলা ইউনিয়ন শিদলাই ইউনিয়ন; পশ্চিমে দেবিদ্বার উপজেলার বড়শালঘর ইউনিয়ন, মুরাদনগর উপজেলার পূর্বধইর পূর্ব ইউনিয়ন আন্দিকোট ইউনিয়ন; উত্তরে জেলার কসবা উপজেলার কুটি ইউনিয়ন এবং পূর্বে জেলার কসবা উপজেলার কায়েমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মাধবপুর ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ব্রাহ্মণপাড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: সুলতান আহমেদ == আরও দেখুন == ব্রাহ্মণপাড়া উপজেলা ব্রাহ্মণপাড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
