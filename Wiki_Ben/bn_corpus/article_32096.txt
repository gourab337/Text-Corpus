'''নোয়াখালী পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == নোয়াখালী পৌরসভা কার্যালয় নোয়াখালী সদর উপজেলার উত্তরাংশে নোয়াখালী পৌরসভার অবস্থান। পৌরসভার পশ্চিমে কাদির হানিফ ইউনিয়ন বিনোদপুর ইউনিয়ন, দক্ষিণে নোয়াখালী ইউনিয়ন, পূর্বে অশ্বদিয়া ইউনিয়ন নিয়াজপুর ইউনিয়ন এবং উত্তরে বেগমগঞ্জ উপজেলার একলাশপুর ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৮৬৭ সালে নোয়াখালী পৌরসভা প্রতিষ্ঠিত হয়। == প্রশাসনিক এলাকা == নোয়াখালী পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম সুধারাম মডেল থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। এটি একটি ''ক'' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: মোহাম্মদ সহিদ উল্যাহ খান সোহেল == আরও দেখুন == নোয়াখালী সদর উপজেলা সুধারাম মডেল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা সদর উপজেলা
