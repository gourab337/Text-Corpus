'''বরুড়া পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বরুড়া উপজেলার মধ্যাংশে বরুড়া পৌরসভার অবস্থান। পৌরসভার উত্তরে খোশবাস দক্ষিণ ইউনিয়ন, উত্তর-পূর্বে আগানগর ইউনিয়ন, পূর্বে ভবানীপুর ইউনিয়ন শিলমুড়ি উত্তর ইউনিয়ন, দক্ষিণ-পূর্বে গালিমপুর ইউনিয়ন, দক্ষিণে শাকপুর ইউনিয়ন এবং পশ্চিমে ঝলম ইউনিয়ন অবস্থিত। == ইতিহাস == বরুড়া উপজেলার ৭নং দেওড়া উত্তর ইউনিয়ন এর সম্পূর্ণ অংশ নিয়ে বরুড়া পৌরসভা গঠন করা হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == বরুড়া পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
