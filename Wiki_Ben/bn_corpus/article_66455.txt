'''ভাউকসার''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বরুড়া উপজেলার মধ্যাংশে ভাউকসার ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে গালিমপুর ইউনিয়ন, দক্ষিণে পয়ালগাছা ইউনিয়ন, পশ্চিমে আদ্রা ইউনিয়ন আড্ডা ইউনিয়ন এবং উত্তরে শাকপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভাউকসার ইউনিয়ন বরুড়া উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
