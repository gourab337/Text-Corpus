'''কেরণখাল''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == কেরণখাল ইউনিয়ন পূর্বে বাড়েরা উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার উত্তরাংশে কেরণখাল ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে মাধাইয়া ইউনিয়ন, দক্ষিণে বাড়েরা ইউনিয়ন, পূর্বে চান্দিনা পৌরসভা এবং উত্তরে দেবিদ্বার উপজেলার সুলতানপুর ইউনিয়ন ভানী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কেরণখাল ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
