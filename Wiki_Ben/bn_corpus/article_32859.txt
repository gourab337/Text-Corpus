'''মাধবপুর পৌরসভা''' বাংলাদেশের সিলেট বিভাগের হবিগঞ্জ জেলার একটি স্থানীয় সরকার সংস্থা। ১৯৯৭ সালে প্রতিষ্ঠিত এই পৌরসভাটি বাংলাদেশের একটি ‘‘ক’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == মাধবপুর পৌরসভাটি সিলেট বিভাগের হবিগঞ্জ জেলার মাধবপুর উপজেলা সদরে অবস্থিত। == ইতিহাস == == ভৌগোলিক উপাত্ত == == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == == নির্বাচিত জন-প্রতিনিধি == বিগত ২০১৫ সালের ৩০ ডিসেম্বর তারিখ অনুষ্ঠিত পৌরসভা নির্বাচনে আওয়ামী লীগ সমর্থিত প্রার্থী হিরেন্দ্র লাল সাহা মেয়র পদে নির্বাচিত হয়েছেন। == জনসংখ্যা উপাত্ত == ২০১১ সালের আদমশুমারী অনুসারে মাধবপুর পৌরসভার লোক সংখ্যা ২১,৯৩০ জন; যাদের ১১,১০৬ জন পুরুষ এবং ১০,৮২৪ জন মহিলা। == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == এখানে ১০ টি প্রাথমিক বিদ্যালয় (সরকারি প্রাথমিক বিদ্যালয় টি, রেজিস্টার্ড বেসরকারি টি, কিন্ডার গার্টেন টি); টি উচ্চ বিদ্যালয় (বেসরকারি বালক টি, বেসরকারি বালিকা টি); টি মাদ্রাসা (আলিয়া টি, কওমী টি) এবং টি কলেজ (বেসরকারি বিশ্ববিদ্যালয়) রয়েছে। == নাগরিক সুযোগ-সুবিধা == এই পৌরসভার নাগরিক সুযোগ-সুবিধা বিভিন্ন ক্ষেত্রে কিছুটা অপ্রতুল; এর মধ্যে পয়োঃনিষ্কাশন ব্যবস্থা প্রধান। == কৃষি অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ব == == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == == বিবিধ == == আরও দেখুন == মাধবপুর উপজেলা; হবিগঞ্জ জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== মাধবপুর পৌরসভা জাতীয় তথ্য বাতায়ন। মাধবপুর পৌরসভা স্থানীয় সরকার প্রকৌশল অধিদপ্তর। উপজেলা জেলার পৌরসভা বিভাগের পৌরসভা
