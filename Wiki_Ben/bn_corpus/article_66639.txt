'''বুড়িচং সদর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার উত্তরাংশে বুড়িচং সদর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে উত্তর-পূর্বে রাজাপুর ইউনিয়ন, পূর্বে বাকশীমূল ইউনিয়ন, দক্ষিণে ষোলনল ইউনিয়ন, পশ্চিমে পীরযাত্রাপুর ইউনিয়ন এবং উত্তর-পশ্চিমে ব্রাহ্মণপাড়া উপজেলার সাহেবাবাদ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বুড়িচং সদর ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। ওয়ার্ডভিত্তিক ইউনিয়নের গ্রামগুলো হল: ওয়ার্ড নং গ্রাম/এলাকা ১নং ওয়ার্ড জগতপুর ২নং ওয়ার্ড ৩নং ওয়ার্ড ৪নং ওয়ার্ড জরইন ৫নং ওয়ার্ড পূনমতি ৬নং ওয়ার্ড হরিপুর ৭নং ওয়ার্ড আরাগ আনন্দপুর ৮নং ওয়ার্ড বুড়িচং উত্তর ৯নং ওয়ার্ড বুড়িচং দক্ষিণ == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠানঃ == == জেনারেল হাই স্কুল ৯টি == ১.বুড়িচং আনন্দ পাইলট সরকারি উচ্চ বিদ্যালয় ১৯২০ ২.বুড়িচং কালি নারায়ণ বালিকা উচ্চ বিদ্যালয় ৩.বুড়িচং হাজি ফজর আলী উচ্চ বিদ্যালয় ৪.বুড়িচং মডেল একাডেমী ৫.হরিপুর আদর্শ উচ্চ বিদ্যালয় ১৯৯৪ ৬.আরাগ আনন্দপুর উচ্চ বিদ্যালয় ৭.পূর্ণমতি এম উচ্চ বিদ্যালয় '''কারিগরিঃ ২টি''' '''১.ফজলুর রহমান মেমোরিয়্যাল কলেজ অব টেকনোল ২০০২''' '''২.হরিপুর আদর্শ উচ্চ বিদ্যালয় ১৯৯৪''' মাদ্রাসাঃ২টি ১.বুড়িচং ইসলামিয়া সুন্নিয়া আলিম মাদ্রাসা ২.জগতপুর এ. ডি. এইস ফাজিল ডিগ্রি মাদ্রাসা। '''কলেজঃ২টি''' ১.জেনারেল কলেজ ১টি, বুুড়িচং এরশাদ ডিগ্রী কলেজ ১৯৮৯ ২.কারিগরি কলেজ ১টি ,ফজলুর রহমান মেমোরিয়্যাল কলেজ অব টেকনোলজি ২০০২ == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার ৩টি == '''১.বুড়িচং বাজার''' '''২.পূর্ণমতি বাজার''' '''৩.জগতপুর বাজার''' == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
