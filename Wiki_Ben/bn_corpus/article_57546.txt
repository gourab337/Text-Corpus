'''তমরদ্দি''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত হাতিয়া উপজেলার একটি ইউনিয়ন। == আয়তন== অত্র ইউনিয়নের মোট আয়তন ৮৪ বর্গ কিঃমিঃ ==জনসংখ্যা== তমরদ্দি ইউনিয়নের বর্তমান জনসংখ্যা ৩৩,৪০০ জন। == অবস্থান সীমানা == হাতিয়া উপজেলার পশ্চিমাংশে তমরদ্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর কিং ইউনিয়ন পূর্বে হাতিয়া পৌরসভা দক্ষিণ-পূর্বে বুড়িরচর ইউনিয়ন, দক্ষিণে সোনাদিয়া ইউনিয়ন এবং পশ্চিমে হাতিয়া চ্যানেল ভোলা জেলার মনপুরা উপজেলার হাজিরহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == তমরদ্দি ইউনিয়ন হাতিয়া উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাতিয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৩নং নির্বাচনী এলাকা নোয়াখালী-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাতিয়া উপজেলা হাতিয়া থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
