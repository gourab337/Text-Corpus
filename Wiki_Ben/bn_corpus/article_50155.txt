'''রতনপুর''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == রতনপুর ইউনিয়নের আয়তন ৬২৯২ একর (২৫.৪৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে রতনপুর ইউনিয়নের মোট জনসংখ্যা ২৬,৭০৩ জন। মোট পরিবার ৫,৩৭৪টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১০৪৯ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার রতনপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে সাতমোড়া ইউনিয়ন, উত্তরে রসুল্লাবাদ ইউনিয়ন শ্যামগ্রাম ইউনিয়ন, পশ্চিমে বাঞ্ছারামপুর উপজেলার দড়িকান্দি ইউনিয়ন ফরদাবাদ ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার মুরাদনগর উপজেলার শ্রীকাইল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রতনপুর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ২১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। এটি ১৩টি মৌজায় বিভক্ত। == শিক্ষা ব্যবস্থা == রতনপুর ইউনিয়নের সাক্ষরতার হার ৭৫.৮৮%। ইউনিয়নে ৩টি মাধ্যমিক বিদ্যালয়, ১টি কারিগরী শিক্ষা প্রতিষ্ঠান ১৫টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == আলী যাকের নাট্যকার। ইরেশ যাকের অভিনেতা। কাজী আনোয়ার হোসেন রাজনীতিবিদ। গোলাম মোস্তফা মারুফ রাজনীতিবিদ; == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: কে এম রুহুল আমিন == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
