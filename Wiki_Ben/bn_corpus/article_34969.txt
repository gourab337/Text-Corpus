'''আদ্রা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বরুড়া উপজেলার পশ্চিমাংশে আদ্রা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আড্ডা ইউনিয়ন, পূর্বে ভাউকসার ইউনিয়ন, দক্ষিণে পয়ালগাছা ইউনিয়ন চাঁদপুর জেলার কচুয়া উপজেলার আশরাফপুর ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার আশরাফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আদ্রা ইউনিয়ন বরুড়া উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
