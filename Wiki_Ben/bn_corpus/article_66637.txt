'''রাজাপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার সর্ব-উত্তরে রাজাপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে দক্ষিণে বাকশীমূল ইউনিয়ন, দক্ষিণ-পশ্চিমে বুড়িচং সদর ইউনিয়ন, পশ্চিমে ব্রাহ্মণপাড়া উপজেলার সাহেবাবাদ ইউনিয়ন, উত্তর-পশ্চিমে ব্রাহ্মণপাড়া উপজেলার ব্রাহ্মণপাড়া সদর ইউনিয়ন, উত্তরে ব্রাহ্মণপাড়া উপজেলার শশীদল ইউনিয়ন এবং উত্তর-পূর্বে পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == রাজাপুর ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == এম গনী রোড, বুড়িচং ব্রাক্ষণপাড়া কুমিল্লা টু সিদলাই সড়ক রেললাইন স্টেশন রাজাপুর কোথাও কোন সাকো নাই। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == গুংগুর নদী। পাইকোটা পদ্মফুলের জলাশয়। দড়িয়াপাড় ঈদগাঁ। হিন্দুরীপুল। ঠুডার পুকুর শংকুচাইল। বারেশ্বর গ্রামের খাল,লাল দালান,নৌকা। == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
