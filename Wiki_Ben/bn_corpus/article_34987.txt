'''বাঘাসুরা ইউনিয়ন''' বাংলাদেশের হবিগঞ্জ জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই ইউনিয়নটি সিলেট বিভাগের হবিগঞ্জ জেলার মাধবপুর উপজেলার অন্তর্গত; যা মাধবপুর উপেজলার শেষ অংশে পড়েছে। এটি মাধবপুর উপজেলা পরিষদ হতে ২২ কি:মি: দূরে এবং ঢাকা-সিলেট মহাসড়কের উভয় পার্শ্বে অবস্থিত। এর উত্তরে লাখাই হবিগঞ্জ সদর উপজেলা, পশ্চিমে জেলার নাসিরনগর উপজেলা, পূর্বে চুনারুঘাট উপজেলা এবং দক্ষিণে ছাতিয়াইন নোয়াপাড়া ইউনিয়ন। এই ইউনিয়নের আয়তন ৫,৮২১ একর। == ইতিহাস == == ভৌগোলিক উপাত্ত == পাহাড়ী মৃত্তিকায় গঠিত এই অঞ্চলে দুলিয়া খাল অদুনা খাল নামে দুটি পাহাড়ী ছড়া রয়েছে। == প্রশাসনিক উপাত্ত == ১০ মৌজা নিয়ে গঠিত এই ইউনিয়টিতে গ্রামের সংখ্যা ৩১টি। গ্রাম কালিকাপুর, রূপনগর, নোয়াগাঁও, হারুনপুর, রামচন্দ্রপুর, বরতল, বাখরনগর, সাতপারিয়া, উজলপুর, বাঘাসুরা, সুন্দরপুর, রতনপুর, পুরাইকলা, ধিতকুরা, সাহপুর, লহিপুর, গঙ্গানগর, কাউররা, দাশপাড়া, রাধাপুর, ফতেহ্‌পুর, তাজপুর, হরিতলা, জয়নগর, উজিরপুর, শিবপুর, কালীনগর, শিবজয়নগর এবং ভাটি শিবজয়নগর। == জনসংখ্যা উপাত্ত == ২০১১ সালের আদম শুমারি অনুযায়ী এই ইউনিয়নের লোকসংখ্যা ২৩,৬১৭ জন; যাদের মধ্যে পুরুষ ১১,৯০৪ জন এবং মহিলা ১১,৭৮৯ জন। এখানে সর্বমোট ৪,৭৮৩টি পরিবারের বসবাস রয়েছে। == নির্বাচিত জন-প্রতিনিধি == == শিক্ষাব্যবস্থা == এখানকার শিক্ষার হার ৫৫%। এখানে মোট শিক্ষা প্রতিষ্ঠানের সংখ্যা নিম্নরূপ: সরকারী প্রাথমিক বিদ্যালয় ১১টি, বেসরকারী প্রাথমিক বিদ্যালয় ২টি, উচ্চ বিদ্যালয় ৩টি, মাদ্রাসা ২টি। == স্বাস্থ্যসেবা == এই ইউনিয়নের ২৮.৫০% বসতি স্যানিটেশন সুবিধার আওতাভূক্ত। == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ত্ব == == দর্শনীয় গুরুত্বপূর্ণ স্থাপনা == হবিগঞ্জ গ্যাস ফিল্ড; ফ্রুটস ভ্যালি; শাহ্‌ সোলেয়মান ফতেহ গাজী বোগদাদীর মাজার ফতেহপুর। == বিবিধ == == আরও দেখুন == মাধবপুর পৌরসভা; মাধবপুর উপজেলা; হবিগঞ্জ জেলা; সিলেট বিভাগ; বাংলাদেশের ইউনিয়নসমূহের তালিকা। == তথ্যসূত্র == == বহিঃসংযোগ == বাঘাসুরা ইউনিয়ন জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
