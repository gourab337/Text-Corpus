'''সুলতানপুর''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == সুলতানপুর ইউনিয়নের আয়তন ৭৩৩৬ একর (২৯.৬৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে সুলতানপুর ইউনিয়নের মোট জনসংখ্যা ৩২,১৭০ জন। মোট পরিবার ৫,৬৫৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১০৮৪ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার সুলতানপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বাসুদেব ইউনিয়ন মাছিহাতা ইউনিয়ন; উত্তরে মাছিহাতা ইউনিয়ন রামরাইল ইউনিয়ন; পশ্চিমে রামরাইল ইউনিয়ন, তিতাস নদী, নবীনগর উপজেলার নাটঘর ইউনিয়ন কাইতলা উত্তর ইউনিয়ন এবং দক্ষিণে তিতাস নদী কসবা উপজেলার মূলগ্রাম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সুলতানপুর ইউনিয়ন সদর উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
