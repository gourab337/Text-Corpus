'''শরীফপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত বেগমগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার শরীফপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে একলাশপুর ইউনিয়ন, উত্তরে চৌমুহনী পৌরসভা হাজীপুর ইউনিয়ন, পূর্বে কাদিরপুর ইউনিয়ন এবং দক্ষিণে নোয়াখালী সদর উপজেলার নিয়াজপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শরীফপুর ইউনিয়ন বেগমগঞ্জ উপজেলার আওতাধীন ১৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == মুরাদপুর সরকারি প্রাথমিক বিদ্যালয় সোনাপুর সরকারি প্রাথমিক বিদ্যালয়। সোনাপুর নুরুল ইসলাম উচ্চ বিদ্যালয়। উত্তর শরিফপুর কারামতিয়া দাখিল মাদ্রাসা। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == শরীফপুর ইউনিয়নের প্রধান হাট-বাজার হল মুরাদপুর বাজার। == দর্শনীয় স্থান == রিয়াজউদ্দিন ভূঁইয়া বাড়ী == জনপ্রতিনিধি == == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
