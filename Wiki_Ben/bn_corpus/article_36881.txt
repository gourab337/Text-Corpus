'''কামারপুকুর ইউনিয়ন''' বাংলাদেশের রংপুর বিভাগের নীলফামারী জেলার সৈয়দপুর উপজেলার একটি স্থানীয় সরকার প্রশাসনিক ইউনিয়ন। == অবস্থান আয়তন == কামারপুকুর ইউনিয়নের আয়তন ৪১৯৭ একর (১১ বর্গ কিলোমিটার)। এই ইউনিয়নে ৩৬৭৪ টি পরিবার বাস করে। == ইতিহাস == == ভৌগোণিক উপাত্ত == == প্রশাসনিক উপাত্ত == এই ইউনিয়টি টি মৌজার টি গ্রামের সমন্বয়ে গঠিত। গ্রাম == জনসংখ্যা উপাত্ত == বাংলাদেশর ১৯৯১ আদমশুমারি অনুযায়ী বুরুঙ্গাবাজার ইউনিয়নের জনসংখ্যা ২০,৩৬২ জন। এর মধ্যে মহিলা ৫১%, এবং পুরুষ ৪৯%। == নির্বাচিত জন-প্রতিনিধি == == শিক্ষাব্যবস্থা == == স্বাস্থ্যসেবা == == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ত্ব == == দর্শনীয় গুরুত্বপূর্ণ স্থাপনা == == বিবিধ == == আরও দেখুন == বাংলাদেশের ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == কামারপুকুর ইউনিয়ন জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন জেলার ইউনিয়ন
