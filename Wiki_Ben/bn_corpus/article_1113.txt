'''ঝালকাঠি সদর''' বাংলাদেশের ঝালকাঠি জেলার অন্তর্গত একটি উপজেলা। == অবস্থান আয়তন == ঝালকাঠী সদর উপজেলার উত্তরে উজিরপুর উপজেলা, দক্ষিণে নলছিটি উপজেলা, পশ্চিমে রাজাপুর উপজেলা, কাঁঠালিয়া উপজেলাও পিরোজপুর সদর উপজেলা এবং পূর্বে বরিশাল সদর উপজেলা অবস্থিত। == ইতিহাস == === নামকরণ === === মুক্তিযুদ্ধে ঝালকাঠি সদর === == ভৌগোলিক উপাত্ত == === ভূপ্রকৃতি === === মৃত্তিকা === === নদ-নদী === == সাংস্কৃতিক বৈশিষ্ঠ্য == === ভাষা === === উৎসব === === খেলাধুলা === == প্রশাসনিক এলাকা == এই উপজেলায় ১০টি ইউনিয়ন রয়েছে; এগুলো হলোঃ ইউনিয়ন, বিনয়কাঠী ইউনিয়ন, নবগ্রাম ইউনিয়ন, ঝালকাঠী, কেওড়া ইউনিয়ন, কীর্তিপাশা ইউনিয়ন, বাসন্ডা ইউনিয়ন, পোনাবালিয়া ইউনিয়ন, গাবখান ধানসিঁড়ি ইউনিয়ন, শেখেরহাট ইউনিয়ন এবং নথুল্লাবাদ ইউনিয়ন। ;নির্বাচনী এলাকা জনপ্রতিনিধি == জনসংখ্যার উপাত্ত == জনসংখ্যার উপাত্ত ১,৯৪,০২০ ;ধর্ম == স্বাস্থ্য == == শিক্ষা == == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == ;সড়কপথ ;রেলপথ ;নৌপথ == কৃতী ব্যক্তিত্ব == আমির হোসেন আমু (জন্মঃ জানুয়ারি ১৯৪০) রাজনীতিবিদ। == দর্শনীয় স্থান স্থাপনা == কীর্ত্তিপাশা জমিদার বাড়ি == বিবিধ == == আরও দেখুন == ঝালকাঠি জেলা; বরিশাল বিভাগ; বাংলাদেশের উপজেলাসমূহ। == তথ্যসূত্র == ==বহিঃসংযোগ== ঝালকাঠি সদর উপজেলা জাতীয় তথ্য বাতায়ন। সদর উপজেলা জেলার উপজেলা বিভাগের উপজেলা
