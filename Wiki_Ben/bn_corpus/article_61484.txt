'''কৃষ্ণনগর''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == কৃষ্ণনগর ইউনিয়নের আয়তন ৫০৬৩ একর (২০.৪৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে কৃষ্ণনগর ইউনিয়নের মোট জনসংখ্যা ৩১,০৭৬ জন। মোট পরিবার ৫,৭২৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৫১৭ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার উত্তরাংশে কৃষ্ণনগর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বড়াইল ইউনিয়ন, তিতাস নদী বিদ্যাকুট ইউনিয়ন; দক্ষিণে তিতাস নদী, নবীনগর পূর্ব ইউনিয়ন, নবীনগর পৌরসভা নবীনগর পশ্চিম ইউনিয়ন; পশ্চিমে বীরগাঁও ইউনিয়ন এবং উত্তরে আশুগঞ্জ উপজেলার লালপুর ইউনিয়ন শরীফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কৃষ্ণনগর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। ইউনিয়নের গ্রামসংখ্যা ১১টি। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == কৃষ্ণনগর ইউনিয়নের প্রধান দুইটি নদী তিতাস নদী পাগলা নদী। == হাট-বাজার == == দর্শনীয় স্থান == কৃষ্ণনগর বড়বাড়ি মধ্যপাড়ার পুরনো বাড়ি (জমিদার ধাঁচের) মধ্যপাড়া চেয়ারম্যান বাড়ীর অভ্যন্তরীণ পুরনো বাড়ি == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: জিল্লুর রহমান == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
