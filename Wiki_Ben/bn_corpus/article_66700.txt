'''গুনাইঘর দক্ষিণ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দেবিদ্বার উপজেলার পশ্চিমাংশে গুনাইঘর দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে গুনাইঘর উত্তর ইউনিয়ন দেবিদ্বার পৌরসভা, পূর্বে দেবিদ্বার পৌরসভা এলাহাবাদ ইউনিয়ন, দক্ষিণে ধামতী ইউনিয়ন, দক্ষিণ-পশ্চিমে রাজামেহার ইউনিয়ন এবং উত্তর-পশ্চিমে মুরাদনগর উপজেলার ধামঘর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গুনাইঘর দক্ষিণ ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
