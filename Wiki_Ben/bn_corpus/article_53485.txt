'''ষাটনল''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত মতলব উত্তর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মতলব উত্তর উপজেলার ষাটনল ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে সাদুল্লাপুর ইউনিয়ন ছেংগারচর পৌরসভা, দক্ষিণে ছেংগারচর পৌরসভা কলাকান্দা ইউনিয়ন, পশ্চিমে মেঘনা নদী মুন্সিগঞ্জ জেলার মুন্সিগঞ্জ সদর উপজেলার আধারা ইউনিয়ন এবং উত্তরে মেঘনা নদী মুন্সিগঞ্জ জেলার গজারিয়া উপজেলার ইমামপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ষাটনল ইউনিয়ন মতলব উত্তর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মতলব উত্তর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬১নং নির্বাচনী এলাকা চাঁদপুর-২ এর অংশ। ইউনিয়নের গ্রামগুলো হল: বড় ষাটনল সটাকি সুগন্ধী ষাটনল মালো পাড়া চর চার আনি আসন আলী প্রধানিয়া কান্দি কন্দু সরকার কান্দি মেহের উল্লা প্রধানিয়া কান্দি পশ্চিম লালপুর পূর্ব লালপুর ইমামপুর মধ্য কালিপুর কালিপুর বাড়ীভাঙ্গা == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == ষাটনল ইউনিয়নের উত্তর পশ্চিম প্রান্ত দিয়ে প্রবাহিত হচ্ছে মেঘনা নদী == হাট-বাজার == ষাটনল ইউনিয়নের প্রধান প্রধান হাট-বাজারগুলো হল কালিপুর বাজার, ষাটনল (বাবুর) বাজার, সটাকী বাজার, কনু মার্কেট এবং আবু মার্কেট। == দর্শনীয় স্থান == ষাটনল পর্যটন কেন্দ্র ষাটনল পর্যটন কেন্দ্র। অত্যন্ত মনরোম পরিবেশে অবস্থিত। প্রতিদিন হাজার হাজার পর্যটক এই পর্যটন কেন্দ্রটি ভ্রমণ করেন। ২০০০ সালে সরকারি ভাবে এটিকে পর্যটন কেন্দ্র হিসাবে ঘোষণা করা হয়। == কৃতী ব্যক্তিত্ব == মেজর জেনারেল (অব) সামছুল হক, সাবেক বাণিজ্য মন্ত্রী। মোজাম্মেল হক বেনু সরকার, শিল্পপতি। নুরুল ইসলাম, সাবেক রাষ্টদুত, মালয়েশিয়া। == জনপ্রতিনিধি == == আরও দেখুন == মতলব উত্তর উপজেলা মতলব উত্তর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উত্তর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
