'''বাদৈর''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == বাদৈর ইউনিয়নের আয়তন ৩০১৮ একর (১২.২১ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বাদৈর ইউনিয়নের মোট জনসংখ্যা ১৮,০১২ জন। মোট পরিবার ৩,৩৬০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৪৭৫ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার উত্তরাংশে বাদৈর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বিনাউটি ইউনিয়ন, দক্ষিণে খাড়েরা ইউনিয়ন, পশ্চিমে মেহারী ইউনিয়ন এবং উত্তরে মূলগ্রাম ইউনিয়ন আখাউড়া উপজেলার ধরখাড় ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাদৈর ইউনিয়ন কসবা উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == সামসুল হক –– বীর প্রতীক খেতাবপ্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
