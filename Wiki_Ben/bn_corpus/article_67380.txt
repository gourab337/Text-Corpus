'''কায়েমপুর''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == কায়েমপুর ইউনিয়নের আয়তন ৪৮২৭ একর (১৯.৫৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে কায়েমপুর ইউনিয়নের মোট জনসংখ্যা ৩৩,১১৯ জন। মোট পরিবার ৬,৪০৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৬৯৫ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার দক্ষিণাংশে কায়েমপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কসবা পৌরসভা কসবা পশ্চিম ইউনিয়ন, উত্তর-পশ্চিমে কুটি ইউনিয়ন, পশ্চিমে কুমিল্লা জেলার ব্রাহ্মণপাড়া উপজেলার মাধবপুর ইউনিয়ন চান্দলা ইউনিয়ন, দক্ষিণ-পশ্চিমে কুমিল্লা জেলার ব্রাহ্মণপাড়া উপজেলার শশীদল ইউনিয়ন, দক্ষিণ-পূর্বে বায়েক ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == কায়েমপুর ইউনিয়ন কসবা উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
