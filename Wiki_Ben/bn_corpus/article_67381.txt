'''বায়েক''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == বায়েক ইউনিয়নের আয়তন ৬২৮২ একর (২৫.৪২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বায়েক ইউনিয়নের মোট জনসংখ্যা ২৯,৯৭৪ জন। মোট পরিবার ৫,৯৩২টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১১৭৯ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার সর্ব-দক্ষিণে বায়েক ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পূর্বে কায়েমপুর ইউনিয়ন, পশ্চিমে কুমিল্লা জেলার ব্রাহ্মণপাড়া উপজেলার শশীদল ইউনিয়ন চান্দলা ইউনিয়ন, দক্ষিণে কুমিল্লা জেলার ব্রাহ্মণপাড়া উপজেলার শশীদল ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ এবং পূর্বে উত্তরে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বায়েক ইউনিয়ন কসবা উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
