'''আখাউড়া উত্তর''' বাংলাদেশের জেলার অন্তর্গত আখাউড়া উপজেলার একটি ইউনিয়ন। == আয়তন == আখাউড়া উত্তর ইউনিয়নের আয়তন ২২৩৩ একর (৯.০৪ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে আখাউড়া উত্তর ইউনিয়নের মোট জনসংখ্যা ১১,৯৫১ জন। মোট পরিবার ২,২৫৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৩২৩ জন। == ইতিহাস == == অবস্থান সীমানা == আখাউড়া উপজেলার সর্ব-উত্তরে আখাউড়া উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে আখাউড়া দক্ষিণ ইউনিয়ন আখাউড়া পৌরসভা; পশ্চিমে আখাউড়া পৌরসভা, সদর উপজেলার বাসুদেব ইউনিয়ন মাছিহাতা ইউনিয়ন; উত্তরে বিজয়নগর উপজেলার সিংগারবিল ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == আখাউড়া উত্তর ইউনিয়ন আখাউড়া উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আখাউড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == আখাউড়া উপজেলা আখাউড়া থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
