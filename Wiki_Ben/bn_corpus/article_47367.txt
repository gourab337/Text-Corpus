'''জামেয়া মিল্লিয়া আহমদিয়া কামিল মাদ্রাসা''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার নাজিরহাট পৌরসভায় অবস্থিত। == ইতিহাস == ১৯৩৯ সালে মাদ্রাসাটি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == প্রতিষ্ঠানের অধ্যক্ষ জনাব মাওলানা কামাল উদ্দীন। == অবকাঠামো == == শিক্ষা কার্যক্রম == এটি একটি কামিল (মাস্টার্স) পর্যায়ের শিক্ষা প্রতিষ্ঠান। প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ১০০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
