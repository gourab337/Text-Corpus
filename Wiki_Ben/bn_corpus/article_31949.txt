(LibreCAD) একটি বিনামূল্যে প্রাপ্ত কম্পিউটার ভিত্তিক ডিজাইনের (ক্যাড) সফটওয়্যার যা দ্বিমাত্রিক (2D) ডিজাইনের জন্য ব্যবহার করা হয়। == অপারেটিং সিস্ট্যাম == এটি লিনাক্স, উইনডোজ, ম্যাক ওএস, ইউনিক্স ইত্যাদি অপারেটিং সিস্ট্যামে কাজ করে। == বিবিধ == এটি অনেকটা অটোক্যাডের মত ফাইল সংরক্ষণ ইমপোর্ট করার ক্ষেত্রে AutoCAD DXF ফাইল সহ অন্যান্য এক্সটেনশন ব্যবহার করে। == বহিঃসংযোগ == Official website of the project (en) Source code repository on GitHub (en) Project website in SourceForge (en) LibreCAD community wiki সফটওয়্যার
