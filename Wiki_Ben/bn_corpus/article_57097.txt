'''নবীপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সেনবাগ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সেনবাগ উপজেলার সর্ব-দক্ষিণে নবীপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বিজবাগ ইউনিয়ন, উত্তর-পশ্চিমে বেগমগঞ্জ উপজেলার রসুলপুর ইউনিয়ন কাদিরপুর ইউনিয়ন, দক্ষিণ-পশ্চিমে কবিরহাট উপজেলার নরোত্তমপুর ইউনিয়ন, দক্ষিণে কবিরহাট উপজেলার বাটইয়া ইউনিয়ন, দক্ষিণ-পূর্বে কোম্পানীগঞ্জ উপজেলার সিরাজপুর ইউনিয়ন এবং পূর্বে ফেনী জেলার দাগনভূঁইয়া উপজেলার ইয়াকুবপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নবীপুর ইউনিয়ন সেনবাগ উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
