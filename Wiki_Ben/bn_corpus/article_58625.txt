'''চর বাদাম''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রামগতি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগতি উপজেলার সর্ব-উত্তরে চর বাদাম ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে আলেকজান্ডার ইউনিয়ন; দক্ষিণে চর আলগী ইউনিয়ন; পূর্বে চর পোড়াগাছা ইউনিয়ন এবং উত্তরে কমলনগর উপজেলার চর কাদিরা ইউনিয়ন, হাজিরহাট ইউনিয়ন পাটারীরহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর বাদাম ইউনিয়ন রামগতি উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগতি থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৭নং নির্বাচনী এলাকা লক্ষ্মীপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রামগতি উপজেলা রামগতি থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
