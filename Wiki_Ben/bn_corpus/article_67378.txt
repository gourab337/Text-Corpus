'''মেহারী''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == মেহারী ইউনিয়নের আয়তন ৫৪৫০ একর (২২.০৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে মেহারী ইউনিয়নের মোট জনসংখ্যা ২৬,৩৩৫ জন। মোট পরিবার ৫,০১৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১১৯৪ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার মেহারী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে কুটি ইউনিয়ন, পূর্বে কসবা পশ্চিম ইউনিয়ন খাড়েরা ইউনিয়ন, উত্তর-পূর্বে বাদৈর ইউনিয়ন, উত্তরে মূলগ্রাম ইউনিয়ন নবীনগর উপজেলার কাইতলা দক্ষিণ ইউনিয়ন এবং পশ্চিমে নবীনগর উপজেলার লাউর ফতেপুর ইউনিয়ন কুমিল্লা জেলার মুরাদনগর উপজেলার আন্দিকোট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মেহারী ইউনিয়ন কসবা উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
