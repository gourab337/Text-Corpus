'''দ্বাদশ গ্রাম''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার উত্তরাংশে দ্বাদশ গ্রাম ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে রাজারগাঁও উত্তর ইউনিয়ন, দক্ষিণে বাকিলা ইউনিয়ন কালচোঁ দক্ষিণ ইউনিয়ন, পূর্বে কালচোঁ উত্তর ইউনিয়ন এবং উত্তরে মতলব দক্ষিণ উপজেলার নারায়ণপুর ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ২০১৩ সালের ২৭ মার্চ রাজারগাঁও উত্তর ইউনিয়ন থেকে ৪টি, বাকিলা ইউনিয়ন থেকে ২টি এবং কালচোঁ উত্তর ইউনিয়ন থেকে ৬টি সর্বমোট ১২টি গ্রাম নিয়ে দ্বাদশ গ্রাম ইউনিয়ন গঠিত হয়। == প্রশাসনিক কাঠামো == দ্বাদশ গ্রাম ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
