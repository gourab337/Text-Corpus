'''পাঁচগাঁও''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == পাঁচগাঁও ইউনিয়নের আয়তন ৫.৬৯ বর্গ কিলোমিটার। এটি আয়তনের দিক থেকে নোয়াখালী জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার পূর্বাংশে পাঁচগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে নোয়াখলা ইউনিয়ন, পশ্চিমে চাটখিল পৌরসভা, উত্তরে মোহাম্মদপুর ইউনিয়ন এবং পূর্বে মোহাম্মদপুর ইউনিয়ন সোনাইমুড়ি উপজেলার জয়াগ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাঁচগাঁও ইউনিয়ন চাটখিল উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
