বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লাকসাম উপজেলার পশ্চিমাংশে মুদাফফরগঞ্জ ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন, পূর্বে কান্দিরপাড় ইউনিয়ন, উত্তরে বাকই ইউনিয়ন বরুড়া উপজেলার পয়ালগাছা ইউনিয়ন এবং পশ্চিমে বরুড়া উপজেলার লক্ষ্মীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মুদাফফরগঞ্জ ইউনিয়ন লাকসাম উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। ইউনিয়নের গ্রামগুলো হল: আউশপাড়া কাঁঠালিয়া ডিমাতলী ধানকুইয়া পশ্চিম কান্দিরপাড় পাশাপুর বাউরতলা সাতবাড়িয়া গোয়ালিয়া বামণ্ডা হলূদিয়া বেতাগাঁও ফুলরা জনাধনপুর কাগৈয়া চিকুনিয়া নাকঝাটিয়া ডুমুরিয়া খাতাপাড়া নগরীপাড়া মুদাফফরগঞ্জ == শিক্ষা ব্যবস্থা == মুদাফফরগঞ্জ ইউনিয়নের সাক্ষরতার হার ৫৪%। ইউনিয়নে ২টি কলেজ, ৪টি মাদ্রাসা, ১টি কলেজিয়েট উচ্চ বিদ্যালয় ৪টি মাধ্যমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: শাহিদুল ইসলাম শাহিন। == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
