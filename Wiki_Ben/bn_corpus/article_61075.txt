'''কচুয়া দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত কচুয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কচুয়া উপজেলার মধ্যাংশে কচুয়া দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কচুয়া উত্তর ইউনিয়ন, পশ্চিমে পালাখাল মডেল ইউনিয়ন, দক্ষিণ-পশ্চিমে কাদলা ইউনিয়ন, দক্ষিণে কচুয়া পৌরসভা কড়ইয়া ইউনিয়ন এবং পূর্বে কুমিল্লা জেলার চান্দিনা উপজেলার জোয়াগ ইউনিয়ন বরকরই ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কচুয়া দক্ষিণ ইউনিয়ন কচুয়া উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (চাঁদপুর) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
