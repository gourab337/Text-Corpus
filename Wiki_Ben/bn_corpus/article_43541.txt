'''বাঙ্গড্ডা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == বাঙ্গড্ডা ইউনিয়নের আয়তন বর্গ কিলোমিটার। == জনসংখ্যা == বাঙ্গড্ডা ইউনিয়নের জনসংখ্যা ২১,৪০৬ জন। == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার সর্ব-উত্তরে বাঙ্গড্ডা ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে রায়কোট ইউনিয়ন পেড়িয়া ইউনিয়ন, পশ্চিমে পেড়িয়া ইউনিয়ন লালমাই উপজেলার বেলঘর দক্ষিণ ইউনিয়ন, উত্তরে চৌদ্দগ্রাম উপজেলার শুভপুর ইউনিয়ন এবং পূর্বে চৌদ্দগ্রাম উপজেলার মুন্সিরহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাঙ্গড্ডা ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। ইউনিয়নের গ্রামগুলো হল: ছোট সাঙ্গিশ্বর নশরতপুর দাড়াচৌ সোনাবেরী রূপাবেরী নূরপুর কৃশিয়ারা কাদবা পদুয়া করের ভোমরা নোয়াপাড়া আঙ্গলখোড়া নিমুড়ী বামারবাগ বাঙ্গড্ডা উত্তর বাঙ্গড্ডা দক্ষিণ গান্দ্বাচি পূর্ব গান্দ্বাচি পশ্চিম শ্যামপুর পূর্ব শ্যামপুর পশ্চিম পরিকোট == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;কলেজ বাঙ্গড্ডা কলেজ == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == ওমর আহাম্মদ মজুমদার –– প্রাক্তন সংসদ সদস্য। == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: শাহজাহান মজুমদার == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
