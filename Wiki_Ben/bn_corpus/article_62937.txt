'''চর ভৈরবী''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাইমচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাইমচর উপজেলার সর্ব-দক্ষিণে চর ভৈরবী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পূর্বে আলগী দুর্গাপুর দক্ষিণ ইউনিয়ন, উত্তরে নীলকমল ইউনিয়ন, পশ্চিমে হাইমচর ইউনিয়ন, দক্ষিণ-পশ্চিমে বরিশাল জেলার হিজলা উপজেলার হিজলা গৌরবদী ইউনিয়ন এবং পূর্বে লক্ষ্মীপুর জেলার রায়পুর উপজেলার উত্তর চর আবাবিল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর ভৈরবী ইউনিয়ন হাইমচর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাইমচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == চরভৈরবী ইউনিয়নের প্রধান হাট-বাজার গুলো হলো চরভৈরবী লঞ্চঘাট বাজার, গরম বাজার, আমতলী বাজার ইত্যাদি। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাইমচর উপজেলা হাইমচর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
