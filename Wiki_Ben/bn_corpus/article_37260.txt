'''হবিগঞ্জ পৌরসভা''' বাংলাদেশের হবিগঞ্জ জেলার একটি স্থানীয় সরকার ব্যবস্থা সংগঠন। এই পৌরসভাটি বাংলাদেশের একটি ‘‘ক’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == হবিগঞ্জ পৌরসভাটি সিলেট বিভাগের হবিগঞ্জ জেলার সদর উপজেলার অন্তর্ভুক্ত। এই পৌর এলাকাটির আয়তন ৯.০৫ বর্গ কিলোমিটার। == ইতিহাস == হবিগঞ্জ পৌরসভাটি গঠিত হয় ১৮৮১ সালে। == ভৌগোলিক উপাত্ত == == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == == নির্বাচিত জন-প্রতিনিধি == বিগত ২০১৫ সালের ৩০ ডিসেম্বর তারিখে অনুষ্ঠিত পৌরসভা নির্বাচনে বিএনপি সমর্থিত প্রার্থী জি কে গৌউছ মেয়র পদে নির্বাচিত হয়েছেন। == জনসংখ্যা উপাত্ত == == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == == নাগরিক সুযোগ-সুবিধা == == কৃষি অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ব == == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == == বিবিধ == == আরও দেখুন == হবিগঞ্জ সদর উপজেলা; হবিগঞ্জ জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== হবিগঞ্জ পৌরসভা অফিসিয়াল ওয়েবসাইট। হবিগঞ্জ পৌরসভা জাতীয় তথ্য বাতায়ন। হবিগঞ্জ পৌরসভা স্থানীয় সরকার প্রকৌশল অধিদপ্তর। সদর উপজেলা জেলার পৌরসভা বিভাগের পৌরসভা
