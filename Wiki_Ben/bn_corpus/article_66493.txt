'''শিলমুড়ি উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বরুড়া উপজেলার পূর্বাংশে শিলমুড়ি উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ভবানীপুর ইউনিয়ন, পশ্চিমে বরুড়া পৌরসভা, দক্ষিণ-পশ্চিমে গালিমপুর ইউনিয়ন, দক্ষিণে দক্ষিণ-পূর্বে শিলমুড়ি দক্ষিণ ইউনিয়ন এবং পূর্বে কুমিল্লা সদর দক্ষিণ উপজেলার বারপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শিলমুড়ি উত্তর ইউনিয়ন বরুড়া উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
