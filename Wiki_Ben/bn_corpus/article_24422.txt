এটি বিশ্বের বিভিন্ন দেশের '''মোবাইল ফোন প্রস্তুতকারকদের তালিকা'''। তালিকাটি '''দেশ''' অনুযায়ী সাজানো আছে। == অস্ট্রেলিয়া == কোগান টেকনোলজিস == ইতালি == ব্রোনদি ওলিভেত্তি ওন্দা মোবাইল কমিউনিকেশন তেলিত == ইন্দোনেশিয়া == নেক্সিয়ান পলিট্রন == চীন == বিবিকে অলকটেল সিইসিটি হাইয়ের মটোরোলা মোবাইলিটি হুয়ায়েই কে-টাচ লেনোভো মেইজু ওপ্পো ডিজিটাল সেগটেল পূর্বে ছিল নিংবো বার্ড সিম্ফোনি মোবাইল জি ফাইভ শাওমি টেকনো টিসিএল কর্পোরেশন জিয়াওমি টেক জুম জেডটিই == চেক প্রজাতন্ত্র == ইয়াবলোকো ভেরজো == জাপান == ফুজিত্‍সু কিওকেরা কমিউনিকেশনস এনইসি এনইসি ক্যাসিও মোবাইল কমিউনিকেশনস প্যানাসনিক সানিও চেটান সনি মোবাইল কমিউনিকেশনস তোশিবা == জার্মানি == এইজি গুন্দিগ মোবাইল টেলেফুংকেন == ডেনমার্ক == লুমিগোন == তাইওয়ান == এসার আসুস ডিবিটেল ডপড ই-টেন এইচটিসি ডিজিটাল কিংকম == তিউনিসিয়া == এভারটেক == দক্ষিণ কোরিয়া == কেটি টেক এলজি প্যানটেক স্যামসাং == নাইজেরিয়া == প্লিরিসমোবাইল == নেদারল্যান্ডস == ফেয়ারফোন জন'স ফোন ফিলিপস == পাকিস্তান == মোবাইল ভয়েস মোবাইল == ফিনল্যান্ড == ইয়োল্লা টুইগ কম (পূর্বে বেনেফোন নামে পরিচিত ছিল) নোকিয়া == ফিলিপাইন == চেরি মোবাইল মাইফোন টর্ক স্টারমোবাইল ক্লাউডফোন == ফ্রান্স == অ্যালক্যাটেল আর্কোস বুল মোবি ওয়্যার (পূর্বে সাগেম ওয়্যার) থমসন ভিইএ ডিজিটাল == বাংলাদেশ == ওয়ালটন ম্যাংগো উই সিম্ফোনি টিং টং == ব্রাজিল == গ্রাদিয়েন্তে == ভারত == লাভা ইন্টারন্যাশনাল লিমিটেড আইবল কার্বোন মোবাইলস মাইক্রোম্যাক্স মোবাইল অনিডা ইলেকট্রনিক্স স্পাইস টেলিকম ভিডিওকন ইনটেক্স হাইটেক == মালয়েশিয়া == এম ডট == মেক্সিকো == কিয়োতো ইলেকট্রনিক্স লানিক্স জোন্দা == যুক্তরাষ্ট্র == অ্যাপল গুগল ব্লু ফায়ারফ্লাই গ্রামিন পাম সনিম == রাশিয়া == গ্রেসো সিত্রোনিকস ইয়োতা == লাতভিয়া == জাস্ট৫ == সংযুক্ত আরব আমিরাত == থুরায়া == সুইডেন == ডরো নউটিজ হ্যান্ডহেল্ড গ্রুপ == স্পেন == হিকসফোন ভিতেলকম ==তথ্যসূত্র== ফোন
