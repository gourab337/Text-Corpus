'''ধরখাড়''' বাংলাদেশের জেলার অন্তর্গত আখাউড়া উপজেলার একটি ইউনিয়ন। == আয়তন == ধরখাড় ইউনিয়নের আয়তন ৮১৮৩ একর (৩৩.১২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ধরখাড় ইউনিয়নের মোট জনসংখ্যা ৩৪,২২১ জন। মোট পরিবার ৬,০৭৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১০৩৩ জন। == ইতিহাস == == অবস্থান সীমানা == আখাউড়া উপজেলার সর্ব-পশ্চিমে ধরখাড় ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে মোগড়া ইউনিয়ন মনিয়ন্দ ইউনিয়ন, দক্ষিণে কসবা উপজেলার বিনাউটি ইউনিয়ন বাদৈর ইউনিয়ন, পশ্চিমে কসবা উপজেলার মূলগ্রাম ইউনিয়ন এবং উত্তরে সদর উপজেলার বাসুদেব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ধরখাড় ইউনিয়ন আখাউড়া উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আখাউড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == মোহাম্মদ জাহাঙ্গীর ওসমান বীর প্রতীক খেতাব প্রাপ্ত মুক্তিযোদ্ধা আসনের সাবেক সাংসদ। == জনপ্রতিনিধি == == আরও দেখুন == আখাউড়া উপজেলা আখাউড়া থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
