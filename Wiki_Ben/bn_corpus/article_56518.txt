'''শরীফপুর''' বাংলাদেশের জেলার অন্তর্গত আশুগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == শরীফপুর ইউনিয়নের আয়তন ১৫৯০ একর (৬.৪৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে শরীফপুর ইউনিয়নের মোট জনসংখ্যা ১৩,৫৪৫ জন। মোট পরিবার ২,৬০৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ২১০৫ জন। == ইতিহাস == শরীফপুর ইউনিয়ন পূর্বে শরীফপুর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == আশুগঞ্জ উপজেলার দক্ষিণাংশে শরীফপুর ইউনিয়নের অবস্থান। উপজেলা সদর থেকে ইউনিয়নের দূরত্ব প্রায় ১১ কিলোমিটার। ইউনিয়নের পূর্বে তারুয়া ইউনিয়ন, উত্তরে আড়াইসিধা ইউনিয়ন লালপুর ইউনিয়ন, পশ্চিমে লালপুর ইউনিয়ন এবং দক্ষিণে নবীনগর উপজেলার কৃষ্ণনগর ইউনিয়ন বড়াইল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শরীফপুর ইউনিয়ন আশুগঞ্জ উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আশুগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। এটি ২টি মৌজায় বিভক্ত। ইউনিয়নের গ্রামগুলো হল: দক্ষিণ তারুয়া খোলাপাড়া টংগীপাড়া লামা শরীফপুর শরীফপুর == শিক্ষা ব্যবস্থা == শরীফপুর ইউনিয়নের সাক্ষরতার হার ৪১.৭০%। ইউনিয়নে ১টি মাধ্যমিক বিদ্যালয়, ১টি দাখিল মাদ্রাসা ৫টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == ;মাধ্যমিক বিদ্যালয় শাহ্ ফরাছত আলী উচ্চ বিদ্যালয় ;মাদ্রাসা শরীফপুর ইসলামিয়া দাখিল মাদ্রাসা ;প্রাথমিক বিদ্যালয় কান্দাপাড়া সরকারি প্রাথমিক বিদ্যালয় খোলাপাড়া দক্ষিণ রেজিস্টার্ড প্রাথমিক বিদ্যালয় তারুয়া দক্ষিণ সরকারি প্রাথমিক বিদ্যালয় == যোগাযোগ ব্যবস্থা == শরীফপুর ইউনিয়নে যোগাযোগের প্রধান সড়ক আশুগঞ্জ-শরীফপুর সড়ক। প্রধান যোগাযোগ মাধ্যম সিএনজি চালিত অটোরিক্সা। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: সাইফউদ্দিন চৌধুরী == আরও দেখুন == আশুগঞ্জ উপজেলা আশুগঞ্জ থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
