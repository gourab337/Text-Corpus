'''গোদাগাড়ী ইউনিয়ন''' রাজশাহী জেলার গোদাগাড়ী উপজেলার একটি ইউনিয়ন। == অবস্থান == ইউনিয়ন সীমারেখা পূর্বে কাকনহাট পৌরসভা, পূর্বে রিসিকুল ইউনিয়ন, পশ্চিমে মোহনপুর ইউপি, দক্ষিণে গোগ্রাম ইউপি, উত্তরে পাকড়ি ইউপি। == যোগাযোগ ব্যবস্থা == উপজেলা সদর হইতে গোদাগাড়ী ইউনিয়নে সড়ক যোগে যাতায়াত ব্যবস্থা রয়েছে। == ইউনিয়নের ইতিহাস == ১৯৪৭ সালে দেশবিভাগের পর ১৯৫৩ সালে ৫২টি মৌজা এবং ৯,১৯৯ একর আয়তন নিয়ে গোদাগাড়ী বোর্ড গঠিত হয়। পরবতিতে ১৯৬৫ সালে এটি ১নং গোদাগাড়ী ইউনিয়ন পরিষদ নামে পরিচিতি লাভ করে। == প্রশাসনিক বিন্যাস == বর্তমানে ৫২ টি মৌজা ৩৭ গ্রাম এবং ২৮.০৫ বর্গ কি: মি আয়তন নিয়ে ১নং গোদাগাড়ী ইউনিয়ন পরিষদ গঠিত। == জনসংখ্যার উপাত্ত == লোক সংখ্যাঃ ১৮৬৫৫। পুরুষ ৯২৭৭ জন মহিলা ৯৩৭৮ জন == শিক্ষা == শিক্ষার হার: শিক্ষার হার ৬০%। শিক্ষা প্রতিষ্ঠান ০১ গোদাগাড়ী ইউনিয়নে কোন কলেজ নেই। উচ্চ বিদ্যালয় এর সংখ্যা ০৪টি। মাদ্রাসাঃ ৪টি রয়েছে। == অর্থনীতি == == দর্শনীয় স্থান == সাফিনা পার্ক সরমংলা ইকো পার্ক। == কৃতী ব্যক্তিত্ব == == আরও দেখুন== == তথ্যসুত্র == == বহিঃসংযোগ == জেলার ইউনিয়ন
