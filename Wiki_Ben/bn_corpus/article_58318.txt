'''ভাটরা''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রামগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগঞ্জ উপজেলার সর্ব-পূর্বে ভাটরা ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে নোয়াগাঁও ইউনিয়ন ভোলাকোট ইউনিয়ন, দক্ষিণে ভোলাকোট ইউনিয়ন, পূর্বে নোয়াখালী জেলার চাটখিল উপজেলার বদলকোট ইউনিয়ন, উত্তর-পূর্বে চাঁদপুর জেলার শাহরাস্তি উপজেলার চিতোষী পশ্চিম ইউনিয়ন এবং উত্তরে চাঁদপুর জেলার শাহরাস্তি উপজেলার সূচীপাড়া দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভাটরা ইউনিয়ন রামগঞ্জ উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৪নং নির্বাচনী এলাকা লক্ষ্মীপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রামগঞ্জ উপজেলা রামগঞ্জ থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
