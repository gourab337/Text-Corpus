'''সুলতানপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == সুলতানপুর ইউনিয়ন পূর্বে ১৪নং ধামতী দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার দক্ষিণাংশে সুলতানপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ভানী ইউনিয়ন, উত্তরে ধামতী ইউনিয়ন, পূর্বে মোহনপুর ইউনিয়ন বরকামতা ইউনিয়ন এবং দক্ষিণে চান্দিনা পৌরসভা চান্দিনা উপজেলার কেরণখাল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সুলতানপুর ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: শফিকুল ইসলাম == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
