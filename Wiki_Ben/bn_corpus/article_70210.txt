'''বড়লেখা পৌরসভা''' বাংলাদেশের সিলেট বিভাগের মৌলভীবাজার জেলার একটি স্থানীয় সরকার সংস্থা। মৌলভীবাজার জেলার বড়লেখা উপজেলার কেন্দ্রস্থলকে ২০০১ সালে এক সরকারি গেজেট অনুসারে পৌলসভা হিসাবে ঘোষণা করে তত্‌কালীন সরকার। এই পৌরসভাটি বাংলাদেশের একটি ‘‘খ’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == বড়লেখা পৌরসভাটি সিলেট বিভাগের মৌলভীবাজার জেলার বড়লেখা উপজেলা সদরে অবস্থিত। জেলা সদর থেকে ৭০ কিলোমিটার দূরে অবস্থিত এই পৌরসভাটির আয়তন ১১.৩০ বর্গ কিলোমিটার। এর উত্তরে বড়লেখা ইউনিয়ন; দক্ষিণে দক্ষিনভাগ উত্তর ইউনিয়ন; পূর্বে বড়লেখা ইউনিয়ন এবং পশ্চিমে বড়লেখা ইউনিয়ন। == ইতিহাস == বড়লেখার উপজেলার কেন্দ্রস্থলের ২০০১ সালের ১৪ জুন ১১ দশমিক ৩০ বর্গকিলোমিটার এলাকা নিয়ে বড়লেখা পৌরসভা ঘোষিত হয়। == ভৌগোলিক উপাত্ত == এই পৌরসভাটি ২২টি গ্রাম এবং ১৬টি মৌজার সমন্বয়ে গঠিত যা ৯টি ওয়ার্ডে বিভক্ত। == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == == নির্বাচিত জন-প্রতিনিধি == == জনসংখ্যা উপাত্ত == == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == == নাগরিক সুযোগ-সুবিধা == == কৃষি অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ব == == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == == বিবিধ == মসজিদ ২৯টি; মন্দির ৪টি; হাট-বাজার ১টি; ব্যাংক শাখা ১১টি; পোস্ট অফিস/সাব পোঃ অফিস ১টি; টেলিফোন এক্সচেঞ্জ ১টি। == আরও দেখুন == বড়লেখা উপজেলা; মৌলভীবাজার জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== বড়লেখা পৌরসভা জাতীয় তথ্য বাতায়ন। উপজেলা জেলার পৌরসভা
