সদর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত ব্রাহ্মণপাড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == ব্রাহ্মণপাড়া উপজেলার দক্ষিণ-মধ্যাংশে ব্রাহ্মণপাড়া সদর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে শশীদল ইউনিয়ন, উত্তরে চান্দলা ইউনিয়ন শিদলাই ইউনিয়ন, পশ্চিমে দুলালপুর ইউনিয়ন এবং দক্ষিণে সাহেবাবাদ ইউনিয়ন বুড়িচং উপজেলার রাজাপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ব্রাহ্মণপাড়া সদর ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ব্রাহ্মণপাড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == মোশাররফ হোসেন খান চৌধুরী বিশ্ববিদ্যালয় কলেজ আবদুল মতিন খসরু মহিলা কলেজ ভগবান সরকারি উচ্চ বিদ্যালয় দীর্ঘভুমি বঙ্গবন্ধু উচ্চ বিদ্যালয় ধান্যদৌল আবদুর রাজ্জাক খান উচ্চ বিদ্যালয় মহালক্ষীপাড়া শরিফ মাধ্যমিক বিদ্যালয় প্রফেসর সেকান্দর আলী বালিকা উচ্চ বিদ্যালয়। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: জসিম উদ্দিন == আরও দেখুন == ব্রাহ্মণপাড়া উপজেলা ব্রাহ্মণপাড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
