'''হাজিরহাট''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত কমলনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কমলনগর উপজেলার দক্ষিণাংশে হাজিরহাট ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে পাটারীরহাট ইউনিয়ন, উত্তরে চর মার্টিন ইউনিয়ন তোরাবগঞ্জ ইউনিয়ন, পূর্বে চর কাদিরা ইউনিয়ন এবং দক্ষিণে রামগতি উপজেলার চর বাদাম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাজিরহাট ইউনিয়ন কমলনগর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কমলনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৭নং নির্বাচনী এলাকা লক্ষ্মীপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == সরকারি উপকূল কলেজ হাজিরহাট ফাযিল মাদ্রাসা সরকারি মিল্লাত একাডেমী তোহা স্মৃতি বালিকা উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == হাজিরহাট বাজার করুনা নগর বাজার == দর্শনীয় স্থান == ভাষা সৈনিক কমরেড মোহাম্মদ তোয়াহার স্মৃতিসৌধ == কৃতী ব্যক্তিত্ব == কমরেড তোয়াহা; ভাষা সৈনিক, রাজনীতিবিদ, সাবেক এমএলএ। আবুল খায়ের; তৎকালীন রামগতি হাতিয়ার সংসদ সদস্য। == জনপ্রতিনিধি == == আরও দেখুন == কমলনগর উপজেলা কমলনগর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
