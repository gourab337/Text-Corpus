'''হানারচর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত চাঁদপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাঁদপুর সদর উপজেলার দক্ষিণাংশে হানারচর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চান্দ্রা ইউনিয়ন, উত্তরে লক্ষ্মীপুর মডেল ইউনিয়ন, পশ্চিমে মেঘনা নদী ইব্রাহিমপুর ইউনিয়ন এবং দক্ষিণে হাইমচর উপজেলার গাজীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হানারচর ইউনিয়ন চাঁদপুর সদর উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাঁদপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == চাঁদপুর সদর উপজেলা চাঁদপুর সদর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
