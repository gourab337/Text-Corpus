বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চৌদ্দগ্রাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চৌদ্দগ্রাম উপজেলার মধ্যাংশে মুন্সিরহাট ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে শুভপুর ইউনিয়ন, উত্তর-পূর্বে ঘোলপাশা ইউনিয়ন, পূর্বে চৌদ্দগ্রাম পৌরসভা, দক্ষিণ-পূর্বে বাতিসা ইউনিয়ন, দক্ষিণে কনকাপৈত ইউনিয়ন এবং পশ্চিমে নাঙ্গলকোট উপজেলার রায়কোট ইউনিয়ন বাঙ্গড্ডা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মুন্সিরহাট ইউনিয়ন চৌদ্দগ্রাম উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চৌদ্দগ্রাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৯নং নির্বাচনী এলাকা কুমিল্লা-১১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চৌদ্দগ্রাম উপজেলা চৌদ্দগ্রাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
