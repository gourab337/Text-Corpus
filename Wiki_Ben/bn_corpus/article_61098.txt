'''গোহট উত্তর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত কচুয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কচুয়া উপজেলার দক্ষিণাংশে গোহট উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কড়ইয়া ইউনিয়ন, দক্ষিণে গোহট দক্ষিণ ইউনিয়ন আশরাফপুর ইউনিয়ন, পূর্বে কুমিল্লা জেলার বরুড়া উপজেলার আদ্রা ইউনিয়ন আড্ডা ইউনিয়ন এবং উত্তরে কুমিল্লা জেলার চান্দিনা উপজেলার জোয়াগ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গোহট উত্তর ইউনিয়ন কচুয়া উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (চাঁদপুর) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
