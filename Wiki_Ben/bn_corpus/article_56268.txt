'''ফুলগাজী''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফুলগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফুলগাজী উপজেলার সর্ব-উত্তরে ফুলগাজী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মুন্সিরহাট ইউনিয়ন দরবারপুর ইউনিয়ন, পূর্বে আমজাদহাট ইউনিয়ন পরশুরাম উপজেলার বক্স মাহমুদ ইউনিয়ন, উত্তরে পরশুরাম উপজেলার চিথলিয়া ইউনিয়ন এবং পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == ফুলগাজী ইউনিয়ন ফুলগাজী উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফুলগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফুলগাজী উপজেলা ফুলগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
