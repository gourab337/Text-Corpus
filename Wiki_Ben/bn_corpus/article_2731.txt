হিন্দি চলচ্চিত্রের অভিনেতাদের একটি তালিকা নিচে দেয়া হল। == == অমিতাভ বচ্চন অজয় দেবগন অনিল কাপুর অশোক কুমার অভিষেক বচ্চন অক্ষয় কুমার অরবিন্দ স্বামী অক্ষয় খান্না == == আমির খান == == ঋত্বিক রোশন ঋষি কাপুর == == কুন্দনলাল সায়গল কিশোর কুমার কমল হাসান কমল সাদানা কুমার গৌরব == == গোবিন্দা গুরু দত্ত == == চাংকি পান্ডে == == জ্যাকি শ্রফ জিতেন্দ্র জয় মুখার্জি জন আব্রাহাম == == *টাইগার শ্রফ == == দিলীপ কুমার দেব আনন্দ == == ধর্মেন্দ্র == == নানা পাটেকর নিরব হোসেন নীল নিতিন মুকেশ ==প== প্রদীপ কুমার ==ফ== ফিরোজ খান ফারুক শেখ ফারহান আখতার == == বিশ্বজিৎ বিনোদ খান্না ববি দেওল বিবেক ওবেরয় == == ভারতভূষণ == == রাজ কাপুর রজনীকান্ত রণধীর কাপুর রনবীর কাপুর রাকেশ রোশন রাজেন্দ্র কুমার ((রাজ কুমার)) রাজেশ খান্না রাজীব কাপুর == == মনোজ কুমার মিঠুন চক্রবর্তী == == শাহরুখ খান শ্যাম কুমার শত্রুঘ্ন সিনহা শশী কাপুর শাম্মী কাপুর শাহিদ কাপুর == স-হ == সঞ্জীব কুমার সালমান খান সঞ্জয় কাপুর সুনীল শেঠি সাইফ আলি খান সঞ্জয় দত্ত সানি দেওল সোরাব মোদী সুনীল দত্ত *সিদ্ধার্থ মালহোত্রা হুসেইন ভাষার চলচ্চিত্র অভিনেতা চলচ্চিত্র অভিনেতা
