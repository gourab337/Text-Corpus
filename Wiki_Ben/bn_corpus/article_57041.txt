'''সোনাইমুড়ি পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == সোনাইমুড়ি উপজেলার মধ্যাংশে সোনাইমুড়ি পৌরসভার অবস্থান। পৌরসভার উত্তরে চাষীরহাট ইউনিয়ন, পশ্চিমে নদনা ইউনিয়ন, দক্ষিণে বজরা ইউনিয়ন এবং পূর্বে বারগাঁও ইউনিয়ন চাষীরহাট ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == সোনাইমুড়ি পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম সোনাইমুড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। এটি একটি ''ক'' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: মোতাহার হোসেন মানিক == আরও দেখুন == সোনাইমুড়ি উপজেলা সোনাইমুড়ি থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
