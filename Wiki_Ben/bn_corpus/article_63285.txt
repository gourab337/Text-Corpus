'''রাজারগাঁও উত্তর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার রাজারগাঁও উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে দ্বাদশ গ্রাম ইউনিয়ন, দক্ষিণে বাকিলা ইউনিয়ন মতলব দক্ষিণ উপজেলার উপাদী দক্ষিণ ইউনিয়ন, পশ্চিমে মতলব দক্ষিণ উপজেলার উপাদী দক্ষিণ ইউনিয়ন উপাদী উত্তর ইউনিয়ন এবং উত্তরে মতলব দক্ষিণ উপজেলার উপাদী উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রাজারগাঁও উত্তর ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
