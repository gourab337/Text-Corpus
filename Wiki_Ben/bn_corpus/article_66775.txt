'''টনকী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার পূর্বাংশে টনকী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পূর্বধইর পশ্চিম ইউনিয়ন চাপিতলা ইউনিয়ন, পশ্চিমে যাত্রাপুর ইউনিয়ন, দক্ষিণে নবীপুর পূর্ব ইউনিয়ন এবং পূর্বে দেবিদ্বার উপজেলার রসুলপুর ইউনিয়ন ইউসুফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == টনকী ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ২২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঙ্গরাবাজার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা বাঙ্গরাবাজার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
