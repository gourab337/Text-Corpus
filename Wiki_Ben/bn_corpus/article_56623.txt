বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার সর্ব-উত্তরে সিন্দুরপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে রাজাপুর ইউনিয়ন; পূর্বে ফেনী সদর উপজেলার পাঁচগাছিয়া ইউনিয়ন শর্শদি ইউনিয়ন; উত্তরে কুমিল্লা জেলার চৌদ্দগ্রাম উপজেলার আলকরা ইউনিয়ন, নাঙ্গলকোট উপজেলার সাতবাড়িয়া ইউনিয়ন বক্সগঞ্জ ইউনিয়ন এবং পশ্চিমে নোয়াখালী জেলার সেনবাগ উপজেলার ডুমুরুয়া ইউনিয়ন কাদরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সিন্দুরপুর ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ১.দরবেশের হাট ফাযিল মাদরাসা। ২.দরবেশের হাট পাবলিক কলেজ। ৩. সুজাতপুর উচ্চ বিদ্যালয়। ৪. ররঘুনাথপুর উচ্চ বিদ্যালয়। ৫. সিন্দুরপুর খাজা আহমদ উচ্চ বিদ্যালয়। ৬. উত্তর কৌশল্যা সরকারী প্রাথমিক বিদ্যালয়। ৭. কৌশল্যা প্রাথমিক বিদ্যালয়। ৮. রঘুনাথপুর প্রাথমিক বিদ্যালয়। ৯. সুজাতপুর প্রাথমিক বিদ্যালয়। ১০. কোরবানপুর প্রাথমিক বিদ্যালয়। ১১. কৌশল্যা বালিকা দাখিল মাদরাসা। ১২. মেরিট কিন্ডার গার্টেন। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ১. দরবেশের হাট বাজার। ২. রঘুনাথপুর বাজার ৩. অলাতলি বাজার ৪. সিন্দুরপুর বাজার == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: নুর নবি। == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
