'''পানিশ্বর''' বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == পানিশ্বর ইউনিয়নের আয়তন ৩৮১০ একর (১৫.৪২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে পানিশ্বর ইউনিয়নের মোট জনসংখ্যা ৩৬,১৮১ জন। এর মধ্যে পুরুষ ১৭,০৮৭ জন এবং মহিলা ১৯,০৯৪ জন। মোট পরিবার ৬,৫৭৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ২৩৪৭ জন। == ইতিহাস == পানিশ্বর ইউনিয়ন পূর্বে পানিশ্বর উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == সরাইল উপজেলার পানিশ্বর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চুণ্টা ইউনিয়ন কালিকচ্ছ ইউনিয়ন, পূর্বে সরাইল সদর ইউনিয়ন সদর উপজেলার বুধল ইউনিয়ন, দক্ষিণে সদর উপজেলার তালশহর পূর্ব ইউনিয়ন, দক্ষিণ-পশ্চিমে আশুগঞ্জ উপজেলার দুর্গাপুর ইউনিয়ন এবং পশ্চিমে মেঘনা নদী কিশোরগঞ্জ জেলার ভৈরব উপজেলার আগানগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পানিশ্বর ইউনিয়ন সরাইল উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে পানিশ্বর ইউনিয়নের সাক্ষরতার হার ৩৬.৫%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
