'''লাউর ফতেপুর''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == লাউর ফতেপুর ইউনিয়নের আয়তন ৩৮৩৮ একর (১৫.৫৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে লাউর ফতেপুর ইউনিয়নের মোট জনসংখ্যা ২০,৭৭১ জন। মোট পরিবার ৪,১২০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৩৩৭ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার দক্ষিণাংশে লাউর ফতেপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে জিনোদপুর ইউনিয়ন, উত্তরে ইব্রাহিমপুর ইউনিয়ন বিটঘর ইউনিয়ন, পূর্বে কাইতলা দক্ষিণ ইউনিয়ন কসবা উপজেলার মেহারী ইউনিয়ন, দক্ষিণে কুমিল্লা জেলার মুরাদনগর উপজেলার আন্দিকোট ইউনিয়ন এবং দক্ষিণ-পশ্চিমে কুমিল্লা জেলার মুরাদনগর উপজেলার আকুবপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == লাউর ফতেপুর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
