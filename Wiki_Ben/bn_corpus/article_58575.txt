'''চর কালকিনি''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত কমলনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কমলনগর উপজেলার সর্ব-পশ্চিমে চর কালকিনি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর লরেন্স ইউনিয়ন, পূর্বে চর মার্টিন ইউনিয়ন, দক্ষিণে সাহেবেরহাট ইউনিয়ন এবং পশ্চিমে মেঘনা নদী ভোলা জেলার দৌলতখান উপজেলার মদনপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর কালকিনি ইউনিয়ন কমলনগর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কমলনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৭নং নির্বাচনী এলাকা লক্ষ্মীপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কমলনগর উপজেলা কমলনগর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
