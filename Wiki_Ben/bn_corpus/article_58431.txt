'''উত্তর হামছাদী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার সর্ব-উত্তরে উত্তর হামছাদী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে দক্ষিণ হামছাদী ইউনিয়ন, দক্ষিণ-পূর্বে পার্বতীনগর ইউনিয়ন, পূর্বে বশিকপুর ইউনিয়ন রামগঞ্জ উপজেলার করপাড়া ইউনিয়ন, উত্তরে রামগঞ্জ উপজেলার লামচর ইউনিয়ন এবং পশ্চিমে রায়পুর উপজেলার বামনী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উত্তর হামছাদী ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লক্ষ্মীপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
