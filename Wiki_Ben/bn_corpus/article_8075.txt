== == অক্ষয়কুমার বড়াল অক্ষয়চন্দ্র সরকার অজিত দত্ত অতুলপ্রসাদ সেন অমিয় চক্রবর্তী অচিন্ত্যকুমার সেনগুপ্ত অরুণ মিত্র অরুণ কুমার সরকার অমিতাভ দাশগুপ্ত অমিতাভ চৌধুরী অনুরাধা মহাপাত্র == == আজিজুর রহমান আজিজুল হাকিম আনোয়ার পাশা আবদুল কাদির আবদুল মান্নান সৈয়দ আবদুল হাই শিকদার আবদুল হাকিম আবিদ আজাদ আবু জাফর ওবায়দুল্লাহ আবুল হাসান আবুল হোসেন আবু হেনা মোস্তফা কামাল আল মাহমুদ আল মুজাহিদী আলাউদ্দিন আল আজাদ আলাওল আলোক সরকার আশিষ সান্যাল আশুতোষ চৌধুরী আসাদ চৌধুরী আসাদ বিন হাফিজ আহসান হাবীব আবিদ আনোয়ার == == ইসমাইল হোসেন সিরাজী == == ঈশ্বরচন্দ্র গুপ্ত == == *একরাম আলি == == ওমর আলী == == কাহ্নপা কাজী নজরুল ইসলাম কাজী কাদের নেওয়াজ কায়কোবাদ কামাল চৌধুরী কামিনী রায় কালিদাস রায় কৃত্তিবাস ওঝা কোরেশী মাগন ঠাকুর == == খান মোহাম্মদ ফারাবী == == গগন হরকরা গিরীন্দ্রমোহিনী দাসী গোবিন্দচন্দ্র দাস গোলাম মোস্তফা গৌড় অভিনন্দ == == ঘনরাম চক্রবর্তী == == চন্দ্রাবতী চণ্ডীদাস চিত্রা লাহিড়ী চৈতালি চট্টোপাধ্যায় == == জগদীশ গুপ্ত জসীম উদ্দীন জয়দেব জয় গোস্বামী জীবনানন্দ দাশ জয়েন উদ্দীন জ্ঞান দাস জিল্লুর রহমান সিদ্দিকী == == তরু দত্ত ত্রিদিব মিত্র তসলিমা নাসরিন == == দেবী রায় দেবেন্দ্রনাথ সেন দৌলত উজির বাহরাম খান দৌলত কাজী দ্বিজেন্দ্রনাথ ঠাকুর দ্বিজেন্দ্রলাল রায় == == == == নবীনচন্দ্র সেন নসরুল্লাহ খাঁ নারায়ণ দেব নির্মলেন্দু গুণ == == প্রেমেন্দ্র মিত্র == == ফকির গরীবুল্লাহ ফজল শাহাবুদ্দীন ফররুখ আহমদ ফালগুনী রায় == == বন্দে আলী মিয়া বড়ু চণ্ডীদাস বিজয়া মুখোপাধ্যায় বিদ্যাপতি বিষ্ণু দে বিহারীলাল চক্রবর্তী বুদ্ধদেব বসু বেলাল চৌধুরী বিনয় মজুমদার বীতশোক ভট্টাচার্য == == ভারতচন্দ্র, রায়গুনাকর == == মতিউল ইসলাম মাহমুদুল মান্নান তারিফ মলয় রায়চৌধুরী মহীউদ্দিন মাইকেল মধুসূদন দত্ত মাহমুদা খাতুন সিদ্দিকা মুকুন্দ দাস মুকুন্দরাম চক্রবর্তী মুহম্মদ নুরুল হুদা মোহাম্মদ নুরুল হুদা মোহাম্মদ মনিরুজ্জামান মোহিতলাল মজুমদার *মোহা হাবিবুল্লাহ == == যতীন্দ্রনাথ সেনগুপ্ত যতীন্দ্রমোহন বাগচী == == রওশন ইজদানী রজনীকান্ত সেন রফিক আজাদ রবীন্দ্রনাথ ঠাকুর রবীন্দ্র গুহ রঙ্গলাল বন্দ্যোপাধ্যায় রাধারমণ দত্ত রামনিধি গুপ্ত রামপ্রসাদ সেন রাশীদুল হাসান রুদ্র মুহম্মদ শহীদুল্লাহ রাম বসু == == লালন শাহ == সমর সেন সমীর রায়চৌধুরী সমুদ্র গুপ্ত সঞ্জয় ভট্টাচার্য সাবিরিদ খান সিকান্দার আবু জাফর সুকান্ত ভট্টাচার্য সুকুমার রায় সুধীন্দ্রনাথ দত্ত সুনীল গঙ্গোপাধ্যায় সুফিয়া কামাল সুভাষ মুখোপাধ্যায় সেলিনা
