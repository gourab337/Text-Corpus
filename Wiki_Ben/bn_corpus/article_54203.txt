'''আণ্ডারচর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত নোয়াখালী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == আণ্ডারচর ইউনিয়নের আয়তন ৪৬.৯৮ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী আণ্ডারচর ইউনিয়নের জনসংখ্যা ৪০,১১০ জন। == অবস্থান সীমানা == নোয়াখালী সদর উপজেলার পশ্চিমাংশে আণ্ডারচর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে কালাদরপ ইউনিয়ন, পূর্বে পূর্ব চর মটুয়া ইউনিয়ন, উত্তরে চর মটুয়া ইউনিয়ন লক্ষ্মীপুর জেলার লক্ষ্মীপুর সদর উপজেলার তেওয়ারীগঞ্জ ইউনিয়ন এবং দক্ষিণ-পশ্চিমে লক্ষ্মীপুর জেলার কমলনগর উপজেলার তোরাবগঞ্জ ইউনিয়ন চর কাদিরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আণ্ডারচর ইউনিয়ন নোয়াখালী সদর উপজেলার আওতাধীন ২০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুধারাম মডেল থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। এটি ৪টি মৌজায় বিভক্ত। ইউনিয়নের গ্রামগুলো হল: পশ্চিম মাইজচরা পূর্ব মাইজচরা আণ্ডারচর কাজিরচর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান: == ;মাধ্যমিক বিদ্যালয় শান্তির হাট উচ্চ বিদ্যালয় ;মাদ্রাসা আণ্ডারচর রহমানিয়া দাখিল মাদ্রাসা (শান্তির হাট) বাংলাবাজার দাখিল মাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == আণ্ডারচর ইউনিয়নের প্রধান প্রধান হাট-বাজারগুলো হল শান্তিরহাট বাজার, বাংলা বাজার, একরামুল করিম চৌধুরী বাজার, চকিদার হাট, কেরে বাজার, কাজির তালুক বাজার এবং চৌরাস্তা বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == ওবায়দুল কাদের –- রাজনীতিবিদ। রুহুল আমিন –- বীরশ্রেষ্ঠ খেতাব প্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি: == চেয়ারম্যানঃ আলী হায়দার বক্সী == আরও দেখুন == নোয়াখালী সদর উপজেলা সুধারাম মডেল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
