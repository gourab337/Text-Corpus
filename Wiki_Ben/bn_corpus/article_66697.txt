'''এলাহাবাদ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == এলাহাবাদ ইউনিয়ন পূর্বে ৭নং জাফরগঞ্জ দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার এলাহাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে জাফরগঞ্জ ইউনিয়ন দেবিদ্বার পৌরসভা, পশ্চিমে গুনাইঘর দক্ষিণ ইউনিয়ন ধামতী ইউনিয়ন, দক্ষিণ-পশ্চিমে সুলতানপুর ইউনিয়ন, দক্ষিণে মোহনপুর ইউনিয়ন এবং পূর্বে বুড়িচং উপজেলার ভারেল্লা ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার মালাপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == এলাহাবাদ ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
