'''বারপাড়া''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা সদর দক্ষিণ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কুমিল্লা সদর দক্ষিণ উপজেলার বারপাড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বিজয়পুর ইউনিয়ন চৌয়ারা ইউনিয়ন, পূর্বে জোড়কানন পশ্চিম ইউনিয়ন, দক্ষিণ-পূর্বে চৌদ্দগ্রাম উপজেলার কাশিনগর ইউনিয়ন, দক্ষিণে লালমাই উপজেলার ভুলইন উত্তর ইউনিয়ন বাগমারা উত্তর ইউনিয়ন এবং পশ্চিমে বরুড়া উপজেলার শিলমুড়ি দক্ষিণ ইউনিয়ন শিলমুড়ি উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বারপাড়া ইউনিয়ন কুমিল্লা সদর দক্ষিণ উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা সদর দক্ষিণ থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সদর দক্ষিণ উপজেলা, কুমিল্লা কুমিল্লা সদর দক্ষিণ থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর দক্ষিণ উপজেলার ইউনিয়ন
