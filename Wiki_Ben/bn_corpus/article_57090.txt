'''সেনবাগ পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == সেনবাগ উপজেলার মধ্যাংশে সেনবাগ পৌরসভার অবস্থান। পৌরসভার উত্তর-পূর্বে কাদরা ইউনিয়ন; উত্তরে ডুমুরুয়া ইউনিয়ন; পশ্চিমে অর্জুনতলা ইউনিয়ন কাবিলপুর ইউনিয়ন; দক্ষিণে বিজবাগ ইউনিয়ন এবং পূর্বে মোহাম্মদপুর ইউনিয়ন, ফেনী জেলার দাগনভূঁইয়া উপজেলার পূর্ব চন্দ্রপুর ইউনিয়ন রাজাপুর ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == সেনবাগ পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
