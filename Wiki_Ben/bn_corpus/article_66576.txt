'''বালিপাড়া ইউনিয়ন''' বাংলাদেশের বরিশাল বিভাগের পিরোজপুর জেলার ইন্দুরকানী উপজেলার একটি ইউনিয়ন প্রশাসনিক এলাকা। == অবস্থান সীমানা == == ইতিহাস == == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == আয়তনঃ ৮৮০৮ একর। জনসংখ্যাঃ পুরুষ- ১৩৫৫৯, মহিলা- ১৩৫৬২। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' ৫৫.১৭ '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' ''মোঃ কবির হোসেন'' তালিকা ক্রমিক চেয়ারম্যানগণের নাম মেয়াদকাল ০১ মৌঃ মুনসুর আলী হাং ১৯৫৪-১৯৭৭ ০২ আব্দুস সোবাহান ১৯৭৭-১৯৮৪ ০৩ মোঃ মতিউর রহমান চুন্নু ১৯৮৪-১৯৮৮ ০৪ ১৯৮৮-১৯৯২ ০৫ আব্দুস সোবাহান ১৯৯২ ১৯৯৮ ০৬ মোঃ মশিউর রহমান মঞ্জু ১৯৯৮ -২০১০ ০৭ ''মোঃ কবির হোসেন'' == আরও দেখুন == পত্তাশি ইউনিয়ন পাড়েরহাট ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == বালিপাড়া ইউনিয়ন- বাংলাদেশ জাতীয় তথ্য বাতায়ন ইন্দুরকানী উপজেলা উপজেলার ইউনিয়ন
