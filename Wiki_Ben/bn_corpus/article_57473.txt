'''চর কাঁকড়া''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কোম্পানীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কোম্পানীগঞ্জ উপজেলার পশ্চিমাংশে চর কাঁকড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে সিরাজপুর ইউনিয়ন, উত্তরে বসুরহাট পৌরসভা, উত্তর-পূর্বে চর হাজারী ইউনিয়ন, পূর্বে রামপুর ইউনিয়ন, দক্ষিণে দক্ষিণ-পশ্চিমে চর ফকিরা ইউনিয়ন এবং পশ্চিমে কবিরহাট উপজেলার ধানশালিক ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর কাঁকড়া ইউনিয়ন কোম্পানীগঞ্জ উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কোম্পানীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কোম্পানীগঞ্জ উপজেলা, নোয়াখালী কোম্পানীগঞ্জ থানা, নোয়াখালী নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (নোয়াখালী) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
