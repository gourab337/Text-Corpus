'''ভূজপুর পাবলিক উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার ভূজপুর ইউনিয়নে অবস্থিত। == ইতিহাস == ২০০৩ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক বাবু সাধন চন্দ্র নাথ। == অবকাঠামো == বিদ্যালয়ে কাঁচা, আধা পাকা নিমার্ণাধীন দ্বিতল ভবন রয়েছে। == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
