'''উপশহর ইউনিয়ন''' বাংলাদেশের যশোর জেলার যশোর সদর উপজেলার অন্তর্গত একটি ইউনিয়ন এই ইউনিয়নের পশ্চিম উত্তর পূর্বদিকে ৪নং নওয়াপাড়া ইউনিয়ন, দক্ষিণ দক্ষিণ-পশ্চিমে ভৈরব নদ অপর পাশে যশোর পৌরসভা। == ইতিহাস == == নামকরণ == == ইউনিয়ন পরিষদ == উপশহর ইউনিয়ন পরিষদ এর বর্তমান চেয়ারম্যান এর নাম কাজী আজগার হোসেন == প্রশাসনিক এলাকা == উপশহর ইউনিয়ন এর আয়তন ১৩৬৯ হেক্টর এবং মৌজার সংখ্যা ৩টি। গ্রাম সমূহের নাম ব্লক বি ব্লক সি ব্লক ডি ব্লক ব্লক এফ ব্লক জি ব্লক এইচ ব্লক == যোগাযোগের ব্যবস্থা == == শিক্ষা == == ইউনিয়নের সরকারি অফিসসমূহ == == ধর্মীয় প্রতিষ্ঠান == == সংগঠন == ক্রীড়া সংগঠনসমূহ আব্দুল্লাহপুর একাদশ ক্লাব ওয়াহেদপুর নবরণী ক্লাব নতুন উপশহর ক্লাব সাংস্কৃতিক সংগঠন দামামা শিল্পী গোষ্ঠী। সিন্দাবাদ শিল্পী গোষ্ঠী। এছাড়ারাও বিভিন্ন এনজিও সমুহ মাঝে মাঝে বিভিন্ন জনসচেতনতা মূলক কার্যক্রমের অংশ হিসেবে নাটিকা, সঙ্গীতানুষ্ঠান ইত্যাদি প্রচার করা হয়। == দর্শনীয় স্থান == == বহিঃসংযোগ == জেলার ইউনিয়ন সদর উপজেলা ইউনিয়নসমূহ
