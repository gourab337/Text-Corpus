'''কবিরহাট পৌরসভা''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == কবিরহাট পৌরসভার আয়তন ১০.৯৩ বর্গ কিলোমিটার। == অবস্থান সীমানা == কবিরহাট উপজেলার মধ্যাংশে কবিরহাট পৌরসভার অবস্থান। পৌরসভার উত্তরে নরোত্তমপুর ইউনিয়ন, পশ্চিমে সুন্দলপুর ইউনিয়ন, দক্ষিণে ঘোষবাগ ইউনিয়ন, পূর্বে ধানশালিক ইউনিয়ন এবং উত্তর-পূর্বে বাটইয়া ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯৯৭ সালের ডিসেম্বর কবিরহাট পৌরসভা প্রতিষ্ঠা লাভ করে। এটি একটি ''খ'' শ্রেণীর পৌরসভা। == প্রশাসনিক এলাকা == কবিরহাট পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম কবিরহাট থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। এটি একটি ''খ'' শ্রেণীর পৌরসভা। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী কবিরহাট পৌরসভার জনসংখ্যা প্রায় ৩০ হাজার। == শিক্ষা == কবিরহাট পৌরসভার সাক্ষরতার হার ৬৪.৯৫%। == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: মোহাম্মদ জহিরুল হক রায়হান == আরও দেখুন == কবিরহাট উপজেলা কবিরহাট থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
