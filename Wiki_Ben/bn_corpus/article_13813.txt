'''দুল্যাতলী''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত লক্ষ্মীছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীছড়ি উপজেলার সর্ব-উত্তরে দুল্যাতলী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে লক্ষ্মীছড়ি ইউনিয়ন; পশ্চিমে মানিকছড়ি উপজেলার তিনটহরী ইউনিয়ন মানিকছড়ি ইউনিয়ন; উত্তরে গুইমারা উপজেলার হাফছড়ি ইউনিয়ন, মহালছড়ি উপজেলার সিন্দুকছড়ি ইউনিয়ন মহালছড়ি ইউনিয়ন এবং পূর্বে রাঙ্গামাটি জেলার নানিয়ারচর উপজেলার সাবেক্ষ্যং ইউনিয়ন ঘিলাছড়ি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দুল্যাতলী ইউনিয়ন লক্ষ্মীছড়ি উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লক্ষ্মীছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীছড়ি উপজেলা লক্ষ্মীছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
