'''সূচীপাড়া দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == শাহরাস্তি উপজেলার সূচীপাড়া দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চিতোষী পশ্চিম ইউনিয়ন, উত্তরে সূচীপাড়া উত্তর ইউনিয়ন, পশ্চিমে হাজীগঞ্জ উপজেলার গন্ধর্ব্যপুর দক্ষিণ ইউনিয়ন, দক্ষিণ-পশ্চিমে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার নোয়াগাঁও ইউনিয়ন এবং দক্ষিণে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার ভাটরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সূচীপাড়া দক্ষিণ ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
