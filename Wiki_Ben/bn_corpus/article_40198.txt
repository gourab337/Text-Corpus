'''পৃথিমপাশা ইউনিয়ন''' বাংলাদেশের সিলেট বিভাগের মৌলভীবাজার জেলার কুলাউড়া উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == == অবস্থান == == প্রশাসনিক এলাকা == == ইতিহাস ঐতিহ্য == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == কুলাউড়া উপজেলার সরকারি ওয়েবসাইট পৃথিমপাশা ইউনিয়ন তথ্য কেন্দ্র জেলার ইউনিয়নসমূহ উপজেলার ইউনিয়নসমূহ
