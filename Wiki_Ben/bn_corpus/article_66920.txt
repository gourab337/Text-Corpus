'''ইলিয়টগঞ্জ উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দাউদকান্দি উপজেলার উত্তর-পূর্বাংশে ইলিয়টগঞ্জ উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে জিংলাতলী ইউনিয়ন, দক্ষিণে ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন, পূর্বে মুরাদনগর উপজেলার বাবুটিপাড়া ইউনিয়ন পাহাড়পুর ইউনিয়ন এবং উত্তরে তিতাস উপজেলার ভিটিকান্দি ইউনিয়ন নারান্দিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইলিয়টগঞ্জ উত্তর ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
