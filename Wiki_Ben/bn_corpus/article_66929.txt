'''দৌলতপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == দৌলতপুর ইউনিয়ন পূর্বে পাঁচগাছিয়া পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দাউদকান্দি উপজেলার দক্ষিণাংশে দৌলতপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে পাঁচগাছিয়া পশ্চিম ইউনিয়ন; উত্তর-পশ্চিমে পদুয়া ইউনিয়ন; উত্তরে গোয়ালমারী ইউনিয়ন বিটেশ্বর ইউনিয়ন; পূর্বে মারুকা ইউনিয়ন; দক্ষিণ-পূর্বে চাঁদপুর জেলার মতলব দক্ষিণ উপজেলার নায়েরগাঁও উত্তর ইউনিয়ন নায়েরগাঁও দক্ষিণ ইউনিয়ন এবং দক্ষিণ-পশ্চিমে ধনাগোদা নদী চাঁদপুর জেলার মতলব উত্তর উপজেলার ইসলামাবাদ ইউনিয়ন দুর্গাপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দৌলতপুর ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
