'''পাইকপাড়া দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত ফরিদগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফরিদগঞ্জ উপজেলার মধ্যাংশে পাইকপাড়া দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পাইকপাড়া উত্তর ইউনিয়ন, পূর্বে পাইকপাড়া উত্তর ইউনিয়ন গুপ্টি পশ্চিম ইউনিয়ন, দক্ষিণে রূপসা উত্তর ইউনিয়ন ফরিদগঞ্জ পৌরসভা এবং পশ্চিমে গোবিন্দপুর উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাইকপাড়া দক্ষিণ ইউনিয়ন ফরিদগঞ্জ উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফরিদগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৩নং নির্বাচনী এলাকা চাঁদপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == গাজীপুর আহমদিয়া সিনিয়র ফাজিল মাদ্রাসা গাজীপুর মুসলিম উচ্চ বিদ্যালয় গাজীপুর সরকারি প্রাথমিক বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == গাজীপুর বাজার *চান্দ্রা বাজার *রূপসা বাজার *কড়ৈইতলী বাজার == দর্শনীয় স্থান == কড়ৈতলী জমিদার বাড়ি == কৃতী ব্যক্তিত্ব == রেজাউল করিম (ফুটবলার) *প্রফেসর ড. মুহাম্মদ মাসুম ইকবাল, ডীন, ব্যবসায় উদ্যেক্তাবৃত্তি অনুষদ, ড্যাফোডিল ইন্টারন্যাশনাল ইউনিভার্সিটি। == জনপ্রতিনিধি == ১নং ওয়ার্ড মেম্বারঃ মোহাম্মদ জসিমউদদীন == আরও দেখুন == ফরিদগঞ্জ উপজেলা ফরিদগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
