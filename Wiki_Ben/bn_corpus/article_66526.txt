'''জোয়াগ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == জোয়াগ ইউনিয়ন পূর্বে বরকরই দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার সর্ব-দক্ষিণে জোয়াগ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বরকরই ইউনিয়ন, পূর্বে বরুড়া উপজেলার চিতড্ডা ইউনিয়ন ঝলম ইউনিয়ন, দক্ষিণে বরুড়া উপজেলার আড্ডা ইউনিয়ন চাঁদপুর জেলার কচুয়া উপজেলার গোহট উত্তর ইউনিয়ন, দক্ষিণ-পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার কড়ইয়া ইউনিয়ন এবং চাঁদপুর জেলার কচুয়া উপজেলার কচুয়া দক্ষিণ ইউনিয়ন কচুয়া উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জোয়াগ ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
