'''শিল্পকলায় একুশে পদক''' বাংলাদেশের শিল্পীদের জন্য একটি জাতীয় এবং সর্বোচ্চ বেসামরিক পুরস্কার। বাংলাদেশের সঙ্গীত, চিত্রকলা, নৃত্যকলা, নাট্যকলা চলচ্চিত্রে অনন্য অবদানের স্বীকৃতি প্রদানের উদ্দেশ্যে ১৯৭৬ সাল থেকে এই ক্ষেত্রে একুশে পদক প্রদান করা হচ্ছে। ভাষা আন্দোলন এর শহীদদের স্মরণে ১৯৭৬ সালে এই পদকের প্রচলন করা হয়। প্রত্যেক পদকপ্রাপ্তকে একটি পদক, একটি সম্মাননা সনদ, একটি রেপ্লিকা এবং পুরস্কারের অর্থমূল্য প্রদান করা হয়ে থাকে। একুশে পদকে ১৮ ক্যারেটের সোনা দিয়ে তৈরি ৩৫ গ্রাম ওজনের একটি মেডেল প্রদান করা হয়; যার ডিজাইন করেছেন নিতুন কুণ্ডু। প্রাথমিকভাবে পুরস্কারের অর্থমূল্য ২৫,০০০ টাকা দেয়া হতো; বর্তমানে এটি লক্ষ টাকায় উন্নীত করা হয়েছে। == বিজয়ীদের তালিকা == +চাবি মরণোত্তর পদক বিজয়ী ===১৯৭০-এর দশক=== বছর চিত্র বিজয়ী বিবরণ সূত্র ১৯৭৭ আবদুল আলীম লোক সঙ্গীতে অবদানের জন্য আলতাফ মাহমুদ সঙ্গীতে অবদানের জন্য গুল মোহাম্মদ খাঁ সঙ্গীতে অবদানের জন্য ফেরদৌসী রহমান সঙ্গীতে অবদানের জন্য জহির রায়হান চলচ্চিত্রে অবদানের জন্য 75px রশিদ চৌধুরী চারুকলায় অবদানের জন্য ১৯৭৮ আভা আলম সঙ্গীতে অবদানের জন্য 75px সফিউদ্দিন আহমেদ চারুকলায় অবদানের জন্য সৈয়দ মোয়াজ্জেম হোসেন চারুকলায় অবদানের জন্য ১৯৭৯ আব্দুল লতিফ সঙ্গীতে অবদানের জন্য শেখ লুতফর রহমান সঙ্গীতে অবদানের জন্য ===১৯৮০-এর দশক=== বছর চিত্র বিজয়ী বিবরণ সূত্র ১৯৮০ বেদারউদ্দিন আহমদ সঙ্গীতে অবদানের জন্য আব্দুল জব্বার সঙ্গীতে অবদানের জন্য হামিদুর রহমান স্থাপত্যশিল্পে অবদানের জন্য 75px মুর্তজা বশীর চারুশিল্পে অবদানের জন্য রণেন কুশারী নাট্যকলায় অবদানের জন্য ১৯৮১ আমিনুল ইসলাম চারুকলায় অবদানের জন্য আবদুল হালিম চৌধুরী সঙ্গীতে অবদানের জন্য মমতাজ আলী খান সঙ্গীতে অবদানের জন্য গওহর জামিল নৃত্যকলায় অবদানের জন্য মোহাম্মদ জাকারিয়া নাট্যকলায় অবদানের জন্য ১৯৮২ ওস্তাদ ফুল মোহাম্মদ সঙ্গীতে অবদানের জন্য এস এম সুলতান চিত্রকলায় অবদানের জন্য জি মান্নান নৃত্যকলায় অবদানের জন্য ১৯৮৩ 75px মোহাম্মদ কিবরিয়া চিত্রকলায় অবদানের জন্য বারীণ মজুমদার সঙ্গীতে অবদানের জন্য ১৯৮৪ ওস্তাদ মীর কাশেম খান সঙ্গীতে অবদানের জন্য 75px সাবিনা ইয়াসমিন সঙ্গীতে অবদানের জন্য 75px কাইয়ুম চৌধুরী চারুকলায় অবদানের জন্য ১৯৮৫ কলিম শরাফী সঙ্গীতে অবদানের জন্য ওস্তাদ আবেদ হোসেন খান সঙ্গীতে অবদানের জন্য 75px সৈয়দ জাহাঙ্গীর চারুকলায় অবদানের জন্য ১৯৮৬ ধীর আলী মিয়া সঙ্গীতে অবদানের জন্য মোবারক হোসেন খান সঙ্গীতে অবদানের জন্য ১৯৮৭ কানাইলাল শীল যন্ত্রসঙ্গীতে অবদানের জন্য 75px ফরিদা পারভীন সঙ্গীতে অবদানের জন্য 75px সৈয়দ মাইনুল হোসেন চারুকলায় অবদানের জন্য ১৯৮৮ আনোয়ার হোসেন চলচ্চিত্রে অবদানের জন্য 75px সুধীন দাশ সঙ্গীতে অবদানের জন্য ১৯৮৯ আবদুর রাজ্জাক চারুকলায় অবদানের জন্য অমলেন্দু বিশ্বাস নাট্যকলায় অবদানের জন্য ===১৯৯০-এর দশক=== বছর চিত্র বিজয়ী বিবরণ সূত্র ১৯৯০ দেবদাস চক্রবর্তী চারুকলায় অবদানের জন্য রাহিজা খানম ঝুনু নৃত্যকলায় অবদানের জন্য খোদা বক্স সাঁই বাউল সঙ্গীতে অবদানের জন্য ১৯৯১ সানজীদা খাতুন সঙ্গীতে অবদানের জন্য মোহাম্মদ আমিনুল হক সঙ্গীতে অবদানের জন্য কাজী আবদুল বাসেত চারুকলায় অবদানের জন্য ১৯৯২ শাহনাজ রহমতুল্লাহ সঙ্গীতে অবদানের জন্য আমজাদ হোসেন চলচ্চিত্রে অবদানের জন্য হাশেম খান চারুকলায় অবদানের জন্য ১৯৯৩ মোহাম্মদ আসাফউদ্দোলাহ সঙ্গীতে অবদানের জন্য ওস্তাদ ফজলুল হক সঙ্গীতে অবদানের জন্য দিলারা জামান নাট্যকলায় অবদানের জন্য রফিকুন নবী চারুকলায় অবদানের জন্য 75px জুয়েল আইচ যাদুশিল্পে অবদানের জন্য ১৯৯৪ আলি মনসুর নাট্যকলায় অবদানের জন্য নীনা হামিদ সঙ্গীতে অবদানের জন্য শাহাদাত হোসেন খান যন্ত্রসঙ্গীতে অবদানের জন্য ১৯৯৫ রওশন জামিল নৃত্যকলায় অবদানের জন্য মুস্তাফা জামান আব্বাসী সঙ্গীতে অবদানের জন্য রথীন্দ্রনাথ রায় সঙ্গীতে অবদানের জন্য ১৯৯৬ ফিরোজ সাঁই সঙ্গীতে অবদানের জন্য ১৯৯৭ 75px নভেরা আহমেদ স্থাপত্যশিল্পে অবদানের জন্য 75px নিতুন কুণ্ডু স্থাপত্যশিল্পে অবদানের জন্য দেবু ভট্টাচার্য সঙ্গীতে অবদানের জন্য শবনম মুশতারী সঙ্গীতে অবদানের জন্য রুনু বিশ্বাস নৃত্যকলায় অবদানের জন্য 75px মমতাজউদ্দীন আহমেদ নাট্যকলায় অবদানের জন্য ১৯৯৮ ফেরদৌসী মজুমদার নাট্যকলায় অবদানের জন্য মাহবুবা রহমান সঙ্গীতে অবদানের জন্য ১৯৯৯ সৈয়দ হাসান ইমাম চলচ্চিত্রে অবদানের জন্য 75px সুভাষ দত্ত চলচ্চিত্রে অবদানের জন্য 75px আলী যাকের নাট্যকলায় অবদানের জন্য মনিরুল ইসলাম চিত্রকলায় অবদানের জন্য হুসনা বানু খানম সঙ্গীতে অবদানের জন্য ফকির আলমগীর সঙ্গীতে অবদানের জন্য আলতামাস আহমেদ নৃত্যকলায় অবদানের জন্য ===২০০০-এর দশক=== বছর চিত্র বিজয়ী বিবরণ সূত্র জন্য রামেন্দু মজুমদার নাট্যকলায় অবদানের জন্য ===২০১০-এর দশক=== বছর চিত্র বিজয়ী বিবরণ সূত্র ২০১০ একেএম আবদুর রউফ চিত্রকলায় অবদানের জন্য ইমদাদ হোসেন চিত্রকলায় অবদানের জন্য 75px পার্থ প্রতিম মজুমদার মূকাভিনয়ে অবদানের জন্য নাসির উদ্দীন ইউসুফ নাট্যকলায় অবদানের জন্য আহমেদ ইমতিয়াজ বুলবুল সঙ্গীতে অবদানের জন্য লায়লা হাসান নৃত্যকলায় অবদানের জন্য ২০১১ আবদুল করিম শাহ সঙ্গীতে অবদানের জন্য আখতার সাদমানী সঙ্গীতে অবদানের জন্য জ্যোৎস্না বিশ্বাস যাত্রা মঞ্চ নাটকে অবদানের জন্য ২০১২ মোবিনুল আজিম চারুকলায় অবদানের জন্য 75px ইনামুল হক নাট্যকলায় অবদানের জন্য 75px মামুনুর রশীদ নাট্যকলায় অবদানের জন্য করুণাময় গোস্বামী চিত্রকলায় অবদানের জন্য ২০১৩ 75px কাদেরী কিবরিয়া সঙ্গীতে অবদানের জন্য 75px বিজয় সরকার সঙ্গীতে অবদানের জন্য জামালউদ্দিন হোসেন চিত্রকলায় অবদানের জন্য বাংলাদেশ উদীচী শিল্পীগোষ্ঠী নাট্যকলায় অবদানের জন্য ২০১৪ রামকানাই দাশ সঙ্গীতে অবদানের জন্য এস এম সোলায়মান নাট্যকলায় অবদানের জন্য সমরজিৎ রায় চৌধুরী চিত্রকলায় অবদানের জন্য কেরামত মওলা নাট্যকলায় অবদানের জন্য ২০১৫ আবদুর রহমান বয়াতী সঙ্গীতে অবদানের জন্য আবুল হায়াত নাট্যকলায় অবদানের জন্য 75px এটিএম শামসুজ্জামান চলচ্চিত্রে অবদানের জন্য ২০১৬ 75px পণ্ডিত অমরেশ রায় চৌধুরী শাস্ত্রীয় সঙ্গীতে অবদানের জন্য শাহীন সামাদ নজরুল সঙ্গীতে অবদানের জন্য জাহানারা আহমেদ নাট্যকলায় চলচ্চিত্রে অবদানের জন্য কাজী আনোয়ার হোসেন চিত্রকলায় অবদানের জন্য আমানুল হক নৃত্যকলায় অবদানের জন্য ২০১৭ 75px সৈয়দ আবদুল্লাহ খালিদ ভাস্কর্যে অবদানের জন্য 75px সুষমা দাস লোকসঙ্গীতে অবদানের জন্য 75px ওস্তাদ আজিজুল ইসলাম যন্ত্রসঙ্গীতে (বংশী) অবদানের জন্য জুলহাস উদ্দিন আহমেদ সঙ্গীতে অবদানের জন্য 75px তানভীর মোকাম্মেল চলচ্চিত্রে অবদানের জন্য 75px সারা জাকের নাট্যকলায় অবদানের জন্য 75px শামীম আরা নিপা নৃত্যকলায় অবদানের জন্য রহমতউল্লাহ আল মাহমুদ সেলিম সঙ্গীতে অবদানের জন্য ২০১৮ শেখ সাদী খান সঙ্গীতে অবদানের জন্য সুজেয় শ্যাম সঙ্গীতে অবদানের জন্য ইন্দ্রমোহন রাজবংশী সঙ্গীতে অবদানের জন্য খুরশিদ আলম সঙ্গীতে অবদানের জন্য মতিউল হক খান সঙ্গীতে অবদানের জন্য 75px হুমায়ুন ফরীদি অভিনয়ে অবদানের জন্য 75px কালিদাস কর্মকার চারুকলা অবদানের জন্য নিখিল সেন নাট্যকলায় অবদানের জন্য মীনু বিল্লাহ নৃত্যে অবদানের জন্য ২০১৯ সুবীর নন্দী সঙ্গীতে অবদানের জন্য 75px আজম খান সঙ্গীতে অবদানের জন্য খায়রুল আনাম শাকিল সঙ্গীতে অবদানের জন্য 75px লাকী ইনাম অভিনয়ে অবদানের জন্য 75px সুবর্ণা মুস্তাফা অভিনয়ে অবদানের জন্য লিয়াকত আলী লাকী অভিনয়ে অবদানের জন্য 75px সাইদা খানম আলোকচিত্রে অবদানের জন্য জামাল উদ্দিন আহমদ চারুকলায় অবদানের জন্য == আরও দেখুন == সাহিত্যে একুশে পদক বিজয়ীদের তালিকা গবেষণায় একুশে পদক বিজয়ীদের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == পদক বৎসর হালনাগাদ করা দরকার এমন নিবন্ধ
