'''মাইসছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মহালছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মহালছড়ি উপজেলার মাইসছড়ি ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে ক্যায়াংঘাট ইউনিয়ন; দক্ষিণে মহালছড়ি ইউনিয়ন; পশ্চিমে সিন্দুকছড়ি ইউনিয়ন, গুইমারা উপজেলার হাফছড়ি ইউনিয়ন, গুইমারা ইউনিয়ন মাটিরাঙ্গা উপজেলার মাটিরাঙ্গা ইউনিয়ন এবং উত্তরে খাগড়াছড়ি সদর উপজেলার খাগড়াছড়ি সদর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মাইসছড়ি ইউনিয়ন মহালছড়ি উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মহালছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মহালছড়ি উপজেলা মহালছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
