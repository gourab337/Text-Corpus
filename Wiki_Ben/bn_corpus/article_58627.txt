বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রামগতি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগতি উপজেলার পশ্চিমাংশে আলেকজান্ডার ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মেঘনা নদী চর আবদুল্যাহ ইউনিয়ন; পূর্বে রামগতি পৌরসভা, চর আলগী ইউনিয়ন চর বাদাম ইউনিয়ন; উত্তরে কমলনগর উপজেলার পাটারীরহাট ইউনিয়ন চর ফলকন ইউনিয়ন এবং পশ্চিমে মেঘনা নদী ভোলা জেলার দৌলতখান উপজেলার মদনপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আলেকজান্ডার ইউনিয়ন রামগতি উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগতি থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৭নং নির্বাচনী এলাকা লক্ষ্মীপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == চর আব্দুল্যাহ্ আল মাদ্রাসাতুল জামিয়াতুল ইসলামিয়া ফাযিল (ডিগ্রি) মাদ্রাসা বালুর চর উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ১/জনতা বাজার == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রামগতি উপজেলা রামগতি থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
