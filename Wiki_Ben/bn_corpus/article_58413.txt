'''লক্ষ্মীপুর পৌরসভা''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার মধ্যাংশে লক্ষ্মীপুর পৌরসভার অবস্থান। পৌরসভার উত্তরে দক্ষিণ হামছাদী ইউনিয়ন; পূর্বে পার্বতীনগর ইউনিয়ন, বাঙ্গাখাঁ ইউনিয়ন লাহারকান্দি ইউনিয়ন; দক্ষিণে টুমচর ইউনিয়ন শাকচর ইউনিয়ন এবং পশ্চিমে চর রুহিতা ইউনিয়ন দালাল বাজার ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯৭৬ সালের সেপ্টেম্বর তৎকালীন ৫নং বাঞ্চানগর ইউনিয়ন লক্ষ্মীপুর পৌরসভায় রূপান্তরিত হয়। পরে এই পৌরসভাটির বিস্তৃতি ঘটে। == প্রশাসনিক এলাকা == লক্ষ্মীপুর পৌরসভায় ১২টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম লক্ষ্মীপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান মেয়র: এম তাহের == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা সদর উপজেলা
