'''ফরিদগঞ্জ দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত ফরিদগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফরিদগঞ্জ উপজেলার দক্ষিণাংশে ফরিদগঞ্জ দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে চর দুঃখিয়া পূর্ব ইউনিয়ন চর দুঃখিয়া পশ্চিম ইউনিয়ন, পশ্চিমে গোবিন্দপুর দক্ষিণ ইউনিয়ন, উত্তরে ফরিদগঞ্জ পৌরসভা এবং পূর্বে রূপসা উত্তর ইউনিয়ন রূপসা দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ফরিদগঞ্জ দক্ষিণ ইউনিয়ন ফরিদগঞ্জ উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফরিদগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৩নং নির্বাচনী এলাকা চাঁদপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == ফরিদগঞ্জ উপজেলা ফরিদগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
