'''মতিগঞ্জ''' বাংলাদেশের ফেনী জেলার অন্তর্গত সোনাগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাগাজী উপজেলার মধ্যাংশে মতিগঞ্জ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মঙ্গলকান্দি ইউনিয়ন, পশ্চিমে চর দরবেশ ইউনিয়ন, দক্ষিণে সোনাগাজী পৌরসভা সোনাগাজী ইউনিয়ন এবং পূর্বে আমিরাবাদ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মতিগঞ্জ ইউনিয়ন সোনাগাজী উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ইউনিয়নে রয়েছে টি হাই স্কুল। আর. এম. হাট. কে উচ্চ বিদ্যালয়। এটি ইউনিয়ন পরিষদ কার্যালয়ের দক্ষিণ পার্শ্ববর্তী অবস্থিত। এটি একটি প্রাচীন শিক্ষা প্রতিষ্ঠান। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাদেশের ইউনিয়ন নিবন্ধ উপজেলার ইউনিয়ন
