'''ধনতলা ইউনিয়ন''' বাংলাদেশের রংপুর বিভাগের ঠাকুরগাঁও জেলার বালিয়াডাঙ্গী উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == ধনতলা ইউনিয়নের আয়তনঃ ৩৫.৬৬ বর্গকি.মি == অবস্থান == == প্রশাসনিক এলাকা == == ইতিহাস == ধনতলা ইউনিয়ন ১৯৭১ সালে প্রতিষ্ঠিত হয়। == জনসংখ্যার উপাত্ত == জন্ম নিবন্ধন মতে, ২২১১০ জন, পুরুষ ১১১২২ জন এবং মহিলা- ১০৯৮৮ জন। মোট পরিবার ৫০০০ টি।। বাংলাদেশর ১৯৯১ সনের আদমশুমারি অনুসারে চাড়োল ইউনিয়নের জনসংখ্যা ১৭,৯৫৯ জন। == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == ধনতলা ইউনিয়নের সরকারি ওয়েবসাইট ==তথ্যসূত্র== জেলার ইউনিয়ন উপজেলার ইউনিয়ন
