'''ফরদাবাদ''' বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == ফরদাবাদ ইউনিয়নের আয়তন ৩২৯৬ একর (১৩.৩৪ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ফরদাবাদ ইউনিয়নের মোট জনসংখ্যা ২১,৯৫২ জন। মোট পরিবার ৪,৩৩৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৬৪৬ জন। == ইতিহাস == ফরদাবাদ ইউনিয়ন পূর্বে রূপসদী পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার ফরদাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ছলিমাবাদ ইউনিয়ন রূপসদী ইউনিয়ন; উত্তরে ছয়ফুল্লাকান্দি ইউনিয়ন দড়িকান্দি ইউনিয়ন; পূর্বে নবীনগর উপজেলার রতনপুর ইউনিয়ন, কুমিল্লা জেলার মুরাদনগর উপজেলার শ্রীকাইল ইউনিয়ন রামচন্দ্রপুর উত্তর ইউনিয়ন এবং দক্ষিণে কুমিল্লা জেলার হোমনা উপজেলার চান্দেরচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ফরদাবাদ ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
