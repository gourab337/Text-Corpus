''ফুলকী ইউনিয়ন'' বাংলাদেশের টাঙ্গাইল জেলার অন্তর্গত বাসাইল উপজেলার একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == ফুলকী ইউনিয়নের মোট আয়তন ২৯ বর্গ কিঃ মিঃ। গ্রাম সংখ্যা ২৪ টি। == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী ফুলকী ইউনিয়নের মোট জনসংখ্যা ৩০,০৩৯ জন। প্শিক্ষার হার ৩০ %। == যোগাযোগ ব্যবস্থা == == হাটবাজারের তালিকা == == হাসপাতাল == == শিক্ষা == == দর্শনীয় স্থান == ময়থা বিল, নুন্দা বিল, নিরাইল বিল, আন্দারমানিক বিল, ময়থা, নেধার খাটরা গ্রামের গামছা, চাদর শাড়ি তৈরী কারখানা, আইসড়া জরাশাহর মাজার == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == ==তথ্যসূত্র== উপজেলার ইউনিয়ন জেলার ইউনিয়ন
