'''চর পাতা''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রায়পুর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রায়পুর উপজেলার উত্তরাংশে চর পাতা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে কেরোয়া ইউনিয়ন, দক্ষিণে কেরোয়া ইউনিয়ন রায়পুর পৌরসভা, পশ্চিমে রায়পুর ইউনিয়ন চাঁদপুর জেলার ফরিদগঞ্জ উপজেলার চর দুঃখিয়া পূর্ব ইউনিয়ন এবং উত্তরে চাঁদপুর জেলার ফরিদগঞ্জ উপজেলার রূপসা দক্ষিণ ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == চর পাতা ইউনিয়ন পূর্বে রায়পুর ইউনিয়নের অংশ ছিল। ১৯৯৬ সালে সংসদ সদস্য হারুনুর রশিদ রায়পুর উপজেলা নির্বাহী অফিসারের মাধ্যমে ৫নং চর পাতা ইউনিয়ন হিসেবে মন্ত্রণালয়ে একটি প্রস্তাব পাঠান। পরে ২০০২ সালের ১০ জুলাই রায়পুর ইউনিয়ন থেকে আলাদা হয়ে ৫নং চর পাতা ইউনিয়ন গঠিত হয়। == প্রশাসনিক কাঠামো == চর পাতা ইউনিয়ন রায়পুর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রায়পুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। এটি ''চর পাতা'' এবং ''গাজীনগর'' দুইটি মৌজা নিয়ে গঠিত। == শিক্ষা ব্যবস্থা == চর পাতা ইউনিয়নে ২টি মাধ্যমিক বিদ্যালয়, ৭টি সরকারি প্রাথমিক বিদ্যালয়, ১টি মাদ্রাসা, ৩টি কিন্ডারগার্টেন ২টি এতিমখানা রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == চর পাতা ইউনিয়নের প্রধান ৩টি হাট-বাজার হল বোয়াডার বাজার, গাজীনগর বাজার এবং আমিন বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রায়পুর উপজেলা রায়পুর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
