'''পূর্ব লক্ষীপ্রসাদ ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত কানাইঘাট উপজেলার একটি ইউনিয়ন। == অবস্থান == == ইতিহাস == == আয়তন জনসংখ্যা == ২৪৮৩৫ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == == দর্শনীয় স্থান == লোভাছড়া == হাট-বাজার == মমতাজগঞ্জবাজার মুলাগুল নয়বাবাজার মুলাগুল বাজার দনাবাজার == কৃতী ব্যক্তিত্ব == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == পূর্ব লক্ষীপ্রসাদ ইউনিয়ন জাতীয় তথ্য বাতায়ন। বাংলাপিডিয়ায় ''কানাইঘাট উপজেলা'' ''এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' উপজেলার ইউনিয়ন বিভাগের ইউনিয়ন জেলার ইউনিয়ন
