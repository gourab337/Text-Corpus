'''আইলহাঁস''' বাংলাদেশের আলমডাঙ্গা উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == মোট আয়তন ৬,১৭৯ একর। == অবস্থান == == প্রশাসনিক এলাকা == == ইতিহাস == == জনসংখ্যার উপাত্ত == জনসংখ্যা নারী এবং পুরুষের সংখ্যা ১৫,৭৬৬ (পনেরো হাজার সাতশত ষেষ্ট্টি) জন। == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == দর্শনীয় স্থান == ঘোলদাড়ী শাহী মসজিদটি বৃহত্তর কুষ্টিয়া চুয়াডাঙ্গা জেলার প্রথম মসজিদ। এই মসজিদটি ঘোদাড়ী বাজর থেকে প্রায় কিঃ মিঃ দক্ষিণ পূর্ব দিকে অবস্থিত। এখান থেকেই এই জেলায় গুলোর প্রথম ইসলাম প্রচার শুরু হয়। বর্তমানে এটিই এই ইউনিয়নের একমাত্র দর্শনীয় স্থান। == আরও দেখুন == == বহিঃসংযোগ == তথ্য সূত্রঃ আইলহাঁস ইউনিয়ান জাতীয় তথ্য বাতায়ন।
