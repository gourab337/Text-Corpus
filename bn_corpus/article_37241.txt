'''আজমিরীগঞ্জ পৌরসভা''' বাংলাদেশের হবিগঞ্জ জেলার একটি স্থানীয় সরকার ব্যবস্থা সংগঠন। এই পৌরসভাটি বাংলাদেশের একটি ‘‘গ’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == আজমিরীগঞ্জ পৌরসভাটি সিলেট বিভাগের হবিগঞ্জ জেলার আজমিরীগঞ্জ উপজেলা সদরে অবস্থিত। এই পৌর এলাকাটির আয়তন সাড়ে বর্গ কিলোমিটার। == ইতিহাস == ২০০৪ সালের ২১ জানুয়ারি তারিখ আজমিরীগঞ্জ উপজেলা সদরকে পৌরসভা হিসাবে ঘোষণা করা হয়। == ভৌগোলিক উপাত্ত == == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == ৯টি ওয়ার্ডে বিভক্ত এই পৌরসভাটির ওয়ার্ড বিন্যাস নিম্নরূপঃ নং ওয়ার্ড ফতেহপুর, কৃষ্ণ নগর, ঝালহাটি; নং ওয়ার্ড নগর আংশিক; নং ওয়ার্ড নগর আংশিক; নং ওয়ার্ড ইলামনগর, আজিমনগর জুম্মাহাটি; নং ওয়ার্ড আজিমনগর লম্বাহাটি; নং ওয়ার্ড নবীন নগর, আজিমনগর মুন্সিহাটি; নং ওয়ার্ড শরীফ নগর; নং ওয়ার্ড আজমিরীগঞ্জ বাজার, পুকুরপাড়; নং ওয়ার্ড জগতপুর, সমীপুর। == নির্বাচিত জন-প্রতিনিধি == বিগত ২০১৫ সালের ৩০ ডিসেম্বর তারিখে সারা দেশে মোট ২৩৪টি পৌরসভায় নির্বাচন অণুষ্ঠিত হলেও বিগত ১১ বছরের অধিক সময় যাবৎ মামলা জটিলতার কারণে পৌরসভা প্রতিষ্ঠার পর হতে এখানে কোন নির্বাচন আয়োজন করা যায়নি। == জনসংখ্যা উপাত্ত == == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == == নাগরিক সুযোগ-সুবিধা == == কৃষি অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ব == == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == == বিবিধ == == আরও দেখুন == আজমিরীগঞ্জ উপজেলা; হবিগঞ্জ জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== আজমিরীগঞ্জ পৌরসভা জাতীয় তথ্য বাতায়ন। আজমিরীগঞ্জ পৌরসভা স্থানীয় সরকার প্রকৌশল অধিদপ্তর। উপজেলা জেলার পৌরসভা বিভাগের পৌরসভা
