অন্য ব্যবহারের জন্য, দেখুন আলীনগর ইউনিয়ন (দ্ব্যর্থতা নিরসন)। '''আলীনগর ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত বিয়ানীবাজার উপজেলার একটি ঐতিহ্যবাহী বৃহত্তম ইউনিয়ন। == অবস্থান == বিয়ানীবাজার উপজেলা হতে ১নং আলীনগর ইউনিয়ন পরিষদের দুরত্ব ২৭ কি:মি:। == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন –''' ২৩ (বর্গ কিঃ মিঃ)। লোকসংখ্যা –২৯,৬৩৩ জন (প্রায়) ১৫,৪৮০ (নারী) ১৫,১৫৩ (পুরুষ)। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার –'''৫৯%। (''২০০১ এরশিক্ষা জরীপ অনুযায়ী'') '''শিক্ষা প্রতিষ্ঠাণ-''' সরকারী প্রাথমিক বিদ্যালয়- ১৩ টি, উত্তরভাগ সরঃপ্রাঃবিঃ খলাগ্রাম সরঃপ্রাঃ বিঃ ঢাকাউত্তর সরঃপ্রাঃবিঃ ব্রাহ্মনগ্রাম সরঃপ্রাঃবিঃ সুরমা সরঃপ্রাঃবিঃ রামধা সরঃপ্রাঃবিঃ ফুলমলিক সরঃপ্রাঃবিঃ রায়খাইল খলাগ্রাম সরঃপ্রাঃবিঃ কনকলস সরঃপ্রাঃবিঃ ১০ চন্দগ্রাম সরঃপ্রাঃবিঃ ১১ হাজী ওয়াসিদ আলী সরঃপ্রাঃবিঃ ১২ কাদিমলিক সরঃপ্রাঃবিঃ ১৩ রাজাপুর সরঃপ্রাঃবিঃ কেজি স্কুলঃ টি মাধ্যমিক বিদ্যালয়ঃ টি, ডি.এম.হাই স্কুল আজাদ চৌধুরী একাডেমী মাদ্রাসাঃ ৬টি (পুরুষ ২টি) (মহিলা ৪টি) কলেজঃ ০১টি, জামে মসজিদঃ ৩৯টি পান্জে খনাঃ ০৮ টি মন্দির/আখড়াঃ ০৫টি == দর্শনীয় স্থান == সুরমা নদীর পারে ছড়িয়ে থাকা পাথর সৈয়দ নবীব আলী সাহেবের পুরাতন বাড়ী সুন্দর সুন্দর মসজিদ এবং মন্দির হাওর মোঘল আমলের জায়গীরদার মসজিদ == গ্রাম সমূহ == গ্রামের সংখ্যা –২১টি। রজাকপুর, খলাগ্রাম উত্তরভাগ নছির খানী টিকর পাড়া, চন্দরপুর,নিজ মোহাম্মদ পুর, ফুল মলিক, চন্দগ্রাম, চন্দগ্রাম পাতন, কাদিমলিক, খাসা হেতিমখানী, করগ্রাম, উজানঢাকি, পুর্ব খলাগ্রাম, কনকলস, রায়খাইল। == হাট-বাজার == রামধা বাজার মদনগন্জ বাজার == খাল নদী == সুরমা নদী কুশিয়ারা নদী আমআমির খাল করতি খাল) কাপাশ খাল == কৃতী ব্যক্তিত্ব == সৈয়দ নবীব আলী সিলেট জেলার প্রথম ডিসি। *সাহেজ মোহাম্মদ, সাহিত্যিক *এডভোকেট আব্দুর রহিম এম.পি. জন == আরও দেখুন == বিয়ানীবাজার সরকারি কলেজ নুরুল ইসলাম নাহিদ তৌফিক-ই-ইলাহী চৌধুরী == তথ্যসূত্র == উপজেলার ইউনিয়ন জেলার ইউনিয়ন বিভাগের ইউনিয়ন
