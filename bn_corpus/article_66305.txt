'''মক্রবপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার উত্তর-মধ্যাংশে মক্রবপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পেড়িয়া ইউনিয়ন, পূর্বে রায়কোট ইউনিয়ন, দক্ষিণে নাঙ্গলকোট পৌরসভা এবং পশ্চিমে হেসাখাল ইউনিয়ন লাকসাম উপজেলার আজগরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মক্রবপুর ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
