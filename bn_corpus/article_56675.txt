'''সোনাগাজী পৌরসভা''' বাংলাদেশের ফেনী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == সোনাগাজী পৌরসভার আয়তন ৫.৩৩ বর্গ কিলোমিটার। == অবস্থান সীমানা == সোনাগাজী উপজেলার মধ্যাংশে সোনাগাজী পৌরসভার অবস্থান। পৌরসভার উত্তরে মতিগঞ্জ ইউনিয়ন, পশ্চিমে চর দরবেশ ইউনিয়ন, দক্ষিণে চর দরবেশ ইউনিয়ন চর চান্দিয়া ইউনিয়ন এবং পূর্বে সোনাগাজী ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == সোনাগাজী পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুযায়ী সোনাগাজী পৌরসভার জনসংখ্যা ২০,৮১০ জন। == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: রফিকুল ইসলাম খোকন == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
