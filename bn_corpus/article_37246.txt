'''চুনারুঘাট পৌরসভা''' বাংলাদেশের হবিগঞ্জ জেলার একটি স্থানীয় সরকার ব্যবস্থা সংগঠন। এই পৌরসভাটি বাংলাদেশের একটি ‘‘ক’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == চুনারুঘাট পৌরসভাটি সিলেট বিভাগের হবিগঞ্জ জেলার চুনারুঘাট উপজেলা সদরে অবস্থিত। এই পৌর এলাকাটির আয়তন ৮.১ বর্গ কিলোমিটার। == ইতিহাস == ২০০৫ সালের ১০ অক্টোবর তারিখ চুনারুঘাট পৌরসভা গঠিত হয়। == ভৌগোলিক উপাত্ত == == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == ৯টি ওয়ার্ডে বিভক্ত এই পৌরসভাটির ওয়ার্ড বিন্যাস নিম্নরূপঃ নং ওয়ার্ড গোগাউড়া আংশিক, মমিনপুর, বাগবাড়ী; নং ওয়ার্ড বড়াইল; নং ওয়ার্ড বড়াইল আংশিক, হাতুন্ডা আংশিক; নং ওয়ার্ড নয়ানী, ডিমাগুরুন্ডা, হাতুন্ডা আংশিক; নং ওয়ার্ড বড়াইল আংশিক, মোহাম্মদীয়া আবাসিক এলাকা, চুনারুঘাট বাজার আংশিক; নং ওয়ার্ড চুনারুঘাট বাজার আংশিক, চন্দনা আংশিক; নং ওয়ার্ড পশ্চিম পাকুড়িয়া, গুচ্ছ গ্রাম; নং ওয়ার্ড আমকান্দি, ধলাইরপাড়; নং ওয়ার্ড হাতুন্ডা আংশিক, চন্দনা আংশিক দেওরগাছ আংশিক। == নির্বাচিত জন-প্রতিনিধি == বিগত ২০১৫ সালের ৩০ ডিসেম্বর তারিখে অনুষ্ঠিত পৌরসভা নির্বাচনে বিএনপি সমর্থিত প্রার্থী নাজিম উদ্দিন সামছু মেয়র পদে নির্বাচিত হয়েছেন। == জনসংখ্যা উপাত্ত == ২০০১ সালের আদমশুমারী অনুসারে এখানে প্রায় ২৪ হাজার লোক বসবাস করে। == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == == নাগরিক সুযোগ-সুবিধা == == কৃষি অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতি ব্যক্তিত্ব == #শাহ্‌ ইসমাঈল তালুকদার,সংগীত শিল্পী পরিচালক। == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == == বিবিধ == == আরও দেখুন == চুনারুঘাট উপজেলা; হবিগঞ্জ জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== চুনারুঘাট পৌরসভা জাতীয় তথ্য বাতায়ন। চুনারুঘাট পৌরসভা স্থানীয় সরকার প্রকৌশল অধিদপ্তর। উপজেলা জেলার পৌরসভা বিভাগের পৌরসভা
