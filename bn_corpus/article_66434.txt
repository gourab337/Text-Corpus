'''আজগরা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == লাকসাম উপজেলার আজগরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে লাকসাম পূর্ব ইউনিয়ন লাকসাম পৌরসভা, পশ্চিমে দক্ষিণে উত্তরদা ইউনিয়ন, দক্ষিণ-পূর্বে নাঙ্গলকোট উপজেলার হেসাখাল ইউনিয়ন এবং পূর্বে নাঙ্গলকোট উপজেলার মক্রবপুর ইউনিয়ন পেড়িয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আজগরা ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
