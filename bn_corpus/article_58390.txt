'''উত্তর চর আবাবিল''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রায়পুর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রায়পুর উপজেলার উত্তর চর আবাবিল ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে উত্তর চর বংশী ইউনিয়ন, পূর্বে দক্ষিণ চর আবাবিল ইউনিয়ন, উত্তরে চাঁদপুর জেলার হাইমচর উপজেলার আলগী দুর্গাপুর দক্ষিণ ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার হাইমচর উপজেলার চর ভৈরবী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উত্তর চর আবাবিল ইউনিয়ন রায়পুর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রায়পুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রায়পুর উপজেলা রায়পুর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
