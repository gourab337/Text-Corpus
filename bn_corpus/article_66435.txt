'''লাকসাম পূর্ব''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। এটি নরপাটি ইউনিয়ন নামেও পরিচিত। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == লাকসাম উপজেলার পূর্বাংশে লাকসাম পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে লাকসাম পৌরসভা, দক্ষিণে আজগরা ইউনিয়ন, পূর্বে নাঙ্গলকোট উপজেলার পেড়িয়া ইউনিয়ন লালমাই উপজেলার বেলঘর দক্ষিণ ইউনিয়ন এবং উত্তরে লালমাই উপজেলার বেলঘর উত্তর ইউনিয়ন পেরুল দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == লাকসাম পূর্ব ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
