'''কমলছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত খাগড়াছড়ি সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == খাগড়াছড়ি সদর উপজেলার কমলছড়ি ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে খাগড়াছড়ি সদর ইউনিয়ন খাগড়াছড়ি পৌরসভা, উত্তরে গোলাবাড়ী ইউনিয়ন, পূর্বে দীঘিনালা উপজেলার মেরুং ইউনিয়ন রাঙ্গামাটি জেলার লংগদু উপজেলার আটারকছড়া ইউনিয়ন এবং দক্ষিণে মহালছড়ি উপজেলার ক্যায়াংঘাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কমলছড়ি ইউনিয়ন খাগড়াছড়ি সদর উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম খাগড়াছড়ি সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == খাগড়াছড়ি সদর উপজেলা খাগড়াছড়ি সদর থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
