বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == ইউনিয়নের আয়তন ২২২৫ একর (৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ইউনিয়নের মোট জনসংখ্যা ১৫,২০১ জন। মোট পরিবার ৩,০২৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৬৮৮ জন। == ইতিহাস == == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার উত্তরাংশে ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দরিয়াদৌলত ইউনিয়ন, দক্ষিণে ছয়ফুল্লাকান্দি ইউনিয়ন তেজখালী ইউনিয়ন, পূর্বে তেজখালী ইউনিয়ন নবীনগর উপজেলার বড়িকান্দি ইউনিয়ন এবং উত্তরে মেঘনা নদী নরসিংদী জেলার নরসিংদী সদর উপজেলার আলোকবালী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
