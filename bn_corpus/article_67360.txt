বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == বড়িকান্দি ইউনিয়নের আয়তন ৩২৬৩ একর (১৩.২০ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বড়িকান্দি ইউনিয়নের মোট জনসংখ্যা ১৮,৭৫১ জন। মোট পরিবার ৩,৭৪১টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৪২০ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার পশ্চিমাংশে বড়িকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে শ্যামগ্রাম ইউনিয়ন; দক্ষিণে ছলিমগঞ্জ ইউনিয়ন; পশ্চিমে বাঞ্ছারামপুর উপজেলার তেজখালী ইউনিয়ন, ইউনিয়ন, মেঘনা নদী নরসিংদী জেলার নরসিংদী সদর উপজেলার আলোকবালী ইউনিয়ন এবং উত্তরে মেঘনা নদী নরসিংদী জেলার রায়পুরা উপজেলার নিলক্ষ্যা ইউনিয়ন চর মধুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বড়িকান্দি ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == মোফাজ্জল হোসেন –– বীর প্রতীক খেতাবপ্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
