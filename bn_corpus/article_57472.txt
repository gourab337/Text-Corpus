'''চর হাজারী''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কোম্পানীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কোম্পানীগঞ্জ উপজেলার পূর্বাংশে চর হাজারী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর পার্বতী ইউনিয়ন, পশ্চিমে বসুরহাট পৌরসভা, দক্ষিণ-পশ্চিমে চর কাঁকড়া ইউনিয়ন, দক্ষিণে রামপুর ইউনিয়ন মুছাপুর ইউনিয়ন এবং পূর্বে ফেনী জেলার সোনাগাজী উপজেলার চর দরবেশ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর হাজারী ইউনিয়ন কোম্পানীগঞ্জ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কোম্পানীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কোম্পানীগঞ্জ উপজেলা, নোয়াখালী কোম্পানীগঞ্জ থানা, নোয়াখালী নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (নোয়াখালী) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
