'''উজানচর''' বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == উজানচর ইউনিয়নের আয়তন ৩২০৮ একর (১২.৯৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে উজানচর ইউনিয়নের মোট জনসংখ্যা ২০,৬৩৫ জন। মোট পরিবার ৪,০৮১টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৫৮৯ জন। == ইতিহাস == উজানচর ইউনিয়ন পূর্বে উজানচর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার সর্ব-দক্ষিণে উজানচর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বাঞ্ছারামপুর পৌরসভা বাঞ্ছারামপুর ইউনিয়ন; পশ্চিমে মানিকপুর ইউনিয়ন, মেঘনা নদী নারায়ণগঞ্জ জেলার আড়াইহাজার উপজেলার কালাপাহাড়িয়া ইউনিয়ন; দক্ষিণে কুমিল্লা জেলার হোমনা পৌরসভা এবং পূর্বে তিতাস নদী, কুমিল্লা জেলার হোমনা উপজেলার ঘাগুটিয়া ইউনিয়ন দুলালপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উজানচর ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
