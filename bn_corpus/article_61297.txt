'''উপাদী উত্তর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত মতলব দক্ষিণ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মতলব দক্ষিণ উপজেলার দক্ষিণাংশে উপাদী উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে উপাদী দক্ষিণ ইউনিয়ন, পশ্চিমে মতলব পৌরসভা, উত্তরে মতলব পৌরসভা খাদেরগাঁও ইউনিয়ন এবং পূর্বে নারায়ণপুর ইউনিয়ন হাজীগঞ্জ উপজেলার রাজারগাঁও উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উপাদী উত্তর ইউনিয়ন মতলব দক্ষিণ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মতলব দক্ষিণ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬১নং নির্বাচনী এলাকা চাঁদপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == মতলব দক্ষিণ উপজেলা মতলব দক্ষিণ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == দক্ষিণ উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
