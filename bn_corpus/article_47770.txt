'''সোনারগাঁও উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার রাঙ্গুনিয়া উপজেলার দক্ষিণ রাজানগর ইউনিয়নের সোনারগাঁও গ্রামে অবস্থিত। == ইতিহাস == ১৯৬৯ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য ১২ জন বিশিষ্ট একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব মোহাম্মদ গিয়াস উদ্দীন। == অবকাঠামো == বিদ্যালয়ের মূল ভবনটি একটি দ্বিতল ভবন। বিদ্যালয়ের সামনে একটি খেলার মাঠ রয়েছে। == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। বর্তমানে চার শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == ফলাফল কৃতিত্ব == ২০১৭ সালের এসএসসি পরীক্ষায় পাশের হার ৯০.৫৭%। জন পরীক্ষার্থী জিপিএ অর্জন করেছে। == আরো দেখুন == রাঙ্গুনিয়া উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
