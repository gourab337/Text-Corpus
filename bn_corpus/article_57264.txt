'''পূর্ব চর মটুয়া''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত নোয়াখালী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নোয়াখালী সদর উপজেলার পশ্চিমাংশে পূর্ব চর মটুয়া ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে পূর্বে কালাদরপ ইউনিয়ন, উত্তরে চর মটুয়া ইউনিয়ন এবং পশ্চিমে আণ্ডারচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পূর্ব চর মটুয়া ইউনিয়ন নোয়াখালী সদর উপজেলার আওতাধীন ১৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুধারাম মডেল থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == নোয়াখালী সদর উপজেলা সুধারাম মডেল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
