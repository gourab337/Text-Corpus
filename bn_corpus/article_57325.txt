'''বাটইয়া''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কবিরহাট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কবিরহাট উপজেলার উত্তর-পূর্বাংশে বাটইয়া ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে নরোত্তমপুর ইউনিয়ন, দক্ষিণ-পশ্চিমে কবিরহাট পৌরসভা, দক্ষিণে ধানশালিক ইউনিয়ন, পূর্বে কোম্পানীগঞ্জ উপজেলার সিরাজপুর ইউনিয়ন এবং উত্তরে সেনবাগ উপজেলার নবীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাটইয়া ইউনিয়ন কবিরহাট উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কবিরহাট থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কবিরহাট উপজেলা কবিরহাট থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
