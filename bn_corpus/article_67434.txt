'''হরষপুর''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == হরষপুর ইউনিয়নের আয়তন ৭০৮৩ একর (২৮.৬৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে হরষপুর ইউনিয়নের মোট জনসংখ্যা ৩৩,৪৫২ জন। মোট পরিবার ৬,৪০২টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১১৬৭ জন। == ইতিহাস == == অবস্থান সীমানা == বিজয়নগর উপজেলার উত্তর-পূর্বাংশে হরষপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে পাহাড়পুর ইউনিয়ন; পশ্চিমে ইছাপুরা ইউনিয়ন, চান্দুরা ইউনিয়ন বুধন্তি ইউনিয়ন; উত্তরে বুধন্তি ইউনিয়ন হবিগঞ্জ জেলার মাধবপুর উপজেলার আদাঐর ইউনিয়ন এবং পূর্বে হবিগঞ্জ জেলার মাধবপুর উপজেলার বহরা ইউনিয়ন, চৌমুহনী ইউনিয়ন ধর্মঘর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হরষপুর ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
