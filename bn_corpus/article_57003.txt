'''বারগাঁও''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সোনাইমুড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাইমুড়ি উপজেলার পূর্বাংশে বারগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চাষীরহাট ইউনিয়ন, উত্তর-পশ্চিমে সোনাইমুড়ি পৌরসভা, পশ্চিমে বজরা ইউনিয়ন, দক্ষিণে নাটেশ্বর ইউনিয়ন, দক্ষিণ-পূর্বে অম্বরনগর ইউনিয়ন এবং উত্তর-পূর্বে সেনবাগ উপজেলার কেশারপাড় ইউনিয়ন ছাতারপাইয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বারগাঁও ইউনিয়ন সোনাইমুড়ি উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাইমুড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == বারগাাঁও ইউনিয়নের প্রধান হাট-বাজার হল কাশিপুর বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাইমুড়ি উপজেলা সোনাইমুড়ি থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
