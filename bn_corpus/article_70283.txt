'''মহাকালি ইউনিয়ন''' বাংলাদেশের মুন্সিগঞ্জ জেলার অন্তর্গত মুন্সীগঞ্জ সদরের একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == মহাকালি ইউনিয়নের মোট আয়তন ২৩৮৪ একর। == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী মহাকালি ইউনিয়নের মোট জনসংখ্যা ১৮৯০৯ জন। এদের মধ্যে ৯৪৯৫ জন পুরূষ এবং ৯৪১৪ জন মহিলা। == ইতিহাস == পাল রাজবংশের প্রতিষ্ঠাতা গোপাল পাল থেকে শুরু করে একাদশ শতাব্দী পর্যন্ত পালবংশ বিক্রমপুর শাসন করে। পরবর্তীতে সেন বংশোদ্ভূত বিক্রম সেনই বিক্রমপুর নগরের স্থপতি। ঐতিহাসিকরা আরও বলেন, মহাকালী প্রতিষ্ঠার পর পরই বিক্রমপুর পরগনার নামের অস্তিত্ব পাওয়া যায়। == শিক্ষা == শিক্ষার হার ৫৬.৮১% == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == দশনীয় স্থান == চোধুরী বাজার মঠ এটি শত বছরের পরানো। এই মঠটি বিভিন্ন জায়গা থেকে লোকজন দেখতে আসে। == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র == বিভাগের উপজেলা
