'''উত্তর ধর্মপুর উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার ধর্মপুর ইউনিয়নের উত্তর ধর্মপুর গ্রামে অবস্থিত। == ইতিহাস == ১৯৬৮ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == পরিচালনা পরিষদ == == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব মোহাম্মদ বেলাল উদ্দীন। == অবকাঠামো == বিদ্যালয়ে দুইটি একতলা ভবন রয়েছে। == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৮৯%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
