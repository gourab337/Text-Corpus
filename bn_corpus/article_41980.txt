'''বলধারা ইউনিয়ন''' ঢাকা বিভাগের মানিকগঞ্জ জেলার অন্তর্গত সিঙ্গাইর উপজেলার একটি ইউনিয়ন। == অবস্থান আয়তন == == ইতিহাস == == ভৌগোলিক উপাত্ত == ইউনিয়নের উপর দিয়ে নূরানী গঙ্গা নামে একটি নদী বয়ে গেছে। এটিই ইউনিয়নের একমাত্র নদী। == প্রশাসনিক উপাত্ত == == জনসংখ্যার উপাত্ত == == শিক্ষা == == কৃষি == == অর্থনীতি == == যোগাযোগ == ==দর্শনীয় স্থান== এই ইউনিয়নের রফিকনগর গ্রামে রয়েছে ভাষা শহিদ রফিক উদ্দিন ‍সৃতি যাদুঘর গ্রন্থাগার। == কৃতী ব্যক্তিত্ব == রফিকউদ্দিন আহমদ (ভাষা আন্দোলনে শহীদ) খান আতাউর রহমান (সাংস্কৃতিক ব্যক্তিত্ব) == তথ্যসূত্র == উপজেলার ইউনিয়ন
