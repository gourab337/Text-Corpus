'''রায়পুর পৌরসভা''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == অবস্থান সীমানা == রায়পুর উপজেলার মধ্যাংশে রায়পুর পৌরসভার অবস্থান। পৌরসভার উত্তরে চর পাতা ইউনিয়ন, পূর্বে কেরোয়া ইউনিয়ন, দক্ষিণে বামনী ইউনিয়ন সোনাপুর ইউনিয়ন এবং পশ্চিমে চর মোহনা ইউনিয়ন রায়পুর ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == রায়পুর পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম রায়পুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == যোগাযোগ ব্যবস্থা == == জনসংখ্যা == == শিক্ষা == == অর্থনীতি == == জনপ্রতিনিধি == == আরও দেখুন == রায়পুর উপজেলা রায়পুর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
