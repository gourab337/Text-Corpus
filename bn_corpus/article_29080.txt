বর্তমানে বাংলাদেশে সর্বমোট ১২টি সিটি কর্পোরেশন রয়েছে। == সাবেক সিটি কর্পোরেশন == ঢাকা সিটি কর্পোরেশন (DCC) == বর্তমান সিটি কর্পোরেশন == বিভাগ সিটি কর্পোরেশন ঢাকা বিভাগ ঢাকা উত্তর সিটি কর্পোরেশন (DNCC) ঢাকা দক্ষিণ সিটি কর্পোরেশন (DSCC) গাজীপুর সিটি কর্পোরেশন (GCC) নারায়ণগঞ্জ সিটি কর্পোরেশন (NCC) চট্টগ্রাম বিভাগ চট্টগ্রাম সিটি কর্পোরেশন (চসিক) কুমিল্লা সিটি কর্পোরেশন (COCC) রাজশাহী বিভাগ রাজশাহী সিটি কর্পোরেশন (RCC) খুলনা বিভাগ খুলনা সিটি কর্পোরেশন (কেসিসি) সিলেট বিভাগ সিলেট সিটি কর্পোরেশন (SCC) বরিশাল বিভাগ বরিশাল সিটি কর্পোরেশন (BCC) রংপুর বিভাগ রংপুর সিটি কর্পোরেশন (RACC) ময়মনসিংহ বিভাগ ময়মনসিংহ সিটি কর্পোরেশন (MCC) ==প্রস্তাবিত সিটি কর্পোরেশনসমূহ== বিভাগ সিটি কর্পোরেশন ঢাকা বিভাগ #ফরিদপুর সিটি কর্পোরেশন (FCC) #টাঙ্গাইল সিটি কর্পোরেশন (TCC) চট্টগ্রাম বিভাগ কক্সবাজার সিটি কর্পোরেশন (CBCC) রাজশাহী বিভাগ #বগুড়া সিটি কর্পোরেশন (BOCC) খুলনা বিভাগ #যশোর সিটি কর্পোরেশন(JCC) #কুষ্টিয়া সিটি কর্পোরেশন (KuCC) রংপুর বিভাগ #দিনাজপুর সিটি কর্পোরেশন (DCC) ময়মনসিংহ বিভাগ #জামালপুর সিটি কর্পোরেশন (JCC) == আরও দেখুন == বাংলাদেশের পৌরসভাসমূহের তালিকা বাংলাদেশের শহরের তালিকা পৌরসভা বিভিন্ন বিষয়ভিত্তিক তালিকা
