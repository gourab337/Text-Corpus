'''উজিরপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চৌদ্দগ্রাম উপজেলার একটি ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == চৌদ্দগ্রাম উপজেলার উত্তর-পূর্বাংশে উজিরপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে কালিকাপুর ইউনিয়ন, পশ্চিমে কাশিনগর ইউনিয়ন, উত্তরে কুমিল্লা সদর দক্ষিণ উপজেলার জোড়কানন পূর্ব ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == উজিরপুর ইউনিয়ন চৌদ্দগ্রাম উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চৌদ্দগ্রাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৯নং নির্বাচনী এলাকা কুমিল্লা-১১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চৌদ্দগ্রাম উপজেলা চৌদ্দগ্রাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
