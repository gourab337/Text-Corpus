বাংলাদেশের মুন্সিগঞ্জ জেলার অন্তর্গত একটি উপজেলা। == অবস্থান == এই উপজেলার উত্তরে কেরানীগঞ্জ উপজেলা ফতুল্লা উপজেলা, দক্ষিণে শ্রীনগর উপজেলা লৌহজং উপজেলা, পশ্চিমে-ঢাকা জেলার নবাবগঞ্জ উপজেলা, পূর্বে- টংগিবাড়ী উপজেলা মুন্সিগঞ্জ সদর উপজেলা। == প্রশাসনিক এলাকা== সিরাজদিখান উপজেলায় মোট ইউনিয়নের সংখ্যা ১৪ টি। চিত্রকোট ইউনিয়ন, শেখর নগর ইউনিয়ন, রাজানগর ইউনিয়ন, কেয়াইন ইউনিয়ন, ভাসাইল ইউনিয়ন, রশুনিয়া ইউনিয়ন, লতব্দি ইউনিয়ন, বালুচর ইউনিয়ন, ইছাপুর ইউনিয়ন, বয়রাগাদি ইউনিয়ন, মালখানগর ইউনিয়ন, মধ্যপাড়া ইউনিয়ন, জৈনসার ইউনিয়ন, কোলা ইউনিয়ন == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == বুদ্ধদেব বসু কবি। সাদেক হোসেন খোকা রাজনীতিবিদ। *রায় বাহাদুর শ্রীনাথ রায় -জমিদার। ফ্লাইট লেফটেন্যান্ট (অবঃ) জামাল উদ্দীন চৌধুরী -সাব-সেক্টর == বিবিধ == == আরও দেখুন == মুন্সিগঞ্জ জেলা; ঢাকা বিভাগ; বাংলাদেশের উপজেলাসমূহ। == তথ্যসূত্র == ==বহিঃসংযোগ== উপজেলা জেলার উপজেলা বিভাগের উপজেলা
