বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কবিরহাট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কবিরহাট উপজেলার নরোত্তমপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বাটইয়া ইউনিয়ন; দক্ষিণে কবিরহাট পৌরসভা সুন্দলপুর ইউনিয়ন; পশ্চিমে নোয়াখালী সদর উপজেলার নিয়াজপুর ইউনিয়ন এবং উত্তরে নোয়াখালী সদর উপজেলার নিয়াজপুর ইউনিয়ন, বেগমগঞ্জ উপজেলার কাদিরপুর ইউনিয়ন সেনবাগ উপজেলার নবীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নরোত্তমপুর ইউনিয়ন কবিরহাট উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কবিরহাট থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কবিরহাট উপজেলা কবিরহাট থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
