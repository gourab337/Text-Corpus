'''প্রকৌশলী দিবস''' বিভিন্ন দেশে পালন করা হয়। == প্রকৌশলী দিবস (আর্জেন্টিনা) == আর্জেন্টিনায় প্রকৌশল দিবস পালন করা হয় ১৬ জুন। লক্ষণীয় "আর্জেন্টিনার প্রকৌশল দিবস" কিন্তু ৮জুন,লুইস অগাস্ট হার্গো নামে এক ব্যক্তি ১৮৭০ সালে আর্জেন্টিনার প্রথম প্রকৌশলী হবার জন্য দিবসটি পালন করা হয় == প্রকৌশলী দিবস (কলম্বিয়া) == কলম্বিয়ায় প্রকৌশল দিবস পালন করা হয় আগস্ট। == প্রকৌশলী দিবস (ভারত) == ভারতে প্রকৌশল দিবস পালন করা হয় ১৫ সেপ্টেম্বর। == প্রকৌশলী দিবস (ইরান) == ইরানে প্রকৌশল দিবস পালন করা হয় ২৪ ফেব্রুয়ারি। == প্রকৌশলী দিবস (মেক্সিকো) == মেক্সিকোতে প্রকৌশল দিবস পালন করা হয় জুলাই। == প্রকৌশল দিবস (ভেনিজুয়েলা) == ভেনিজুয়েলাতে প্রকৌশল দিবস পালন করা হয় ২৮ অক্টোবর। == প্রকৌশলী দিবস (পানামা) == পানামাতে প্রকৌশল দিবস পালন করা হয় ২৬ জানুয়ারী। == প্রকৌশলী দিবস (ইতালি) == ইতালীতে প্রকৌশল দিবস পালন করা হয় ১৫ জুন। == প্রকৌশল দিবস (তাইওয়ান) == তাইওয়ানে প্রকৌশল দিবস পালন করা হয় জুন। == প্রকৌশল দিবস (যুক্তরাষ্ট্র) == যুক্তরাষ্ট্রে প্রকৌশল দিবস পালন করা হয় জুন। == প্রকৌশলী দিবস (পেরু) == পেরুতে প্রকৌশল দিবস পালন করা হয় জুন। == প্রকৌশলী দিবস (চিলি) == চিলিতে প্রকৌশল দিবস পালন করা হয় ১৪ মে। == প্রকৌশলী দিবস (নেপাল) == নেপালে প্রকৌশল দিবস পালন করা হয় ১৮/১৯ জুলাই(৩ শ্রাবণ)। == তথ্যসূত্র == == বহি:সংযোগ ==
