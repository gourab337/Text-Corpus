'''দরবস্ত ইউনিয়ন''' বাংলাদেশের সিলেট জেলার জৈন্তাপুর উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' '''শিক্ষা প্রতিষ্ঠাণ''' জৈন্তিয়া ডিগ্রী কলেজ সেন্ট্রাল জৈন্তা হাই স্কুল == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' বাহারুল আলম বাহার চেয়ারম্যানগণের তালিকা == আরও দেখুন == == তথ্যসূত্র == উপজেলার ইউনিয়ন
