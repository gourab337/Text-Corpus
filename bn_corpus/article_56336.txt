'''ফাজিলপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার ফাজিলপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে ফরহাদনগর ইউনিয়ন; পশ্চিমে ছনুয়া ইউনিয়ন; উত্তর-পশ্চিমে মোটবী ইউনিয়ন; উত্তর-পূর্বে ছাগলনাইয়া উপজেলার রাধানগর ইউনিয়ন এবং পূর্বে মুহুরী নদী, ছাগলনাইয়া উপজেলার শুভপুর ইউনিয়ন ঘোপাল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ফাজিলপুর ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
