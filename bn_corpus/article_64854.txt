'''পূর্ব পৈলনপুর ইউনিয়ন''' বাংলাদেশের সিলেট জেলার বালাগঞ্জ উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == বালাগঞ্জ উপজেলা সদর হতে দুরত্ব প্রায় ১২কিমিঃ। == ইতিহাস == পৈলনপুর ইউনিয়ন নামে একটি ইউনিয়ন ছিল পরবর্তীতে পৈলনপুর ইউনিয়নকে দুভাগে ভাগ করা হয় একটি পশ্চিম পৈলনপুর ইউনিয়ন অপরটি পূর্ব পৈলনপুর ইউনিয়ন। পূর্ব পৈলনপুর ইউনিয়নের গালিমপুর গ্রামে ১৯৭১সালে মুক্তি যুদ্বের সময় পাক হানাদার বাহিনীর নির্মম আক্রমনে অসখ্য মানুষ প্রান হারায়। == আয়তন জনসংখ্যা == '''আয়তন:''' ৫০৯২.৪৫ একর। জনসংখ্যা: ১১৫২৯ জন। == গ্রাম সমূহ == ঐয়া, কিত্তেজালালপুর, পশ্চিম ইছাপুর, পূর্ব ইছাপুর, ফাজিলপুর, পৈলনপুর পশ্চিম পাড়া, পৈলনপুর পূর্ব পাড়া, ভাটপাড়া, গালিমপুর, জালালপুর, সাদেকপুর, রশিদপুর, হামছাপুর। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হারঃ-''' ৩৯.১৭% '''শিক্ষা প্রতিষ্ঠান-''' উচ্চ বিদ্যালয়ের সংখ্যা ১টি সরকারী প্রাথমিক বিদ্যালয় ৬টি বেসরকারী রেজি: প্রাথমিক বিদ্যালয় ২টি দাখিল মাদ্রাসা ১টি কওমী মাদ্রাসা ৪টি মসজিদ ১৮টি গালিমপুর ভৈরব মন্দির ১টি ঈদগাহ ৩টি কবরস্তান ৮টি শ্বসান ১টি == দর্শনীয় স্থান == কুশিয়ারা নদী স্মৃতি সৌধ, ১৯৭১ সালের মুক্তি যুদ্ধের শহীদদের স্বরনে নির্মিত। == খাল নদী == খাল নদীতে পরিবেস্টিত একটি ইউনিয়ন ইউনিয়নের দক্ষিন দিকে প্রবাহিত কুশিয়ারা নদী বষাকালে নদী == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান'''- মো: আব্দুল মতিন ====== পুরাতন চেয়ারম্যান বৃন্দ ====== '''ক্রম নং''' নাম''' সময়কাল ০১ মোঃ রইছ উদ্দিন ২৪ ফেব্রুয়ারি ১৯৭৭ মার্চ ১৯৮৪ ০২ আব্দুল কাহের (আদর মিয়া) মার্চ ১৯৮৪ ৩০ এপ্রিল ১৯৯২ ০৩ আব্দুর রহমান ৩০ এপ্রিল ১৯৯২ ২৩ এপ্রিল ২০০৩ ০৪ মখলূছ মিয়া ২৩ এপ্রিল ২০০৩ ১০ আগস্ট ২০১১ ০৫ সান উল্ল্যা ১০ আগস্ট ২০১১ ১৭ আগস্ট ২০১৬ ০৬ '''মো: আব্দুল মতিন''' ১৭ আগস্ট ২০১৬ ''পদাধিকারী'' == আরও দেখুন == বালাগঞ্জ ইউনিয়ন পশ্চিম পৈলনপুর ইউনিয়ন বালাগঞ্জ উপজেলা == তথ্যসূত্র == উপজেলার ইউনিয়ন == বহিঃসংযোগ == বালাগঞ্জ উপজেলা ''-'' বাংলাপিডিয়ায় প্রাতিষ্ঠানিক ওয়েবসাইট বাংলাদেশ জাতীয় তথ্য বাতায়ন।
