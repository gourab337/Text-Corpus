বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == তিতাস উপজেলার ভিটিকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কলাকান্দি ইউনিয়ন, পশ্চিমে বলরামপুর ইউনিয়ন নারান্দিয়া ইউনিয়ন, দক্ষিণে দাউদকান্দি উপজেলার ইলিয়টগঞ্জ উত্তর ইউনিয়ন এবং পূর্বে মুরাদনগর উপজেলার পাহাড়পুর ইউনিয়ন ছালিয়াকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভিটিকান্দি ইউনিয়ন তিতাস উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
