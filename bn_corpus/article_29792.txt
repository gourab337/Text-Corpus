'''গাজীপুর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাইমচর উপজেলার একটি ইউনিয়ন। == আয়তন == গাজীপুর ইউনিয়নের আয়তন ২.৫ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের আদমশুমারী অনুযায়ী গাজীপুর ইউনিয়নের জনসংখ্যা ৬,১৮০ জন। == অবস্থান সীমানা == হাইমচর উপজেলার গাজীপুর ইউনিয়নের অবস্থান। উপজেলা সদর থেকে ইউনিয়নের দূরত্ব ২০ কিলোমিটার। ইউনিয়নের দক্ষিণে নীলকমল ইউনিয়ন আলগী দুর্গাপুর দক্ষিণ ইউনিয়ন, পূর্বে আলগী দুর্গাপুর উত্তর ইউনিয়ন, উত্তরে চাঁদপুর সদর উপজেলার হানারচর ইউনিয়ন এবং পশ্চিমে মেঘনা নদী চাঁদপুর সদর উপজেলার ইব্রাহিমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গাজীপুর ইউনিয়ন হাইমচর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাইমচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। এটি ৯টি মৌজায় বিভক্ত। ইউনিয়নের গ্রামগুলো হল: গাজীপুর মনিপুর গাজীপুর কুতুবপুর গাজীপুর বাজাপ্তি == শিক্ষা ব্যবস্থা == গাজীপুর ইউনিয়নের সাক্ষরতার হার ৪৯%। ইউনিয়নে ২টি মাদ্রাসা, ১টি প্রাথমিক বিদ্যালয় ২টি আনন্দ স্কুল রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == উপজেলা সদর থেকে গাজীপুর ইউনিয়নে যোগাযোগের প্রধান মাধ্যম নৌ-পরিবহন বা ইঞ্জিন চালিত নৌকা। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == মেঘনা নদী == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোহাম্মদ হাবিবুর রহমান গাজী তালিকা চেয়ারম্যানের নাম দায়িত্বকাল মোহাম্মদ পাটোয়ারী ১৯৭৪-১৯৭৯ ছিদ্দিকুর রহমান ১৯৭৯-১৯৮৫ মুনছুর আলী গাজী ১৯৮৫-১৯৯০ হাফেজ আহম্মদ ১৯৯২-১৯৯৫ শাহজাহান মিয়া ১৯৯৬-২০০৮ মোহাম্মদ ইসমাঈল গাজী ২০০৮-বর্তমান == আরও দেখুন == হাইমচর উপজেলা হাইমচর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
