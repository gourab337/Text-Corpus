'''দাউদকান্দি উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দাউদকান্দি উপজেলার দাউদকান্দি উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে দাউদকান্দি পৌরসভা, পূর্বে উত্তরে তিতাস উপজেলার মজিদপুর ইউনিয়ন, উত্তর-পশ্চিমে মেঘনা উপজেলার গোবিন্দপুর ইউনিয়ন লুটেরচর ইউনিয়ন এবং পশ্চিমে মেঘনা নদী মুন্সিগঞ্জ জেলার গজারিয়া উপজেলার বাউশিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দাউদকান্দি উত্তর ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
