'''লক্ষ্মীপুর মডেল''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত চাঁদপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাঁদপুর সদর উপজেলার দক্ষিণাংশে লক্ষ্মীপুর মডেল ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে হানারচর ইউনিয়ন চান্দ্রা ইউনিয়ন, পূর্বে বালিয়া ইউনিয়ন, উত্তরে চাঁদপুর পৌরসভা, উত্তর-পশ্চিমে ইব্রাহিমপুর ইউনিয়ন এবং পশ্চিমে মেঘনা নদী শরিয়তপুর জেলার ভেদরগঞ্জ উপজেলার চর সেনসাস ইউনিয়ন অবস্থিত। == ইতিহাস == লক্ষ্মীপুর মডেল ইউনিয়ন পূর্বে চাঁদপুর সদর উপজেলার ১০নং সাখুয়া ইউনিয়ন নামে পরিচিত ছিল। ইউনিয়নকে মডেল ইউনিয়ন ঘোষণা করা হলে ইউনিয়নের নাম পরিবর্তন করে ১০নং লক্ষ্মীপুর মডেল ইউনিয়ন পরিষদ রাখা হয়। == প্রশাসনিক কাঠামো == লক্ষ্মীপুর মডেল ইউনিয়ন চাঁদপুর সদর উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাঁদপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সাখুয়া ইউনিয়ন চাঁদপুর সদর উপজেলা চাঁদপুর সদর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
