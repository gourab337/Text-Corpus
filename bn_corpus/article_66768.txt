বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বাবুটিপাড়া ইউনিয়ন পূর্বে ২১নং পাহাড়পুর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == মুরাদনগর উপজেলার সর্ব-দক্ষিণে বাবুটিপাড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পাহাড়পুর ইউনিয়ন, পূর্বে দেবিদ্বার উপজেলার রাজামেহার ইউনিয়ন ভানী ইউনিয়ন, দক্ষিণে চান্দিনা উপজেলার বাতাঘাসী ইউনিয়ন এবং পশ্চিমে দাউদকান্দি উপজেলার ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন ইলিয়টগঞ্জ উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাবুটিপাড়া ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ২১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মুরাদনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা মুরাদনগর থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
