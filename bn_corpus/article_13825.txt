'''বড়নাল''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মাটিরাঙ্গা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মাটিরাঙ্গা উপজেলার মধ্যাংশে বড়নাল ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে আমতলী ইউনিয়ন, পূর্বে পানছড়ি উপজেলার উল্টাছড়ি ইউনিয়ন, উত্তরে তবলছড়ি ইউনিয়ন এবং পশ্চিমে তবলছড়ি ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বড়নাল ইউনিয়ন মাটিরাঙ্গা উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মাটিরাঙ্গা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;কলেজ তবলছড়ি গ্রীন হিল কলেজ ;মাধ্যমিক বিদ্যালয় তবলছড়ি কদমতলী উচ্চ বিদ্যালয় বড়নাল মুক্তিযোদ্ধা উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == সড়ক পথই যোগাযোগের একমাত্র মাধ্যম। পূর্বে উক্ত ইউনিয়নের পশ্চিমে অবস্থিত ফেনী নদী দিয়ে চলাচল থাকলেও সড়ক পথের উন্নয়ন নিরাপত্তার বিষয় সহ সামগ্রিক অসুবিধার কারণে এই জলপথ এখন আর ব্যবহৃত হয়না। == খাল নদী == তৈলাফাং, ফেনী নদী == হাট-বাজার == ডাকবাংলা বাজার, বড়নাল বাজার == দর্শনীয় স্থান == বড়গ্রামের ঝর্ণা বড়কুম == জনপ্রতিনিধি == == আরও দেখুন == মাটিরাঙ্গা উপজেলা মাটিরাঙ্গা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
