'''টামটা দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == টামটা দক্ষিণ ইউনিয়নের আয়তন ১৫.৮৪ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের আদমশুমারী অনুযায়ী টামটা দক্ষিণ ইউনিয়নের জনসংখ্যা ২৯,২৫৯ জন। == অবস্থান সীমানা == শাহরাস্তি উপজেলার টামটা দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে টামটা উত্তর ইউনিয়ন, পূর্বে শাহরাস্তি পৌরসভা মেহের দক্ষিণ ইউনিয়ন, দক্ষিণে হাজীগঞ্জ উপজেলার গন্ধর্ব্যপুর উত্তর ইউনিয়ন এবং পশ্চিমে হাজীগঞ্জ উপজেলার বড়কুল পূর্ব ইউনিয়ন হাজীগঞ্জ পৌরসভা অবস্থিত। == ইতিহাস == == প্রশাসনিক কাঠামো == টামটা দক্ষিণ ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == টামটা দক্ষিণ ইউনিয়নে যোগাযোগের প্রধান সড়ক সড়ক। সব ধরনের যানবাহনে যোগাযোগ করা যায়। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
