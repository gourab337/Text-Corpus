'''মাজেদা হক উচ্চ বিদ্যালয়''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার মায়ানী ইউনিয়নে বিদ্যালয়টি অবস্থিত। == ইতিহাস == অত্র সমাজের গণ্যমান্য ব্যক্তিবর্গের প্রচেষ্টায় ১৯৮৬ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। ১৯৮৮ সালে অষ্টম শ্রেণীর স্বীকৃতি এবং ১৯৯২ সালে এসএসসি পরীক্ষা দেয়ার অনুমতি লাভ করে। ১৯৯৪ সালে বিদ্যালয়টি এমপিও ভুক্ত হয়। == পরিচালনা ব্যবস্থা == == == বিদ্যালয়ের প্রধান শিক্ষক জনাব দিদারুল আলম। বর্তমানে দুই শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == অবকাঠামো == == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা কার্যক্রম রয়েছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৮৮.৫৭%। == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
