'''বামনী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রায়পুর উপজেলার একটি ইউনিয়ন। == আয়তন == বামনী ইউনিয়নের আয়তন ১২ বর্গ কিলোমিটার। == জনসংখ্যা == বামনী ইউনিয়নের জনসংখ্যা ৪০,৮৪৪ জন। == অবস্থান সীমানা == রায়পুর উপজেলার পূর্বাংশে বামনী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কেরোয়া ইউনিয়ন রায়পুর পৌরসভা, পশ্চিমে সোনাপুর ইউনিয়ন, দক্ষিণে লক্ষ্মীপুর সদর উপজেলার দালাল বাজার ইউনিয়ন এবং পূর্বে লক্ষ্মীপুর সদর উপজেলার দক্ষিণ হামছাদী ইউনিয়ন উত্তর হামছাদী ইউনিয়ন অবস্থিত। == ইতিহাস == বামনী পূর্বে ৪নং সোনাপুর ৬নং কেরোয়া ‌ইউনিয়নের অন্তর্ভুক্ত ছিল। ১৯৩৭ সালে ৭নং বামনী আলাদা ইউনিয়ন হিসেবে প্রতিষ্ঠা লাভ করে। == প্রশাসনিক কাঠামো == বামনী ইউনিয়ন রায়পুর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রায়পুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। ইউনিয়নের গ্রামগুলো হল: বামনী কলাকোপা সাইচা সাগরদী কাঞ্চনপুর শিবপুর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == বামনী ইউনিয়নের প্রধান হাট-বাজার হল কাজির দীঘির পাড় বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == মোহাম্মদউল্লাহ সাবেক রাষ্ট্রপতি। কর্নেল নুরুননবী প্রতিষ্ঠাতা, মাইলস্টোন কলেজ। শহীদ ডাঃ মিলন স্বৈরশাসন বিরোধী আন্দোলনে শহীদ। ডাঃ শামসুল আলম মিলন স্বৈরশাসন বিরোধী আন্দোলনে শহীদ। == জনপ্রতিনিধি == == আরও দেখুন == রায়পুর উপজেলা রায়পুর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
