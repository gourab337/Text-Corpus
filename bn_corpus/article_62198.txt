'''শাহ মাহমুদপুর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত চাঁদপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাঁদপুর সদর উপজেলার পূর্বাংশে শাহ মাহমুদপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে আশিকাটি ইউনিয়ন মৈশাদী ইউনিয়ন, দক্ষিণে ফরিদগঞ্জ উপজেলার বালিথুবা পশ্চিম ইউনিয়ন, পূর্বে রামপুর ইউনিয়ন হাজীগঞ্জ উপজেলার বাকিলা ইউনিয়ন এবং উত্তরে মতলব দক্ষিণ উপজেলার উপাদী দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শাহ মাহমুদপুর ইউনিয়ন চাঁদপুর সদর উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাঁদপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == চাঁদপুর সদর উপজেলা চাঁদপুর সদর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
