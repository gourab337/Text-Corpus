'''পাহাড়পুর''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == পাহাড়পুর ইউনিয়নের আয়তন ৮১৬৮ একর (৩৩.০৫ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে পাহাড়পুর ইউনিয়নের মোট জনসংখ্যা ৩৫,৬৯৩ জন। মোট পরিবার ৬,৯৮৫টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১০৮০ জন। == ইতিহাস == == অবস্থান সীমানা == বিজয়নগর উপজেলার পূর্বাংশে পাহাড়পুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে বিষ্ণুপুর ইউনিয়ন; পশ্চিমে বিষ্ণুপুর ইউনিয়ন, চম্পকনগর ইউনিয়ন ইছাপুরা ইউনিয়ন; উত্তরে হরষপুর ইউনিয়ন এবং পূর্বে হবিগঞ্জ জেলার মাধবপুর উপজেলার ধর্মঘর ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == পাহাড়পুর ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
