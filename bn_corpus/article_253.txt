'''ডোমিনিকান প্রজাতন্ত্র''' রেপুব্লিকা দোমিনিকানা IPA re'puβlika domini'kana) ক্যারিবিয়ান দ্বীপপুঞ্জে অবস্থিত একটি রাষ্ট্র। এটি হিস্পানিওলা দ্বীপের দুই-তৃতীয়াংশ জুড়ে অবস্থিত। এর রাজধানীর নাম সান্টো ডোমিংগো। == ইতিহাস == == রাজনীতি == == প্রশাসনিক অঞ্চলসমূহ == == ভূগোল == == অর্থনীতি == dominikan er ortoniti khub yi valo == জনসংখ্যা == == সংস্কৃতি == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == দ্বীপপুঞ্জ আমেরিকার রাষ্ট্র
