'''সমিতিরহাট উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার সমিতিরহাট ইউনিয়নের নিশ্চিন্তাপুর গ্রামে অবস্থিত। == ইতিহাস == ১৯৬৫ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == পরিচালনা পরিষদ == বিদ্যালয় পরিচালনার জন্য ১০ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব মোহাম্মদ মঈন উদ্দীন। এছাড়া আরো জন শিক্ষক-শিক্ষিকা বিদ্যালয়ে কর্মরত আছেন। == অবকাঠামো == বিদ্যালয়ে তিনটি একতলা ভবন রয়েছে। == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। বর্তমানে ছয় শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
