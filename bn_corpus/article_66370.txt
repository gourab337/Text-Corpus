'''লাকসাম পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লাকসাম উপজেলার মধ্যাংশে লাকসাম পৌরসভার অবস্থান। পৌরসভার পশ্চিমে কান্দিরপাড় ইউনিয়ন, দক্ষিণ-পশ্চিমে গোবিন্দপুর ইউনিয়ন, দক্ষিণে উত্তরদা ইউনিয়ন আজগরা ইউনিয়ন, পূর্বে লাকসাম পূর্ব ইউনিয়ন লালমাই উপজেলার পেরুল দক্ষিণ ইউনিয়ন এবং উত্তরে লালমাই উপজেলার পেরুল দক্ষিণ ইউনিয়ন অবস্থিত। == ইতিহাস == == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == লাকসাম পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
