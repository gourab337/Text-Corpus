'''নিলখী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == হোমনা উপজেলার মধ্যাংশে নিলখী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ঘাগুটিয়া ইউনিয়ন দুলালপুর ইউনিয়ন, পূর্বে ঘারমোড়া ইউনিয়ন, দক্ষিণে ঘারমোড়া ইউনিয়ন মাথাভাঙ্গা ইউনিয়ন এবং পশ্চিমে মাথাভাঙ্গা ইউনিয়ন হোমনা পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == নিলখী ইউনিয়ন হোমনা উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
