'''পাপুয়া নিউগিনি''' (তোক পিসিন Papua Niugini, ইংরেজি Papua New Guinea) প্রশান্ত মহাসাগরের একটি দ্বীপ রাষ্ট্র। এটি ওশেনিয়া অঞ্চলে অবস্থিত। == ইতিহাস == প্রথম বিশ্বযুদ্ধের সময় জার্মানীর নিকট থেকে অস্ট্রেলিয়া জার্মান নিউ গিনি অধিকার করে। ১৯৭৫ সালে এটি অস্ট্রেলিয়ার নিকট থেকে স্বাধীনতা লাভ করে। == রাজনীতি == == প্রশাসনিক অঞ্চলসমূহ == == ভূগোল == == অর্থনীতি == == জনসংখ্যা == == সংস্কৃতি == == শিক্ষাব্যবস্থা == == বিজ্ঞান প্রযুক্তি == == পরিবহন ব্যবস্থা == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == রাষ্ট্র
