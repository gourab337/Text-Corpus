'''দারোরা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == দারোরা ইউনিয়নের আয়তন ১২ বর্গ কিলোমিটার। == জনসংখ্যা == দারোরা ইউনিয়নের জনসংখ্যা ১৫,৯৩০ জন। == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার দক্ষিণ-মধ্যাংশে দারোরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে জাহাপুর ইউনিয়ন, পূর্বে ধামঘর ইউনিয়ন, দক্ষিণে পাহাড়পুর ইউনিয়ন এবং পশ্চিমে ছালিয়াকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দারোরা ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ১৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মুরাদনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। ইউনিয়নের গ্রামগুলো হল: দারোরা উত্তর পদুয়া দীঘিরপাড় কেমতলি দক্ষিণ পুস্কুনিরপাড় পালাসুতা নোয়াকান্দি হাটেরপাড় কাজিয়াতল উত্তর পুকুরপাড় রতননগর পুটিয়াজুরি কাজিয়াতল দক্ষিণ শব্দলকান্দা == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == দারোরা ইউনিয়নের প্রধান হাট-বাজার হল দারোরা বাজার। == দর্শনীয় স্থান == আশেক হাজী (রহ.) এর মাজার; এই মাজারে প্রতি বছর ৮ই পৌষ ওরশ অনুষ্ঠিত হয় যা হাজারো মানুষের মিলন মেলায় রূপ নেয়। কথিত আছে এই বিখ্যাত ছুফি সাধক পায়ে হেটে পবিত্র হজ্জ পালন করেছিলেন তাই এখনো তার বাড়ীটি হাজী বাড়ী নামেই সুপরিচিত। == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা মুরাদনগর থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
