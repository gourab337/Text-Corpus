'''ভবানীপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == ভবানীপুর ইউনিয়ন পূর্বে ভবানীপুর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার উত্তর-পূর্বাংশে ভবানীপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে শিলমুড়ি উত্তর ইউনিয়ন, পশ্চিমে বরুড়া পৌরসভা, উত্তরে আগানগর ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার কালিরবাজার ইউনিয়ন এবং পূর্বে কুমিল্লা সদর দক্ষিণ উপজেলার বিজয়পুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভবানীপুর ইউনিয়ন বরুড়া উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == মাধ্যমিক বিদ্যালয় বাতাইছড়ি উচ্চ বিদ্যালয় মাদ্রাসা বাতাইছড়ি দাখিল মাদ্রাসা জালগাঁও সরকারি প্রাথমিক বিদ্যালয়। লক্ষ্মীপুর সঃ প্রাঃ বিদ্যালয়। ভবানীপুর সঃ প্রাঃ বিদ্যালয়। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ১/ বাতাইছড়ি নতুন বাজার (সুজাতনগর) ২/ বাতাইছড়ি পুরাতন বাজার ৩/ জালগাঁও ফকির মার্কেট ৪/ ভবানীপুর সুপার মার্কেট == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
