'''আড়াইসিধা''' বাংলাদেশের জেলার অন্তর্গত আশুগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == আড়াইসিধা ইউনিয়নের আয়তন ১৪৬৯ একর (৫.৯৪ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে আড়াইসিধা ইউনিয়নের মোট জনসংখ্যা ২০,৭২৭ জন। মোট পরিবার ৩,৭১৫টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৩৪৮৭ জন। == ইতিহাস == আড়াইসিধা ইউনিয়ন পূর্বে আড়াইসিধা দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == আশুগঞ্জ উপজেলার মধ্যাংশে আড়াইসিধা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আশুগঞ্জ সদর ইউনিয়ন, পশ্চিমে চর চারতলা ইউনিয়ন, দক্ষিণে লালপুর ইউনিয়ন শরীফপুর ইউনিয়ন এবং পূর্বে তারুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আড়াইসিধা ইউনিয়ন আশুগঞ্জ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আশুগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == আবদুল কাদির –– একুশে পদক এবং বাংলা একাডেমি সাহিত্য পুরস্কার প্রাপ্ত কবি, সাহিত্য সমালোচক এবং ছান্দসিক। == জনপ্রতিনিধি == == আরও দেখুন == আশুগঞ্জ উপজেলা আশুগঞ্জ থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
