বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মেঘনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == চালিভাঙ্গা ইউনিয়নের জনসংখ্যা ১১,০৫৮ জন। == ইতিহাস == চন্দনপুর ইউনিয়ন থেকে আলাদা করে ৭টি গ্রাম নিয়ে চালিভাঙ্গা ইউনিয়ন প্রতিষ্ঠিত হয়। == অবস্থান সীমানা == মেঘনা উপজেলার সর্ব-পশ্চিমে চালিভাঙ্গা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে মেঘনা নদী, বড়কান্দা ইউনিয়ন চন্দনপুর ইউনিয়ন; উত্তর-পূর্বে মেঘনা নদী নারায়ণগঞ্জ জেলার আড়াইহাজার উপজেলার কালাপাহাড়িয়া ইউনিয়ন; উত্তরে মেঘনা নদী নারায়ণগঞ্জ জেলার আড়াইহাজার উপজেলার খাগকান্দা ইউনিয়ন সোনারগাঁও উপজেলার বারদী ইউনিয়ন; পশ্চিমে মেঘনা নদী নারায়ণগঞ্জ জেলার সোনারগাঁও উপজেলার বৈদ্যেরবাজার ইউনিয়ন পিরোজপুর ইউনিয়ন এবং দক্ষিণে মেঘনা নদী মুন্সিগঞ্জ জেলার গজারিয়া উপজেলার বালুয়াকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চালিভাঙ্গা ইউনিয়ন মেঘনা উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মেঘনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। ইউনিয়নের গ্রামগুলো হল: বড়ইয়াকান্দি মৈশারচর রামপ্রসাদেরচর ফরাজিকান্দি নলচর চালিভাঙ্গা টিটিরচর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোহাম্মদ লতিফ সরকার == আরও দেখুন == মেঘনা উপজেলা মেঘনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
