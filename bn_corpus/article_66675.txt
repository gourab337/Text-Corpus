'''সাহেবাবাদ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত ব্রাহ্মণপাড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == ব্রাহ্মণপাড়া উপজেলার দক্ষিণাংশে সাহেবাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ব্রাহ্মণপাড়া সদর ইউনিয়ন, পশ্চিমে দুলালপুর ইউনিয়ন মালাপাড়া ইউনিয়ন, দক্ষিণে বুড়িচং উপজেলার পীরযাত্রাপুর ইউনিয়ন, দক্ষিণ-পূর্বে বুড়িচং উপজেলার বুড়িচং সদর ইউনিয়ন এবং পূর্বে বুড়িচং উপজেলার রাজাপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সাহেবাবাদ ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ব্রাহ্মণপাড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == সাহেবাবাদ ডিগ্রি কলেজ, সাহেবাবাদ ইসলামিয়া ফাজিল মাদ্রাসা, সাহেবাবাদ লতিফ ইসমাইল উচ্চ বিদ্যালয়, জিরুইন বহুমুখী উচ্চ বিদ্যালয়, টাকই উচ্চ বিদ্যালয়, টাটেরা মহিলা দাখিল মাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == দরিয়ারপাড় ঈদগাহ কুমিল্লা জেলার সর্ববৃহত ঈদগাহ। == জনপ্রতিনিধি == মরহুম ইদ্রিছ মিয়া, সাবেক চেয়ারম্যান গনি মিয়া, সাবেক চেয়ারম্যান মিজানুর রহমান খান, সাবেক চেয়ারম্যান ফরিদ উদ্দিন, সাবেক চেয়ারম্যান জসিম উদ্দিন নান্নু, সাবেক চেয়ারম্যান বর্তমান চেয়ারম্যান: মোস্তফা ছরোয়ার খান == কৃতি ব্যাক্তিত্ব == মরহুম ড.আলী নওয়াজ মরহুম ড.সিরাজুল ইসলাম আনম ইউসুফ ভুইয়া, মুখ্য সচিব মরহুম আবদুল খালেক,প্রথম পুলিশ মহাপরিদর্শক বাংলাদেশ ব্যারিস্টার আবদুল্লাহ আল মামুন মরহুম ইসমাইল ইঞ্জিনিয়ার == আরও দেখুন == ব্রাহ্মণপাড়া উপজেলা ব্রাহ্মণপাড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
