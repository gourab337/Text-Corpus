'''আমতা ইউনিয়ন''' বাংলাদেশের ঢাকা জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই ইউনিয়নটি ঢাকা বিভাগের ঢাকা জেলার অন্তর্গত ধামরাই উপজেলায় অবস্থিত। == ইতিহাস == == ভৌগোলিক উপাত্ত == == প্রশাসনিক উপাত্ত == == জনসংখ্যার উপাত্ত == শিক্ষার দিক থেকে আমতা ইউনিয়ন অনেকটা এগিয়ে আছে। == কৃষি == == অর্থনীতি == == যোগাযোগ == == কৃতী ব্যক্তিত্ব == সাধক ভবা পাগলা == আরও দেখুন == ==তথ্য সূত্র== == বহিঃসংযোগ == উপজেলার ইউনিয়ন
