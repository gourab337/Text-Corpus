'''আনন্দপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফুলগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফুলগাজী উপজেলার সর্ব-দক্ষিণে আনন্দপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মুন্সিরহাট ইউনিয়ন, পূর্বে জিএমহাট ইউনিয়ন, দক্ষিণ-পূর্বে ছাগলনাইয়া উপজেলার পাঠাননগর ইউনিয়ন, দক্ষিণ-পশ্চিমে ফেনী সদর উপজেলার কাজিরবাগ ইউনিয়ন এবং পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == আনন্দপুর ইউনিয়ন ফুলগাজী উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফুলগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == বাংলাদেশের রাজধানী ঢাকার সায়েদাবাদ বা বন্দরনগরী চট্টগ্রাম থেকে সরাসরি ফেনীর মহিপাল হয়ে ফুলগাজী পরশুরামের যেকোন বাহনে আনন্দপুর ইউনিয়নসহ যেকোন জায়গায় পৌঁছানো যায়। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফুলগাজী উপজেলা ফুলগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
