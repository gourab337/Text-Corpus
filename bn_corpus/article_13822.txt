'''কবাখালী''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত দীঘিনালা উপজেলার একটি ইউনিয়ন। == আয়তন == কবাখালী ইউনিয়নের আয়তন ১১,৫২০ একর (৪৬.৬২ বর্গ কিলোমিটার)। == জনসংখ্যা == == অবস্থান সীমানা == দীঘিনালা উপজেলার দক্ষিণাংশে কবাখালী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে দীঘিনালা ইউনিয়ন; পশ্চিমে বোয়ালখালী ইউনিয়ন; দক্ষিণে মেরুং ইউনিয়ন এবং পূর্বে রাঙ্গামাটি জেলার বাঘাইছড়ি উপজেলার রূপকারী ইউনিয়ন, বঙ্গলতলী ইউনিয়ন সাজেক ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কবাখালী ইউনিয়ন দীঘিনালা উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দীঘিনালা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == কবাখালী ইউনিয়নের মধ্য দিয়ে প্রবাহিত হচ্ছে মাইনী নদী। == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দীঘিনালা উপজেলা দীঘিনালা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
