'''মোগড়া''' বাংলাদেশের জেলার অন্তর্গত আখাউড়া উপজেলার একটি ইউনিয়ন। == আয়তন == মোগড়া ইউনিয়নের আয়তন ৪৫৭৮ একর (১৮.৫৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে মোগড়া ইউনিয়নের মোট জনসংখ্যা ২৭,৫৭১ জন। মোট পরিবার ৫,১৮৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৪৮৮ জন। == ইতিহাস == == অবস্থান সীমানা == আখাউড়া উপজেলার মধ্যাংশে মোগড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আখাউড়া দক্ষিণ ইউনিয়ন সদর উপজেলার বাসুদেব ইউনিয়ন, পশ্চিমে ধরখাড় ইউনিয়ন, দক্ষিণে মনিয়ন্দ ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == মোগড়া ইউনিয়ন আখাউড়া উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আখাউড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == আখাউড়া উপজেলা আখাউড়া থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
