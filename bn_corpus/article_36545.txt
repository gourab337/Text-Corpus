'''নবীগঞ্জ পৌরসভা''' বাংলাদেশের হবিগঞ্জ জেলার একটি স্থানীয় সরকার ব্যবস্থা সংগঠন। এই পৌরসভাটি বাংলাদেশের একটি ‘‘ক’’ শ্রেনীভূক্ত পৌরসভা। == অবস্থান আয়তন == নবীগঞ্জ পৌরসভাটি সিলেট বিভাগের হবিগঞ্জ জেলার নবীগঞ্জ উপজেলা সদরে অবস্থিত। মানচিত্রে এর অবস্থান উত্তর অক্ষাংশ ২৪‘-৩৩“ থেকে ২৪‘-৩৪“ পূর্ব দ্রাঘিমাংশ ৯১-৩০‘ থেকে ৯১-৩২‘। এর আয়তন ৯.০৭ বর্গ কিলোমিটার। == ইতিহাস == == ভৌগোলিক উপাত্ত == == শিল্প সংস্কৃতি == == প্রশাসনিক উপাত্ত == == নির্বাচিত জন-প্রতিনিধি == বিগত ২০১৫ সালের ৩০ ডিসেম্বর তারিখে অনুষ্ঠিত পৌরসভা নির্বাচনে বিএনপি সমর্থিত প্রার্থী ছাবির আহমদ চৌধুরী মেয়র পদে নির্বাচিত হয়েছেন। == জনসংখ্যা উপাত্ত == নবীগঞ্জ পৌরসভার মোট জনসংখ্যা ৩২,৫৮৬ জন। পুরুষ: ১৬,৮৫১ জন, নারী: ১৫,৭৩৫ জন। == স্বাস্থ্যসেবা == == শিক্ষা ব্যবস্থা == == কলেজ == নবীগঞ্জ পৌরসভায় দুটি কলেজ *নবীগঞ্জ সরকারি কলেজ *উইমেন্স আইডিয়াল কলেজ (মহিলা কলেজ) ==উচ্চ বিদ্যালয়== নবীগঞ্জ পৌরসভায় ১টি সরকারি ৪টি এমপি ভুক্ত ২টি নন এমপি ভুক্ত উচ্ছ বিদ্যালয় রয়েছে। *নবীগন্জ জেকে মডেল সরকারী উচ্চ বিদ্যালয় *হিরা মিয়া বালিকা উচ্ছ বিদ্যালয় *হোমল্যান্ড আইডিয়াল স্কুল *নবীগঞ্জ মডেল উচ্চ বিদ্যালয় *ম্যাপল হাই স্কুল *আরমান উল্লা ইসলামী একাডেমী। *কলেজ হাই স্কুল। *নবীগঞ্জ পৌর আইডিয়াল স্কুল। == নাগরিক সুযোগ-সুবিধা == == কৃষি অর্থনীতি == == যোগাযোগ == == যানবাহন == ইত্যাদি == রোড == রোড রোড রোড রোড রোড রোড == কৃতি ব্যক্তিত্ব == *অধ্যাপক তাফাজ্জুল ইসলাম চৌধুরী -রাজনিতীবিদ শিক্ষাবিদ। অনুদৈপায়ন == দর্শনীয় বৈশিষ্ট্যপূর্ণ স্থান এবং স্থাপনা == #নবীগঞ্জ কেন্দ্রীয় শহিদ মিনার #স্মৃতিসৌধ #নবীগঞ্জ সরকারি কলেজ == বিবিধ == == আরও দেখুন == নবীগঞ্জ উপজেলা; হবিগঞ্জ জেলা; বাংলাদেশের পৌরসভাসমূহের তালিকা। == তথ্যসূত্র == ==বহিঃসংযোগ== নবীগঞ্জ পৌরসভা জাতীয় তথ্য বাতায়ন। নবীগঞ্জ পৌরসভা স্থানীয় সরকার প্রকৌশল অধিদপ্তর। উপজেলা জেলার পৌরসভা বিভাগের পৌরসভা
