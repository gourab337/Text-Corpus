'''কাঞ্চননগর বহুমুখী উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার কাঞ্চননগর ইউনিয়নে অবস্থিত। == ইতিহাস == বিদ্যালয়টি ১৯৭১ সালে প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == প্রতিষ্ঠানের অধ্যক্ষ জনাব মোহাম্মদ আবদুস সাত্তার। এছাড়া আরো ১১ জন শিক্ষক-শিক্ষিকা বিদ্যালয়ে কর্মরত আছেন। == অবকাঠামো == প্রতিষ্ঠান ভবনটি একটি দ্বিতল ভবন। == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৮০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
