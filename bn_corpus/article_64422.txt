'''পূর্ব দিঘীরপার ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত কানাইঘাট উপজেলার একটি ইউনিয়ন। == অবস্থান == == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন-''' ৮৩৭২ একর, '''জনসংখ্যা-''' ১৬৪২২ জন পুরুষ- ৮১৭৮, মহিলা- ৮২৪৪। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == == হাট-বাজার == মাত্র একটি- সড়কের বাজার == খাল নদী == মাত্র একটি নদী নাম: সুরমা নদী '''খাল:''' ১-দর্পনগর ২- দনবাগা খাল ৩- ইদগাখাল ৪- নওয়াখাল। == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == পূর্ব দিঘীরপার ইউনিয়ন জাতীয় তথ্য বাতায়ন। বাংলাপিডিয়ায় ''কানাইঘাট উপজেলা'' ''এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' উপজেলার ইউনিয়ন
