'''ইউনিভার্সিটি অব সান ডিয়েগো''' লা জোলাতে অবস্থিত একটি পাবলিক বিশ্ববিদ্যালয়। ২১৪১ একর জমি জুড়ে অবস্থিত। ১৯৬০ সালে এটি প্রতিষ্ঠিত হয়। এই বিশ্ববিদ্যালয়ে ২০০'র বেশি উপস্নাতক স্নাতক ডিগ্রি প্রোগ্রাম চালু আছে। == ইতিহাস == == ক্যাম্পাস == ==সুবিধাসমূহ== == অ্যাকাডেমিকস এবং প্রশাসন == == র‍্যাংকিং == == বিখ্যাত শিক্ষার্থী == ব্রুশ বিউটলার, চিকিৎসাবিজ্ঞানে নোবেল পুরস্কার ২০১১ == বিখ্যাত শিক্ষক == ফ্রান্সিস ক্রিক, চিকিৎসাবিজ্ঞানে নোবেল পুরস্কার ১৯৬২, আলবার্ট ল্যাস্কার অ্যাওয়ার্ড ফর বেসিক মেডিকেল রিসার্চ (১৯৬০), রয়েল মেডেল (১৯৭২), কপলি মেডেল (১৯৭৫), আলবার্ট মেডেল (১৯৮৭) রবার্ট এঙ্গেল, অর্থনীতিতে নোবেল পুরস্কার ২০০৩ মারিয়া পদার্থবিজ্ঞানে নোবেল পুরস্কার ১৯৬৩ ক্লাইভ গ্রেঞ্জার, অর্থনীতিতে নোবেল পুরস্কার ২০০৩ ওয়াল্টার কোন, রসায়নে নোবেল পুরস্কার ১৯৯৮, অলিভার বাকলি কনডেন্সড ম্যাটার প্রাইজ (১৯৬১), ন্যাশনাল মেডেল অব সায়েন্স (১৯৮৮) রজার ওয়াই. তিসিয়েন, রসায়নে নোবেল পুরস্কার ২০০৮, গোল্ডেন গুজ অ্যাওয়ার্ড (২০১২), বি উইলসন মেডেল (২০০৮), ওলফ প্রাইজ ইন মেডিসিন (২০০৪) হ্যারল্ড ক্লেটন ইউরি, রসায়নে নোবেল পুরস্কার ১৯৩৪, উইলার্ড গিবস অ্যাওয়ার্ড (১৯৩৪), ডেভি মেডেল (১৯৪০), ফ্রাঙ্কলিন মেডেল (১৯৪৩), মেডেল ফর মেরিট (১৯৪৬), জে লরেন্স স্মিথ মেডেল (১৯৬২), ন্যাশনাল মেডেল অব সায়েন্স (১৯৬৪), গোল্ড মেডেল অব দ্য রয়েল সোসাইটি (১৯৬৬), প্রিস্টলি মেডেল (১৯৭৩) আরউইন মার্ক জ্যাকবস, ন্যাশনাল মেডেল অব টেকনোলজি অ্যান্ড ইনোভেশন ১৯৯৪, আইইই মেডেল অব অনার ২০১৩, আইইইই আলেক্সান্ডার গ্রাহাম বেল মেডেল (১৯৯৫), মার্কনি প্রাইজ (২০১১), আইইইই/আরএসই ওলফসন জেমস ক্লার্ক ম্যাক্সওয়েল অ্যাওয়ার্ড (২০০৭) জ্যাক কেইল ওলফ, আইইইই কোজি কোবায়াশি কম্পিউটার্স অ্যান্ড কমিউনিকেশন্স অ্যাওয়ার্ড ১৯৯৮, ক্লদ শ্যানন অ্যাওয়ার্ড ২০০১, আইইইই রিচার্ড ডব্লিউ হ্যামিং মেডেল ২০০৪, মার্কনি প্রাইজ ২০১১ == তথ্যসূত্র == যুক্তরাষ্ট্রের বিশ্ববিদ্যালয় বিশ্ববিদ্যালয়
