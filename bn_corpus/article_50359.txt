'''খান সাহেব শেখ মোশাররফ হোসেন''' হলেন বাংলাদেশের একজন প্রখ্যাত রাজনীতিবিদ সমাজ সেবক, যিনি পূর্ব পাকিস্তান প্রাদেশিক আইনসভার সদস্য ছিলেন। == জন্ম পারিবারিক পরিচিতি == শেখ মোশাররফ হোসেন ছিলেন বাংলাদেশের স্থপতি বঙ্গবন্ধু শেখ মুজিবুর রহমানের চাচা। তার পুত্র শেখ কবির হোসেন বাংলাদেশ প্রাইভেটাইজেশন বোর্ড এবং বাংলাদেশ রেড ক্রিসেন্ট সোসাইটির সাবেক চেয়ারম্যান। == শিক্ষাজীবন == == কর্মজীবন == == রাজনৈতিক জীবন == শেখ মোশাররফ হোসেন পূর্ব পাকিস্তান প্রাদেশিক আইনসভার সদস্য ছিলেন। == মুক্তিযুদ্ধে অবদান == == মৃত্যু == == স্মারক স্বীকৃতি == জনসেবায় অসাধারণ অবদানের জন্য তত্কালীন ব্রিটিশ সরকার তাকে '''খান বাহাদুর''' উপাধীতে ভূষিত করেন। তার নামে টুঙ্গিপাড়া উপজেলায় 'খান সাহেব শেখ মোশাররফ হোসেন স্কুল এ্যান্ড কলেজ' রয়েছে। == আরও দেখুন == == কোটালীপাড়ায় খান সাহেব শেখ মোশাররফ হোসেন সড়কের ভিত্তিপ্রস্তর স্থাপন == == বহিঃসংযোগ == জেলার রাজনীতিবিদ
