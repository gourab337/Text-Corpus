'''সুখচর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত হাতিয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাতিয়া উপজেলার সর্ব-উত্তরে সুখচর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে হাতিয়া চ্যানেল চর কিং ইউনিয়ন; দক্ষিণে হাতিয়া পৌরসভা; পূর্বে নলচিরা ইউনিয়ন এবং উত্তরে মেঘনা নদী, সুবর্ণচর উপজেলার চর জুবলী ইউনিয়ন চর ওয়াপদা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সুখচর ইউনিয়ন হাতিয়া উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাতিয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৩নং নির্বাচনী এলাকা নোয়াখালী-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাতিয়া উপজেলা হাতিয়া থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
