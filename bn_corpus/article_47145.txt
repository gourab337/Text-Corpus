'''নিজামপুর মাওলানা আবদুল গণি (রহ.) দাখিল মাদ্রাসা''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার ওয়াহেদপুর ইউনিয়নের মাইজগাঁও গ্রামে মাদ্রাসাটি অবস্থিত। == ইতিহাস == মাদ্রাসাটি ১৯৯০ সালে প্রতিষ্ঠিত হয়। == পরিচালনা ব্যবস্থা == == == মাদ্রাসাটির অধ্যক্ষ জনাব মোহাম্মদ ইউনুছ নজিরী। বর্তমানে আড়াই শতাধিক শিক্ষার্থী মাদ্রাসায় অধ্যয়নরত আছে। == অবকাঠামো == মাদ্রাসা ভবনটি একটি আধাপাকা ভবন। == পর্যায় == মাদ্রাসায় দাখিল পর্যায় পর্যন্ত পাঠদান করা হয়। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯৮%। == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
