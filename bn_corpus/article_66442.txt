'''আগানগর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == আগানগর ইউনিয়ন পূর্বে ভবানীপুর উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার উত্তর-পূর্বাংশে আগানগর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে খোশবাস উত্তর ইউনিয়ন খোশবাস দক্ষিণ ইউনিয়ন, দক্ষিণ-পশ্চিমে বরুড়া পৌরসভা, দক্ষিণে ভবানীপুর ইউনিয়ন এবং পূর্বে উত্তরে কুমিল্লা আদর্শ সদর উপজেলার কালিরবাজার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আগানগর ইউনিয়ন বরুড়া উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
