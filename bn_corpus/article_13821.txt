'''দীঘিনালা''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত দীঘিনালা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দীঘিনালা উপজেলার মধ্যাংশে দীঘিনালা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বাবুছড়া ইউনিয়ন, পশ্চিমে বোয়ালখালী ইউনিয়ন, দক্ষিণে কবাখালী ইউনিয়ন এবং পূর্বে রাঙ্গামাটি জেলার বাঘাইছড়ি উপজেলার সাজেক ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দীঘিনালা ইউনিয়ন দীঘিনালা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দীঘিনালা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == দীঘিনালা ইউনিয়নের মধ্য দিয়ে প্রবাহিত হচ্ছে মাইনী নদী। == হাট-বাজার == দীঘিনালা ইউনিয়নের প্রধান হাট-বাজার হল দীঘিনালা থানা বাজার। == দর্শনীয় স্থান == ত্রিপুরা মহারাজ গোবিন্দ মাণিক্য কর্তৃক খননকৃত দীঘি == জনপ্রতিনিধি == == আরও দেখুন == দীঘিনালা উপজেলা দীঘিনালা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
