বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == কান্দিরপাড় ইউনিয়ন পূর্বে পশ্চিমগাঁও ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == লাকসাম উপজেলার মধ্যাংশে কান্দিরপাড় ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন মুদাফফরগঞ্জ ইউনিয়ন, উত্তরে বাকই ইউনিয়ন, উত্তর-পূর্বে লালমাই উপজেলার পেরুল দক্ষিণ ইউনিয়ন, পূর্বে লাকসাম পৌরসভা এবং দক্ষিণে গোবিন্দপুর ইউনিয়ন মনোহরগঞ্জ উপজেলার মৈশাতুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কান্দিরপাড় ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
