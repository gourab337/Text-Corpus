'''রাধানগর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মেঘনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মেঘনা উপজেলার পূর্বাংশে রাধানগর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে চন্দনপুর ইউনিয়ন মানিকারচর ইউনিয়ন, দক্ষিণ-পশ্চিমে কাঁঠালিয়া নদী গোবিন্দপুর ইউনিয়ন, দক্ষিণে কাঁঠালিয়া নদী তিতাস উপজেলার সাতানী ইউনিয়ন, পূর্বে কাঁঠালিয়া নদী হোমনা উপজেলার মাথাভাঙ্গা ইউনিয়ন, উত্তর-পূর্বে কাঁঠালিয়া নদী হোমনা পৌরসভা এবং উত্তরে মেঘনা নদী নারায়ণগঞ্জ জেলার আড়াইহাজার উপজেলার কালাপাহাড়িয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রাধানগর ইউনিয়ন মেঘনা উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মেঘনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মেঘনা উপজেলা মেঘনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
