''বাসাইল ইউনিয়ন'' বাংলাদেশের টাঙ্গাইল জেলার অন্তর্গত বাসাইল উপজেলার একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == বাসাইল ইউনিয়নের মোট আয়তন ৬৪৪৬ (একর)। গ্রাম সংখ্যা ১০ টি। == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী বাসাইল ইউনিয়নের মোট জনসংখ্যা ২৬৫৭০ জন। প্শিক্ষার হার ৪২%। == যোগাযোগ ব্যবস্থা == == হাটবাজারের তালিকা == বাসাইল হাট, ইশ্বরগঞ্জ বাজার, রাশড়া বাজার == হাসপাতাল == == শিক্ষা == == দর্শনীয় স্থান == চাপড়া বিলের বাসুলিয়া পার্ক। == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == ==তথ্যসূত্র== উপজেলার ইউনিয়ন জেলার ইউনিয়ন
