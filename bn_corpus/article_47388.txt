'''ভূজপুর শরিয়তুল উলুম দাখিল মাদ্রাসা''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার ভূজপুর ইউনিয়নে অবস্থিত। == ইতিহাস == মাওলানা ফজলুল করিম সাহেব মাদ্রাসায় জায়গা দান শিক্ষা বিস্তারে এলাকার বিত্তশালী লোকদের আর্থিক সহায়তায় ১৯৩৭ সালে মাদ্রাসাটি প্রতিষ্ঠা করেন। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == প্রতিষ্ঠানের অধ্যক্ষ জনাব মোহাম্মদ মোরশেদ খান। == অবকাঠামো == মাদ্রাসায় একটি তিনতলা বিশিষ্ট এতিমখানা শ্রেণীকক্ষ, একটি একতলা শ্রেণীকক্ষ এবং দুইতলা অফিস কক্ষ রয়েছে। == শিক্ষা কার্যক্রম == এটি একটি দাখিল (মাধ্যমিক) পর্যায়ের শিক্ষা প্রতিষ্ঠান। প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৯০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
