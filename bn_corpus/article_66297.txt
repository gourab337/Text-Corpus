'''নাঙ্গলকোট পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার মধ্যাংশে নাঙ্গলকোট পৌরসভার অবস্থান। পৌরসভার পশ্চিমে হেসাখাল ইউনিয়ন, উত্তরে মক্রবপুর ইউনিয়ন, উত্তর-পূর্বে রায়কোট দক্ষিণ ইউনিয়ন এবং পূর্বে দক্ষিণে মৌকরা ইউনিয়ন অবস্থিত। == ইতিহাস == নাঙ্গলকোট উপজেলার ৫নং নাঙ্গলকোট ইউনিয়ন এর দক্ষিণ-পূর্বাংশ নিয়ে নাঙ্গলকোট পৌরসভা প্রতিষ্ঠিত হয়। ফলে নাঙ্গলকোট ইউনিয়নকে বিলুপ্ত করে পৌরসভার উত্তরের অংশ ৫নং মক্রবপুর ইউনিয়ন এবং পশ্চিমের অংশ ১২নং হেসাখাল ইউনিয়ন নামে দুইটি আলাদা ইউনিয়ন গঠন করা হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == নাঙ্গলকোট পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == মেয়রঃ জনাব আব্দুল মালেক == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
