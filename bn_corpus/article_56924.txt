বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার সর্ব-পূর্বে মোহাম্মদপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বদলকোট ইউনিয়ন, দক্ষিণ-পশ্চিমে চাটখিল পৌরসভা পাঁচগাঁও ইউনিয়ন, দক্ষিণ-পূর্বে সোনাইমুড়ি উপজেলার জয়াগ ইউনিয়ন, পূর্বে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার সরসপুর ইউনিয়ন, উত্তর-পূর্বে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার বাইশগাঁও ইউনিয়ন এবং উত্তরে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার হাসনাবাদ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মোহাম্মদপুর ইউনিয়ন চাটখিল উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
