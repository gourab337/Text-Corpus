'''দৌলতপুর ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত বিশ্বনাথ উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == বিশ্বনাথ উপজেলা সদর হতে সড়ক পথে কিঃ মিঃ দুরত্বে অবস্থিত। সীমানাঃ পুবে রামপাশা ইউনিয়ন, পশ্চিমে ছাতক উপজেলা, উত্তরে দশঘর ইউনিয়ন, দক্ষিণে দেওকলস ইউনিয়ন। == ইতিহাস == সিলেট জেলার বিশ্বনাথ উপজেলার ঐতিহ্যবাহী ইউনিয়ন ৫নং দৌলতপুর ইউনিয়ন পরিষদ। স্থাপিত ১৯৬৪ সালে। == গ্রাম সমূহ == গ্রাম সংখ্যাঃ ৫৩ টি। == আয়তন জনসংখ্যা == আয়তন- ৪১.৬ বর্গ কিঃ মিঃ। মোট জনসংখ্যা- ৩০৫৭৩ জন, পুরুষ ১৬০৫৬ জন, মহিলা ১৪৫১৮ জন। ভোটার সংখ্যাঃ ১৬২৪৩ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার-''' ৪৫% '''শিক্ষা প্রতিষ্ঠাণ''' প্রাথমিক বিদ্যালয়ঃ সরকারী ১৮ টি বেসরকারী ০১ টি কমিউনিটি ১০ টি মাধ্যমিক বিদ্যালয়ঃ সেরকারী ০২ টি মাদ্রাসাঃ আলীয়া ০৩ টি অন্যান্য ০৪ টি মসজিদ ৫০ টি মন্দির ০৩ টি == হাট বাজার == ০৪ টি। পুরাতন হাবড়া বাজার, দশপাইকা বাজার, সিংঙ্গেরকাছ বাজার। == খাল নদী == মাকুন্দা নদী মাটিজুরা নদী চন্ড্রিখাল == দর্শনীয় স্থান == লেনকাদার হাওর == কৃতী ব্যক্তিত্ব == ডঃ যতীন্দ্রমোহন ভট্রার্চায্য দৌলতপুর ইউনিয়নাধীন পশ্চিম গাঁও’র কৃতি সন্তান।১৯০৫ সালের আগষ্ট পশ্চিম গাঁও গ্রামে জন্ম গ্রহণ করেন। চৌধুরী ১৯১৬ খ্রীঃ দৌলতপুর ইউনিয়নাধীন সিংঙ্গেরকাছ মৌজার মাঝগাঁও গ্রামে জন্ম গ্রহণ করেন। == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান'''- মোহাম্মদ আমির আলী ক্রম নং চেয়ারম্যানের নাম সময়কাল ০১ মো: জায়ফর আলী পূর্ব পাকিস্তান ০২ রাশিদ আলী (ভারপ্রাপ্ত) পূর্ব পাকিস্তান ০৩ খুশের্দ আলী পূর্ব পাকিস্তান ০৪ ইব্রাহিম খান সরকার মনোনীত রিলিফ চেয়ারম্যান ০৫ আ.ন.ম শফিকুল হক সরকার মনোনীত রিলিফ চেয়ারম্যান ০৬ রমুজ খান ১৯৭৩-১৯৭৭ ০৭ সুনা মিয়া চৌধুরী ১৯৭৭-১৯৮২ ০৮ নুরুল গণি ১৯৮২-১৯৮৭ ০৯ আবদুল হাই ১৯৮৭-১৯৯২ ১০ আবারক আলী ১৯৯২-১৯৯৭ ১১ মশরফ হোসেন ১৯৭৯-২০০৩ ১২ মো: আমির আলী ২০০৩-২০১১ ১৩ মোঃ আব্বাস আলী ২০১১-২০১৬ ১৪ মোহাম্মদ আমির আলী ১৮ সেপ্টেম্বর ২০১৬- চলমান == আরও দেখুন == বিশ্বনাথ উপজেলা সিলেট বিভাগ সিলেট জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বিশ্বনাথ উপজেলা বাংলাপিডিয়া ৫নং দৌলতপুর ইউনিয়ন পরিসদ- বাংলাদেশ জাতীয় তথ্য বাতায়ন উপজেলার ইউনিয়ন
