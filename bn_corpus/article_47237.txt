'''গুইমারা''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত গুইমারা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == গুইমারা উপজেলার সর্ব-উত্তরে গুইমারা ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে হাফছড়ি ইউনিয়ন রামগড় উপজেলার পাতাছড়া ইউনিয়ন, পশ্চিমে মাটিরাঙ্গা উপজেলার মাটিরাঙ্গা ইউনিয়ন, উত্তরে মাটিরাঙ্গা পৌরসভা মাটিরাঙ্গা উপজেলার মাটিরাঙ্গা ইউনিয়ন এবং পূর্বে মহালছড়ি উপজেলার মাইসছড়ি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গুইমারা ইউনিয়ন ছিল মাটিরাঙ্গা উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। বর্তমানে এটি গুইমারা উপজেলার আওতাধীন। ইউনিয়নের প্রশাসনিক কার্যক্রম গুইমারা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == গুইমারা উপজেলা গুইমারা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
