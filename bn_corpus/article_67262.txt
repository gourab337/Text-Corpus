'''নবীনগর পৌরসভা''' বাংলাদেশের জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == নবীনগর পৌরসভার আয়তন ৩৯০৭ একর (১৫.৮১ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নবীনগর পৌরসভার মোট জনসংখ্যা ৫৩,১৫৭ জন। মোট পরিবার ১০,৫৩৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ৩৩৬২ জন। == অবস্থান সীমানা == নবীনগর উপজেলার মধ্যাংশে নবীনগর পৌরসভার অবস্থান। পৌরসভার পশ্চিমে নবীনগর পশ্চিম ইউনিয়ন, দক্ষিণে শ্রীরামপুর ইউনিয়ন ইব্রাহিমপুর ইউনিয়ন, পূর্বে নবীনগর পূর্ব ইউনিয়ন এবং উত্তরে তিতাস নদী কৃষ্ণনগর ইউনিয়ন অবস্থিত। == ইতিহাস == ১৯৯৯ সালের ১২ সেপ্টেম্বর নবীনগর উপজেলার নবীনগর পূর্ব, শ্রীরামপুর ইব্রাহিমপুর ইউনিয়নত্রয়ের অংশ নিয়ে নবীনগর পৌরসভা প্রতিষ্ঠিত হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == নবীনগর পৌরসভা একটি 'ক' শ্রেণীর পৌরসভা। পৌরসভার প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। এটি ৯টি ওয়ার্ডে বিভক্ত। ওয়ার্ডভিত্তিক পৌরসভার এলাকাসমূহ হল: ওয়ার্ড নং আওতাধীন এলাকা ১নং ওয়ার্ড আলমনগর ২নং ওয়ার্ড পশ্চিম নবীনগর ৩নং ওয়ার্ড খাজানগর, গোপীনাথপুর, উত্তর নবীনগর ৪নং ওয়ার্ড নবীনগর মধ্য ৫নং ওয়ার্ড আলিয়াবাদ, মাঝিকাড়া ৬নং ওয়ার্ড জল্লা, উত্তর নারায়ণপুর ৭নং ওয়ার্ড দক্ষিণ নারায়ণপুর ৮নং ওয়ার্ড পূর্ব দোলাবাড়ী, নবীপুর, পূর্ব ভোলাচং ৯নং ওয়ার্ড ভেলানগর, পশ্চিম দোলাবাড়ী, পশ্চিম ভোলাচং, সোহাতা == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
