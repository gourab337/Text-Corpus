'''দৌলখাঁড়''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার দক্ষিণাংশে দৌলখাঁড় ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বটতলী ইউনিয়ন, উত্তরে জোড্ডা পূর্ব ইউনিয়ন, পূর্বে ঢালুয়া ইউনিয়ন বক্সগঞ্জ ইউনিয়ন এবং দক্ষিণে নোয়াখালী জেলার সেনবাগ উপজেলার ডুমুরুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দৌলখাঁড় ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == এমরান কবির চৌধুরী—কুমিল্লা বিশ্ববিদ্যালয় এর বর্তমান উপাচার্য। সৈয়দ মাহমুদ ২২তম বিচারপতি। == জনপ্রতিনিধি == == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
