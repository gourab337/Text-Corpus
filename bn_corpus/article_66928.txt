'''পাঁচগাছিয়া পশ্চিম''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দাউদকান্দি উপজেলার পাঁচগাছিয়া পশ্চিম ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পদুয়া ইউনিয়ন, পূর্বে দৌলতপুর ইউনিয়ন, দক্ষিণে ধনাগোদা নদী চাঁদপুর জেলার মতলব উত্তর উপজেলার দুর্গাপুর ইউনিয়ন এবং পশ্চিমে ধনাগোদা নদী চাঁদপুর জেলার মতলব উত্তর উপজেলার বাগানবাড়ী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাঁচগাছিয়া পশ্চিম ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
