বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == ইয়াকুবপুর ইউনিয়নের আয়তন বর্গ কিলোমিটার। == জনসংখ্যা == ইয়াকুবপুর ইউনিয়নের জনসংখ্যা প্রায় ২০ হাজার। == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার সর্ব-দক্ষিণে ইয়াকুবপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে রামনগর ইউনিয়ন, পূর্বে দাগনভূঁইয়া ইউনিয়ন, দক্ষিণে নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার চর পার্বতী ইউনিয়ন এবং পশ্চিমে নোয়াখালী জেলার সেনবাগ উপজেলার নবীপুর ইউনিয়ন বিজবাগ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইয়াকুবপুর ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;মাদ্রাসা ইসহাকিয়া আলিম মাদ্রাসা ;মাধ্যমিক বিদ্যালয় জাহানারা উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == ইয়াকুবপুর ইউনিয়নে যোগাযোগের প্রধান সড়ক রোড। == খাল নদী == ইয়াকুবপুর ইউনিয়ের উপর দিয়ে প্রবাহিত হচ্ছে দাগনভূঁইয়া খাল। == হাট-বাজার == ইয়াকুবপুর ইউনিয়নের প্রধান হাট-বাজার হল এতিমখানা বাজার == দর্শনীয় স্থান == কূরবার দিঘি == জনপ্রতিনিধি == == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
