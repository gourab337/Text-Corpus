বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == মাথাভাঙ্গা ইউনিয়ন পূর্বে হোমনা দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == হোমনা উপজেলার মাথাভাঙ্গা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে হোমনা পৌরসভা, পূর্বে নিলখী ইউনিয়ন ঘারমোড়া ইউনিয়ন, দক্ষিণে তিতাস উপজেলার সাতানী ইউনিয়ন এবং পশ্চিমে কাঁঠালিয়া নদী মেঘনা উপজেলার রাধানগর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মাথাভাঙ্গা ইউনিয়ন হোমনা উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
