'''সোনাগাজী''' বাংলাদেশের ফেনী জেলার অন্তর্গত সোনাগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাগাজী উপজেলার সোনাগাজী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আমিরাবাদ ইউনিয়ন মতিগঞ্জ ইউনিয়ন; পশ্চিমে সোনাগাজী পৌরসভা চর চান্দিয়া ইউনিয়ন; দক্ষিণে নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার মুছাপুর ইউনিয়ন সন্দ্বীপ চ্যানেল এবং পূর্বে ফেনী নদী, চট্টগ্রাম জেলার মীরসরাই উপজেলার ইছাখালী ইউনিয়ন ওসমানপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সোনাগাজী ইউনিয়ন সোনাগাজী উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাদেশের ইউনিয়ন নিবন্ধ উপজেলার ইউনিয়ন
