'''এতবারপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == এতবারপুর ইউনিয়ন পূর্বে ৭নং চান্দিনা পশ্চিম ইউনিয়ন নামে পরিচিত ছিল। ইউনিয়নের উত্তরাংশের কিছু অংশ চান্দিনা পৌরসভার আওতাধীন হওয়ায় বাকি অংশকে ৭নং এতবারপুর ইউনিয়ন নামে নতুন নামকরণ করা হয়। == অবস্থান সীমানা == চান্দিনা উপজেলার পূর্বাংশে এতবারপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বরকইট ইউনিয়ন, উত্তরে চান্দিনা পৌরসভা, পশ্চিমে বাড়েরা ইউনিয়ন মাইজখার ইউনিয়ন এবং দক্ষিণে বরুড়া উপজেলার খোশবাস উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == এতবারপুর ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
