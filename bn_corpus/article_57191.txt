'''কালাদরপ''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত নোয়াখালী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নোয়াখালী সদর উপজেলার মধ্যাংশে কালাদরপ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে এওজবালিয়া ইউনিয়ন, উত্তরে দাদপুর ইউনিয়ন আণ্ডারচর ইউনিয়ন, পশ্চিমে পূর্ব চর মটুয়া ইউনিয়ন, দক্ষিণ-পশ্চিমে লক্ষ্মীপুর জেলার কমলনগর উপজেলার চর কাদিরা ইউনিয়ন এবং দক্ষিণে সুবর্ণচর উপজেলার চর ওয়াপদা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কালাদরপ ইউনিয়ন নোয়াখালী সদর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুধারাম মডেল থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নোয়াখালী সদর উপজেলা সুধারাম মডেল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
