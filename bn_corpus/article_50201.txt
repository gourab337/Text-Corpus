'''কচুয়া পৌরসভা''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == কচুয়া পৌরসভার আয়তন ১১.২১ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী কচুয়া পৌরসভার জনসংখ্যা ২৭,০২৪ জন। এর মধ্যে পুরুষ ১৩,৩৭১ জন এবং মহিলা ১৩,৬৫৩ জন। == অবস্থান সীমানা == কচুয়া উপজেলার মধ্যাংশে কচুয়া পৌরসভার অবস্থান। পৌরসভার উত্তরে কচুয়া দক্ষিণ ইউনিয়ন, পশ্চিমে কচুয়া দক্ষিণ ইউনিয়ন কাদলা ইউনিয়ন এবং দক্ষিণে পূর্বে কড়ইয়া ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == ১৯৯৮ সালের ১০ মে কচুয়া পৌরসভা প্রতিষ্ঠিত হয়। প্রতিষ্ঠাকালে ''খ'' শ্রেণির পৌরসভা থাকলেও ২০১৫ সালের সেপ্টেম্বর এটি ''ক'' শ্রেণিতে উন্নীত হয়। == প্রশাসনিক এলাকা == কচুয়া পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। == যোগাযোগ ব্যবস্থা == == শিক্ষা == কচুয়া পৌরসভার সাক্ষরতার হার ৬৩.৮০%। == অর্থনীতি == মূলত কৃষি, ব্যবসা এবং বৈদেশিক রেমিটেন্স থেকে পৌরসভার অর্থনীতি নির্ভরশীল। == জনপ্রতিনিধি == পৌর মেয়র: নাজমুল আলম স্বপন == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা, চাঁদপুর
