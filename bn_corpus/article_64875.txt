'''কুড়ার বাজার ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত বিয়ানীবাজার উপজেলার একটি ঐতিহ্যবাহী বৃহত্তম ইউনিয়ন। == অবস্থান সীমানা == বিয়ানীবাজার উপজেলা সদর থেকে ইউনিয়ন পরিষদের দুরত্ব কিঃমিঃ। বিয়ানীবাজার হতে বৈরাগী বাজার হয়ে কুড়ার বাজার ইউনিয়ন ইউনিয়নের অবস্থান। == ইতিহাস == == গ্রাম সমূহ == গ্রামের সংখ্যা ১৩ টি। '''গ্রাম সমূহের নামঃ–''' দেউলগ্রাম, গোবিন্দশ্রী, আঙ্গুরা মোহাম্মদপুর, উত্তর আকাখাজানা, আঙ্গুরা মুহাম্মদপুর, আকাখাজানা, লাঊঝারী, পূর্ব লাউঝারী, গড়রবন্দ, আরিজ খা টিলা, খশির, খশিরবন্দ, আঙ্গারজুর। == আয়তন জনসংখ্যা == '''আয়তন–''' ১৬.৯ বর্গ কিঃ মিঃ। ৩৬৯৫৭ জন (''১৮ ফেব্রুয়ারী, ২০১২সালের জন্ম নিবন্ধন অনুযায়ী'')। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার –''' ৪৮.৪%। (''২০০১ এর শিক্ষা জরীপ অনুযায়ী'') '''শিক্ষা প্রতিষ্ঠাণ''' সরকারী প্রাথমিক বিদ্যালয়- ১১টি বে-সরকারী রেজিঃ প্রাঃ বিদ্যালয়- ৩টি আনরেজিঃ প্রাঃ বিদ্যালয়- ১টি আলিম মাদ্রাসা- ১টি দাখিল মাদ্রাসা- ১টি কওমী/টাইটেল মাদ্রাসা –৪ টি উচ্চ বিদ্যালয়- ৫টি নিম্ন মাধ্যমিক বিদ্যালয়- ১টি কে,জি স্কুল- ৩টি,কলেজ-০২টি == হাট-বাজার == গোবিন্দশ্রী বাজার ফাড়ির বাজার কুড়ার বাজার বৈরাগী বাজার == খাল নদী == কুশিয়ারা নদী সোনাই নদী লুলা নদী কুড়া খাল করতীর খাল ভেউলর খাল গোবিন্দশ্রী খাল == দর্শনীয় স্থান == জামিয়া মাদানিয়া আঙ্গুরা মুহাম্মদপুর। বালু মহাল *নদীর চর == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান'''- ''এ এফ এম আবু তাহের'' ====== চেয়ারম্যানগণের তালিকা ====== ক্রম নং চেয়ারম্যানের নাম সময়কাল ০১ আব্দুল মন্নান (গোবিন্দশ্রী) ০২ মুজম্মিল আলী (গোবিন্দশ্রী) ০৩ মোঃ আব্দুল আহাদ কলা মিয়া (আকাখাজানা) ০৪ মোঃ রফিকুল বারী (খশির) ০৫ মোঃ গজম্বর আলী ধলা মিয়া (আঙ্গুরা মোহাম্মদপুর) ০৬ মোঃ গজম্বর আলী ধলা মিয়া (আঙ্গুরা মোহাম্মদপুর) ০৭ মোঃ আব্দুল লতিফ (দেউলগ্রাম) ০৮ মোঃ নাজিম উদ্দিন (গোবিন্দশ্রী) ০৯ মোঃ আলকাছ আলী (পূর্ব লাউঝারী) ১০ '''এ এফ এম আবু তাহের''' ''পদাধিকারী'' == আরও দেখুন == সিলেট জেলা আলীনগর ইউনিয়ন, বিয়ানীবাজার বিয়ানীবাজার উপজেলা == তথ্যসূত্র == == বহিঃসংযোগ == বিয়ানীবাজার উপজেলা ''-'' বাংলাপিডিয়ায় প্রাতিষ্ঠানিক ওয়েবসাইট বাংলাদেশ জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
