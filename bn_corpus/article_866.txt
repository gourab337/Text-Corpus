'''কুষ্টিয়া সদর উপজেলা''' বাংলাদেশের কুষ্টিয়া জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == স্থানাঙ্কে কুষ্টিয়া সদরের অবস্থান এখানে প্রায় ৬৬,৭৮৭ ঘরবাড়ী রয়েছে এবং এর আয়তন ৩১৬.২৬ কিমি²। উপজেলার উত্তরে বহমান পদ্মা, গড়াই নদী এবং পাবনা জেলা, দক্ষিণে ঝিনাইদহ জেলার শৈলকুপা উপজেলা হরিণাকুন্ডু উপজেলা, পূর্বে কালীগঙ্গা নদী কুমারখালী উপজেলা এবং পশ্চিমে কুষ্টিয়া জেলার মিরপুর উপজেলা। == ইতিহাস == এন৭০৪ কুষ্টিয়া অতীতে নদীয়া জেলার অন্তর্গত ছিলো। ;নামকরণ ;মুক্তিযুদ্ধে == ভৌগোলিক উপাত্ত == ;ভূপ্রকৃতি ;মৃত্তিকা ;নদ-নদী == সাংষ্কৃতিক বৈশিষ্ট্য == ;ভাষা ;উৎসব ;খেলাধুলা == প্রশাসনিক এলাকা == ১৪৪টি মৌজা ১৫৭টি গ্রাম নিয়ে গঠিত কুষ্টিয়া সদরে ১২টি ইউনিয়ন (+১টি আদালতে মামলাধীন) রয়েছে। ;নির্বাচনী এলাকা জনপ্রতিনিধি == জনসংখ্যার উপাত্ত == ২০১১ সালের বাংলাদেশ আদমশুমারীর তথ্যানুযায়ী, কুষ্টিয়া সদরের মোট জনসংখ্যা ৪,২৩,৮১৮। এদের মধ্যে পুরুষ ৫১.৭৭% মহিলা ৪৮.২৩%। এই উপজেলা। এই উপজেলাতে আঠারোর্ধ জনসংখ্যা ৩,১৩৩২৪ জন (২০০১)। কুষ্টিয়া সদরের শিক্ষার হার ৩৩.৭% (৭+ বছর বয়সী), যেখানে জাতীয় গড় ৩৭.৪৩% == স্বাস্থ্য == == শিক্ষা == এখানে রয়েছেঃ মেডিক্যাল কলেজঃ ১টি, সরকারী কলেজঃ ৮টি, বেসরকারী কলেজঃ ৩০টি, সরকারী উচ্চ বিদ্যালয়ঃ ৩টি, বেসরকারী উচ্চ বিদ্যালয়ঃ ১৭৩টি, বেসরকারী নিম্নমাধ্যমিক বিদ্যালয়ঃ ৩৮টি, সরকারী প্রাথমিক বিদ্যালয়ঃ ৩৩০টি, বেসরকারী প্রাথমিক বিদ্যালয়ঃ ২৭৫টি, কিন্ডারগার্টেনঃ ৩৯টি, মাদ্রাসাঃ ৩৭টি, কারিগরী প্রশিক্ষণ কেন্দ্রঃ ২টি, আইন কলেজঃ ১টি, প্রতিবন্ধীদের বিদ্যালয়ঃ ১টি এবং শিক্ষক প্রশিক্ষণ কেন্দ্রঃ ২টি। == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == কেন্দ্রীয় বাস টার্মিনাল, কুষ্টিয়া সদর, কুষ্টিয়া। কুষ্টিয়া সদর উপজেলার সাথে বাস এবং ট্রেন যোগাযোগ আছে। ;সড়কপথ ;রেলপথ == কৃতী ব্যক্তিত্ব == মাহাবুব-উল-আলম হানিফ এমপি যুগ্ন সাধারণ সম্পাদক বাংলাদেশ আওয়ামী লীগ। == দর্শনীয় স্থান স্থাপনা == চাপাইগাছি বিল নাজিরপুর। ==আনসার ক্যাম্প== বটতৈল চারমাইলে আনসার ক্যাম্প অবস্থিত। == বিবিধ == == আরও দেখুন == কুষ্টিয়া জেলা; খুলনা বিভাগ; বাংলাদেশের উপজেলাসমূহ। == তথ্যসুত্র == ==বহিঃসংযোগ== কুষ্টিয়া সদর উপজেলা জাতীয় তথ্য বাতায়ন। সদর উপজেলা জেলার উপজেলা বিভাগের উপজেলা
