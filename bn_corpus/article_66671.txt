'''চান্দলা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত ব্রাহ্মণপাড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == ব্রাহ্মণপাড়া উপজেলার উত্তর-মধ্যাংশে চান্দলা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মাধবপুর ইউনিয়ন, পশ্চিমে শিদলাই ইউনিয়ন, দক্ষিণে ব্রাহ্মণপাড়া সদর ইউনিয়ন এবং পূর্বে শশীদল ইউনিয়ন জেলার কসবা উপজেলার কায়েমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চান্দলা ইউনিয়ন ব্রাহ্মণপাড়া উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ব্রাহ্মণপাড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোস্তফা আলী শাহীন == আরও দেখুন == ব্রাহ্মণপাড়া উপজেলা ব্রাহ্মণপাড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
