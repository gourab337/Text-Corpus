'''দোল্লাই নবাবপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == দোল্লাই নবাবপুর ইউনিয়ন পূর্বে গল্লাই দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার দোল্লাই নবাবপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে গল্লাই ইউনিয়ন, উত্তরে মহিচাইল ইউনিয়ন, পূর্বে মাইজখার ইউনিয়ন, দক্ষিণ-পূর্বে বরকরই ইউনিয়ন এবং দক্ষিণ-পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার কচুয়া উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দোল্লাই নবাবপুর ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
