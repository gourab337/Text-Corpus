'''বুড়িশ্চর জিয়াউল উলুম ফাজিল মাদ্রাসা''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার হাটহাজারী উপজেলার বুড়িশ্চর ইউনিয়নে অবস্থিত। == ইতিহাস == ১৯৫৬ সালে মাদ্রাসাটি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == মাদ্রাসা পরিচালনার জন্য জনাব মোহাম্মদ মোয়াজ্জম হোসেনকে সভাপতি করে ১৩ জন বিশিষ্ট একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == মাদ্রাসার অধ্যক্ষ জনাব সৈয়দ মোহাম্মদ ফরিদ উদ্দিন। == অবকাঠামো == মাদ্রাসার মূল ভবনটি একটি ত্রিতল ভবন। মাদ্রাসার সামনে একটি খেলার মাঠ রয়েছে। == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। প্রতিষ্ঠানে ফাজিল (ডিগ্রী) পর্যায় পর্যন্ত শিক্ষালাভের সুযোগ রয়েছে। বর্তমানে ১২ শতাধিক শিক্ষার্থী অধ্যয়নরত আছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ১০০%। == আরো দেখুন == হাটহাজারী উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
