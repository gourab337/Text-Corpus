'''কালকিনি পৌরসভা''' বাংলাদেশের মধ্যাঞ্চলের মাদারিপুর জেলার কালকিনি উপজেলার অন্তর্গত একটি প্রথম শ্রেণীর পৌরসভা। == আয়তন == কালকিনি পৌরসভা আয়তন ৬৬৭১.৮৪৬ একর (২৭ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী কালকিনি পৌরসভায় সর্বমোট ৪৬,২৫২ (পুরুষ- ২৩,৮২২; মহিলা- ২২,৪৩০) পৌর নাগরিক পৌর এলাকায় বসবাস করেন। == অবস্থান সীমানা == মাদারিপুর জেলার দক্ষিণ প্রান্তে কালকিনি পৌরসভার অবস্থান। জেলা সদর থেকে পৌরসভার দূরত্ব ২২ কিলোমিটার। পৌরসভার উত্তরে আলীনগর বালীগ্রাম ইউনিয়ন, পূর্বে এনায়েতনগর, শিকারমংগল চরদৌলতখান ইউনিয়ন, দক্ষিণে রমজানপুর ইউনিয়ন, খাঞ্জাপুর ইউনিয়ন, গৌরনদী উপজেলা এবং পশ্চিমে গোপালপুর ইউনিয়ন অবস্থিত। ভৌগলিক অবস্থানঃ ২৩°০৪'২০.৯" উত্তর অক্ষাংশ এবং ৯০°১৪'০৫" পূর্ব দ্রাঘিমাংশ। == প্রতিষ্ঠাকাল == ১৯৯৭ খ্রিষ্টাব্দে কালকিনি পৌরসভা স্থাপিত হয়। যা পরবর্তীতে ১০১৬ খ্রিষ্টাব্দের ১২ জানুয়ারি প্রথম শ্রেনীতে উন্নিত করা হয়। == নামকরণ == == প্রশাসনিক এলাকা == কালকিনি পৌরসভার প্রশাসনিক কার্যক্রম কালকিনি উপজেলার আওতাধীন। এটি জাতীয় সংসদের ২২০নং নির্বাচনী এলাকা মাদারীপুর-৩ এর অংশ। পৌরসভায় ৯টি ওয়ার্ড, ৩৮টি মহল্লা রয়েছে। == শিক্ষা ব্যবস্থা == কালকিনি পৌরসভার সাক্ষরতার হার ৫১.৪০%। পৌরসভায় ১টি কলেজ, ৪টি মাদ্রাসা, ৭টি মাধ্যমিক বিদ্যালয়, ৩টি কওমী মাদ্রাসা, ২৮টি প্রাথমিক বিদ্যালয় রয়েছে। == যোগাযোগ ব্যবস্থা == কালকিনি পৌরসভায় যোগাযোগের প্রধান সড়ক কালকিনি-ভুরঘাটা সড়ক। যে কোন যানবাহনে যোগাযোগ করা যায়। এছাড়া মোট রাস্তা কাঁচা ১৬৫.০০ কি.মি., সিসি/আরসিসি ১৯.০০ কি.মি., সলিং ১৩.০০ কি.মি., এইচ বি বি ৯.০০ কি.মি. এবং কার্পেটিং ৫৫.০০ কি.মি.। ব্রীজ ৬৮টি এবং কালভার্ট ৩৫টি। == স্বাস্থ্য == কালকিনি পৌরসভায় ১টি সরকারি হাসপাতাল, ৭টি বেসরকারি হাসপাতাল এবং ১টি স্বাস্থ্য পরিবার পরিকল্পনা কেন্দ্র, ৪৯টি ইপিআই সেন্টার রয়েছে। == ধর্মীয় উপাসনালয় == কালকিনি পৌরসভায় ১৩৭টি মসজিদ, ১০টি ঈদগাহ, ৯টি মন্দির রয়েছে। == খাল নদী == পালরদী == পত্র-পত্রিকা সাময়িকী == অন্ লাইন পত্রিকা: কলকিনি ২৪ ডট কম, আমাদের মাদারীপুর ডট কম; সাপ্তাহিক: সাপ্তাহিক কালকিনি বার্তা; অবলুপ্ত: বালারঞ্জিকা (১৮৬৩)। == আরও দেখুন == কালকিনি উপজেলা মাদারিপুর জেলা মাদারিপুর পৌরসভা == তথ্যসূত্র == == বহিঃসংযোগ == পৌরসভা বিভাগের পৌরসভা জেলার পৌরসভা
