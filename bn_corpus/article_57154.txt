'''কাদিরপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত বেগমগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার কাদিরপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে শরীফপুর ইউনিয়ন হাজীপুর ইউনিয়ন, উত্তরে উত্তর-পূর্বে রসুলপুর ইউনিয়ন, দক্ষিণ-পূর্বে সেনবাগ উপজেলার নবীপুর ইউনিয়ন এবং দক্ষিণে কবিরহাট উপজেলার নরোত্তমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কাদিরপুর ইউনিয়ন বেগমগঞ্জ উপজেলার আওতাধীন ১৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
