'''ছয়ানী''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত বেগমগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার ছয়ানী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আলাইয়ারপুর ইউনিয়ন, উত্তর-পূর্বে জিরতলী ইউনিয়ন, পূর্বে রাজগঞ্জ ইউনিয়ন, দক্ষিণে নোয়াখালী সদর উপজেলার দাদপুর ইউনিয়ন চর মটুয়া ইউনিয়ন এবং পশ্চিমে লক্ষ্মীপুর জেলার লক্ষ্মীপুর সদর উপজেলার চর শাহী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছয়ানী ইউনিয়ন বেগমগঞ্জ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ছয়ানী ইমামিয়া ফাজিল মাদ্রাসা ছয়ানী উচ্চ বিদ্যালয় ছয়ানী বালিকা উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ছয়ানী ইউনিয়নের প্রধান হাট-বাজার হল ছয়ানী বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
