'''হাজীপুর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত বেগমগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == বেগমগঞ্জ উপজেলার পূর্বাংশে হাজীপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে দুর্গাপুর ইউনিয়ন, পশ্চিমে চৌমুহনী পৌরসভা, দক্ষিণে শরীফপুর ইউনিয়ন, পূর্বে কাদিরপুর ইউনিয়ন এবং উত্তর-পূর্বে রসুলপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাজীপুর ইউনিয়ন বেগমগঞ্জ উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বেগমগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭০নং নির্বাচনী এলাকা নোয়াখালী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বেগমগঞ্জ উপজেলা বেগমগঞ্জ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
