বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা আদর্শ সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == কুমিল্লা আদর্শ সদর উপজেলার জগন্নাথপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পাঁচথুবী ইউনিয়ন, পশ্চিমে কুমিল্লা সিটি কর্পোরেশন কুমিল্লা সদর দক্ষিণ উপজেলার চৌয়ারা ইউনিয়ন, দক্ষিণে কুমিল্লা সদর দক্ষিণ উপজেলার গলিয়ারা ইউনিয়ন এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == জগন্নাথপুর ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা কোতোয়ালী থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৪নং নির্বাচনী এলাকা কুমিল্লা-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মামুনুর রশিদ মামুন == আরও দেখুন == কুমিল্লা আদর্শ সদর উপজেলা কোতোয়ালী থানা, কুমিল্লা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == আদর্শ সদর উপজেলার ইউনিয়ন
