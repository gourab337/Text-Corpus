'''হেয়াকো বনানী উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার অন্তর্গত দাঁতমারা ইউনিয়নের হেয়াকো গ্রামে অবস্থিত। == ইতিহাস == ১৯৭৫ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জনাব মজিবুল হক মজুমদার সভাপতি করে ১১ সদস্যের আব্দুর রশীদ সরকার সহ একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব মির্জা মোহাম্মদ বখতিয়ার উদ্দীন। এছাড়া বিদ্যালয় আরো ১২ জন শিক্ষক-শিক্ষিকা কর্মরত আছেন। == অবকাঠামো == বিদ্যালয়ের ১টি দ্বিতল পাকা ভবন ১টি আধাপাকা ভবন রয়েছে। == শিক্ষা কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব বিদ্যালয়টি 2019 সালের এসএসসি পাশের হার 76.63 == == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
