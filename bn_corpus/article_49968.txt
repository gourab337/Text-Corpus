'''মার্চ ২০১৭'''-এ বিশ্বব্যাপী সংঘটিত ঘটনার সংক্ষিপ্ত-সার। == মার্চ ২০১৭ (বুধবার) == ইংরেজ ক্রিকেটার আম্পায়ার জন হ্যাম্পশায়ার মৃত্যুবরণ করেন। == মার্চ ২০১৭ (শনিবার) == মার্কিন চিকিৎসক অঙ্গ প্রতিস্থাপন বিশেষজ্ঞ থমাস স্টারজল মৃত্যুবরণ করেন। == মার্চ ২০১৭ (মঙ্গলবার) == ** ভারতীয় বাঙালি লোকসঙ্গীত শিল্পী কালিকাপ্রসাদ ভট্টাচার্য মৃত্যুবরণ করেন। ** জার্মান বংশোদ্ভূত মার্কিন নোবেল বিজয়ী পদার্থবিজ্ঞানী হ্যান্স গেয়র্গ ডেমেল্ট মৃত্যুবরণ করেন। == ১১ মার্চ ২০১৭ (শনিবার) == বাংলাদেশী কূটনীতিক মোহাম্মদ মিজারুল কায়েস মৃত্যুবরণ করেন। == ১৭ মার্চ ২০১৭ (শুক্রবার) == নোবেল বিজয়ী সেন্ট লুসিয় কবি নাট্যকার ডেরেক ওয়ালকট‌ মৃত্যুবরণ করেন। == ১৯ মার্চ ২০১৭ (রবিবার) == বাংলাদেশী গল্পকার, ঔপন্যাসিক, শিশুসাহিত্যিক জুবাইদা গুলশান আরা মৃত্যুবরণ করেন। == ২৭ মার্চ ২০১৭ (সোমবার) == বাংলাদেশী চলচ্চিত্র অভিনেতা মিজু আহমেদ মৃত্যুবরণ করেন। == ২৯ মার্চ ২০১৭ (বুধবার) == রুশ বংশোদ্ভূত মার্কিন নোবেল বিজয়ী তাত্ত্বিক পদার্থবিজ্ঞানী আলেক্সেই আলেক্সেভিচ আব্রিকোসোভ মৃত্যুবরণ করেন। == তথ্যসূত্র ==
