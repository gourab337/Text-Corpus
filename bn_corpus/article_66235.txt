'''পেরুল উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লালমাই উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লালমাই উপজেলার মধ্যাংশে পেরুল উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বাগমারা দক্ষিণ ইউনিয়ন, পশ্চিমে বাকই উত্তর ইউনিয়ন, দক্ষিণে পেরুল দক্ষিণ ইউনিয়ন এবং পূর্বে ভুলইন দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পেরুল উত্তর ইউনিয়ন লালমাই উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লালমাই থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লালমাই উপজেলা লালমাই থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
