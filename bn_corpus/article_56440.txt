জর্দানের বিখ্যাত মানুষের নাম সমূহ নীচে উল্লেখ করা হলোঃ == রাজনৈতিক নেতা== শেরিফ হুসেন ইবনে আলী রাজা প্রথম আবদুল্লাহ, জর্ডান রাজা তালাল বিন আবদুল্লাহ রাজা প্রথম হুসেন রাজা দ্বিতীয় আবদুল্লাহ (জর্ডান) ওয়াসফি আল তাল রাণী নূর রানী রানিয়া == রাজনীতিবিদ == মাহা আলী ইসমাঈল বাবুক, জর্দানের রাজধানী আম্মান এর প্রথম মেয়র (১৯০৯-১৯১১)। ফাহাদ এনসৌর ফয়সাল আল-ফায়েজ থৌকান হিন্দওয়ায়ী, সাবেক মন্ত্রী আওন খাসাওয়ানে, সাবেক প্রধানমন্ত্রী আন্তর্জাতিক ন্যায়বিচার আদালত এর সাবেক বিচারপতি। যুবরাজ রাশেদ আল খুজাই আবদেসসালাম আল-মাজালি ইনাম আল-মুফতি, জর্ডানের প্রথম নারী, যিনি সরকারি অফিসে কর্মরত ছিলেন। আলী, আবু আল-রাঘব আলী সুহিমাত শেখ আতাউল্লাহ সুহিমাত সালাহ সুহিমাত, এমপি তারেক সুহিমাত বাহজাত তালহুনি সাবেক প্রধানমন্ত্রী ফায়াজ তারাউয়ানে == ঔপন্যাসিক, কবি, গবেষক লেখক == নাসের আব্দেল আজিজ এলিয়ান সামের লিবদে গবেষক, লেখক সুলেইমান মুসা ইতিহাসবিদ, লেখক হায়দার মাহমুদ কবি, লেখক আবদেল-রহমান মুনিফ ঔপন্যাসিক সামির রাইমুনি কবি, কর্মী মোস্তফা ওয়াহবী (মোস্তফা ওয়াহবী আল তাল) কবি কামেল রিহানী কবি, সাংবাদিক == সামরিক বাহিনী == ক্যাপ্টেন মুয়াত আল-কাসাবেহ জর্ডানের বিমান বাহিনীর বৈমানিক, বন্দী এবং জিম্মি করে ইসলামিক স্টেট তাকে জীবন্ত পুড়িয়ে হত্যা করে হাবিস আল-মাজালি আবদেসসালাম আল-মাজালি মেজর জেনারেল ইব্রাহিম খাসাওয়ানে- জর্ডানের রয়েল চিকিৎসা সেবার সাবেক উপ-পরিচালক জেনারেল মুহাম্মদ সুহিমাত (মৃত্যু ১৯৬৮) == চিকিৎসক == আবদেসসালাম আল-মাজালি দাউদ হানানিয়া তারেক সুহিমাত আব্দাল্লাহ দারাদেখ == ক্রীড়াবিদ == আমের দীব দিমা এবং লামা হাত্তাব আল্ট্রাম্যারাথন রানার্স == সঙ্গীতশিল্পী == মাহমুদ রাদাইদেহ জাদে দিরানি == শিল্পী == জর্জ আলিফ মুহান্না আল দুরা উইজদান আলী মাহমুদ ত্বাহা == সাবেক রাষ্ট্রদূত == নাবিল তালহৌনী == ব্যবসায়ী == ইমান মুথলাক মোহাম্মদ সাহেদাহ == অন্যান্য == শিরীন অদি (জন্ম ১৯৭০), দৃশ্যসংক্রান্ত শিল্পী মার্গো হাদ্দাদ (জন্ম ১৯৮৮), অভিনেত্রী মোনা সৌদি (জন্ম ১৯৪৫), ভাস্কর == তথ্যসূত্র == অনুযায়ী মানুষের তালিকা
