'''হোমনা পৌরসভা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হোমনা উপজেলার পশ্চিমাংশে হোমনা পৌরসভার অবস্থান। পৌরসভার দক্ষিণে মাথাভাঙ্গা ইউনিয়ন; পূর্বে নিলখী ইউনিয়ন; উত্তরে ঘাগুটিয়া ইউনিয়ন, তিতাস নদী জেলার বাঞ্ছারামপুর উপজেলার উজানচর ইউনিয়ন; পশ্চিমে মেঘনা নদী নারায়ণগঞ্জ জেলার আড়াইহাজার উপজেলার কালাপাহাড়িয়া ইউনিয়ন এবং দক্ষিণ-পশ্চিমে কাঁঠালিয়া নদী মেঘনা উপজেলার রাধানগর ইউনিয়ন অবস্থিত। == ইতিহাস == হোমনা উপজেলার তৎকালীন হোমনা উত্তর ইউনিয়ন এর সম্পূর্ণ অংশ নিয়ে হোমনা পৌরসভা প্রতিষ্ঠিত হয়। == নামকরণ == == প্রতিষ্ঠাকাল == == প্রশাসনিক এলাকা == হোমনা পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
