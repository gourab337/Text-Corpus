'''মহাজনহাট ফজলুর রহমান স্কুল এন্ড কলেজ''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার ধুম ইউনিয়নে এটি অবস্থিত। == ইতিহাস == == পরিচালনা ব্যবস্থা == == == অধ্যাপক মোহাম্মদ সোহরাব হোসেন বর্তমানে প্রতিষ্ঠানের ভারপ্রাপ্ত অধ্যক্ষ হিসেবে দায়িত্ব পালন করছেন। == অবকাঠামো == == কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। বর্তমানে উচ্চ মাধ্যমিক পর্যায় পর্যন্ত শিক্ষা ব্যবস্থা চালু আছে। == কৃতিত্ব ফলাফল == == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
