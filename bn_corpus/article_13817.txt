'''লোগাং''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত পানছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == পানছড়ি উপজেলার লোগাং ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চেঙ্গী ইউনিয়ন, দক্ষিণে উল্টাছড়ি ইউনিয়ন, পশ্চিমে মাটিরাঙ্গা উপজেলার তাইন্দং ইউনিয়ন এবং উত্তরে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == লোগাং ইউনিয়ন পানছড়ি উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম পানছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == লোগাং উচ্চ বিদ্যালয় লোগাং বাজার আদর্শ উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == লোগাং ইউনিয়নের পূর্ব সীমান্ত দিয়ে প্রবাহিত হচ্ছে চেঙ্গী নদী লোগাং খাল। == হাট-বাজার == লোগাং ইউনিয়নের প্রধান হাট-বাজার হল লোগাং বাজার। == দর্শনীয় স্থান == ১.নিঝুম পার্ক ২.ভগবানটিলা ৩.স্বপ্নফুরী পার্ক == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: প্রত্যুত্তর চাকমা == আরও দেখুন == পানছড়ি উপজেলা পানছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
