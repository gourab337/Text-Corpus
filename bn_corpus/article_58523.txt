'''কুশাখালী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার কুশাখালী ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে তেওয়ারীগঞ্জ ইউনিয়ন, উত্তরে মান্দারী ইউনিয়ন দিঘলী ইউনিয়ন, উত্তর-পূর্বে চর শাহী ইউনিয়ন, পূর্বে নোয়াখালী জেলার নোয়াখালী সদর উপজেলার চর মটুয়া ইউনিয়ন আণ্ডারচর ইউনিয়ন এবং দক্ষিণে নোয়াখালী জেলার নোয়াখালী সদর উপজেলার চর মটুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কুশাখালী ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ১৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
