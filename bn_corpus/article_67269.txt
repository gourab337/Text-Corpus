'''বিদ্যাকুট''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == বিদ্যাকুট ইউনিয়নের আয়তন ৬০০৩ একর (২৪.২৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বিদ্যাকুট ইউনিয়নের মোট জনসংখ্যা ২৯,৩৬৬ জন। মোট পরিবার ৫,৩৩৯টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১২০৯ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার উত্তর-পূর্বাংশে বিদ্যাকুট ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে নাটঘর ইউনিয়ন; দক্ষিণে শিবপুর ইউনিয়ন; দক্ষিণ-পশ্চিমে নবীনগর পূর্ব ইউনিয়ন; পশ্চিমে তিতাস নদী, কৃষ্ণনগর ইউনিয়ন এবং উত্তরে তিতাস নদী, বড়াইল ইউনিয়ন সদর উপজেলার সাদেকপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বিদ্যাকুট ইউনিয়ন নবীনগর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
