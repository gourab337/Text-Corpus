'''চর ইসলামপুর''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == চর ইসলামপুর ইউনিয়নের আয়তন ৫৫৪০ একর (২২.৪২ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে চর ইসলামপুর ইউনিয়নের মোট জনসংখ্যা ১৬,৪৩৮ জন। মোট পরিবার ২,৬৬৪টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৭৩৩ জন। == ইতিহাস == == অবস্থান সীমানা == বিজয়নগর উপজেলার পশ্চিমাংশে চর ইসলামপুর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চান্দুরা ইউনিয়ন, ইছাপুরা ইউনিয়ন চম্পকনগর ইউনিয়ন; দক্ষিণে পত্তন ইউনিয়ন; পশ্চিমে পত্তন ইউনিয়ন সরাইল উপজেলার শাহবাজপুর ইউনিয়ন এবং উত্তরে সরাইল উপজেলার শাহবাজপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর ইসলামপুর ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == রুমিন ফারহানা –– রাজনীতিবিদ এবং আইনজীবি। == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
