'''শরিয়তপুর সদর''' বাংলাদেশের শরিয়তপুর জেলার অন্তর্গত একটি উপজেলা। == অবস্থান == == প্রশাসনিক এলাকা == == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == ==ঐতিহাসিক স্থান== রুদ্রকর জমিদার বাড়ি রুদ্রকর মঠ == কৃতী ব্যক্তিত্ব == এনামুল হক শামীম, বাংলাদেশ আওয়ামীলীগের কেন্দ্রীয় কার্যনির্বাহী কমিটির সদস্য। -সরদার নাসির উদ্দিন কালু, সাবেক সাংসদ। এইচ এম মনির, কবি লেখক। == আরো দেখুন== বাংলাদেশের জেলাসমূহ == বিবিধ == ==তথ্যসূত্র== ==বহিঃসংযোগ== জেলার উপজেলা সদর উপজেলা
