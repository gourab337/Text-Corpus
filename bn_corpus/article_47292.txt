'''দৌলতপুর আবদুল বারী চৌধুরী উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার নাজিরহাট পৌরসভার দৌলতপুর এলাকায় অবস্থিত। == ইতিহাস == চট্টগ্রামের ব্যবসায়ী ব্রিটিশ বিরোধী আন্দোলনের অন্যতম পুরোধা জনাব আবদুল বারী চৌধুরী বিদ্যালয় প্রতিষ্ঠা করেন। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব মোহাম্মদ শহিদুল ইসলাম। == অবকাঠামো == == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা আবদুল বারী চৌধুরী == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
