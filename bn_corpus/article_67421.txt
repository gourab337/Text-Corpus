'''আখাউড়া পৌরসভা''' বাংলাদেশের জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == আখাউড়া পৌরসভার আয়তন ৮.৩৩ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে আখাউড়া পৌরসভার জনসংখ্যা ৩৬,২৬২ জন। মোট পরিবার ৭,৪৫৩টি। == অবস্থান সীমানা == আখাউড়া উপজেলার উত্তরাংশে আখাউড়া পৌরসভার অবস্থান। আখাউড়া উপজেলা সদর পৌরসভায় অবস্থিত। পৌরসভার উত্তর আখাউড়া উত্তর ইউনিয়ন, পূর্বে আখাউড়া উত্তর ইউনিয়ন আখাউড়া দক্ষিণ ইউনিয়ন, দক্ষিণে আখাউড়া দক্ষিণ ইউনিয়ন এবং পশ্চিমে সদর উপজেলার বাসুদেব ইউনিয়ন অবস্থিত। == প্রতিষ্ঠাকাল == আখাউড়া উপজেলার আখাউড়া উত্তর আখাউড়া দক্ষিণ ইউনিয়নের কিছু অংশ নিয়ে ১৯৯৯ সালের ১২ ডিসেম্বর আখাউড়া পৌরসভা প্রতিষ্ঠিত হয়। == প্রশাসনিক এলাকা == আখাউড়া পৌরসভা একটি 'ক' শ্রেণীর পৌরসভা। পৌরসভার প্রশাসনিক কার্যক্রম আখাউড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। পৌরসভা ৯টি ওয়ার্ডে বিভক্ত। ওয়ার্ডভিত্তিক পৌরসভার এলাকাসমূহ হল: ওয়ার্ড নং প্রশাসনিক এলাকা ১নং ওয়ার্ড দুর্গাপুর ২নং ওয়ার্ড টানপাড়া, উত্তর খড়মপুর, টেংরাপাড়া ৩নং ওয়ার্ড আখাউড়া উত্তর-পূর্ব কর্নার, দক্ষিণ খড়মপুর, রেলওয়ে কুমারপাড়া কলোনী, রেলওয়ে পশ্চিম কলোনী, রেলওয়ে পূর্ব কলোনী ৪নং ওয়ার্ড উত্তর আখাউড়া, মসজিদপাড়া, খালাজুড়া (আংশিক) ৫নং ওয়ার্ড বড় বাজার, দক্ষিণ আখাউড়া, পশ্চিম রাধানগর, সড়ক বাজার, মালদারপাড়া ৬নং ওয়ার্ড নারায়ণপুর, পূর্ব রাধানগর, কলেজপাড়া, খালাজুড়া (আংশিক) ৭নং ওয়ার্ড চন্দনসার, দেবগ্রাম শান্তিনগর, দক্ষিণ রাধানগর, সুবার পুকুর পাড়, উত্তর দেবগ্রাম, লালবাজার ৮নং ওয়ার্ড দক্ষিণ দেবগ্রাম ৯নং ওয়ার্ড পূর্ব দেবগ্রাম, তারাগন == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == আখাউড়া পৌরসভায় যোগাযোগের প্রধান সড়ক ঢাকা-আগরতলা সড়ক। সব ধরণের যানবাহনে যোগাযোগ করা যায়। এছাড়া রেল যোগেও ঢাকা, চট্টগ্রাম সহ দেশের বিভিন্ন জেলা শহর থেকে যাতায়াত করা যায়। == নদ-নদী == আখাউড়া পৌরসভার পশ্চিম প্রান্ত দিয়ে প্রবাহিত হচ্ছে তিতাস নদী। == দর্শনীয় স্থান == আখাউড়া রেলওয়ে জংশন –– বাংলাদেশের সর্ববৃহৎ রেলওয়ে জংশন। আখাউড়া স্মৃতি সৌধ কেন্দ্রীয় পৌর শহীদ মিনার খড়মপুর মাজার শরীফ তিতন শাহ (রহ.) মাজার (টানপাড়া) শের আলী শাহ (রহ.) মাজার (তারাগন) == কৃতি ব্যক্তিত্ব == আবদুর রহমান –– বীর প্রতীক খেতাবপ্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি == পৌর মেয়র: মোহাম্মদ তাকজিল খলিফা == আরও দেখুন == আখাউড়া উপজেলা আখাউড়া থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
