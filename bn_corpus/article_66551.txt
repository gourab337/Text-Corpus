'''মোক্ষপুর ইউনিয়ন''' বাংলাদেশের ময়মনসিংহ জেলার ত্রিশাল উপজেলার একটি ইউনিয়ন।'''''' == অবস্থান সীমানা == == ইতিহাস == == প্রশাসনিক এলাকা == ওয়াহেদপুর শিবনগর নারায়নপুর কানিবিল আব্দুল্লাহপুর টানচারা জিন্নতপুর বুড়িরপাড় রামনগর সৈয়দপুর নোয়াকান্দি হাদিপুর পুটিয়াপাড়া মাছুয়াবাদ রাজাপুর খালপাড় সুবিল পশ্চিম পোমকাড়া রাঘবপুর == আয়তন জনসংখ্যা == আয়তনঃ ৭০০৭ একর। জনসংখ্যাঃ পুরুষ ১১৬৬৫জন। মহলা ১১৪৬৭ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' ৪৫.৪৫ '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == সংযোগানন্দ গিরি স্মৃতি মন্দির পঞ্চবটী মহাশশ্মান হযরত কালাই শাহ (রঃ) এর মাজার == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' তালিকা ক্রমিক চেয়ারম্যানগণের নাম মেয়াদকাল ০১ ০২ ০৩ ০৪ ০৫ == আরও দেখুন == আমিরাবাড়ী ইউনিয়ন রামপুর ইউনিয়ন, ত্রিশাল সাখূয়া ইউনিয়ন, ত্রিশাল == তথ্যসূত্র == == বহিঃসংযোগ == মোক্ষপুর ইউনিয়ন- বাংলাদেশ জাতীয় তথ্য বাতায়ন ত্রিশাল উপজেলা উপজেলার ইউনিয়ন
