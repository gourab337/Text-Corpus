'''ময়নামতি''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার দক্ষিণাংশে ময়নামতি ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে মোকাম ইউনিয়ন, উত্তর-পশ্চিমে ভারেল্লা দক্ষিণ ইউনিয়ন, উত্তরে গোমতী নদী পীরযাত্রাপুর ইউনিয়ন, পূর্বে গোমতী নদী ষোলনল ইউনিয়ন, দক্ষিণ-পূর্বে কুমিল্লা আদর্শ সদর উপজেলার দুর্গাপুর উত্তর ইউনিয়ন কুমিল্লা সেনানিবাস এবং দক্ষিণে কুমিল্লা আদর্শ সদর উপজেলার কালিরবাজার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ময়নামতি ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
