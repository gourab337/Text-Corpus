'''চম্পকনগর''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == চম্পকনগর ইউনিয়নের আয়তন ৩৭২৭ একর (১৫.০৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে চম্পকনগর ইউনিয়নের মোট জনসংখ্যা ১৬,৩৮৮ জন। মোট পরিবার ৩,১৫২টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১০৮৭ জন। == ইতিহাস == চম্পকনগর ইউনিয়ন পূর্বে ইছাপুরা দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বিজয়নগর উপজেলার মধ্যাংশে চম্পকনগর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ইছাপুরা ইউনিয়ন, পূর্বে পাহাড়পুর ইউনিয়ন বিষ্ণুপুর ইউনিয়ন, দক্ষিণে পত্তন ইউনিয়ন এবং পশ্চিমে পত্তন ইউনিয়ন চর ইসলামপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চম্পকনগর ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
