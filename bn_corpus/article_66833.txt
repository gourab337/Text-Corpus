'''চান্দেরচর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == চান্দেরচর ইউনিয়ন পূর্বে চান্দেরচর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == হোমনা উপজেলার উত্তর-পূর্বাংশে চান্দেরচর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দুলালপুর ইউনিয়ন আছাদপুর ইউনিয়ন; দক্ষিণে ভাসানিয়া ইউনিয়ন মুরাদনগর উপজেলার রামচন্দ্রপুর দক্ষিণ ইউনিয়ন; পূর্বে মুরাদনগর উপজেলার রামচন্দ্রপুর দক্ষিণ ইউনিয়ন, রামচন্দ্রপুর উত্তর ইউনিয়ন শ্রীকাইল ইউনিয়ন এবং উত্তরে জেলার বাঞ্ছারামপুর উপজেলার ফরদাবাদ ইউনিয়ন ছলিমাবাদ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চান্দেরচর ইউনিয়ন হোমনা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
