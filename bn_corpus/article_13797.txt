বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত খাগড়াছড়ি সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == ভাইবোনছড়া ইউনিয়নের জনসংখ্যা প্রায় ৮০ হাজার। নারী-পুরুষের অনুপাত ১০০:১০৩। == অবস্থান সীমানা == খাগড়াছড়ি সদর উপজেলার সর্ব-উত্তরে ভাইবোনছড়া ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে পেরাছড়া ইউনিয়ন, পশ্চিমে মাটিরাঙ্গা উপজেলার গোমতি ইউনিয়ন, উত্তরে পানছড়ি উপজেলার লতিবান ইউনিয়ন এবং পূর্বে খাগড়াছড়ি পৌরসভা দীঘিনালা উপজেলার বোয়ালখালী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভাইবোনছড়া ইউনিয়ন খাগড়াছড়ি সদর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম খাগড়াছড়ি সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;কলেজ ভাইবোনছড়া কলেজ ;মাধ্যমিক বিদ্যালয় মিলেনিয়াম ভাইবোনছড়া উচ্চ বিদ্যালয় মুনিগ্রাম উচ্চ বিদ্যালয় ;নিম্ন মাধ্যমিক বিদ্যালয় ভাইবোনছড়া নিম্ন মাধ্যমিক বালিকা বিদ্যালয় ;প্রাথমিক বিদ্যালয় ভাইবোনছড়ার মুখ সরকারি প্রাথমিক বিদ্যালয় ভাইবোনছড়া সরকারি প্রাথমিক বিদ্যালয় ;কিন্ডারগার্টেন ভাইবোনছড়া আইডিয়াল স্কুল == যোগাযোগ ব্যবস্থা == ভাইবোনছড়া ইউনিয়নে যোগাযোগের একমাত্র মাধ্যম হচ্ছে সড়ক পথ। খাগড়াছড়ির চেঙ্গী স্কয়ার স্বনির্ভর এলাকা থেকে বাস, সিএনজি, অটোরিকশা বা মোটরবাইকযোগে ভাইবোনছড়া যাওয়া যায়। == খাল নদী == খাগড়াছড়ি, পানছড়ি, মহালছড়ির মত ভাইবোনছড়া গ্রামটিও গড়ে উঠেছে চেঙ্গী নদীর তীরে। চেঙ্গী নদী ছাড়া ইউনিয়নে আর কোন নদী নাই। == হাট-বাজার == ভাইবোনছড়া ইউনিয়নের প্রশাসনিক কেন্দ্র ভাইবোনছড়া বাজার। ইউনিয়নের প্রধান বাজারে হাট বসে মঙ্গলবার শনিবার। == দর্শনীয় স্থান == মায়াবিনী লেক ভাইবোনছড়া গার্ডার ব্রীজ রাবার বাগান == জনপ্রতিনিধি == == আরও দেখুন == খাগড়াছড়ি সদর উপজেলা খাগড়াছড়ি সদর থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
