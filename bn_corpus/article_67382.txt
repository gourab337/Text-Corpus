বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == গোপীনাথপুর ইউনিয়নের আয়তন ৭৬০৯ একর (৩০.৭৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে গোপীনাথপুর ইউনিয়নের মোট জনসংখ্যা ৩৫,৬৬৭ জন। মোট পরিবার ৬,৬৯০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১১৫৮ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার সর্ব-পূর্বে গোপীনাথপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কসবা পৌরসভা বিনাউটি ইউনিয়ন, উত্তরে আখাউড়া উপজেলার মনিয়ন্দ ইউনিয়ন এবং পূর্বে দক্ষিণে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == গোপীনাথপুর ইউনিয়ন কসবা উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == গোপীনাথপুর আলহাজ্ব শাহ্ আলম বিশ্ববিদ্যালয় কলেজ গোপীনাথপুর মালেক জোবেদা মেডিকেল কলেজ গোপীনাথপুর শহীদ বাবুল উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == গোপীনাথপুর ইউনিয়নের প্রধান হাট-বাজার হল গোপীনাথপুর বাজার। == দর্শনীয় স্থান == গোপীনাথপুর বাজার কেন্দ্রীয় জামে মসজিদ == কৃতি ব্যক্তিত্ব == আনোয়ার হোসেন –– বীর উত্তম খেতাবপ্রাপ্ত বীর মুক্তিযোদ্ধা। == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
