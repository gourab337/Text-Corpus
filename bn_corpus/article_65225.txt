'''মাইজগাঁও ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত ফেঞ্চুগঞ্জ উপজেলার একটি ইউনিয়ন। == অবস্থান == ফেঞ্চুগঞ্জ উপেজলার কিলোমিটার দক্ষিণে ফরিদপুর নয়াবাজার হইতে পূর্বে হাটুভাঙ্গা গ্রামে উপেজলা স্বাস্থ্য কেন্দ্র হাটুভাঙ্গা দারুস সুন্নাহ মাদ্রাসার পাশে অবস্থিত। == ইতিহাস == এই ইউনিয়নের কার্যক্রম ১৯৬০ ইংরেজি সন থেকে শুরু হয়। ইউনিয়নের প্রথম চেয়ারম্যান ছিলেন মোঃ সাজ্জাদুর রহমান চৌধুরী == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == আয়তন- ৫০৮৭ একর। জনসংখ্যা- পুরুষ ১৫০৫৪ জন। মহিলা- ১৪১৭৮ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার-''' ৬০.৮১%। ====== শিক্ষা প্রতিষ্ঠাণ ====== == নদী খাল == গাড়ুলীছড়া খাল == হাট-বাজার == কচুয়াবহর পাল বাড়ি বাজার ফরিদপুর নয়াবাজার সারকারখানা বাজার মাইজগাঁও, বাজার একাংশ == দর্শনীয় স্থান == ফেঞ্চুগঞ্জ বিদ্যুৎ কেন্দ্র বারাকাতুল্লাহ ইলেকট্রো ডায়নামিক্স শাহজালাল ফার্টিলাইজার কোম্পানী লিমিটেড ন্যাচারেল গ্যাস ফার্টিলাইজার ফ্যাক্টরী লিমিটিড == কৃতী ব্যক্তিত্ব == '''মাহমুদ উস সামাদ চৌধুরী''' == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান- ''ছুফিয়ানুল করিম চৌধুরী''''' তালিকা ক্রম নং চেয়ারম্যানের নাম সময়কাল ০১ মো: সজ্জাদ আলী চৌধুরী ১৯৬০ হতে ১৯৬৫ ০২ দেলোয়ার হোসেন চৌধুরী ১৯৬৫ হতে১৯৭৪ ০৩ মো সজ্জাদ আলী চৌধুরী ১৯৭৪ হতে ১৯৭৭ ০৪ মো: রসু মিয়া ১৯৭৭ হতে ১৯৮৪ ০৫ মো: মুহিবুর রহমান ১৯৮৪ হতে১৯৮৮ ০৬ মো: আজিজ-উস-সামাদ চৌধুরী ১৯৮৮ হতে১৯৯২ ০৭ আব্দুল বাছিত চৌধুরী (ভারপ্রাপ্ত) ১৯৯২ হতে ১৯৯২ ০৮ ছুফিয়ানুল করিম চৌধুরী ১৯৯২ হতে ১৯৯৮ ০৯ মো: তজমুল আলী ১৯৯৮ হতে ১৯৯৯ ১০ আকরাম হোসেন (ভারপ্রাপ্ত) ০২-০২-১৯৯৯ হতে ১২-০৬-১৯৯৯ ১১ মো: ফয়জুল ইসলাম মানিক ১৩-০৬-১৯৯৯ হতে ০৯-০৪-২০০৩ ১২ মো নুরুল ইসলাম(বাছিত) ১০-০৪-২০০৩ হতে ০৩-০৯-২০১৬ ১৩ আকরাম হোসেন (ভারপ্রাপ্ত) ৪-০৯-২০১৬ হতে ১৫-১০-২০১৬ ১৪ মো নুরুল ইসলাম (বাছিত) ১৬-১০-২০১৬ হতে ২০-০৫-২০১৮ ১৫ '''ছুফিয়ানুল করিম চৌধুরী''' '''বর্তমান''' == আরও দেখুন == উত্তর কুশিয়ারা ইউনিয়ন উত্তর ফেঞ্চুগঞ্জ ইউনিয়ন ফেঞ্চুগঞ্জ ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == ফেঞ্চুগঞ্জ উপজেলা -বাংলাপিডিয়া মাইজগাঁও ইউনিয়ন -বাংলাদেশ জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
