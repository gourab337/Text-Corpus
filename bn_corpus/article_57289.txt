বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সুবর্ণচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সুবর্ণচর উপজেলার সর্ব-দক্ষিণে মোহাম্মদপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর ক্লার্ক ইউনিয়ন, পশ্চিমে মেঘনা নদী লক্ষ্মীপুর জেলার রামগতি উপজেলার চর গাজী ইউনিয়ন, দক্ষিণে সন্দ্বীপ চ্যানেল হাতিয়া উপজেলার চর ঈশ্বর ইউনিয়ন এবং পূর্বে চট্টগ্রাম জেলার সন্দ্বীপ উপজেলার কালাপানিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মোহাম্মদপুর ইউনিয়ন সুবর্ণচর উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুবর্ণচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সুবর্ণচর উপজেলা সুবর্ণচর থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
