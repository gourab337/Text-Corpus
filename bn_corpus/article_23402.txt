'''বাদিয়াখালী ইউনিয়ন''' বাংলাদেশের গাইবান্ধা সদর উপজেলার অন্তর্গত একটি ইউনিয়ন। == আয়তন == এই ইউনিয়নের আয়তন ৫৭৪৮ একর (৯ বর্গ কিলোমিটার) == অবস্থান == গাইবান্ধা জেলা শহর থেকে ১৩ কিঃ মিঃ দক্ষিণে গাইবান্ধা সাঘাটা ফুলছড়ি সড়কে ঘেষেবাদিয়াখালী স্টেশন রোডে নুরুলগঞ্জ হাটের পশ্চিম পার্শ্বে তালুক রিফাইতপুর মৌজায় সদর উপজেলার বাদিয়াখালী ইউনিয়ন অবস্থিত। ইউনিয়নের পূর্বে উদাখালী ইউ.পি, দক্ষিণে পদুম শহর ইউ.পি, পশ্চিমে মনোহরপুর ইউ.পি, এবং উত্তরে বোয়ালী ইউ.পি। == প্রশাসনিক এলাকা == == ইতিহাস == == নামকরণ == কথিত আছে অনেক আগে বাদিয়ারা বসবাস ব্যবসা বাণিজ্য করত সে থেকেই এলাকার নাম করন হয় বাদিয়াখালী। == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == ইউনিয়নের বেশির ভাগ লোক কৃষি কাজ করে। == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র ==
