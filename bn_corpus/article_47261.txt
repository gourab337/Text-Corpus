'''গুলতাজ মেমোরিয়াল স্কুল এন্ড কলেজ''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি পৌরসভায় অবস্থিত। == ইতিহাস == ২০১০ সালে এটি প্রতিষ্ঠিত হয়। == পরিচালনা পরিষদ == == == প্রতিষ্ঠানের অধ্যক্ষ জনাব আবু আহমেদ। == অবকাঠামো == প্রতিষ্ঠান ভবনটি একটি ত্রিতল ভবন। == কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। উচ্চ মাধ্যমিক পর্যায়ে অধ্যয়নের সুযোগ রয়েছে প্রতিষ্ঠানে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৭৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
