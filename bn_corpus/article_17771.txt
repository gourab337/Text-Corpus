'''নারান্দিয়া ইউনিয়ন''' বাংলাদেশের টাঙ্গাইল জেলার অন্তর্গত কালিহাতি উপজেলার একটি ইউনিয়ন == আয়তন == এই ইউনিয়নের আয়তন ৩৮৯৫ একর (১৫,.৭৬বর্গ কিলোমিটার) বা ৩৮.৮৫ বর্গমাইল == ভৌগলিক উপাত্ত == == অবস্থান == পূর্বে বাংড়া ইউনিয়ন ঘাটাইল উপজেলা, পশ্চিমে সল্লা ইউনিয়ন, দক্ষিনে বাংড়া এলেঙ্গা পৌরসভা, উত্তরে ভূঞাপুর ঘাটাইল উপজেলা। == জনসংখ্যা == মোট ৩১৮৮১ জন- পুরুষ ১৬,২০০ জন, মহিলা: ১৫,৬৮১ জন। == শিক্ষা == == শিক্ষা প্রতিষ্ঠান == সরকারী প্রা: বিদ্যালয় ১৬ টি, উচ্চ বিদ্যালয় টি, স্কুল এন্ড কলেজ টি, আলীয়া মাদ্রাসা টি, বিএম কলেজ টি, কৃষি প্রশিক্ষনায়তন কলেজ টি এম কে জামান প্রিপারেটরি স্কুল, নারান্দিয়া সরকারি প্রাথমিক বিদ্যালয়, নারান্দিয়া টিআরকেএন উচ্চ মাধ্যমিক বিদ্যালয় এন্ড কলেজ, নগরবাড়ী কৃষি কলেজ, কারিগরি কলেজ == ইতিহাস == == হাট বাজার == নারান্দিয়া বাজার, সয়া হাট, পালিমা বাজার। == গ্রাম এবং মৌজা == মৌজার সংখ্যা ২৭ টি, গ্রামের সংখ্যা ৪৩ টি, খানার সংখ্যা ৭২৯৬ টি == কৃতী ব্যক্তিত্ব == == বিবিধ == স্বাস্থ্য পরিবার কল্যাণ কেন্দ্র টি, উপস্বাস্থ্য কেন্দ্র ১টি, মসজিদের সংখ্যা ৪৭ টি, মন্দিরের সংখ্যা ১২ টি, ডাকঘর: টি, এনজিও ১৫ টি, ভূমি অফিস টি, ব্যাংক টি, ক্লাবের সংখ্যা ২৩টি (রেজিষ্টিকৃত ৩টি), পাঠাগারের সংখ্যা ১২ টি == তথ্যসূত্র == ১. নারান্দিয়া উপজেলার ইউনিয়ন
