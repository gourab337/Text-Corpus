'''বেলছড়ি''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মাটিরাঙ্গা উপজেলার একটি ইউনিয়ন। == নামকরণ == এক সময়ে এই এলাকাটি প্রচুর বেলের জন্য প্রসিদ্ধ ছিল, যে কারণে এই ইউনিয়নের নাম বেলছড়ি নামকরণ করা হয়। == আয়তন == == জনসংখ্যা == বেলছড়ি ইউনিয়নের মোট জনসংখ্যা ১২,৬৯৬ জন। '''গ্রামভিত্তিক জনসংখ্যাঃ-''' ১নং ওয়ার্ডে ১৪৯১ জন ২নং ওয়ার্ডে ১০৬২ জন ৩নং ওয়ার্ডে ১৩৪৮ জন ৪নং ওয়ার্ডে ১৮৮৪ জন ৫নং ওয়ার্ডে ১৫৭৩ জন ৬নং ওয়ার্ডে ২৩৮৬ জন ৭নং ওয়ার্ডে ৮৫৭ জন ৮নং ওয়ার্ডে ১১৬৮ জন ৯নং ওয়ার্ডে ৯২৭ জন == অবস্থান সীমানা == মাটিরাঙ্গা উপজেলার দক্ষিণাংশে বেলছড়ি ইউনিয়নের অবস্থান। মাটিরাঙ্গা সদর উপজেলা থেকে ১৪ কিলোমিটার দক্ষিণে এই ইউনিয়ন অবস্থিত। ইউনিয়নের দক্ষিণে মাটিরাঙ্গা ইউনিয়ন, পূর্বে খাগড়াছড়ি সদর উপজেলার পেরাছড়া ইউনিয়ন, উত্তরে গোমতি ইউনিয়ন এবং পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বেলছড়ি ইউনিয়ন মাটিরাঙ্গা উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মাটিরাঙ্গা থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। বেলছড়ি ইউনিয়নের মোট ভোটার সংখ্যা: ৭৫৩২ জন। যার মধ্যে পুরুষ ভোটার ৩৮৭০ জন এবং মহিলা ভোটার রয়েছে ৩৬৬২ জন। এই ইউনিয়নে মোট ২,৬৬৮ টি পরিবার ৫১ টি গ্রাম/পাড়া তে বসবাস করে। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == এই ইউনিয়নে কাঁচা রাস্তা ১২ কি.মি., ব্রীজ সলিং: ১৮ কি.মি. পাঁকা রাস্তা ১২ কি.মি.। এছাড়া ৪টি ব্রীজ ২২ টি কাল্ভারট রয়েছে। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মাটিরাঙ্গা উপজেলা মাটিরাঙ্গা থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
