'''ফতুল্লা''' বাংলাদেশের নারায়ণগঞ্জ সদর উপজেলার অন্তর্গত একটি থানা। == অবস্থান == নারায়নগঞ্জ সদর উপজেলার একটি থানা ফতুল্লা। == প্রশাসনিক এলাকা == == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == শাহাদাত হোসেন, পেসার জাতীয় ক্রিকেট দলের সাবেক খেলোয়াড় *ডা কে খান এম ডি কলকাতা মেডিকেল কলেজ == দর্শনীয় স্থীন== == বিবিধ == == আরও দেখুন == নারায়ণগঞ্জ সদর উপজেলা; নারায়নগঞ্জ জেলা; ঢাকা বিভাগ। == তথ্যসূত্র == == বহিঃসংযোগ == জেলার জনবহুল স্থান সদর উপজেলা
