'''পাইন্দং উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার অন্তর্গত পাইন্দং ইউনিয়নে অবস্থিত। == ইতিহাস == ১৯৬২ সালে এটি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জনাব মোহাম্মদ রেজাউল করিমকে সভাপতি করে ১২ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক বাবু সঞ্জীব কুসুম চৌধুরী। এছাড়া আরো ১৩ জন শিক্ষক-শিক্ষিকা প্রতিষ্ঠানে কর্মরত আছেন। == অবকাঠামো == == শিক্ষা কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। বর্তমানে শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৭৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
