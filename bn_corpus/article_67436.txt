'''বিষ্ণুপুর''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == বিষ্ণুপুর ইউনিয়নের আয়তন ৪৪১২ একর (১৭.৮৫ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বিষ্ণুপুর ইউনিয়নের মোট জনসংখ্যা ১৬,৬৭২ জন। মোট পরিবার ৩,৪৩৭টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ৯৩৪ জন। == ইতিহাস == বিষ্ণুপুর ইউনিয়ন পূর্বে সিংগারবিল উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বিজয়নগর উপজেলার দক্ষিণাংশে বিষ্ণুপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পাহাড়পুর ইউনিয়ন, পশ্চিমে চম্পকনগর ইউনিয়ন পত্তন ইউনিয়ন, দক্ষিণে সিংগারবিল ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == বিষ্ণুপুর ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
