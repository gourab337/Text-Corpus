'''বরকামতা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বরকামতা ইউনিয়ন পূর্বে ১৫নং বরকামতা দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার বরকামতা ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে সুলতানপুর ইউনিয়ন, উত্তরে মোহনপুর ইউনিয়ন, পূর্বে বুড়িচং উপজেলার মোকাম ইউনিয়ন এবং দক্ষিণে চান্দিনা পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == বরকামতা ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ১৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
