'''বাঙ্গরা পূর্ব''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মুরাদনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == মুরাদনগর উপজেলার উত্তর-মধ্যাংশে বাঙ্গরা পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে পূর্বধইর পশ্চিম ইউনিয়ন, উত্তরে আকুবপুর ইউনিয়ন, উত্তর-পশ্চিমে শ্রীকাইল ইউনিয়ন, পশ্চিমে বাঙ্গরা পশ্চিম ইউনিয়ন এবং দক্ষিণে চাপিতলা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাঙ্গরা পূর্ব ইউনিয়ন মুরাদনগর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঙ্গরাবাজার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫১নং নির্বাচনী এলাকা কুমিল্লা-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মুরাদনগর উপজেলা বাঙ্গরাবাজার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
