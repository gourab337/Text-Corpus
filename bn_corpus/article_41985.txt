'''জামশা ইউনিয়ন''' ঢাকা বিভাগের মানিকগঞ্জ জেলার অন্তর্গত সিঙ্গাইর উপজেলার একটি ইউনিয়ন। == অবস্থান আয়তন == ইউনিয়নের আয়তন ৩১.৮৩ বর্গ কিলোমিটার। == ইতিহাস == == ভৌগোলিক == ইউনিয়নের পাশ দিয়ে বয়ে গেছে কালীগঙ্গা নদী। == প্রশাসন == এই ইউনিয়নের ওয়ার্ড সংখ্যা ০৯টি। মৌজার সংখ্যা ১০ টি, গ্রাম সংখ্যা ১৪টি। == জনসংখ্যা == ২০১১ সালের আদমশুমারী অনুযায়ী ইউনিয়নের লোকসংখ্যা ২৪,৭৪০ জন। == শিক্ষা == == কৃষি == == অর্থনীতি == == যোগাযোগ == == কৃতী ব্যক্তিত্ব == কান্দু খান == তথ্যসূত্র == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
