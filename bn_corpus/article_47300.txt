'''শান্তিরহাট উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার দাঁতমারা ইউনিয়নে অবস্থিত। == ইতিহাস == বিদ্যালয়টি আলহাজ্ব মাওলানা মোঃ ইউনুছ মেম্বারের একান্ত প্রচেষ্টায় এবং এলাকার গণ্যমান্য ব্যক্তিবর্গের সহায়তায় ১৯৭৯ সালে প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জানে আলম চেয়ারম্যানকে সভাপতি করে ১১ জন বিশিষ্ট একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == == অবকাঠামো == বিদ্যলয়ে ১টি দ্বিতল, ১টি একতলা ২টি আধাপাকা ভবন আছে। এছাড়া ১টি ছাত্রাবাস ১টি খেলার মাঠ আছে। == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == ২০১৯ সালে পাশের হার ৮২.২৩% == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
