'''মৈশাতুয়া''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মনোহরগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মনোহরগঞ্জ উপজেলার উত্তরাংশে মৈশাতুয়া ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ঝলম উত্তর ইউনিয়ন ঝলম দক্ষিণ ইউনিয়ন, দক্ষিণে খিলা ইউনিয়ন, পূর্বে লাকসাম উপজেলার গোবিন্দপুর ইউনিয়ন, উত্তরে লাকসাম উপজেলার কান্দিরপাড় ইউনিয়ন এবং উত্তর-পশ্চিমে লাকসাম উপজেলার মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মৈশাতুয়া ইউনিয়ন মনোহরগঞ্জ উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মনোহরগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মনোহরগঞ্জ উপজেলা মনোহরগঞ্জ থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
