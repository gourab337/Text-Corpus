'''রংপুর সদর''' বাংলাদেশের রংপুর জেলার অন্তর্গত একটি উপজেলা। == অবস্থান == রংপুর সদর উপজেলার আয়তন ৩৩০.৩৩ বর্গ কিঃমিঃ। উত্তরে গংগাচড়া উপজেলা, দক্ষিণে মিঠাপুকুর উপজেলা, পূর্বে কাউনিয়া পীরগাছা উপজেলা, পশ্চিমে তারাগঞ্জ উপজেলা বদরগঞ্জ উপজেলা। উপজেলা শহর ঘাঘট নদীর তীরে অবস্থিত। ১৫টি ওয়ার্ড ১০৯ মহল্লা নিয়ে গঠিত। আয়তন ৪২.৫৭ বর্গ কিঃমিঃ। == প্রশাসনিক এলাকা == রংপুর সদর থানা সৃষ্টি হয় ১৮৭৭ সালে। বর্তমানে এটি উপজেলা। পৌরসভা ১, ওয়ার্ড ১৫, ইউনিয়ন ১২, মৌজা ২৭০, গ্রাম ৩১১। == ইতিহাস == == জনসংখ্যার উপাত্ত == জনসংখ্যাঃ ২৫১৬৯৯; পুরুষ ৫২.৩১%, মহিলা ৪৭.৬৯%। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিঃমিঃ ৫৯১৩ জন। == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == ==ঐতিহাসিক নিদর্শন ঐতিহ্য== তাজহাট জমিদার বাড়ি ==তথ্যসূত্র== == আরও দেখুন == ২০১৭ রংপুর দাঙ্গা ==বহিঃসংযোগ== জেলার উপজেলা বিভাগের উপজেলা সদর উপজেলা
