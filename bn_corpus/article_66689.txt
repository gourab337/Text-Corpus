'''ইউসুফপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == ইউসুফপুর ইউনিয়ন পূর্বে ২নং বড়শালঘর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দেবিদ্বার উপজেলার উত্তরাংশে ইউসুফপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বড়শালঘর ইউনিয়ন, পূর্বে ব্রাহ্মণপাড়া উপজেলার শিদলাই ইউনিয়ন, দক্ষিণে সুবিল ইউনিয়ন রসুলপুর ইউনিয়ন এবং পশ্চিমে মুরাদনগর উপজেলার টনকী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইউসুফপুর ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
