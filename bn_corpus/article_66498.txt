'''পয়ালগাছা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বরুড়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == পয়ালগাছা ইউনিয়ন পূর্বে পয়ালগাছা উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বরুড়া উপজেলার দক্ষিণাংশে পয়ালগাছা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আদ্রা ইউনিয়ন ভাউকসার ইউনিয়ন, পূর্বে গালিমপুর ইউনিয়ন, দক্ষিণ-পূর্বে লাকসাম উপজেলার মুদাফফরগঞ্জ ইউনিয়ন, দক্ষিণে লক্ষ্মীপুর ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার কচুয়া উপজেলার আশরাফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পয়ালগাছা ইউনিয়ন বরুড়া উপজেলার আওতাধীন ১৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বরুড়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৬নং নির্বাচনী এলাকা কুমিল্লা-৮ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বরুড়া উপজেলা বরুড়া থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
