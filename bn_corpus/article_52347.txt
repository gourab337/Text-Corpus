'''বশিকপুর''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == বশিকপুর ইউনিয়নের আয়তন ১৪ বর্গ কিলোমিটার। == জনসংখ্যা == বশিকপুর ইউনিয়নের জনসংখ্যা ৪৯,৭৪২ জন। == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার উত্তরাংশে বশিকপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে দত্তপাড়া ইউনিয়ন, বাঙ্গাখাঁ ইউনিয়ন পার্বতীনগর ইউনিয়ন; পশ্চিমে উত্তর হামছাদী ইউনিয়ন; উত্তরে রামগঞ্জ উপজেলার করপাড়া ইউনিয়ন নোয়াখালী জেলার চাটখিল উপজেলার সাহাপুর ইউনিয়ন এবং পূর্বে নোয়াখালী জেলার চাটখিল উপজেলার রামনারায়ণপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বিরাহিমপুর তালতলা মোড় রোকনপুর গ্রামের সুপারি বিক্রেতা বশিকপুর ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == বশিকপুর ডিএসইউ কামিল মাদরাসা বিরাহিমপুর মাদরাসাতুল বানাত আল ইসলামীয়া দারুল ইরফান ইন্টারন্যাশনাল একাডেমি, বশিকপুর বশিকপুর স্কুল এন্ড কলেজ বশিকপুর মহিলা আলিম মাতরাসা রোকনপুর উচ্চ বিদ্যালয় বিরাহিমপুর উচ্চ বিদ্যালয় নন্দীগ্রাম উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
