'''জোড্ডা পশ্চিম''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার জোড্ডা পশ্চিম ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে আদ্রা দক্ষিণ ইউনিয়ন, পূর্বে জোড্ডা পূর্ব ইউনিয়ন, দক্ষিণে বটতলী ইউনিয়ন নোয়াখালী জেলার সেনবাগ উপজেলার ছাতারপাইয়া ইউনিয়ন এবং পশ্চিমে মনোহরগঞ্জ উপজেলার নাথেরপেটুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জোড্ডা পশ্চিম ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা== ২০১১ সালের আদমশুমারী অনুসারে এখানে শিক্ষার হার ৬৫%; যার মধ্যে পুরুষ ৬৮% এবং মহিলা ৬২% == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == চড়িয়া বাজার। == দর্শনীয় স্থান == রাজাপাড়া সালাউদ্দিন আহমেদ ভাইয়ের বাংলো। == কৃতি ব্যক্তিত্ব == মো সালাউদ্দিন আহমেদ (রাজনীতিবিদ সমাজ সেবক) মো নঈম নিজাম (সাংবাদিক রাজনীতি বিদ) আলহাজ্ব আবদুল গফুর ভূঁইয়া রাজনীতিবিদ, সমাজ সংস্কারক, সাবেক সাংসদ প্রতিষ্ঠাতা আবদুল গফুর ভূঁইয়া কলেজ *মোবাশের আলম ভুইয়া রাজনীতিবিদ, সমাজ সংস্কারক, সমাজ সেবক, == জনপ্রতিনিধি == জনাব মাসুদ রানা। == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
