'''চণ্ডিপুর''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রামগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগঞ্জ উপজেলার পশ্চিমাংশে চণ্ডিপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ইছাপুর ইউনিয়ন, পূর্বে রামগঞ্জ পৌরসভা, দক্ষিণ-পূর্বে লামচর ইউনিয়ন, দক্ষিণ-পশ্চিমে রায়পুর উপজেলার কেরোয়া ইউনিয়ন এবং পশ্চিমে চাঁদপুর জেলার ফরিদগঞ্জ উপজেলার রূপসা দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চণ্ডিপুর ইউনিয়ন রামগঞ্জ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৪নং নির্বাচনী এলাকা লক্ষ্মীপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রামগঞ্জ উপজেলা রামগঞ্জ থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
