বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা আদর্শ সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == কুমিল্লা আদর্শ সদর উপজেলার সর্ব-পশ্চিমে কালিরবাজার ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে কুমিল্লা সেনানিবাস কুমিল্লা সদর দক্ষিণ উপজেলার বিজয়পুর ইউনিয়ন, দক্ষিণে বরুড়া উপজেলার ভবানীপুর ইউনিয়ন, দক্ষিণ-পূর্বে বরুড়া উপজেলার আগানগর ইউনিয়ন, পশ্চিমে চান্দিনা উপজেলার বরকইট ইউনিয়ন, উত্তর-পশ্চিমে বুড়িচং উপজেলার মোকাম ইউনিয়ন এবং উত্তর-পূর্বে বুড়িচং উপজেলার ময়নামতি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কালিরবাজার ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা কোতোয়ালী থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৪নং নির্বাচনী এলাকা কুমিল্লা-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কুমিল্লা আদর্শ সদর উপজেলা কোতোয়ালী থানা, কুমিল্লা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == আদর্শ সদর উপজেলার ইউনিয়ন
