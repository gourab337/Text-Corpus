'''তোরাবগঞ্জ''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত কমলনগর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কমলনগর উপজেলার উত্তরাংশে তোরাবগঞ্জ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে চর কাদিরা ইউনিয়ন, দক্ষিণে হাজিরহাট ইউনিয়ন, পশ্চিমে চর মার্টিন ইউনিয়ন চর লরেন্স ইউনিয়ন এবং উত্তরে লক্ষ্মীপুর সদর উপজেলার তেওয়ারীগঞ্জ ইউনিয়ন নোয়াখালী জেলার নোয়াখালী সদর উপজেলার আণ্ডারচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == তোরাবগঞ্জ ইউনিয়ন কমলনগর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কমলনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৭নং নির্বাচনী এলাকা লক্ষ্মীপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ;মাদ্রাসা তোরাবগঞ্জ হামেদিয়া ফাজিল মাদ্রাসা সুফিয়া দারুল আমান ইসলামিয়া দাখিল মাদ্রাসা ;মাধ্যমিক বিদ্যালয় উদয়ন আদর্শ উচ্চ বিদ্যালয় তোরাবগঞ্জ উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == লক্ষ্মীপুর থেকে মিনি বাস বা টেক্সি যোগে দক্ষিণে তোরাবগঞ্জ বাজার। == খাল নদী == == হাট-বাজার == তোরাবগঞ্জ ইউনিয়নের প্রধান হাট-বাজার তোরাবগঞ্জ বাজার। এছাড়াও রয়েছে হাজী ফাজিল মিয়ার হাট, ইসলামগঞ্জ বাজার এবং রহিমগঞ্জ বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == আরও দেখুন == কমলনগর উপজেলা কমলনগর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
