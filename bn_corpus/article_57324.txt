'''ধানশালিক''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত কবিরহাট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কবিরহাট উপজেলার পূর্বাংশে ধানশালিক ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বাটইয়া ইউনিয়ন; পশ্চিমে কবিরহাট পৌরসভা ঘোষবাগ ইউনিয়ন; দক্ষিণে চাপরাশিরহাট ইউনিয়ন এবং পূর্বে কোম্পানীগঞ্জ উপজেলার চর ফকিরা ইউনিয়ন, চর কাঁকড়া ইউনিয়ন সিরাজপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ধানশালিক ইউনিয়ন কবিরহাট উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কবিরহাট থানার আওতাধীন। এটি জাতীয় সংসদের ২৭২নং নির্বাচনী এলাকা নোয়াখালী-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == লামছি প্রসাদ তাকিয়া বাজার, ধানশালিক বাজার, জনতা বাজার, চরমন্ডলিয়া বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == চেয়ারম্যান- এয়াকুব নবী। == আরও দেখুন == কবিরহাট উপজেলা কবিরহাট থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
