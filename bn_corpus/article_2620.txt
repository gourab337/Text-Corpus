বাংলাদেশের পটুয়াখালী জেলার অন্তর্গত একটি প্রশাসনিক এলাকা। == অবস্থান == এই উপজেলার উত্তরে- বাকেরগঞ্জ উপজেলা, পূর্বে পায়রা নদী, দক্ষিণে বরগুনা সদর উপজেলা পশ্চিমে বেতাগি উপজেলা অবস্থিত। == প্রশাসনিক এলাকা == মির্জাগঞ্জ উপজেলার ০৬ টি ইউনিয়ন হল মাধবখালী ইউনিয়ন মির্জাগঞ্জ ইউনিয়ন আমড়াগাছিয়া ইউনিয়ন দেউলী সুবিদখালী ইউনিয়ন কাকড়াবুনিয়া ইউনিয়ন এবং মসজিদবাড়ীয়া ইউনিয়ন == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == এয়ার ভাইস মার্সাল (অবঃ) আলতাফ হোসেন চৌধুরী, ইয়ার উদ্দিন খলিফা। == বিবিধ == == আরও দেখুন == ==তথ্যসূত্র== ==বহিঃসংযোগ== উপজেলা জেলার উপজেলা বিভাগের উপজেলা
