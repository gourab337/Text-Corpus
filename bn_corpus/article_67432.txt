'''ইছাপুরা''' বাংলাদেশের জেলার অন্তর্গত বিজয়নগর উপজেলার একটি ইউনিয়ন। == আয়তন == ইছাপুরা ইউনিয়নের আয়তন ৩১৫৮ একর (১২.৭৮ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ইছাপুরা ইউনিয়নের মোট জনসংখ্যা ১৪,৩০৮ জন। মোট পরিবার ২,৫৪০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১১২০ জন। == ইতিহাস == ইছাপুরা ইউনিয়ন পূর্বে ইছাপুরা উত্তর ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বিজয়নগর উপজেলার মধ্যাংশে ইছাপুরা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চান্দুরা ইউনিয়ন, পূর্বে হরষপুর ইউনিয়ন পাহাড়পুর ইউনিয়ন, দক্ষিণে চম্পকনগর ইউনিয়ন এবং পশ্চিমে চর ইসলামপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইছাপুরা ইউনিয়ন বিজয়নগর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বিজয়নগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == বিজয়নগর উপজেলা বিজয়নগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
