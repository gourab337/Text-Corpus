'''ভারেল্লা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত বুড়িচং উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == বুড়িচং উপজেলার ভারেল্লা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে পীরযাত্রাপুর ইউনিয়ন, দক্ষিণে ভারেল্লা দক্ষিণ ইউনিয়ন, পশ্চিমে দেবিদ্বার উপজেলার মোহনপুর ইউনিয়ন এলাহাবাদ ইউনিয়ন এবং উত্তরে গোমতী নদী ব্রাহ্মণপাড়া উপজেলার মালাপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ভারেল্লা ইউনিয়ন বুড়িচং উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বুড়িচং থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৩নং নির্বাচনী এলাকা কুমিল্লা-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বুড়িচং উপজেলা বুড়িচং থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
