'''গোবিন্দপুর দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত ফরিদগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফরিদগঞ্জ উপজেলার পশ্চিমাংশে গোবিন্দপুর দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে চর দুঃখিয়া পশ্চিম ইউনিয়ন ফরিদগঞ্জ দক্ষিণ ইউনিয়ন; পূর্বে ফরিদগঞ্জ দক্ষিণ ইউনিয়ন ফরিদগঞ্জ পৌরসভা; উত্তরে গোবিন্দপুর উত্তর ইউনিয়ন এবং পশ্চিমে চাঁদপুর সদর উপজেলার বালিয়া ইউনিয়ন, চান্দ্রা ইউনিয়ন হাইমচর উপজেলার আলগী দুর্গাপুর উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গোবিন্দপুর দক্ষিণ ইউনিয়ন ফরিদগঞ্জ উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফরিদগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৩নং নির্বাচনী এলাকা চাঁদপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == ফরিদগঞ্জ উপজেলা ফরিদগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
