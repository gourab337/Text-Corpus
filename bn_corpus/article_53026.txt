'''ছলিমগঞ্জ''' বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == ছলিমগঞ্জ ইউনিয়নের আয়তন ২২৯৮ একর (৯.৩০ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ছলিমগঞ্জ ইউনিয়নের মোট জনসংখ্যা ১৬,৪৬৪ জন। মোট পরিবার ৩,৩২৬টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৭৭০ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার পশ্চিমাংশে ছলিমগঞ্জ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বড়িকান্দি ইউনিয়ন, পূর্বে শ্যামগ্রাম ইউনিয়ন এবং দক্ষিণে পশ্চিমে বাঞ্ছারামপুর উপজেলার তেজখালী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছলিমগঞ্জ ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। ওয়ার্ডভিত্তিক ইউনিয়নের গ্রামগুলো হল: ওয়ার্ড নং গ্রামের নাম ১নং ওয়ার্ড বাড্ডা ২নং ওয়ার্ড চর বাড্ডা ৩নং ওয়ার্ড বন্দে বাহেরচর ৪নং ওয়ার্ড কাজীরগাঁও ৫নং ওয়ার্ড কাদৈর ৬নং ওয়ার্ড বাড়াইল ৭নং ওয়ার্ড নিলখী ৮নং ওয়ার্ড রাজনগর ৯নং ওয়ার্ড বাড়াইল কৈবর্ত্যপাড়া == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
