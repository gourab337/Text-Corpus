বাংলাদেশের জেলার অন্তর্গত নবীনগর উপজেলার একটি ইউনিয়ন। == আয়তন == শ্রীরামপুর ইউনিয়নের আয়তন ৩৬৫৪ একর (১৪.৭৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে শ্রীরামপুর ইউনিয়নের মোট জনসংখ্যা ১৯,১৬৯ জন। মোট পরিবার ৩,৮৪২টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১২৯৬ জন। == ইতিহাস == == অবস্থান সীমানা == নবীনগর উপজেলার মধ্যাংশে শ্রীরামপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে নবীনগর পশ্চিম ইউনিয়ন নবীনগর পৌরসভা, পূর্বে নবীনগর পৌরসভা, দক্ষিণ-পূর্বে ইব্রাহিমপুর ইউনিয়ন, দক্ষিণ-পশ্চিমে রসুল্লাবাদ ইউনিয়ন এবং পশ্চিমে শ্যামগ্রাম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == শ্রীরামপুর ইউনিয়ন নবীনগর উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নবীনগর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৭নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == কাজী আকবর উদ্দিন সিদ্দিক –– মুক্তিযুদ্ধের সংগঠক এবং গণপরিষদ সদস্য। == জনপ্রতিনিধি == == আরও দেখুন == নবীনগর উপজেলা নবীনগর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
