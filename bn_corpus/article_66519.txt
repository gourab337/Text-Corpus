'''বরকইট''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বরকইট ইউনিয়ন পূর্বে ৮নং চান্দিনা পূর্ব ইউনিয়ন নামে পরিচিত ছিল। ইউনিয়নের উত্তরাংশের কিছু অংশ চান্দিনা পৌরসভার আওতাধীন হওয়ায় বাকি অংশকে ৮নং বরকইট ইউনিয়ন নামে নতুন নামকরণ করা হয়। == অবস্থান সীমানা == চান্দিনা উপজেলার সর্ব-পূর্বে বরকইট ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে এতবারপুর ইউনিয়ন, উত্তরে চান্দিনা পৌরসভা বুড়িচং উপজেলার মোকাম ইউনিয়ন, পূর্বে কুমিল্লা আদর্শ সদর উপজেলার কালিরবাজার ইউনিয়ন বরুড়া উপজেলার আগানগর ইউনিয়ন এবং দক্ষিণে বরুড়া উপজেলার খোশবাস উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বরকইট ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। ইউনিয়নের গ্রামগুলো হল: বরকইট খিরাসার মোহনপুর শালচর খৈচারা তাম্বলিয়া == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == বরকইট উদয়ন উচ্চ বিদ্যালয় বরকইট সরকারি প্রাথমিক বিদ্যালয় খিরাসার মোহনপুর দাখিল মাদ্রাসা খিরাসার মোহনপুর সরকারি প্রাথমিক বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == বরকইট বাজার শালচর বাজার খিরাসার বাজার == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
