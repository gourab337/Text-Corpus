'''জাহানপুর আমজাদ আলী আবদুল হাদী ইন্সটিটিউশন''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার অন্তর্গত জাফতনগর ইউনিয়নের জাহানপুর গ্রামে অবস্থিত। == ইতিহাস == ১৯৩২ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জনাব জনাব হোসাইন মোহাম্মদ আবু তৈয়বকে সভাপতি করে ১১ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব আবুল বশর সিকদার। == অবকাঠামো == == শিক্ষা কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৮০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
