'''জয়পুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == জয়পুর ইউনিয়নের আয়তন ৬.৪১ বর্গ কিলোমিটার। এটি আয়তনের দিক থেকে কুমিল্লা জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == হোমনা উপজেলার দক্ষিণাংশে জয়পুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ঘারমোড়া ইউনিয়ন, উত্তরে ঘারমোড়া ইউনিয়ন ভাসানিয়া ইউনিয়ন, পূর্বে ভাসানিয়া ইউনিয়ন, দক্ষিণ-পূর্বে তিতাস উপজেলার কলাকান্দি ইউনিয়ন এবং দক্ষিণে তিতাস উপজেলার বলরামপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জয়পুর ইউনিয়ন হোমনা উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
