বাংলাদেশের চাঁদপুর জেলার অন্তর্গত চাঁদপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == ইব্রাহিমপুর ইউনিয়নের আয়তন ৪৭.৭৭ বর্গ কিলেমিটার। এটি আয়তনের দিক থেকে চাঁদপুর জেলার সবচেয়ে বড় ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == চাঁদপুর সদর উপজেলার পশ্চিমাংশে ইব্রাহিমপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চাঁদপুর পৌরসভা; পূর্বে লক্ষ্মীপুর মডেল ইউনিয়ন, মেঘনা নদী হানারচর ইউনিয়ন; দক্ষিণে হাইমচর উপজেলার নীলকমল ইউনিয়ন এবং পশ্চিমে মেঘনা নদী শরিয়তপুর জেলার ভেদরগঞ্জ উপজেলার চর সেনসাস ইউনিয়ন, তারাবুনিয়া দক্ষিণ ইউনিয়ন তারাবুনিয়া উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ইব্রাহিমপুর ইউনিয়ন চাঁদপুর সদর উপজেলার আওতাধীন ১১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাঁদপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬২নং নির্বাচনী এলাকা চাঁদপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == চাঁদপুর সদর উপজেলা চাঁদপুর সদর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
