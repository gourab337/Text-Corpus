'''সুয়াবিল উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার সুয়াবিল ইউনিয়নে অবস্থিত। == ইতিহাস == ১৯৯০ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব কাজী ফজলুল হক। এছাড়া আরো ২৫ জন শিক্ষক-শিক্ষিকা বিদ্যালয়ে কর্মরত আছেন। == অবকাঠামো == বিদ্যালয় ভবনটি একটি ত্রিতল ভবন। == শিক্ষা কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। বর্তমানে বার শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৮৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
