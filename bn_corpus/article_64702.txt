অন্য ব্যবহারের জন্য, দেখুন রঘুনাথপুর ইউনিয়ন। '''রঘুনাথপুর ইউনিয়ন''' বাংলাদেশের গোপালগঞ্জ জেলার সদর উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন-''' ৫৭৯৭ একর। '''মোট জনসংখ্যা-''' ২৭,৬৫১। মোট পুরুষ- ১৪৩০২ জন, মোট মহিলা- ১৩৩৪৯ জন (''২০০১সালের আদম সুমারী অনুযায়ী'')। == গ্রাম সমূহ == '''গ্রাম ৭টি''' রঘুনাথপুর সিলনা সিঙ্গারকুল পারকুশী নকড়িরচর আন্ধার কোঠা দীঘারকুল == হাট-বাজার == সিলনা বাজার সিলনা হাট == জনপ্রতিনিধি == == আরও দেখুন == শামসুল হক ফরিদপুরী শেখ মুজিবুর রহমান শেখ হাসিনা == তথ্যসূত্র == == বহিঃসংযোগ == রঘুনাথপুর ইউনিয়ন, গোপালগঞ্জ সদর উপজেলা -বাংলাদেশ জাতীয় তথ্য বাতায়ন। গোপালগঞ্জ সদর -বাংলাপিডিয়া। ''বাংলাদেশের ইউনিয়ন বিষয়ক এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' সদর উপজেলার ইউনিয়ন
