'''বড়তাকিয়া জাহেদিয়া দাখিল মাদ্রাসা''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার খৈয়াছড়া ইউনিয়নের পূর্ব খৈয়াছড়া গ্রামে মাদ্রাসাটি অবস্থিত। == ইতিহাস == মাদ্রাসাটি ১৯৯৯ সালে প্রতিষ্ঠিত হয়। বার আউলিয়ার সর্দার হযরত শাহ জাহেদ (রহ.) এর নামে মাদ্রাসাটির নামকরণ করা হয়। == পরিচালনা ব্যবস্থা == মাদ্রাসা পরিচালনার জন্য জনাব মুহাম্মদ আবদুল হাকিমকে সভাপতি করে ১৩ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == == মাদ্রাসার অধ্যক্ষ জনাব মোহাম্মদ আলাউদ্দীন। বর্তমানে দুই শতাধিক শিক্ষার্থী মাদ্রাসায় অধ্যয়নরত আছে। == অবকাঠামো == == পর্যায় == মাদ্রাসায় দাখিল পর্যায় পর্যন্ত পাঠদান করা হয়। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯৭.৭%। == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
