'''কাদলা''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত কচুয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কচুয়া উপজেলার মধ্যাংশে কাদলা ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে সহদেবপুর পশ্চিম ইউনিয়ন পালাখাল মডেল ইউনিয়ন, উত্তর-পূর্বে কচুয়া দক্ষিণ ইউনিয়ন, পূর্বে কচুয়া পৌরসভা, দক্ষিণ-পূর্বে কড়ইয়া ইউনিয়ন, দক্ষিণে হাজীগঞ্জ উপজেলার হাটিলা পশ্চিম ইউনিয়ন কালচোঁ উত্তর ইউনিয়ন এবং পশ্চিমে মতলব দক্ষিণ উপজেলার নারায়ণপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কাদলা ইউনিয়ন কচুয়া উপজেলার আওতাধীন ৮নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (চাঁদপুর) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
