'''দরবেশপুর''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রামগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগঞ্জ উপজেলার দক্ষিণাংশে দরবেশপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে করপাড়া ইউনিয়ন লামচর ইউনিয়ন, পশ্চিমে লামচর ইউনিয়ন, উত্তরে রামগঞ্জ পৌরসভা ভাদুর ইউনিয়ন, উত্তর-পূর্বে ভোলাকোট ইউনিয়ন এবং পূর্বে নোয়াখালী জেলার চাটখিল উপজেলার পরকোট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দরবেশপুর ইউনিয়ন রামগঞ্জ উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৪নং নির্বাচনী এলাকা লক্ষ্মীপুর-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মিজানুর রহমান == আরও দেখুন == রামগঞ্জ উপজেলা রামগঞ্জ থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
