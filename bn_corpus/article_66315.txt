'''পার্বতীপুর ইউনিয়ন''' বাংলাদেশের রাজশাহী বিভাগের চাঁপাইনবাবগঞ্জ জেলার গোমস্তাপুর উপজেলার ইউনিয়ন, একটি প্রশাসনিক এলাকা। == অবস্থান == চাঁপাইনবাবগঞ্জ জেলা শহর হতে উত্তরে ৩২ কি: মি: দুরে গোমস্তাপুর উপজেলা সদর হতে দক্ষিণে ১৬ কি: মি: দুরে রাস্তার পূর্ব জিনারপুর গড়বাড়ী বাজার সংলগ্ন পার্বতীপুর ইউনিয়ন পরিষদ টি অবস্তিত। == ইতিহাস == == প্রশাসনিক এলাকা == বড়দাদপুর রাইহোগ্রাম জিওল পাবর্তীপুর মাগুর শহর চানপুর সাতছালা হাটখোলা দিঘা পার্বতীপুর আড্ডা মালপুর ফুটনগর খরপা সোনাবর জিনারপুর শেরপুর শেখপুরা বিজয়সানজুরা আনুরা কাঁশরইল সুবইল দেওপুরা দামদীপুর বড়াইল গড়বাড়ী ভাটখৈর গোপিনাথপুর মৃধাপাড়া মেহেরপুর কৃষ্ণপুর করমজা ধর্মপুর টিকরামপুর খামারবাড়ী গোপিনাথপুর মাদ্রাসা পাড়া ছোটদাদপুর আন্ধরইল নিসকালীপুর আনারপুর নিমইল == আয়তন জনসংখ্যা == আয়তন- ১৮৭৪৭ একর। জনসংখ্যা- পুরুষ, ১৫৯৭৪ জন। মহিলা, ১৫৪৪৬ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' ৩৬.৯৪ '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == == জনপ্রতিনিধি == '''বর্তমান লিয়াকত আলী খান == আরও দেখুন == রহনপুর ইউনিয়ন গোমস্তাপুর ইউনিয়ন চৌডালা ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == পার্বতীপুর ইউনিয়ন -বাংলাদেশ জাতীয় তথ্য বাতায়ন গোমস্তাপুর উপজেলা বাংলাপিডিয়া উপজেলার ইউনিয়ন
