'''রায়শ্রী দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == শাহরাস্তি উপজেলার পূর্বাংশে রায়শ্রী দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে রায়শ্রী উত্তর ইউনিয়ন, পশ্চিমে মেহের দক্ষিণ ইউনিয়ন, দক্ষিণে চিতোষী পশ্চিম ইউনিয়ন চিতোষী পূর্ব ইউনিয়ন, দক্ষিণ-পূর্বে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার ঝলম দক্ষিণ ইউনিয়ন এবং পূর্বে কুমিল্লা জেলার মনোহরগঞ্জ উপজেলার ঝলম উত্তর ইউনিয়ন লাকসাম উপজেলার মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রায়শ্রী দক্ষিণ ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == বিজয়পুর উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
