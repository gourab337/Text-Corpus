'''পাঠাননগর''' বাংলাদেশের ফেনী জেলার অন্তর্গত ছাগলনাইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ছাগলনাইয়া উপজেলার সর্ব-পশ্চিমে পাঠাননগর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে মহামায়া ইউনিয়ন, ছাগলনাইয়া পৌরসভা রাধানগর ইউনিয়ন; দক্ষিণে ফেনী সদর উপজেলার মোটবী ইউনিয়ন; পশ্চিমে ফেনী সদর উপজেলার কাজিরবাগ ইউনিয়ন ফুলগাজী উপজেলার আনন্দপুর ইউনিয়ন এবং উত্তরে ফুলগাজী উপজেলার জিএমহাট ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাঠাননগর ইউনিয়ন ছাগলনাইয়া উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ছাগলনাইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ছাগলনাইয়া উপজেলা ছাগলনাইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
