'''হাইবারনেট''' ওআরএম (সংক্ষিপ্ত মধ্যে হাইবারনেট) জাভা প্রোগ্রামিং ভাষার জন্য একটি ম্যাপিং সরঞ্জাম। এটি একটি রিলেশনাল ডাটাবেস বস্তু ভিত্তিক ডোমেন মডেল ম্যাপিং জন্য একটি কাঠামো প্রদান করে। হাইবারনেট হ'ল সরাসরি, স্থায়ী ডেটাবেসকে উচ্চ-স্তরের বস্তুর হ্যান্ডলিং ফাংশনগুলির সাথে অ্যাক্সেস করে প্রতিস্থাপনের প্রতিবন্ধকতা মেলামেশে সমস্যা পরিচালনা করে। ==জাভা স্থিরতা API== নিজস্ব "নেটিভ" API ছাড়াও, হাইবারনেটটি জাভা পার্সিসেন্স API (JPA) স্পেসিফিকেশনের একটি বাস্তবায়ন। যেমন, জাভা SE অ্যাপ্লিকেশন, জাভা EE অ্যাপ্লিকেশন সার্ভার ইত্যাদি সহ যে কোন পরিবেশে এটি সহজেই ব্যবহার করা যেতে পারে। ==প্রকল্পের বিন্যাস== আপনাকে "/ src main resources" এর অধীনে লাগাতে হবে। আপনি সমস্ত মডেল-ম্যাপিং ফাইলগুলি একই ডিরেক্টরি অধীনে স্থাপন করুন যা আপনি POJO মডেল ক্লাসগুলি সংজ্ঞায়িত করে। ==অন্তর্বর্তী উপাদানসমূহ== তুলনামূলক ম্যাপিং অবজেক্ট টীকা এনটিটি ম্যানেজার হাইবারনেট ৩.২ সংস্করণ থেকে জাভা স্থিরতা API কাজের জন্য এর ব্যবহার হয় এনভার্স বস্তু গ্রিড ম্যাপার শার্ডস অনুসন্ধান সরঞ্জামসমূহ ভ্যালিডেটার মেটামডেল জেনারেটর এন হাইবারনেট ==সংস্করণের সমন্বয়== হাইবারনেট জাভা জেপিএ বা ১১ ২.২ ৫.৪ বা ১১ ২.২ ৫.৩ বা ১১ ২.২ ৫.২ ২.১ ৫.১.১৭ ২.১ ৫.১.১৬ বা ২.১ ৫.১.৩ বা বা ২.১ বা বা ২.১ ==আরো দেখুন== ==References== ==External links== Hibernate source code on zGrepCode প্ল্যাটফর্ম mapping enterprise platform বিষয়শ্রেণী:Red Hat software software frameworks
