'''মোল্লাকান্দি ইউনিয়ন''' বাংলাদেশের মুন্সিগঞ্জ জেলার অন্তর্গত মুন্সীগঞ্জ সদরের একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == ইউনিয়নের মোট আয়তন ৩৮৪৯ একর (৬.০২ বর্গমাইল)। মোল্লাকান্দি ইউনিয়নের দক্ষিনে মুন্সীগঞ্জ সদর উপজেলা, পূর্বে আধারা ইউনিয়ন, উত্তরে চরকেওয়ার ইউনিয়ন, দক্ষিণে শিলই ইউনিয়ন এবং পশ্চিমে কাঠাদিয়া ইউনিয়ন অবস্থিত। == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী মোল্লাকান্দি ইউনিয়নের মোট জনসংখ্যা ছিল ২৫৭৮৩ জন। এর মধ্যে ১৩২৮০ জন পুরুষ জন মহিলা। == ইতিহাস == == শিক্ষা == শিক্ষার হার ৪০.০১%। প্রাথমিক বিদ্যালয় ০৯টি, বেসরকারী রেজিঃ প্রাথমিক ০১টি, জুনিয়ার স্কুল ০১টি, বেসরকারী জুনিয়র স্কুল ০১টি, এবতেদায়ী মাদ্রাসা ০৩টি। == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র == বিভাগের উপজেলা
