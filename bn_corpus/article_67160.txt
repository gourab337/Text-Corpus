বাংলাদেশের জেলার অন্তর্গত বাঞ্ছারামপুর উপজেলার একটি ইউনিয়ন। == আয়তন == ছয়ফুল্লাকান্দি ইউনিয়নের আয়তন ৩৬৯৭ একর (১৪.৯৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে ছয়ফুল্লাকান্দি ইউনিয়নের মোট জনসংখ্যা ২৫,২৭১ জন। মোট পরিবার ৫,৩৪৫টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৬৮৯ জন। == ইতিহাস == ছয়ফুল্লাকান্দি ইউনিয়ন পূর্বে ছয়ফুল্লাকান্দি পশ্চিম ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == বাঞ্ছারামপুর উপজেলার পূর্বাংশে ছয়ফুল্লাকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে তেজখালী ইউনিয়ন; পশ্চিমে দরিয়াদৌলত ইউনিয়ন; দক্ষিণে ছলিমাবাদ ইউনিয়ন, রূপসদী ইউনিয়ন ফরদাবাদ ইউনিয়ন এবং পূর্বে দড়িকান্দি ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ছয়ফুল্লাকান্দি ইউনিয়ন বাঞ্ছারামপুর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম বাঞ্ছারামপুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৮নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == বাঞ্ছারামপুর উপজেলা বাঞ্ছারামপুর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
