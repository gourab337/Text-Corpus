'''টিন''' (ইংরেজি:Tin, উচ্চারন:/ˈtɪn/ tin) পর্যায় সারণীর ৫০তম মৌলিক পদার্থ। টিনের আণবিক সংকেত ''Sn''। টিনের ল্যাটিন নাম ''স্ট্যানাম'' (Latin: stannum) থেকেই এই সংকেতকরন। পর্যায় সারনীর গ্রুপ ১৪তে এই ধাতব মৌলিক পদার্থের অবস্থান। == রাসায়নিক বৈশিষ্ট্যসমূহ == === গাঠনিক ধর্ম === ===রাসায়নিক ধর্ম=== === আইসোটোপ সমূহ === == নামকরণ == == ইতিহাস == == রসায়ন == == প্রাচুর্যতা == == উৎপাদন == == ব্যবহারিক প্রয়োগ == == আরও দেখুন == == তথ্যসূত্র == == গ্রন্থপুঞ্জি == == বহিঃসংযোগ == পদার্থ
