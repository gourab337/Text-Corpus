'''বেলঘর উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লালমাই উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লালমাই উপজেলার বেলঘর উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে পেরুল দক্ষিণ ইউনিয়ন, উত্তরে ভুলইন দক্ষিণ ইউনিয়ন, পূর্বে চৌদ্দগ্রাম উপজেলার শ্রীপুর ইউনিয়ন শুভপুর ইউনিয়ন, দক্ষিণে বেলঘর দক্ষিণ ইউনিয়ন এবং দক্ষিণ-পশ্চিমে লাকসাম উপজেলার লাকসাম পূর্ব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বেলঘর উত্তর ইউনিয়ন লালমাই উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লালমাই থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লালমাই উপজেলা লালমাই থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
