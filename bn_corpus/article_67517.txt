'''কালিকচ্ছ''' বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == কালিকচ্ছ ইউনিয়নের আয়তন ৬৮৪৩ একর (২৭.৬৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে কালিকচ্ছ ইউনিয়নের মোট জনসংখ্যা ৩০,৬০২ জন। এর মধ্যে পুরুষ ১৪,৬১০ জন এবং মহিলা ১৫,৯৯২ জন। মোট পরিবার ৫,৮৪৪টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১১০৫ জন। == ইতিহাস == == অবস্থান সীমানা == সরাইল উপজেলার মধ্যাংশে কালিকচ্ছ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে নোয়াগাঁও ইউনিয়ন, দক্ষিণে সরাইল সদর ইউনিয়ন পানিশ্বর ইউনিয়ন, পশ্চিমে চুণ্টা ইউনিয়ন এবং উত্তরে পাকশিমুল ইউনিয়ন নাসিরনগর উপজেলার কুণ্ডা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কালিকচ্ছ ইউনিয়ন সরাইল উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে কালিকচ্ছ ইউনিয়নের সাক্ষরতার হার ৪৩.৪%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == অখিলচন্দ্র নন্দী –– ভারতীয় উপমহাদেশের ব্রিটিশ বিরোধী স্বাধীনতা আন্দোলনের বিপ্লবী। উল্লাসকর দত্ত –– ভারতীয় উপমহাদেশের ব্রিটিশ বিরোধী স্বাধীনতা আন্দোলনের বিপ্লবী। == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
