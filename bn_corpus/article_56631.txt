বাংলাদেশের ফেনী জেলার অন্তর্গত দাগনভূঁইয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == দাগনভূঁইয়া উপজেলার পূর্বাংশে মাতুভূঁইয়া ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে দাগনভূঁইয়া ইউনিয়ন, পশ্চিমে দাগনভূঁইয়া পৌরসভা পূর্ব চন্দ্রপুর ইউনিয়ন, উত্তরে জায়লস্কর ইউনিয়ন, পূর্বে ফেনী সদর উপজেলার বালিগাঁও ইউনিয়ন এবং দক্ষিণ-পূর্বে সোনাগাজী উপজেলার চর মজলিশপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মাতুভূঁইয়া ইউনিয়ন দাগনভূঁইয়া উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাগনভূঁইয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == মাতুভূঁইয়া ইউনিয়নের প্রধান হাট-বাজার হল বেকের বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাগনভূঁইয়া উপজেলা দাগনভূঁইয়া থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
