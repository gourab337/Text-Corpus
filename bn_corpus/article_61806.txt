'''জহিরাবাদ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত মতলব উত্তর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মতলব উত্তর উপজেলার জহিরাবাদ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে ফরাজিকান্দি ইউনিয়ন ফতেপুর পশ্চিম ইউনিয়ন, উত্তরে মোহনপুর ইউনিয়ন, পশ্চিমে এখলাছপুর ইউনিয়ন মেঘনা নদী এবং দক্ষিণে মেঘনা নদী চাঁদপুর সদর উপজেলার রাজরাজেশ্বর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জহিরাবাদ ইউনিয়ন মতলব উত্তর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মতলব উত্তর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬১নং নির্বাচনী এলাকা চাঁদপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == মতলব উত্তর উপজেলা মতলব উত্তর থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উত্তর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
