'''চর শাহী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার পূর্বাংশে চর শাহী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চন্দ্রগঞ্জ ইউনিয়ন হাজিরপাড়া ইউনিয়ন, পশ্চিমে দিঘলী ইউনিয়ন, দক্ষিণ-পশ্চিমে কুশাখালী ইউনিয়ন, দক্ষিণ-পূর্বে নোয়াখালী জেলার নোয়াখালী সদর উপজেলার চর মটুয়া ইউনিয়ন এবং পূর্বে নোয়াখালী জেলার বেগমগঞ্জ উপজেলার ছয়ানী ইউনিয়ন আলাইয়ারপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর শাহী ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
