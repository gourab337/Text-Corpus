'''খাড়েরা''' বাংলাদেশের জেলার অন্তর্গত কসবা উপজেলার একটি ইউনিয়ন। == আয়তন == খাড়েরা ইউনিয়নের আয়তন ৩২৯৩ একর (১৩.৩৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে খাড়েরা ইউনিয়নের মোট জনসংখ্যা ২০,৫৩৭ জন। মোট পরিবার ৩,৯৪৫টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ১৫৪১ জন। == ইতিহাস == == অবস্থান সীমানা == কসবা উপজেলার মধ্যাংশে খাড়েরা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বিনাউটি ইউনিয়ন, উত্তরে বাদৈর ইউনিয়ন, পশ্চিমে মেহারী ইউনিয়ন এবং দক্ষিণে কসবা পশ্চিম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == খাড়েরা ইউনিয়ন কসবা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কসবা থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৬নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == কসবা উপজেলা কসবা থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
