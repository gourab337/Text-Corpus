'''আমিরাবাড়ী ইউনিয়ন''' বাংলাদেশের ময়মনসিংহ জেলার ত্রিশাল উপজেলার একটি ইউনিয়ন।'''''' == অবস্থান সীমানা == == ইতিহাস == == প্রশাসনিক এলাকা == ১। গুজিয়াম কাকড়াচড়া ২। গুজিয়াম ধনিয়ারচালা, ৩। গুজিয়াম সরধনবাড়ী ৪। গুজিয়াম কাঠালীবন্ধ ৫। আমিরাবাড়ী নামপাড়া ৬। আমিরাবাড়ী টানপাড়া ৭। নারায়নপুর ৮। গোপালপুর ৯। ছনকান্দা ১০। চরখী বাড়ী ১১। ১ম খন্ড বড়গাঁও ১২। কুর্শানগর ১৩। বড়গাঁও == আয়তন জনসংখ্যা == আয়াতনঃ ৫১১১ একর (২০.৭০ বর্গ কি: মি:) মোট জনসংখ্যাঃ ১৮৮১৫। (পুরুষঃ ৯৬০১, মহিলাঃ ৯২১৪) == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার:''' ৪০.৮২। '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == কাশিগঞ্জ বাজার ৪০০ বছরের পুরানো বাজার == কৃতী ব্যক্তিত্ব == হোসেন আলী -সাবেক ডেপুটি ম্যাজিষ্ট, সচিব মন্ত্রী। == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' ''মোঃ আনিছুর রহমান'' তালিকা ক্রমিক চেয়ারম্যানগণের নাম মেয়াদকাল ০১ এডভোকেট আব্দুল হামিদ খান ০২ মিয়া নূরুল ইসলাম বাদশা ০৩ মোঃ এনামুল হক ভূইয়া ০৪ শ্রী সুরঞ্জন দেবনাথ (বলাই) ০৫ ''মোঃ আনিছুর রহমান'' বর্তমান == আরও দেখুন == মোক্ষপুর ইউনিয়ন রামপুর ইউনিয়ন, ত্রিশাল সাখূয়া ইউনিয়ন, ত্রিশাল == তথ্যসূত্র == == বহিঃসংযোগ == আমিরাবাড়ী ইউনিয়ন- বাংলাদেশ জাতীয় তথ্য বাতায়ন ত্রিশাল উপজেলা উপজেলার ইউনিয়ন
