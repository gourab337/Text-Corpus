এটি অক্ষর অনুযায়ী সাজানো '''পাকিস্তানি অভিনেতাগণের একটি তালিকা'''। == == আকমল খান আজফার রেহমান আজাব গুত আদনান খান আদনান জাফর আদনান মালিক আদনান সিদ্দিকি আদিল মুরাদ আদীল হাশমি আদীল হুসাইন আনোয়ার সোলানগি আফজাল খান আবদুল্লাহ ইজাজ আব্দুর রশিদ কারদার আবিদ আলি আবিদ কাশ্মিরি আবিদ খান আলমজেব মুজাহিদ আলী জাফর আসাদ মালিক আসিফ রাজা মির আহমেদ জাহানজেব আহসান খান == ই/ঈ == ইউসুফ খান ইকবাল থিবা ইজহার কাজি ইজাজ দুরানি ইমরান আব্বাস নাকভি ইমরান আসলাম ইমরান খান ইরফান খুসাত ইলিয়াস কাশ্মিরি ইসমাইল তারা ইসমাইল শাহিদ ইয়াসির নেওয়াজ ইয়াসির হুসাইন == উ/ঊ/এ/ও == উমার শরিফ উসমান পীরজাদা এমি খান ওমার শাহজাদ ওয়াকার জাকা ওয়াসীম আব্বাস ওয়াহীদ মুরাদ == ক/খ/গ/ঘ == কাজি ওয়াজিদ কাদের খান কাভি খান কামাল ইরানি কাশিফ খান কাশিফ মেহমুদ খালিদ আব্বাস দার গোলাম মহিউদ্দিন গোহার রশীদ == == জামশেদ আনসারি জাভেদ শেখ জামাল শাহ জিয়া মহীদ্দিন জুনাইদ খান জুহাব খান == ট/ত/দ/ন == তারিক আজিজ তারিক মুস্তাফা তৌকীর নাসির দানিশ তাইমূর দানিশ নেওয়াজ নাঈম হক নাঈম হাশমি নাজির আহমেদ খান নাজীবুল্লাহ আনজুম নাসিম ভিকি নূর হাসান রিজভি নোমান ইজাজ নোমান মাসুদ == == মালিক আনুখা মাহমুদ আলি মিকাল জুলফিকার মুনাওয়ার জারিফ মুর্তাজা হাসান মেহমুদ আখতার মেহমুদ আসলাম মোহসিন আব্বাস হায়দার মোহসিন খান মোহসিন মির্জা মোহাম্মাদ আলি মোহিব মির্জা == শ/স == শহিদ খান শাকিল হুসাইন খান শান শাহিদ শাফি মুহাম্মাদ শাহ শামিল খান শামূন আব্বাসি শাহূদ আলভি শেহজাদ শেখ সন্তোষ কুমার সাঈদ খান রঙ্গীলা সাজিদ হাসান সাজ্জাদ আলি সাদ হারূন সামি খান সামি শাহ সালমান শাহিদ সালাহউদ্দিন তূরফানি সাহির লোদি সিকান্দার জামান খান সিকান্দার রিজভি সিকান্দার সানাম সেলিম নাসির সেলীম শেখ == == হাবিব-উর-রেহমান হামজা আলি আব্বাসি হামিদ রানা হামিদ শেখ হাসান নিয়াজি হাসাম খান হুমায়ুন সাঈদ হুসাইন জাহাঙ্গীর == আরও দেখুন == পাকিস্তানি অভিনেত্রীগণের তালিকা তালিকা অনুযায়ী অভিনেতাগণের তালিকা
