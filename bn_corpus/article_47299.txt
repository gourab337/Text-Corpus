'''ফতেপুর লতিফ রহমান উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার জাফতনগর ইউনিয়নের ফতেপুর গ্রামে অবস্থিত। == ইতিহাস == ১৯৭২ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য ১১ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক জনাব কে এম মহিউদ্দীন। এছাড়া আরো জন শিক্ষক-শিক্ষিকা বিদ্যালয়ে কর্মরত আছেন। == অবকাঠামো == == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৭৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
