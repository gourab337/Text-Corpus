লক্ষদ্বীপের একটি দ্বীপ এটি '''ভারতের দ্বীপসমূহের তালিকা'''। ভারতে মোট ১,২০৮টি দ্বীপ (জনবসতিহীন দ্বীপ সহ) রয়েছে। == আন্দামান নিকোবর দ্বীপপুঞ্জ == === আন্দামান দ্বীপপুঞ্জ === উত্তর আন্দামান মধ্য আন্দামান দক্ষিণ আন্দামান লিটিল আন্দামান ইত্যাদি ===নিকোবর দ্বীপপুঞ্জ=== লিটিল নিকোবর গ্রেট নিকোবর == অন্ধ্রপ্রদেশ == ভবাণী দ্বীপ শ্রীহরিকোটা == আসাম == মাজুলি দ্বীপ == দমন দিউ == দিউ দ্বীপ == গোয়া == == গুজরাট == বেট দ্বারকা == কর্ণাটক == == কেরালা == == লাক্ষাদ্বীপ == কাবারট্টি মিনিকয় নেত্রানি == মহারাষ্ট্র == == ওড়িশা == আব্দুল কালাম দ্বীপ == পুদুচেরি == == পশ্চিমবঙ্গ == সাগর দ্বীপ দক্ষিণ তালপট্টি দ্বীপ কাকদ্বীপ নয়াচর চুকসার দ্বীপ ঘোড়ামারা দ্বীপ হেনরী আইল্যান্ড নতুন দ্বীপ জম্বু দ্বীপ == তথ্যসূত্র == ভূগোল
