বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == লাকসাম উপজেলার দক্ষিণাংশে গোবিন্দপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কান্দিরপাড় ইউনিয়ন, উত্তর-পূর্বে লাকসাম পৌরসভা, পূর্বে উত্তরদা ইউনিয়ন, দক্ষিণে মনোহরগঞ্জ উপজেলার খিলা ইউনিয়ন এবং পশ্চিমে মনোহরগঞ্জ উপজেলার মৈশাতুয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গোবিন্দপুর ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
