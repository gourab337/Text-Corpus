অন্য ব্যবহারের জন্য, দেখুন আলীনগর ইউনিয়ন (দ্ব্যর্থতা নিরসন)।আলীনগর ইউনিয়ন বাংলাদেশের বরিশাল বিভাগের ভোলা জেলার সদর উপজেলার একটি ইউনিয়ন। == অবস্থান == ভোলা সদর থেকে মাত্র ২.৫০ কিঃমিঃ দুরত্বে ০৮নং আলীনগর ইউনিয়ন পরিষদ কমপ্লেক্স ভবন। == ইতিহাস == == আয়তন জনসংখ্যা == '''আয়তন-''' ২৫.০০ বর্গ কিলোমিটার। '''সর্বমোট জনসংখ্যা-''' ৩৫৫১৪জন। == শিক্ষার হার == '''শিক্ষার হার-''' ৫০.৪১% == অর্থনীতি == এলকার শতকরা ৮০% লোকজন কৃষি সেচ কাজের উপর নির্ভর শীল। প্রতি বছর সেচ কৃষি খাত থেকে প্রচুর পরিমান কৃষকগন আয় করে থাকেন == দর্শনীয় স্থান == বীরশ্রেষ্ঠ মোস্তফা কামাল স্মৃতি জাদুঘর == গ্রামসমূহ == কাঠালী ছিফলী সাচিয়া রুহিতা মৌটুপী রুহিতা == হাট-বাজার == আলীনগর মাদ্রাসা বাজার পন্ডিতের পুল বাজার বেপারী বাজার নজু বেপারী বাজার তালতলী বাজার == খাল নদী == গজারীয়া খাল মাদ্রাসা বাজার খাল == কৃতী ব্যক্তিত্ব == বীরশ্রেষ্ঠ মোহাম্মদ মোস্তাফা কামাল == আরও দেখুন == আলীনগর ইউনিয়ন, বিয়ানীবাজার আলীনগর ইউনিয়ন, গোমস্তাপুর আলীনগর ইউনিয়ন, কমলগঞ্জ == তথ্যসূত্র == ''এই নিবন্ধটি অসম্পূর্ণ। আপনি চাইলে এটিকে সম্প্রসারিত করে উইকিপিডিয়াকে সাহায্য করতে পারেন।'' সদর উপজেলার ইউনিয়ন
