'''নাজিরহাট বালিকা উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার নাজিরহাট পৌরসভার দৌলতপুর এলাকায় অবস্থিত। == ইতিহাস == ১৯৭৫ সালে জনাব এম বাদশাহ আলম বিদ্যালয়টি প্রতিষ্ঠা করেন। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জনাব এম আনোয়ারুল আজিমকে সভাপতি করে ১১ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক বাবু দেবাশীষ কুমার নাথ। == অবকাঠামো == বিদ্যালয় ভবনটি একটি ত্রিতল ভবন। == শিক্ষা কার্যক্রম == এটি একটি বালিকা মাধ্যমিক বিদ্যালয়। বর্তমানে পাঁচ শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৮০%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
