'''গন্ধর্ব্যপুর দক্ষিণ''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার সর্ব-দক্ষিণে গন্ধর্ব্যপুর দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বড়কুল পূর্ব ইউনিয়ন গন্ধর্ব্যপুর উত্তর ইউনিয়ন, পূর্বে শাহরাস্তি উপজেলার সূচীপাড়া উত্তর ইউনিয়ন সূচীপাড়া দক্ষিণ ইউনিয়ন, দক্ষিণে লক্ষ্মীপুর জেলার রামগঞ্জ উপজেলার নোয়াগাঁও ইউনিয়ন কাঞ্চনপুর ইউনিয়ন এবং পশ্চিমে ফরিদগঞ্জ উপজেলার গুপ্টি পূর্ব ইউনিয়ন সুবিদপুর পূর্ব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গন্ধর্ব্যপুর দক্ষিণ ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
