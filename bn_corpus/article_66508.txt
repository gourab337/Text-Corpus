'''বাতাঘাসী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == বাতাঘাসী ইউনিয়ন পূর্বে সুহিলপুর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == চান্দিনা উপজেলার বাতাঘাসী ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে সুহিলপুর ইউনিয়ন, দক্ষিণে গল্লাই ইউনিয়ন, পূর্বে মহিচাইল ইউনিয়ন মাধাইয়া ইউনিয়ন, উত্তর-পূর্বে দেবিদ্বার উপজেলার ভানী ইউনিয়ন, উত্তরে মুরাদনগর উপজেলার বাবুটিপাড়া ইউনিয়ন এবং উত্তর-পশ্চিমে দাউদকান্দি উপজেলার ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাতাঘাসী ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
