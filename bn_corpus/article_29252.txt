এটি বাঙালি সাহিত্যিকদের বর্ণানুক্রমিক তালিকা। নামের অদ্যাক্ষর অনুযায়ী এই তালিকাটি তৈরী করা হয়েছে, এবং এই তালিকাটি কোন নির্দিষ্ট দেশ বা সময়কালের জন্য নয়। কবিতা, প্রবন্ধ-নিবন্ধ, উপন্যাস, নাটক, ছড়া, শিশুসাহিত্য, সমালোচনা সাহিত্য, কল্পবিজ্ঞান কিংবা সায়েন্স ফিকশন, জীবনী ইত্যাদিসহ সাহিত্যের যেকোনো শাখায় যিনি উল্লেখযোগ্য কাজ করেছেন এবং তিনি বাঙালি, তাঁকে তালিকায় রাখার চেষ্টা করা হয়েছে। 'উল্লেখযোগ্য কাজ' শব্দবন্ধটার মানদন্ড নির্ধারণে বাংলাদেশ পশ্চিমবঙ্গের বিশেষ বিশেষ বিশেষ বিবেচনায় নেয়া হয়েছে। ==অ== অক্ষয়কুমার বড়াল অক্ষয়চন্দ্র সরকার অক্ষয়কুমার দত্ত অক্ষয়চন্দ্র চৌধুরী অচিন্ত্যকুমার সেনগুপ্ত অজয় রায় অজিত দত্ত অজিতেশ বন্দোপাধ্যায় অজিতকুমার গুহ অজিতকুমার চক্রবর্তী অতীন বন্দ্যোপাধ্যায় অতুলপ্রসাদ সেন অতুলচন্দ্র গুপ্ত অদ্বৈত মল্লবর্মণ অনিতা অগ্নিহোত্রী অনিরুদ্ধ আলম অনিল মুখার্জি অনির্বাণ অনুরূপা দেবী অন্নদাশঙ্কর রায় অন্নদাচরণ খাস্তগীর অবধূত দুলালচন্দ্র মুখোপাধ্যায় অবনীন্দ্রনাথ ঠাকুর অভিজিৎ তরফদার অমর মিত্র অমলেন্দু দে অমিতাভ ঘোষ অমিতাভ দাশগুপ্ত অমূল্যকুমার দাশগুপ্ত অমিয় চক্রবর্তী অমিয় চৌধুরী অমিয় ভূষণ মজুমদার অমৃতলাল বসু অমর্ত্য সেন অরুণ মিত্র অরুণ রায় অরুণকুমার মিত্র অরূপরতন ভট্টাচার্য অলোকরঞ্জন দাশগুপ্ত অশোক মিত্র অসিতকুমার বন্দ্যোপাধ্যায় == == আকবর আলী আকবর হোসেন (সাহিত্যিক) আখতারুজ্জামান ইলিয়াস আজিজুর রহমান আজিজুল হাকিম আতিকুল হক চৌধুরী আনিস চৌধুরী আনিসুজ্জামান আনিসুল হক আনোয়ার পাশা আবু ইসহাক আবু জাফর শামসুদ্দীন আবুবকর সিদ্দিক আবুল কালাম মোহাম্মদ যাকারিয়া আবুল কাসেম (ভাষা সৈনিক) আবুল কাসেম ফজলুল হক আবুল বাশার আবদুল কাদির আবদুল গাফফার চৌধুরী আবদুল জব্বার আবদুল মান্নান সৈয়দ আবদুল হক (প্রাবন্ধিক) আবদুল হক চৌধুরী আবদুল হাই শিকদার আবদুল হাকিম আবদুল্লাহ আল মামুন আবদুল্লাহ আল মুতী শরফুদ্দিন আবদুশ শাকুর আবিদ আজাদ আবিদ আনোয়ার আবু জাফর ওবায়দুল্লাহ আবু হাসান শাহরিয়ার আবুল ফজল (সাহিত্যিক) আবুল হাসান আবুল হোসেন আবু হেনা মোস্তফা কামাল আরজ আলী মাতুব্বর আবুল মনসুর আহমেদ আমজাদ হোসেন আমীরুল ইসলাম আলী আনোয়ার আলী ইমাম আল মাহমুদ আল মুজাহিদী আলাউদ্দিন আল আজাদ আলাওল আশাপূর্ণা দেবী আশুতোষ চৌধুরী আশুতোষ মুখোপাধ্যায় আসকার ইবনে শাইখ আসাদ চৌধুরী আসাদ বিন হাফিজ আসাদ্দর আলী আহমদ শরীফ আহমেদ ছফা আহসান হাবীব আ. ন. ম. বজলুর রশীদ ==এ== এম হারুন-অর-রশিদ == == ইনামুল হক ইবরাহীম খাঁ ইমদাদুল হক মিলন ইসমাইল হোসেন সিরাজী == == ঈশ্বরচন্দ্র গুপ্ত ঈশ্বরচন্দ্র বিদ্যাসাগর == == ওমর আলী ওয়াহিদুল হক == == কমলকুমার মজুমদার করুণানিধান বন্দ্যোপাধ্যায় *কাজী ইমদাদুল হক কাজী কাদের নেওয়াজ কাজী নজরুল ইসলাম কাজী মোতাহার হোসেন কামাল চৌধুরী কামিনী রায় কালকূট কালিচন্দ্র রায় চৌধুরী কালিদাস রায় কালীনাথ দত্ত কালীপ্রসন্ন সিংহ কাহ্নপা কায়কোবাদ কুমুদরঞ্জন মল্লিক কোরেশী মাগন ঠাকুর কৃত্তিবাস ওঝা কৃষ্ণদয়াল বসু কৃষ্ণমোহন বন্দ্যোপাধ্যায় ক্ষীরোদপ্রসাদ বিদ্যাবিনোদ == == খান মোহাম্মদ ফারাবী খেলারাম চক্রবর্তী *খান মোহাম্মদ মঈনউদ্দীন == == গগন হরকরা গিরিশচন্দ্র ঘোষ গিরীন্দ্রমোহিনী দাসী গোবিন্দচন্দ্র দাস গোলাম মোস্তফা গোলাম মোহাম্মদ গৌড় অভিনন্দ == == চন্দ্রাবতী চণ্ডীদাস চণ্ডীচরণ মুনশী চিত্তরঞ্জন মাইতি == == জগদীশ গুপ্ত জসীম উদ্দীন জহির রায়হান জয়দেব জয় গোস্বামী জয়েন উদ্দীন জাহানারা ইমাম জিল্লুর রহমান সিদ্দিকী জীবনানন্দ দাশ জ্ঞান দাস ঠাকুর == == ঝড়েশ্বর চট্টোপাধ্যায় == == তনিকা সরকার তপন চক্রবর্তী তপন বন্দ্যোপাধ্যায় তপন রায়চৌধুরী তরু দত্ত তসলিমা নাসরিন তারকনাথ গঙ্গোপাধ্যায় তারকমোহন দাশ তারাপদ রায় তারাপদ সাঁতরা তারাশঙ্কর বন্দ্যোপাধ্যায় তাহমিমা আনাম তিলোত্তমা মজুমদার ত্রিদিব দস্তিদার ত্রিদিব মিত্র == == দক্ষিণারঞ্জন মিত্র মজুমদার দাদ আলী দানেশ পন্ডিত দিদারুল আলম দিব্যেন্দু পালিত দিলওয়ার দীনেশ দাশ দীনেশচন্দ্র সেন দীনেশচন্দ্র চট্টোপাধ্যায় দীনেশচন্দ্র ভট্টাচার্য দুদ্দু শাহ দুলেন্দ্র ভৌমিক দেবাশিস বন্দ্যোপাধ্যায় দেবী রায় দেবেশ রায় দেবেন্দ্রনাথ সেন দেওয়ান গোলাম মোর্তাজা দৌলত উজির বাহরাম খান দৌলত কাজী দ্বারকানাথ বিদ্যাভূষণ দ্বিজ বংশী দাস দ্বিজেন শর্মা দ্বিজেন্দ্রনাথ ঠাকুর দ্বিজেন্দ্রলাল রায় == == নবারুণ ভট্টাচার্য নবীনচন্দ্র সেন নরেন্দ্র দেব নসরুল্লাহ খাঁ নারায়ণ দেব নারায়ণ সান্যাল নাসরীন জাহান নিমাই ভট্টাচার্য নির্মলেন্দু গুণ নিশিকান্ত রায় চৌধুরী নীলমণি মুখোপাধ্যায় নীহাররঞ্জন গুপ্ত নীহাররঞ্জন রায় == == প্রফুল্ল রায় প্রেমেন্দ্র মিত্র == == ফকরুল আলম ফকির গরীবুল্লাহ ফজল শাহাবুদ্দীন ফররুখ আহমদ ফয়েজ আহমেদ ফালগুনী রায় ফাল্গুনী মুখোপাধ্যায় == == বঙ্কিমচন্দ্র চট্টোপাধ্যায় বন্দে আলী মিয়া বলদেব পালিত বলেন্দ্রনাথ ঠাকুর বড়ু চণ্ডীদাস বাণী বসু বাসন্তী দুলাল নাগচৌধুরী বাশার মাহমুদ বিজয় গুপ্ত বিজয়চন্দ্র মজুমদার বিজয়া মুখোপাধ্যায় বিনয় মজুমদার বিভূতিভূষণ বন্দ্যোপাধ্যায় বিভূতিভূষণ মুখোপাধ্যায় বিদ্যাপতি বিপিনবিহারী নন্দী বিমল কর বিমল মিত্র বিষ্ণু দে বিহারীলাল চক্রবর্তী বুদ্ধদেব গুহ বুদ্ধদেব বসু বেলাল চৌধুরী বেনজির আহমেদ == == ভারতচন্দ্র, রায়গুনাকর ভগীরথ মিশ্র == == মঈনুল আহসান সাবের মকবুলা মনজুর মঙ্গলাচরণ চট্টোপাধ্যায় মণিভূষণ ভট্টাচার্য মণীন্দ্র গুপ্ত মণীন্দ্র রায় মণীশ ঘটক মতি নন্দী মতিউল ইসলাম মদন মোহন তর্কালঙ্কার মনজুরে মওলা মঞ্জু সরকার মন্দাক্রান্তা সেন মনিরউদ্দীন ইউসুফ মনোজ মিত্র মনোমোহন বসু মফিদুল হক মবিন উদ্দিন আহমেদ মমতাজউদ্দীন আহমেদ মমতাজুর রহমান তরফদার মলয় রায়চৌধুরী মহাদেব সাহা মহাশ্বেতা দেবী মহীউদ্দিন মহেন্দ্রনাথ গুপ্ত ময়ুখ চৌধুরী মাইকেল মধুসূদন দত্ত মান্নান হীরা মানিক বন্দ্যোপাধ্যায় মাহবুব আলম মাহবুব উল আলম চৌধুরী মাহবুব তালুকদার মাহবুব সাদিক মাহফুজুর রহমান মাহমুদা খাতুন সিদ্দিকা মাহমুদুল হক মাহাবুব উল আলম মিন্নাত আলী মিহির সেনগুপ্ত মীর মশাররফ হোসেন মুকুন্দ দাস মুকুন্দরাম চক্রবর্তী মুনতাসীর মামুন মুন্সী রইসউদ্দীন মুনির চৌধুরী মুশাররাফ করিম মুস্তফা নুরুল ইসলাম মুহম্মদ আবদুল হাই মুহম্মদ আসাদ্দর আলী মুহম্মদ এনামুল হক মুহম্মদ কবির মুহম্মদ কুদরাত-এ-খুদা মুহম্মদ জাফর ইকবাল মুহম্মদ নুরুল হুদা মুহম্মদ মনসুর উদ্দিন *মুহাম্মদ মোজাম্মেল হক মুহম্মদ শহীদুল্লাহ মুহাম্মদ আকরাম খাঁ মুহাম্মদ ইব্রাহীম মুহাম্মদ হাবিবুর রহমান মৃণাল সেন মৈত্রেয়ী দেবী মোজাম্মেল হোসেন মিন্টু মোতাহের হোসেন চৌধুরী মোফাজ্জল হায়দার চৌধুরী মোবারক হোসেন খান মোবাশ্বের আলী মোহাম্মদ আবদুল কাইউম মোহাম্মদ আবদুল জববার মোহাম্মদ ওয়াজেদ আলী মোহাম্মদ কায়কোবাদ মোহাম্মদ নাসিরউদ্দিন মোহাম্মদ নাসিরউদ্দীন মোহাম্মদ নুরুল হুদা মোহাম্মদ মনিরুজ্জামান মোহাম্মদ মনিরুজ্জামান মিয়া মোহাম্মদ রফিক মোহিতলাল মজুমদার মোহাম্মদ মাহফুজউল্লাহ্ মোহাম্মদ মোদাব্বের মোহাম্মদ লুতফর রহমান মৌলভী আবদুল করিম == == যতীন্দ্রনাথ সেনগুপ্ত যতীন্দ্রমোহন বাগচী যোগীন্দ্রনাথ সরকার যোগেশচন্দ্র রায় বিদ্যানিধি == == রওশন ইজদানী রঙ্গলাল বন্দ্যোপাধ্যায় রজনীকান্ত সেন রফিক আজাদ রবীন্দ্র গুহ রবীন্দ্রনাথ ঠাকুর রমাপদ চৌধুরী রমেশচন্দ্র মজুমদার রাণী চন্দ রাধাগোবিন্দ নাথ রাধারমণ দত্ত রামনিধি গুপ্ত রামপ্রসাদ সেন রাশীদুল হাসান রুদ্র মুহম্মদ শহীদুল্লাহ রূপরাম চক্রবর্তী == == লালন শাহ লীলা মজুমদার == == শওকত আলী শওকত ওসমান শক্তি চট্টোপাধ্যায় শঙ্খ ঘোষ শক্তিপদ রাজগুরু শরৎচন্দ্র চট্টোপাধ্যায় শরদিন্দু বন্দ্যোপাধ্যায় শরদিন্দু দস্তিদার শংকর শঙ্কর সেনগুপ্ত শচীন্দ্রনাথ বসু শহীদ কাদরী শান্তিরঞ্জন ভট্টাচার্য শামসুদ্দীন আবুল কালাম শামসুর রাহমান শামসুল আলম শাহ মুহম্মদ সগীর শাহাদাৎ হোসেন শিবনারায়ণ রায় শিবনাথ শাস্ত্রী শিতালং শাহ শীর্ষেন্দু মুখোপাধ্যায় শেখ ভানু শেখ মুত্তালিব শৈলেন ঘোষ শ্যামল গঙ্গোপাধ্যায় শ্রীপারাবত শ্রীকুমার বন্দ্যোপাধ্যায় শিবরাম চক্রবর্তী == == সজনীকান্ত দাস সতীকান্ত গুহ সতীনাথ ভাদুড়ী সত্যজিৎ রায় সত্যেন্দ্রনাথ দত্ত সঞ্জীব চট্টোপাধ্যায় সন্দীপন চট্টোপাধ্যায় সন্ধ্যাকর নন্দী সমরেশ বসু সমরেশ মজুমদার সমরেন্দ্রনাথ সেন সমর সেন সমীর রায়চৌধুরী সমুদ্র গুপ্ত সাবিরিদ খান সিকান্দার আবু জাফর সুকান্ত গঙ্গোপাধ্যায় সুকান্ত ভট্টাচার্য সুকুমার রায় সুকুমার বসু সুকুমার সেন সুখময় ভট্টাচার্য সুচিত্রা ভট্টাচার্য সুধীন্দ্রনাথ দত্ত সুনীতিকুমার চট্টোপাধ্যায় সুনীল গঙ্গোপাধ্যায় সুফিয়া কামাল সুবিমল বসাক সুবিমল মিশ্র সুবোধ ঘোষ সুবোধচন্দ্র চক্রবর্তী সুভাষ মুখোপাধ্যায় সুরেশচন্দ্র বন্দ্যোপাধ্যায় সুশোভন সরকার সেলিনা পারভীন সেলিনা হোসেন সৈয়দ আলী আহসান সৈয়দ ইসমাইল হোসেন সিরাজী সৈয়দ ওয়ালিউল্লাহ সৈয়দ মুজতবা আলী সৈয়দ মুস্তাফা সিরাজ সৈয়দ শামসুল হক সৈয়দ শাহনুর সৈয়দ সুলতান সৈয়দ হামজা == == হর্ষ দত্ত হরিশ চন্দ্র মিত্র হাসন রাজা হাসান আজিজুল হক হাসান ফকরী হাসান হাফিজুর রহমান হিমানী বন্দ্যোপাধ্যায় হুমায়ুন আজাদ হুমায়ুন কবীর হুমায়ূন আহমেদ হেমচন্দ্র বন্দ্যোপাধ্যায় হেমেন্দ্র কুমার রায় == আরও দেখুন == বাঙালি কবিদের তালিকা বাংলা ঔপন্যাসিক তালিকা বাঙালি সাহিত্যিকদের তালিকা (কালানুক্রমিক) বাংলাদেশী কবি তালিকা বাংলা কবি তালিকা == তথ্যসূত্র == সাহিত্য উপন্যাস কবিতা কবি ঔপন্যাসিক বিজ্ঞান লেখক নাট্যকার লেখক সাহিত্যিক লেখক
