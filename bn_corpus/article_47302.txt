'''ভূজপুর ন্যাশনাল স্কুল এন্ড কলেজ''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার ভূজপুর ইউনিয়নে অবস্থিত। == ইতিহাস == ভূজপুর ন্যাশনাল উচ্চ বিদ্যালয়ে ২০১৩ সালে একাদশ দ্বাদশ শ্রেণী ভর্তির মাধ্যমে কলেজ প্রতিষ্ঠা হয়। == ব্যবস্থাপনা == == শিক্ষকবৃন্দ == প্রতিষ্ঠানের অধ্যক্ষ জনাব মুরতুজা এনামুল হক। == অবকাঠামো == প্রতিষ্ঠান ভবনটি একটি দ্বিতল ভবন। == শিক্ষা কার্যক্রম == প্রতিষ্ঠানে সহ-শিক্ষা ব্যবস্থা রয়েছে। উচ্চ মাধ্যমিক পর্যায় পর্যন্ত শিক্ষালাভের সুযোগ রয়েছে প্রতিষ্ঠানে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৯৫%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
