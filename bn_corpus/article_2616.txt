'''পটুয়াখালী সদর''' বাংলাদেশের পটুয়াখালী জেলার অন্তর্গত একটি প্রশাসনিক এলাকা। == অবস্থান == এই উপজেলার উত্তরে দুমকি উপজেলা, পূর্বে বাউফল উপজেলা গলাচিপা উপজেলা, দক্ষিণে গলাচিপা উপজেলা বরগুনা জেলার আমতলী উপজেলা এবং পশ্চিমে মির্জাগঞ্জ উপজেলা অবস্থিত। == প্রশাসনিক এলাকা == এই উপজেলার ইউনিয়ন সমুহঃ বদরপুর ইউনিয়ন লাউকাঠী ইউনিয়ন লোহালিয়া ইউনিয়ন কালিকাপুর ইউনিয়ন ছোট বিঘাই ইউনিয়ন জৈনকাঠী ইউনিয়ন বড় বিঘাই ইউনিয়ন আউলিয়াপুর ইউনিয়ন মরিচবুনিয়া ইউনিয়ন মাদারবুনিয়া ইউনিয়ন ইটবাড়ীয়া ইউনিয়ন এবং কমলাপুর ইউনিয়ন == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == ==তথ্যসূত্র== == আরও দেখুন == পটুয়াখালী জেলা ==বহিঃসংযোগ== সদর উপজেলা জেলার উপজেলা বিভাগের উপজেলা
