'''দিদার ইসলাম''' (জন্ম: ১৯৬৭) বাংলাদেশের একজন প্রখ্যাত প্রকৌশলী বিজ্ঞানী। তিনি কুইক রেডিওর আবিস্কারক হিসাবে সমধিক পরিচিত। == জন্ম বাল্যকাল == দিদারের জন্ম বাংলাদেশের মাগুরা জেলার সদর উপজেলার হাজিপুর গ্রামে। তার পিতার নাম ''আমিরুল ইসলাম'' এবং মাতা ''রওশন আরা''। == শিক্ষাজীবন == গ্রামের স্কুলে পড়াশোনা সমাপনান্তে দিদার যশোর জিলা স্কুল ঝিনাইদহ ক্যাডেট কলেজে পড়াশোনা করেন। এরপর বাংলাদেশ প্রকৌশল বিশ্ববিদ্যালয় (বুয়েট) থেকে ১৯৯২ সালে তড়িৎকৌশলে স্নাতক পাস করেন। == কর্মজীবন == পড়াশোনা শেষ করে তিনি বাংলাদেশ প্রকৌশল বিশ্ববিদ্যালয় (বুয়েট)-এ শিক্ষক হিসাবে যোগদান করেন। == অবদান == == সম্মাননা == == গ্রন্থাবলী == == আরও দেখুন == বাংলাদেশ প্রকৌশল বিশ্ববিদ্যালয় ঝিনাইদহ ক্যাডেট কলেজ কুইক রেডিও == সহায়ক গ্রন্থপঞ্জি == == তথ্যসূত্র == == বহি:সংযোগ == জেলার ব্যক্তি জন্ম ব্যক্তি বিজ্ঞানী উদ্যোক্তা ব্যক্তিত্ব জিলা স্কুলের প্রাক্তন শিক্ষার্থী
