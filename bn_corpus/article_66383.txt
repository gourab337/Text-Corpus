'''বাকই''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লাকসাম উপজেলার সর্ব-উত্তরে বাকই ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মুদাফফরগঞ্জ ইউনিয়ন কান্দিরপাড় ইউনিয়ন, পূর্বে লালমাই উপজেলার পেরুল দক্ষিণ ইউনিয়ন, উত্তরে লালমাই উপজেলার বাকই উত্তর ইউনিয়ন এবং পশ্চিমে বরুড়া উপজেলার গালিমপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাকই ইউনিয়ন লাকসাম উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
