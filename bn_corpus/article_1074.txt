'''ক্ষেতলাল উপজেলা''' বাংলাদেশের জয়পুরহাট জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই উপজেলার উত্তরে পাঁচবিবি উপজেলা, পূর্বে কালাই উপজেলা বগুড়া জেলার শিবগঞ্জ উপজেলা, দক্ষিনে আক্কেলপুর উপজেলা বগুড়া জেলার দুপচাঁচিয়া উপজেলা পশ্চিমে জয়পুরহাট সদর উপজেলা == ইতিহাস == ১৭৯৩ সালে ক্ষেতলাল অবিভক্ত দিনাজপুর জেলার থানা হিসেবে প্রতিষ্ঠিত হয়। == ভৌগোলিক উপাত্ত == == ভাষা সংষ্কৃতি == সবাই বাংলা ভাষায় কথা বলে। সব মুসলিম তবে কিছু কিছু হিন্দু আছে। == প্রশাসনিক এলাকা == ইউনিয়ন সমূহ আলমপুর ইউনিয়ন তুলশীগংগা ইউনিয়ন (অবলুপ্ত ক্ষেতলাল সদর) বড়তারা ইউনিয়ন বড়াইল ইউনিয়ন মামুদপুর ইউনিয়ন == স্বাস্থ্য == উন্নত চিকিৎসা সেবার জন্য বর্তমানে কাজ চলছে == শিক্ষা == নিম্ন মাধ্যমিক শিক্ষায় মেয়েরা এগিয়ে আছে। == কৃষি == এখানে প্রচুর পরিমাণ আলু ধান উৎপাদিত হয়। এই থানায় উৎপাদিত ফসল অর্থনীতিতে গুরুত্বপূর্ণ ভুমিকা রাখে। == অর্থনীতি == সারা বাংলাদেশের প্রায় একতৃতীয়াংশ আলুর চাহিদা এখান থেকে যোগান দেওয়া হয়ে থাকে। == যোগাযোগ ব্যবস্থা == খুব ভালো। কিন্তু রেলযোগাযোগ নাই। == কৃতি ব্যক্তিত্ব == জরিপ চলছে,পরে জানানো হবে। == দর্শনীয় স্থান স্থাপনা == ১. হিন্দা মসজিদ ২. অাছরাঙ্গা দিঘী ৩. বিলের ঘাট(রাখালীয়া ব্রীজ) ৪. উপজেলা চত্বর ৫.বিনাই বড়াইল ধানের ক্ষেত তুলসিগঙ্গা হারাবতি নদী == বিবিধ == == আরও দেখুন == বাংলাদেশের উপজেলাসমূহ; জয়পুরহাট জেলা; রাজশাহী বিভাগ। == তথ্যসুত্র == ==বহিঃসংযোগ== জেলার উপজেলা বিভাগের উপজেলা উপজেলা
