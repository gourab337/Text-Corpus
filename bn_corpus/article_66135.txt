'''কাশিনগর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চৌদ্দগ্রাম উপজেলার একটি ইউনিয়ন। ==আয়তন== == জনসংখ্যা == == অবস্থান সীমানা == চৌদ্দগ্রাম উপজেলার কাশিনগর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে উজিরপুর ইউনিয়ন, দক্ষিণ-পূর্বে কালিকাপুর ইউনিয়ন, দক্ষিণে শ্রীপুর ইউনিয়ন, পশ্চিমে লালমাই উপজেলার ভুলইন উত্তর ইউনিয়ন, উত্তর-পশ্চিমে কুমিল্লা সদর দক্ষিণ উপজেলার বারপাড়া ইউনিয়ন এবং উত্তরে কুমিল্লা সদর দক্ষিণ উপজেলার জোড়কানন পশ্চিম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কাশিনগর ইউনিয়ন চৌদ্দগ্রাম উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চৌদ্দগ্রাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৯নং নির্বাচনী এলাকা কুমিল্লা-১১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চৌদ্দগ্রাম উপজেলা চৌদ্দগ্রাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
