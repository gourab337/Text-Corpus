'''আদ্রা উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার পশ্চিমাংশে আদ্রা উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে হেসাখাল ইউনিয়ন, দক্ষিণে আদ্রা দক্ষিণ ইউনিয়ন, পশ্চিমে মনোহরগঞ্জ উপজেলার খিলা ইউনিয়ন এবং উত্তরে লাকসাম উপজেলার উত্তরদা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আদ্রা উত্তর ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ১৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == ১. শাকতলী উচ্চ বিদ্যালয় ২. বেলঘর গোসাই বাজার উচ্চ বিদ্যালয় ৩. দক্ষিণ শাকতলী সরকারি প্রাথমিক বিদ্যালয় ৪. হলি কেয়ার একাডেমি, শাকতলী ৫. শিশু বিদ্যা নিকেতন, আলীগঞ্জ বাজার ৬. আদ্রা সরকারি প্রাথমিক বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
