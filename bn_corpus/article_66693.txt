'''রসুলপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দেবিদ্বার উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দেবিদ্বার উপজেলার উত্তরাংশে রসুলপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ইউসুফপুর ইউনিয়ন, পূর্বে দক্ষিণে সুবিল ইউনিয়ন, দক্ষিণ-পশ্চিমে দেবিদ্বার পৌরসভা এবং পশ্চিমে মুরাদনগর উপজেলার নবীপুর পূর্ব ইউনিয়ন টনকী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রসুলপুর ইউনিয়ন দেবিদ্বার উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দেবিদ্বার থানার আওতাধীন। এটি জাতীয় সংসদের ২৫২নং নির্বাচনী এলাকা কুমিল্লা-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দেবিদ্বার উপজেলা দেবিদ্বার থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
