'''মির্জানগর''' বাংলাদেশের ফেনী জেলার অন্তর্গত পরশুরাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == পরশুরাম উপজেলার সর্ব-উত্তরে মির্জানগর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে চিথলিয়া ইউনিয়ন, পূর্বে পরশুরাম পৌরসভা এবং উত্তরে পশ্চিমে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == মির্জানগর ইউনিয়ন পরশুরাম উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম পরশুরাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == *মির্জানগর তৌহিদ একাডেমী। *কালিকাপুর বাশারাত উল্লাহ উচ্চ বিদ্যালয়। *সুবার বাজার ইসলামিয়া ফাজিল মাদ্রাসা। *বটতলী ইসলামিয়া আলীম মাদ্রাসা। *ডি.এম. সাহেব নগর সরকারী প্রাথমিক বিদ্যালয়। *মির্জানগর সরকারি প্রাথমিক বিদ্যালয়। *সম্রাট একাডেমী *সুবার বাজার বিদ্যানিকেতন। == যোগাযোগ ব্যবস্থা == == খাল নদী == *সিলোনিয়া *মুহুরি নদী *মির্জানগর খাল == হাট-বাজার == *সুবার বাজার। *বটতলী বাজার। *তারামিয়া বাজার। *তুলাতলী বাজার। *সাহেব নগর *মেলাঘর == দর্শনীয় স্থান == *মহেশপুস্করনী রাবার বাগান। *মুহুরির চর। *জয়চাঁদপুর। == জনপ্রতিনিধি == ইউনিয়ন চেয়ারম্যান জনাব নুরুজ্জামান ভুট্টো। == আরও দেখুন == পরশুরাম উপজেলা পরশুরাম থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
