== ১৮৫৮ == '''জুলাই:''' ময়মনসিংহ শহরে স্বায়ত্তশাসন প্রতিষ্ঠিত হয়। == ১৮৫৯ == ময়মনসিংহ শহরের অধিবাসীরা স্বায়ত্তশাসন প্রত্যাহারের আবেদন জানান। == ১৮৬০ == কিশোরগঞ্জ মহকুমা স্থাপিত হয়। == ১৮৬১ ৬২ == ইংরেজ ওয়াইজ সাহেবের সাথে চাকলাদারদের বড় রকমের দাঙ্গা হয়। == ১৮৬৩ == ময়মনসিংহ সদর থানায় দাতব্য চিকিৎসালয় স্থাপন। == ১৯৭১ == ডিসেম্বর ১০ পাকিস্তানী সেনাবাহিনীকে পরাজিত করার মাধ্যমে ময়মনসিংহ শহর মুক্তি লাভ করে। == তথ্যসূত্র == ময়মনসিংহের ইতিহাস (বই) শ্রীকেদারনাথ মজুমদার == আরও দেখুন == ময়মনসিংহ ময়মনসিংহ জেলা ময়মনসিংহ সদর উপজেলা
