'''চারিগ্রাম ইউনিয়ন''' ঢাকা বিভাগের মানিকগঞ্জ জেলার অন্তর্গত সিঙ্গাইর উপজেলার একটি ইউনিয়ন। == অবস্থান আয়তন == চারিগ্রাম ইউনিয়নের উত্তরে সিংগাইর সদর ইউনিয়ন পরিষদ, দক্ষিণে নবাবগঞ্জ ইউনিয়ন পরিষদ, পূর্বে সায়েস্তা ইউনিয়ন পরিষদ, এবং পশ্চিমে জামশা ইউনিয়ন পরিষদ। ইউনিয়নের আয়তন ১৪.৭০ বর্গ কিলোমিটার। == ইতিহাস == == ভৌগোলিক উপাত্ত == == প্রশাসনিক উপাত্ত == ওয়ার্ড সংখ্যাঃ ০৯ টি (ঘ) মৌজার সংখ্যাঃ ১৯ টি == জনসংখ্যার উপাত্ত == ২০১১ সালের আদম শুমারি অনুযায়ী ইউনিয়নের লোকসংখ্যা ২৫,৪৩৯ জন (পুরষ ১২৭৩৪ জন মহিলা -১২৭০৫ জন)। == শিক্ষা == == কৃষি == == অর্থনীতি == == যোগাযোগ == == কৃতী ব্যক্তিত্ব == == তথ্যসূত্র == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
