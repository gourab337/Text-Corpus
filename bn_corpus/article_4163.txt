'''ময়মনসিংহ সদর''' বাংলাদেশের ময়মনসিংহ জেলার অন্তর্গত একটি উপজেলা। == অবস্থান == এই উপজেলার উত্তরে তারাকান্দা উপজেলা, দক্ষিণে ত্রিশাল উপজেলা, পূর্বে গৌরীপুর উপজেলা ঈশ্বরগঞ্জ উপজেলা, পশ্চিমে মুক্তাগাছা উপজেলা ফুলবাড়িয়া উপজেলা অবস্থিত। == প্রশাসনিক এলাকা == ময়মনসিংহ সদর উপজেলায় ১৩টি ইউনিয়ন রয়েছে। সেগুলো হচ্ছে অষ্টধার, কুষ্টিয়া, বোররচর, পরানগঞ্জ, সিরতা, চরঈশ্বরদিয়া, চরনিলক্ষীয়া, আকুয়া, খাগডহর, দাপুনিয়া, ঘাগড়া, ভাবখালী এবং বয়রা ইউনিয়ন। == নদীসমূহ == ময়মনসিংহ সদর উপজেলার দাপুনিয়া বাজার থেকে তোলা সুতিয়া নদীর দৃশ্য ময়মনসিংহ সদর উপজেলায় অনেকগুলো নদী আছে। সেগুলো হচ্ছে পুরাতন ব্রহ্মপুত্র নদী, পাগারিয়া নদী, সুতিয়া নদী, নাগেশ্বরী নদী, সিরখালি নদী, চোরখাই নদী, বাড়েরা নদী হিংরাজানি নদী। == শিক্ষা == এখানে ১টি বিশ্ববিদ্যালয় (বাংলাদেশ কৃষি ২টি মেডিক্যাল কলেজ (ময়মনসিংহ মেডিকেল কলেজ কম্যুনিটি বেজড মেডিক্যাল কলেজ), ১টি প্রকৌশল কলেজ (ময়মনসিংহ ইঞ্জিনিয়ারিং কলেজ), ১টি ক্যাডেট কলেজ (ময়মনসিংহ গার্লস ক্যাডেট কলেজ) ২টি শিক্ষক প্রশিক্ষণ কলেজ আনন্দ মোহন কলেজ ছাড়াও আরো অসংখ্য শিক্ষা প্রতিষ্ঠান রয়েছে। == ইতিহাস == == জনসংখ্যার উপাত্ত == ২০১১ সালের আদমশুমারি অনুযায়ী মোট জনসংখ্যা ৬,৭৪,৪৫২জন। এর মধ্যে পুরুষ ৩,৫০,৩৭২ জন এবং মহিলা ৩,২৪,০৮০ জন। লোক সংখ্যার ঘনত্ব ১,৭৩৬ জন/ বর্গ কিলোমিটার। == অর্থনীতি == == কৃষি == ময়মনসিংহ সদর ধান উৎপাদনে বিখ্যাত। এখানে ধানের পাশাপাশি পাট সহ মৌসুমী সব সবজি ফল উৎপাদন হয়। == স্বাস্থ্য == ময়মনসিংহ মেডিকেল কলেজ হাসপাতাল (১০০০ শয্যা) বেসরকারি ক্লিনিক == সাহিত্য == ব্রহ্মপুত্র নদীর তীরে আছে কাজী নজরুল ইসলাম এর স্মৃতি বিজরিত বট গাছ। কবি নির্মলেন্দু গুণ, আবদুল হাই মাশরেকী, আবু ফাতেমা মোহাম্মদ আবু ইসহাক ময়মনসিংহ এর সন্তান। আন্তর্জাতিক বাংলা ভাষা সাহিত্য পরিষদের সদর দপ্তর এখানেই অবস্থিত। == সংস্কৃতি == ময়মনসিংহে রয়েছে শিল্পী জয়নুল আবেদীন এর স্মৃতি বিজড়িত মৃত্যুঞ্জয় স্কুল। তিনি ময়মনসিংহের কৃতি সন্তান। তাঁর কর্মের নিদর্শনগুলো রাখা আছে ব্রহ্মপুত্র নদীর পাড়ে শিল্পাচার্য জয়নুল আবেদিন সংগ্রহশালা। ময়মনসিংহ লোক গীতির জন্য বিখ্যাত। == ধর্ম == == ব্যবসা-বাণিজ্য == == ব্যাংক == জনতা ব্যাংক, রুপালি ব্যাংক, সোনালি ব্যাংক, পূবালি ব্যাংক, কৃষি ব্যাংক। এছাড়াও রয়েছে প্রায় সব বেসরকারি ব্যাংকের শাখা। == সংবাদপত্র == সাপ্তাহিক মোমেনশাহী == গ্রন্থাগার == শহরের সার্কিট হাউজ মাঠের কাছে জেলা সরকারি গণগ্রন্থাগার অবস্থিত। == সংগঠন == ;ক্লাব পন্ডিতপাড়া এসি, মুকুল ফৌজ, আল হেলাল এসি, নব জাগরন বয়েস ক্লাব == বিবিধ == == আরও দেখুন == ময়মনসিংহ জেলা; ময়মনসিংহ বিভাগ; বাংলাদেশের উপজেলাসমূহ। == তথ্যসূত্র ==
