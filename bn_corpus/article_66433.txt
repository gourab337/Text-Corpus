'''উত্তরদা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == লাকসাম উপজেলার দক্ষিণাংশে উত্তরদা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে আজগরা ইউনিয়ন, উত্তরে আজগরা ইউনিয়ন লাকসাম পৌরসভা, পশ্চিমে গোবিন্দপুর ইউনিয়ন এবং দক্ষিণে মনোহরগঞ্জ উপজেলার খিলা ইউনিয়ন নাঙ্গলকোট উপজেলার আদ্রা উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == উত্তরদা ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
