'''চর জব্বর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সুবর্ণচর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সুবর্ণচর উপজেলার উত্তরাংশে চর জব্বর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দক্ষিণ-পশ্চিমে চর ওয়াপদা ইউনিয়ন, দক্ষিণ-পূর্বে পূর্বে চর জুবলী ইউনিয়ন এবং উত্তরে নোয়াখালী সদর উপজেলার ধর্মপুর ইউনিয়ন এওজবালিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর জব্বর ইউনিয়ন সুবর্ণচর উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সুবর্ণচর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭১নং নির্বাচনী এলাকা নোয়াখালী-৪ এর অংশ। ইউনিয়নের গ্রামগুলো হল: জাহাজমারা চর হাসান চর রশিদ পশ্চিম চর জব্বর চর জব্বর চর পানা উল্যাহ উত্তর ব্যগ্যা == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: তরিকুল ইসলাম == আরও দেখুন == সুবর্ণচর উপজেলা সুবর্ণচর থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
