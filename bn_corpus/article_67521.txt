'''সরাইল সদর''' বাংলাদেশের জেলার অন্তর্গত সরাইল উপজেলার একটি ইউনিয়ন। == আয়তন == সরাইল সদর ইউনিয়নের আয়তন ৩৪৬৮ একর (১৪.০৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে সরাইল সদর ইউনিয়নের মোট জনসংখ্যা ৪৬,৯২৭ জন। এর মধ্যে পুরুষ ২২,৫২৬ জন এবং মহিলা ২৪,৪০১ জন। মোট পরিবার ৮,৯৩০টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে প্রায় ৩৩৪৪ জন। == ইতিহাস == == অবস্থান সীমানা == সরাইল উপজেলার দক্ষিণাংশে সরাইল সদর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে পানিশ্বর ইউনিয়ন, উত্তরে কালিকচ্ছ ইউনিয়ন, পূর্বে নোয়াগাঁও ইউনিয়ন, দক্ষিণ-পূর্বে সদর উপজেলার মজলিশপুর ইউনিয়ন এবং দক্ষিণে সদর উপজেলার বুধল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সরাইল সদর ইউনিয়ন সরাইল উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সরাইল থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == ২০১১ সালের আদমশুমারি অনুসারে সরাইল সদর ইউনিয়নের সাক্ষরতার হার ৫০.৭%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সরাইল উপজেলা সরাইল থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
