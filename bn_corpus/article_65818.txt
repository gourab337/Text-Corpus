'''আলীরগাঁও ইউনিয়ন''' বাংলাদেশের সিলেট জেলার গোয়াইনঘাট উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == ৫নং আলীরগাঁও ইউনিয়ন ১৯৫১ ইংরেজীতে ইউনিয়ন পরিষদ গঠনের আগে একটি গ্রামছিল। আলীরগাও গ্রামের নাম অনুসারে আলীরগাঁও ইউনিয়নের নামকরণ করা হয়। ১৯৭১ সালের ২৮ নভেম্বর গভীর রাতে পাকবাহিনী উপজেলার আলীরগাঁও ইউনিয়নের উজুহাত গ্রামে অতর্কিত হামলা করলে ২৫ জন মুক্তিযোদ্ধা শহীদ হন। == প্রশাসনিক এলাকা == == আয়তন জনসংখ্যা == আয়তনঃ ২১৫৯৮ একর। জনসংখ্যাঃ- পুরুষ, ১৬৮৪৬ জন। মহিলা, ১৬২৩০ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হারঃ-''' ১৭.৪৮%। '''শিক্ষা প্রতিষ্ঠাণ''' == দর্শনীয় স্থান == জাওর বিল == খাল নদী == নদী নাই, খালঃ- খাইয়া খাল পুড়া খাই খাল চাবরা বলি হান্দারি বিল আরও অনেক == হাট-বাজার == বাঘের সড়ক বাজার বারহাল বাজার লাফনাউট বাজার বাঘের সড়ক বাজার == কৃতী ব্যক্তিত্ব == মো: আতি উল্লাহ, পিএইচডি), শা:বি: ইংরেজী প্রভাষক মো: দেলোয়ার, এডভোকেট, সিলেট জজকোর্ট আছান উল্লাহ সিলেট, ডিসি অফিস == জনপ্রতিনিধি == ''বর্তমান চেয়ারম্যান-'' '''মো: গোলাম কিবরিয়া হেলাল''' চেয়ারম্যানগণের তালিকা == আরও দেখুন == ১নং রুস্তমপুর ইউনিয়ন ২নং পশ্চিম জাফলং ইউনিয়ন ৩নং পুর্ব জাফলং ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == গোয়াইনঘাট উপজেলা -বাংলাপিডিয়া আলীরগাঁও ইউনিয়ন -বাংলাদেশ জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
