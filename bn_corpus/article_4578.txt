'''স্বাধীন সামোয়া রাষ্ট্র''' (সামোয়ান Malo Tutoʼatasi Samoa ''মালো তুতোʼআতাসি সামোয়া'', ইংরেজি Independent State of Samoa স্টেট্‌ অভ্‌ সামোয়া'') দক্ষিণ প্রশান্ত মহাসাগরে অবস্থিত স্বাধীন দ্বীপরাষ্ট্র। এটি নিউজিল্যান্ডের প্রায় ২৯০০ কিমি উত্তর-পূর্বে অবস্থিত। প্রায় ৪৮০ কিমি দীর্ঘ সামোয়ান দ্বীপপুঞ্জের বৃহত্তর পশ্চিম ভাগ নিয়ে (১৭১ ডিগ্রি পশ্চিম দ্রাঘিমার পশ্চিমে) রাষ্ট্রটি গঠিত। বিভাজন রেখাটির পূর্বে অবস্থিত দ্বীপগুলি মার্কিন শাসনাধীন অঞ্চল মার্কিন সামোয়া গঠন করেছে। রাষ্ট্রটি বহুদিন যাবৎ পশ্চিম সামোয়া নামে পরিচিত ছিল। ১৯৯৭ সালে এটির নাম সরকারীভাবে বদলে সামোয়া রাখা হয়। ১৯৪৫ সালে দ্বিতীয় বিশ্বযুদ্ধের শেষ থেকে ১৯৬২ সালে স্বাধীনতা লাভ পর্যন্ত সামোয়া জাতিসঙ্ঘের একটি ট্রাস্ট এলাকা ছিল, যার দেখাশোনা করত নিউজিল্যান্ড। রাষ্ট্রটির রাজধানী, বৃহত্তম শহর বাণিজ্যিক কেন্দ্রের নাম আপিয়া। == ইতিহাস == == রাজনীতি == == প্রশাসনিক অঞ্চলসমূহ == == ভূগোল == == অর্থনীতি == == জনসংখ্যা == == সংস্কৃতি == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == '''সামোয়ান সরকারি ওয়েবসাইট''' Samoa News '''Samoalive Newsline সামোয়ার সর্বশেষ তথ্য Open Directory Project ''Samoa'' directory category Map: district subdivision Samoa, Hundred Years Ago And Long Before by George Turner, an eText available from Project Gutenberg
