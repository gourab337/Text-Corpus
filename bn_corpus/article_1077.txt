'''জয়পুরহাট সদর উপজেলা''' বাংলাদেশের জয়পুরহাট জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই উপজেলার উত্তরে পাঁচবিবি উপজেলা হাকিমপুর উপজেলা, দক্ষিণাংশে আক্কেলপুর উপজেলা, পশ্চিমে বদলগাছী উপজেলা, পূর্বে কালাই উপজেলা ক্ষেতলাল উপজেলা। == ইতিহাস == == ভৌগোলিক উপাত্ত == == ভাষা সংষ্কৃতি == == প্রশাসনিক এলাকা == পৌরসভা ১টি ইউনিয়ন ৯টি মোহাম্মাদাবাদ ইউনিয়ন ধলাহার ইউনিয়ন দোগাছি ইউনিয়ন ভাদশা ইউনিয়ন পুরানাপৈল ইউনিয়ন আমদই ইউনিয়ন বম্বু ইউনিয়ন জামালপুর ইউনিয়ন চকবরকত ইউনিয়ন == জনসংখ্যার উপাত্ত == জয়পুরহাট সদর উপজেলার মোট জনসংখ্যা ২,৫৬,৬৯১ জন। == স্বাস্থ্য == == শিক্ষা == জয়পুরহাট কে বি উচ্চ বিদ্যালয়। == কৃষি == == অর্থনীতি == == যোগাযোগ ব্যবস্থা == == কৃতী ব্যক্তিত্ব == কাবীর চৌধুরী চলচ্চিত্র পরিচালক। == দর্শনীয় স্থান স্থাপনা == পাগলা দেওয়ান বধ্যভূমি কড়ই কাদিরপুর বধ্যভূমি বারশিবালয় মন্দির == বিবিধ == == আরও দেখুন == বাংলাদেশের উপজেলাসমূহ; জয়পুরহাট জেলা; রাজশাহী বিভাগ। == তথ্যসুত্র == ==বহিঃসংযোগ== জেলার উপজেলা বিভাগের উপজেলা সদর উপজেলা
