'''কালিদহ''' বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == কালিদহ ইউনিয়নের আয়তন ৪৭৩৬ একর (১৯.১৭ বর্গ কিলোমিটার)। == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার মধ্যাংশে কালিদহ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে কাজিরবাগ ইউনিয়ন, পশ্চিমে ফেনী পৌরসভা বালিগাঁও ইউনিয়ন, দক্ষিণ-পশ্চিমে ধলিয়া ইউনিয়ন, দক্ষিণ-পূর্বে লেমুয়া ইউনিয়ন ছনুয়া ইউনিয়ন এবং পূর্বে মোটবী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কালিদহ ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ৬নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। ইউনিয়নের গ্রামগুলো হল: ভালুকিয়া আলোকদিয়া গোহাড়ুয়া মাইজবাড়িয়া তুলাবাড়িয়া কালিদহ ছিলোনিয়া যাত্রা সিদ্দিক গোবিন্দপুর মহেশপুর চেয়োরিয়া == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == কালীদহ এস সি উচ্চ বিদ্যালয় গোবিন্দপুর উচ্চ বিদ্যালয় তুলাবাড়িয়া উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
