'''কলাকান্দি''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == তিতাস উপজেলার উত্তর-পূর্বাংশে কলাকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বলরামপুর ইউনিয়ন, দক্ষিণে ভিটিকান্দি ইউনিয়ন, দক্ষিণ-পূর্বে মুরাদনগর উপজেলার ছালিয়াকান্দি ইউনিয়ন, পূর্বে মুরাদনগর উপজেলার জাহাপুর ইউনিয়ন, উত্তরে হোমনা উপজেলার ভাসানিয়া ইউনিয়ন এবং উত্তর-পশ্চিমে হোমনা উপজেলার জয়পুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == কলাকান্দি ইউনিয়ন তিতাস উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
