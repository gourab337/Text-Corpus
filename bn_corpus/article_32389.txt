বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সেনবাগ উপজেলার একটি ইউনিয়ন। == আয়তন == মোহাম্মদপুর ইউনিয়নের আয়তন ৩০৯৯ একর। == জনসংখ্যা == == অবস্থান সীমানা == সেনবাগ উপজেলার মধ্যাংশে মোহাম্মদপুর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পশ্চিমে সেনবাগ পৌরসভা, দক্ষিণে বিজবাগ ইউনিয়ন এবং পূর্বে ফেনী জেলার দাগনভূঁইয়া উপজেলার রামনগর ইউনিয়ন পূর্ব চন্দ্রপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মোহাম্মদপুর ইউনিয়ন সেনবাগ উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। ইউনিয়নে ৪টি গ্রাম রয়েছে: মোহাম্মদপুর রাজারামপুর শাহপুর আহম্মদপুর == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == মোহাম্মদপুর রামেন্দ্র মডেল উচ্চ বিদ্যালয় শেরে বাংলা উচ্চ বিদ্যালয় জয়নগর উচ্চ বিদ্যালয় রাজারামপুর বশিরিয়া আলিম মাদ্রাসা কল্যাণ্দী ফয়জুল উলুম মাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
