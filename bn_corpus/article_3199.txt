'''থুলিয়াম''' একটি মৌলিক পদার্থ। এর পারমাণবিক সংখ্যা হলো ৬৯। এটি পর্যায়সারণীর ৬ষ্ঠ পর্যায়ের ব্লকে অবস্থিত। কক্ষ তাপমাত্রায় এটি একটি কঠিন ধাতব পদার্থ, যার বর্ণ রূপালী ধূসর। == আবিষ্কার == == বৈশিষ্ট্য == == যৌগসমূহ == == রাসায়নিক বিক্রিয়া == == ব্যবহার == == আরও দেখুন == == তথ্যসূত্র == == বহিঃসংযোগ == পদার্থ
