'''হাটপুকুরিয়া ঘাটলাবাগ''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত চাটখিল উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == চাটখিল উপজেলার মধ্যাংশে হাটপুকুরিয়া ঘাটলাবাগ ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে রামনারায়ণপুর ইউনিয়ন নোয়াখলা ইউনিয়ন, পূর্বে নোয়াখলা ইউনিয়ন চাটখিল পৌরসভা, উত্তরে চাটখিল পৌরসভা পরকোট ইউনিয়ন এবং পশ্চিমে পরকোট ইউনিয়ন সাহাপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাটপুকুরিয়া ঘাটলাবাগ ইউনিয়ন চাটখিল উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চাটখিল থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চাটখিল উপজেলা চাটখিল থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
