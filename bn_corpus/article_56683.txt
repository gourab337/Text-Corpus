'''চর দরবেশ''' বাংলাদেশের ফেনী জেলার অন্তর্গত সোনাগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাগাজী উপজেলার পশ্চিমাংশে চর দরবেশ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে বগাদানা ইউনিয়ন; পূর্বে মঙ্গলকান্দি ইউনিয়ন, মতিগঞ্জ ইউনিয়ন, সোনাগাজী পৌরসভা চর চান্দিয়া ইউনিয়ন; দক্ষিণে ছোট ফেনী নদী নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার মুছাপুর ইউনিয়ন এবং পশ্চিমে ছোট ফেনী নদী, নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার মুছাপুর ইউনিয়ন, চর হাজারী ইউনিয়ন চর পার্বতী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর দরবেশ ইউনিয়ন সোনাগাজী উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাদেশের ইউনিয়ন নিবন্ধ উপজেলার ইউনিয়ন
