'''শাহরাস্তি পৌরসভা''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == ১.৮৬ বর্গ কিলোমিটার এলাকা নিয়ে শাহরাস্তি পৌরসভা প্রতিষ্ঠিত হলেও পরবর্তীতে সম্প্রসারিত হয়ে পৌরসভার বর্তমান আয়তন ৩.৫০ বর্গ কিলোমিটার। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী শাহরাস্তি পৌরসভার জনসংখ্যা ৬২,৬৫০ জন। == অবস্থান সীমানা == শাহরাস্তি উপজেলার উত্তরাংশে শাহরাস্তি পৌরসভার অবস্থান। পৌরসভার উত্তরে মেহের উত্তর ইউনিয়ন টামটা উত্তর ইউনিয়ন, পশ্চিমে টামটা দক্ষিণ ইউনিয়ন, দক্ষিণে মেহের দক্ষিণ ইউনিয়ন এবং পূর্বে মেহের উত্তর ইউনিয়ন অবস্থিত। == ইতিহাস == == নামকরণ == == প্রতিষ্ঠাকাল == ১৯৯৮ সালের ১৫ অক্টোবর শাহরাস্তি পৌরসভা প্রতিষ্ঠিত হয়। প্রতিষ্ঠাকালে ''গ'' শ্রেণীর পৌরসভা হিসেবে প্রতিষ্ঠিত হলেও ২০১২ সালের ১১ জুলাই পৌরসভাটি ''খ'' শ্রেণীতে এবং ২০১৪ সালের ২৩ জুন ''ক'' শ্রেণীতে উন্নীত করা হয়। == প্রশাসনিক এলাকা == শাহরাস্তি পৌরসভায় ১২টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == শাহরাস্তি পৌরসভার সাক্ষরতার হার ৯৩.৪০%। পৌরসভায় ২টি কলেজ, ৪টি মাদ্রাসা, ৬টি মাধ্যমিক বিদ্যালয় এবং ১০টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == অর্থনীতি == মূলত কৃষি, ব্যবসা এবং বৈদেশিক রেমিটেন্স থেকে পৌরসভার অর্থনীতি নির্ভরশীল। == দর্শনীয় স্থান == রাস্তি শাহ (রা.) এর মাজার শ্রী শ্রী মহের কালীবাড়ি == কৃতী ব্যক্তিত্ব == ড. এম.এ সাত্তার, বেইস প্রতিষ্ঠাতা। == জনপ্রতিনিধি == পৌর মেয়র: আব্দুল লতিফ == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
