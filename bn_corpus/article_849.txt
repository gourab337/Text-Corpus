'''শরণখোলা উপজেলা''' বাংলাদেশের বাগেরহাট জেলার একটি প্রশাসনিক এলাকা। == অবস্থান আয়তন == এই উপজেলার উত্তরে মোড়েলগঞ্জ উপজেলা, দক্ষিণে বঙ্গোপসাগর, পূর্বে মঠবাড়িয়া উপজেলা পাথরঘাটা উপজেলা, পশ্চিমে মোংলা উপজেলা। == প্রশাসনিক এলাকা == শরণখোলা উপজেলাটি চারটি ইউনিয়ন নিয়ে গঠিত। এই ইউনিয়নগুলো হচ্ছে- ধানসাগর ইউনিয়ন খোন্তাকাটা ইউনিয়ন রায়েন্দা ইউনিয়ন সাউথখালী ইউনিয়ন == ইতিহাস == == জনসংখ্যার উপাত্ত == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == মাওলানা কে এম ইউসুফ(বিশিষ্ট আলেম সাবেক মন্ত্রী == বিবিধ == == আরও দেখুন == বাগেরহাট জেলা; খুলনা বিভাগ; বাংলাদেশের উপজেলাসমূহ। == তথ্যসূত্র == ==বহিঃসংযোগ== উপজেলা জেলার উপজেলা বিভাগের উপজেলা
