'''মাছিহাতা''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == মাছিহাতা ইউনিয়নের আয়তন ৭৮৯৯ একর (৩১.৯৭ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে মাছিহাতা ইউনিয়নের মোট জনসংখ্যা ৪০,৫০৯ জন। মোট পরিবার ৭,৫৬৮টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১২৬৭ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার দক্ষিণাংশে মাছিহাতা ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে বাসুদেব ইউনিয়ন সুলতানপুর ইউনিয়ন, পশ্চিমে সুলতানপুর ইউনিয়ন রামরাইল ইউনিয়ন, উত্তর-পশ্চিমে পৌরসভা, উত্তরে সুহিলপুর ইউনিয়ন বিজয়নগর উপজেলার পত্তন ইউনিয়ন এবং পূর্বে বিজয়নগর উপজেলার সিংগারবিল ইউনিয়ন আখাউড়া উপজেলার আখাউড়া উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মাছিহাতা ইউনিয়ন সদর উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
