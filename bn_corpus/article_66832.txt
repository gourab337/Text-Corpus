'''দুলালপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == দুলালপুর ইউনিয়ন পূর্বে ঘাগুটিয়া পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == হোমনা উপজেলার উত্তরাংশে দুলালপুর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে ঘাগুটিয়া ইউনিয়ন; দক্ষিণে নিলখী ইউনিয়ন, ঘারমোড়া ইউনিয়ন আছাদপুর ইউনিয়ন; পূর্বে আছাদপুর ইউনিয়ন চান্দেরচর ইউনিয়ন; উত্তরে জেলার বাঞ্ছারামপুর উপজেলার ছলিমাবাদ ইউনিয়ন এবং উত্তর-পশ্চিমে জেলার বাঞ্ছারামপুর উপজেলার বাঞ্ছারামপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দুলালপুর ইউনিয়ন হোমনা উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
