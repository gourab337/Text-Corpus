'''মান্দারী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == মান্দারী ইউনিয়নের আয়তন ২১,৭১২ একর (৮৭.৮৭ বর্গ কিলোমিটার)। এটি লক্ষ্মীপুর সদর উপজেলার একটি ঐতিহ্যবাহী বৃহত্তর ইউনিয়ন। == জনসংখ্যা == মান্দারী ইউনিয়নের লোক সংখ্যা ৪৮,০০০ জন। এর মধ্যে পুরুষ ২৩,৮০০ জন এবং মহিলা ২৪,২০০ জন। == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার মধ্যাংশে মান্দারী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে দত্তপাড়া ইউনিয়ন বাঙ্গাখাঁ ইউনিয়ন, পশ্চিমে লাহারকান্দি ইউনিয়ন, দক্ষিণে তেওয়ারীগঞ্জ ইউনিয়ন কুশাখালী ইউনিয়ন এবং পূর্বে দিঘলী ইউনিয়ন হাজিরপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মান্দারী ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ১৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == মান্দারী ইউনিয়নের সাক্ষরতার হার ৬২%। ইউনিয়নে ১টি কলেজ, ১টি আলিম মাদ্রাসা, ২টি মাধ্যমিক বিদ্যালয়, ৭টি অন্যান্য মাদ্রাসা ১২টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == ;মাদ্রাসা মান্দারী ইসলামিয়া আলিম মডেল মাদ্রাসা গন্ধর্ব্যপুর ইসলামিয়া আলিম মাদ্রাসা ;মাধ্যমিক বিদ্যালয় দক্ষিণ মান্দারী ফাতেমা বহুমুখী উচ্চ বিদ্যালয় মান্দারী মাধ্যমিক উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == মান্দারী ইউনিয়নে যোগাযোগের প্রধান মাধ্যম রিক্সা এবং সিএনজি চালিত অটোরিক্সা। == খাল নদী == মান্দারী ইউনিয়নের রতনপুর মৌজার উপর দিয়ে রহমতখালী খাল এবং গন্ধব্যপুর মৌজার উপর দিয়ে ওয়াপদার খাল প্রবাহিত হচ্ছে। == হাট-বাজার == মান্দারী ইউনিয়নের প্রধান প্রধান হাট-বাজারগুলো হল মান্দারী বাজার, আমিন বাজার, ওয়াপদা বাজার, গোধূলীর হাট এবং চতলিয়া বাজার। == দর্শনীয় স্থান == মান্দারী বড় জামে মাসজিদ == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
