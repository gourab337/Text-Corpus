'''বাসুদেব''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == বাসুদেব ইউনিয়নের আয়তন ৬৪৪১ একর (২৬.০৭ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে বাসুদেব ইউনিয়নের মোট জনসংখ্যা ৩৫,৮৭৩ জন। মোট পরিবার ৬,৫১৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ১৩৭৬ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার বাসুদেব ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মাছিহাতা ইউনিয়ন; পশ্চিমে সুলতানপুর ইউনিয়ন, তিতাস নদী কসবা উপজেলার মূলগ্রাম ইউনিয়ন; দক্ষিণে তিতাস নদী, আখাউড়া উপজেলার ধরখাড় ইউনিয়ন মোগড়া ইউনিয়ন এবং পূর্বে আখাউড়া উপজেলার আখাউড়া দক্ষিণ ইউনিয়ন, আখাউড়া পৌরসভা আখাউড়া উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাসুদেব ইউনিয়ন সদর উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
