'''হালিশহর''' বাংলাদেশের চট্টগ্রাম জেলার একটি মেট্রোপলিটন থানা। এটি চট্টগ্রাম সিটি কর্পোরেশনের আওতাধীন। == আয়তন == হালিশহর থানার আয়তন ১১.৬৩ বর্গ কিলোমিটার। == প্রতিষ্ঠাকাল == ২০০০ সালের ২৭ মে ডবলমুরিং থানা পাহাড়তলী থানার কিছু অংশ নিয়ে হালিশহর থানা গঠিত হয়। == জনসংখ্যা == ২০১১ সালের পরিসংখ্যান অনুযায়ী হালিশহর থানার লোকসংখ্যা ১,৪৬,৯৮৯ জন। এর মধ্যে পুরুষ ৭৮,৩৩৬ জন এবং মহিলা ৬৮,৬৫৩ জন। == অবস্থান সীমানা == চট্টগ্রাম সিটি কর্পোরেশনের পশ্চিমাংশে হালিশহর থানার অবস্থান। এর উত্তরে পাহাড়তলী থানা, পূর্বে ডবলমুরিং থানা, দক্ষিণে বন্দর থানা এবং পশ্চিমে বঙ্গোপসাগর। == প্রশাসনিক এলাকা == হালিশহর থানার আওতাধীন চট্টগ্রাম সিটি কর্পোরেশনের এলাকাসমূহ হল: ১১নং দক্ষিণ কাট্টলী ওয়ার্ডের দক্ষিণাংশ ২৪নং উত্তর আগ্রাবাদ ওয়ার্ডের পশ্চিমাংশ ২৫নং রামপুর ওয়ার্ড ২৬নং উত্তর হালিশহর ওয়ার্ড == সংসদীয় আসন == সংসদীয় আসন জাতীয় নির্বাচনী এলাকা সংসদ সদস্য রাজনৈতিক দল ২৮৭ চট্টগ্রাম-১০ চট্টগ্রাম সিটি কর্পোরেশনের ৮নং শুলকবহর, ১১নং দক্ষিণ কাট্টলী, ১২নং সরাইপাড়া, ১৩নং পাহাড়তলী, ১৪নং লালখান বাজার, ২৪নং উত্তর আগ্রাবাদ, ২৫নং রামপুর ২৬নং উত্তর হালিশহর ওয়ার্ড ডাঃ আফছারুল আমীন বাংলাদেশ আওয়ামী লীগ == শিক্ষা == হালিশহর থানার আওতাধীন এলাকার সাক্ষরতার হার ৬৩.১০%। ;উল্লেখযোগ্য শিক্ষা প্রতিষ্ঠান গার্হস্থ্য অর্থনীতি কলেজ এসওএস ট্রেড স্কুল ফিজিক্যাল ট্রেনিং কলেজ == == আগ্রাবাদ একসেস রোড জেনারেল হাসপাতাল ডিও পয়েন্ট হাসপাতাল রাজ ডেন্টাল ক্লিনিক রয়েল চক্ষু হাসপাতাল == গুরুত্বপূর্ণ স্থাপনা == বর্ডার সিকিউরিটি গার্ড সেক্টর হেড কোয়ার্টার আর্টিলারী ট্রেনিং সেন্টার এবং স্কুল (বাংলাদেশ সেনাবাহিনীর একমাত্র আর্টিলারী স্কুল) চট্টগ্রাম জেলা পুলিশ লাইন এস এস শিশু পল্লী বিহারী ক্যাম্প == আবাসিক এলাকা == হালিশহর মূলত আবাসিক এলাকা হিসাবে পরিচিত। উত্তর হালিশহরে রয়েছে হালিশহর আবাসিক এলাকা, যা হালিশহর হাউজিং এস্টেট নামে পরিচিত। হালিশহর হাউজিং এস্টেট ১০ টি ব্লকে বিভক্তঃ A-Block, B-Block, G-Block, H-Block, I-Block, J-Block, K-Block, L-Block, New A-Block এবং New I-Block. সবচেয়ে বড় ব্লক B-Block, যা বঙ্গোপসাগর উপকূলবর্তী। == হাটবাজার শপিং সেন্টার == বড়পুল বাজার ছোটপুল বাজার আনন্দীপুর বাজার ফইল্যাথলী
