'''জগতপুর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == জগতপুর ইউনিয়ন পূর্বে জগতপুর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == তিতাস উপজেলার জগতপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মজিদপুর ইউনিয়ন কড়িকান্দি ইউনিয়ন, পূর্বে বলরামপুর ইউনিয়ন, উত্তর-পূর্বে হোমনা উপজেলার ঘারমোড়া ইউনিয়ন, উত্তরে সাতানী ইউনিয়ন এবং পশ্চিমে কাঁঠালিয়া নদী মেঘনা উপজেলার গোবিন্দপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == জগতপুর ইউনিয়ন তিতাস উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
