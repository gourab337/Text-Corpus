'''বাইশগাঁও''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত মনোহরগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == মনোহরগঞ্জ উপজেলার পশ্চিমাংশে বাইশগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে ঝলম দক্ষিণ ইউনিয়ন, পূর্বে খিলা ইউনিয়ন লক্ষণপুর ইউনিয়ন, দক্ষিণে সরসপুর ইউনিয়ন নোয়াখালী জেলার চাটখিল উপজেলার মোহাম্মদপুর ইউনিয়ন এবং পশ্চিমে হাসনাবাদ ইউনিয়ন চাঁদপুর জেলার শাহরাস্তি উপজেলার চিতোষী পূর্ব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাইশগাঁও ইউনিয়ন মনোহরগঞ্জ উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মনোহরগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মনোহরগঞ্জ উপজেলা মনোহরগঞ্জ থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
