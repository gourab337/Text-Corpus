'''হেসাখাল''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত নাঙ্গলকোট উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == নাঙ্গলকোট উপজেলার হেসাখাল ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে মক্রবপুর ইউনিয়ন, নাঙ্গলকোট পৌরসভা মৌকরা ইউনিয়ন; দক্ষিণে জোড্ডা পূর্ব ইউনিয়ন; পশ্চিমে আদ্রা দক্ষিণ ইউনিয়ন আদ্রা উত্তর ইউনিয়ন এবং উত্তর-পশ্চিমে লাকসাম উপজেলার আজগরা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হেসাখাল ইউনিয়ন নাঙ্গলকোট উপজেলার আওতাধীন ১২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম নাঙ্গলকোট থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == হেসাখাল ইউনিয়নের প্রতিটি স্কুল কলেজে ভালো মানের পাঠ দেওয়া হয়। == শিক্ষা প্রতিষ্ঠান == ১) হেসাখাল বাজার উচ্চ বিদ্যালয় ২) দায়েমছাতি উচ্চ বিদ্যালয় ৩) হেসাখাল বাজার নঈম নিজাম ডিগ্রি কলেজ ৪) হেসাখাল বাজার সরকারি প্রাথমিক বিদ্যালয় ৫) হেসাখাল সরকারি প্রাথমিক বিদ্যালয় ৬) উরুকচাইল সরকারি প্রাথমিক বিদ্যালয় ৭) হেসাখাল পদুয়ারপাড়া সরকারি প্রাথমিক বিদ্যালয় ৮) পাটোয়ার সরকারি প্রাথমিক বিদ্যালয় ৯) যুব উদ্যোগ শিশু নিকেতন,হেসাখাল। ১০) প্রত্যাশা ১১) হেসাখাল বাজার দাখিল মাদ্রাসা == যোগাযোগ ব্যবস্থা == হেসাখাল ইউনিয়ন এর মেইন কেন্দ্র হেসাখাল বাজার। নাংগলকোট থেকে হেসাখাল বাজারে যাতায়াত করার জন্য রিকশা, অটোরিকশা আছে। লাকসাম থেকে হেসাখাল বাজারে যাতায়াত করার জন্য সি এন জি আছে। == খাল নদী == হেসাখাল বাজারের উপর দিয়ে একটা খাল বয়ে গেছে, এটা অনেকে গাংগরী খাল হিসাবে চিনে == হাট-বাজার == ১) হেসাখাল বাজার ২) দায়েমছাতি বাজার ৩) ওমরগঞ্জ বাজার == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == মাষ্টার শহিদ, মাষ্টার শাহাদাত হোসেন ভূইয়া, আলহাজ্ব মাহফুজুল হক মোল্লা, রবিউল মেম্বার সাবেক) বাবুল মেম্বার সাবেক) ফরহাদ মেম্বার মাসুদুল হক মেম্বার সাবেক) == জনপ্রতিনিধি == চেয়ারম্যান জালাল আহমেদ ভূইয়া == আরও দেখুন == নাঙ্গলকোট উপজেলা নাঙ্গলকোট থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
