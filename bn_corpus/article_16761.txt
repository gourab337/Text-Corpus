'''সিমুলা''' হলো দুইটি সিম্যুলেশন প্রোগ্রামিং ভাষার নাম; যারা সিমুলা এবং সিমুলা ৬৭ হিসাবে পরিচিত এবং ১৯৬০-এর দশকে অসলোতে অবস্থিত নরওয়েজীয়ান কম্পিউটার সেন্টারে উলাহ্‌-ইয়োহান ডাল ক্রিস্টেন নিগার্ড কর্তৃক উদ্ভাবিত। এটি ALGOL 60-এর একটি বিশ্বস্ত সুপারসেট। সিমুলা ৬৭ যে সব নতুন বিষয়ের সূচনা করে সেগুলো হলো: ক্লাস, সাব-ক্লাস, ভার্চুয়াল মেথড, ডিসক্রিট ইভেন্ট সিমুলেশন (Discrete event simulation) এবং গারবেজ কালেকশন। সিমুলাকে প্রথম অবজেক্ট ওরিয়েন্টেড প্রোগ্রামিং ভাষা হিসেবে বিবেচনা করা হয়। এর নামকরণ থেকেই বুঝা যায় যে, সিমুলা ডিজাইন করা হয়েছে সিমুলেশন এর জন্য। সিমুলা বিভিন্ন অ্যাপ্লিকেশনের সিমুলেশন এর জন্য ব্যবহার করা হয়, যেমন: ভি এল এস আই ডিজাইন, প্রসেস মডেলিং, প্রোটোকল, অ্যালগরিদম। এছাড়াও অন্যান্য অ্যাপ্লিকেশন যেমন, টাইপসেটিং, কম্পিউটার গ্রাফিক্স, শিক্ষা ইত্যাদি। সি++, জাভা, সি# সিমুলা-টাইপ অবজেক্টের ব্যবহার দেখে সিমুলা প্রোগ্রামিং ভাষার সহজেই প্রভাব বুঝা যায়। == আরও দেখুন == বেটা, সিমুলার আধুনিক উত্তরসূরী। == টীকা == == উৎস == IBM System 360/370 Compiler and Historical Documentation The Simula Standard and other historical documentation by Peter Sylvester == অতিরিক্ত পাঠ == == বহিঃসংযোগ == with source code example with some notes on Demos by Graham Birtwistle based on the 1997 seminar "30 Years of Object Oriented Programming (OOP)" includes tutorials, documentation, and links in English and in French ভাষা
