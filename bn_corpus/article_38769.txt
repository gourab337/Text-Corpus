'''ডুমুরুয়া''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সেনবাগ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সেনবাগ উপজেলার উত্তরাংশে ডুমুরুয়া ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে কেশারপাড় ইউনিয়ন, দক্ষিণে অর্জুনতলা ইউনিয়ন সেনবাগ পৌরসভা, পূর্বে কাদরা ইউনিয়ন ফেনী জেলার দাগনভূঁইয়া উপজেলার সিন্দুরপুর ইউনিয়ন এবং উত্তরে কুমিল্লা জেলার নাঙ্গলকোট উপজেলার বক্সগঞ্জ ইউনিয়ন দৌলখাঁড় ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ডুমুরুয়া ইউনিয়ন সেনবাগ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সেনবাগ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৯নং নির্বাচনী এলাকা নোয়াখালী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == ডুমুরুয়া ইউনিয়নের প্রধান ৩টি হাট-বাজার হল গাজিরহাট বাজার, অশ্বদিয়া বাজার এবং ফকিরহাট বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সেনবাগ উপজেলা সেনবাগ থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
