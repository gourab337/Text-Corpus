'''সহদেবপুর ইউনিয়ন''' বাংলাদেশের টাঙ্গাইল জেলার অন্তর্গত কালিহাতি উপজেলার একটি ইউনিয়ন। == ভৌগলিক উপাত্ত == সহদেবপুর ইউনিয়নের মোট আয়তন ৪৭৪২ একর।ঘরবাড়ির সংখ্যা ৬৬৯২ টি। == জনসংখ্যার উপাত্ত == বাংলাদেশের ২০১১ সালের আদমশুমারী অনুযায়ী সহদেবপুর ইউনিয়নের মোট জনসংখ্যা ২৭০৩৬ জন।এদের মধ্যে ১৩১৩৮ জন পুরূষ এবং ১৩৮৯৮ জন মহিলা। প্রতি ব:কি: ১৪০৯ জন লোক বাস করে। == ইতিহাস == == শিক্ষা == == অর্থনীতি == == কৃতী ব্যক্তিত্ব == == বিবিধ == == আরও দেখুন == == বহিঃসংযোগ == == তথ্যসূত্র == উপজেলার ইউনিয়ন
