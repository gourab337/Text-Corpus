'''গোলাবাড়ী''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত খাগড়াছড়ি সদর উপজেলার একটি ইউনিয়ন। == আয়তন == গোলাবাড়ী ইউনিয়নের আয়তন ৯.৩২ বর্গ কিলোমিটার। এটি আয়তনের দিক থেকে খাগড়াছড়ি জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == খাগড়াছড়ি সদর উপজেলার মধ্য-পূর্বাংশে গোলাবাড়ী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে পেরাছড়া ইউনিয়ন, পশ্চিমে খাগড়াছড়ি পৌরসভা, দক্ষিণে কমলছড়ি ইউনিয়ন এবং পূর্বে দীঘিনালা উপজেলার মেরুং ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == গোলাবাড়ী ইউনিয়ন খাগড়াছড়ি সদর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম খাগড়াছড়ি সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == খাগড়াছড়ি সদর উপজেলা খাগড়াছড়ি সদর থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
