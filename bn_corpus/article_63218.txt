'''হাজীগঞ্জ সদর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত হাজীগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == হাজীগঞ্জ উপজেলার মধ্যাংশে হাজীগঞ্জ সদর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে বড়কুল পশ্চিম ইউনিয়ন, হাজীগঞ্জ পৌরসভা হাটিলা পশ্চিম ইউনিয়ন, উত্তরে কালচোঁ দক্ষিণ ইউনিয়ন, পশ্চিমে কালচোঁ দক্ষিণ ইউনিয়ন, বাকিলা ইউনিয়ন চাঁদপুর সদর উপজেলার রামপুর ইউনিয়ন, দক্ষিণে ফরিদগঞ্জ উপজেলার সুবিদপুর পশ্চিম ইউনিয়ন এবং দক্ষিণ-পূর্বে ফরিদগঞ্জ উপজেলার সুবিদপুর পূর্ব ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == হাজীগঞ্জ সদর ইউনিয়ন হাজীগঞ্জ উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাজীগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == হাজীগঞ্জ উপজেলা হাজীগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
