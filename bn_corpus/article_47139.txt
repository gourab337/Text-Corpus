'''মীরসরাই লতিফিয়া কামিল মাদ্রাসা''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই পৌরসভায় মাদ্রাসাটি অবস্থিত। == ইতিহাস == == পরিচালনা ব্যবস্থা == == == == অবকাঠামো == == পর্যায় == মাদ্রাসায় কামিল (মাস্টার্স) পর্যায় পর্যন্ত পাঠদান করা হয়। == কৃতিত্ব ফলাফল == == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
