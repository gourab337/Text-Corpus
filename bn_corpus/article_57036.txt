বাংলাদেশের নোয়াখালী জেলার অন্তর্গত সোনাইমুড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাইমুড়ি উপজেলার আমিশাপাড়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে দেওটি ইউনিয়ন, পূর্বে সোনাপুর ইউনিয়ন বেগমগঞ্জ উপজেলার গোপালপুর ইউনিয়ন, দক্ষিণে বেগমগঞ্জ উপজেলার আমানউল্যাপুর ইউনিয়ন, দক্ষিণ-পশ্চিমে লক্ষ্মীপুর জেলার লক্ষ্মীপুর সদর উপজেলার চন্দ্রগঞ্জ ইউনিয়ন এবং পশ্চিমে চাটখিল উপজেলার খিলপাড়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আমিশাপাড়া ইউনিয়ন সোনাইমুড়ি উপজেলার আওতাধীন ১০নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাইমুড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৮নং নির্বাচনী এলাকা নোয়াখালী-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাইমুড়ি উপজেলা সোনাইমুড়ি থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
