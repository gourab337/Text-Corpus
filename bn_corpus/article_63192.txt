'''মেহের উত্তর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত শাহরাস্তি উপজেলার একটি ইউনিয়ন। == আয়তন == মেহের উত্তর ইউনিয়নের আয়তন ৬.৩১ বর্গ কিলোমিটার। এটি আয়তনের দিক থেকে চাঁদপুর জেলার সবচেয়ে ছোট ইউনিয়ন। == জনসংখ্যা == == অবস্থান সীমানা == শাহরাস্তি উপজেলার উত্তরাংশে মেহের উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে টামটা উত্তর ইউনিয়ন, দক্ষিণে শাহরাস্তি পৌরসভা মেহের দক্ষিণ ইউনিয়ন, পূর্বে রায়শ্রী উত্তর ইউনিয়ন এবং উত্তরে কচুয়া উপজেলার আশরাফপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মেহের উত্তর ইউনিয়ন শাহরাস্তি উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম শাহরাস্তি থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৪নং নির্বাচনী এলাকা চাঁদপুর-৫ এর অংশ। ওয়ার্ডভিত্তিক ইউনিয়নের গ্রামগুলো হল: ওয়ার্ড নং গ্রামের নাম ১নং ওয়ার্ড দেবীপুর ২নং ওয়ার্ড নায়নগর ৩নং ওয়ার্ড তারাপুর ৪নং ওয়ার্ড কাকৈড়তলা ৫নং ওয়ার্ড বানিয়াচোঁ ৬নং ওয়ার্ড খনেশ্বর ৭নং ওয়ার্ড পতিচোঁ, বরুলিয়া ৮নং ওয়ার্ড দুর্গাপুর, কদমতলী ৯নং ওয়ার্ড নয়নপুর, শেখখুনি == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == বানিয়াচোঁ জে বি উচ্চ বিদ্যালয় কাকৈড়তলা জি কে মাদ্রাসা কাকৈড়তলা সরকারি প্রাথমিক বিদ্যালয় খনেশ্বর ব্র্যাক স্কুল শাহরাস্তি একাডেমি == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == মেহের উত্তর ইউনিয়নের প্রধান দুইটি হাট-বাজার হল কাকৈড়তলা বাজার এবং শাহরাস্তি রেলস্টেশন বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোহাম্মদ মনির হোসেন == আরও দেখুন == শাহরাস্তি উপজেলা শাহরাস্তি থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
