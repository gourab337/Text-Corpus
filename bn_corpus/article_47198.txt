'''আহমদিয়া হাবিবিয়া গণিয়া দাখিল মাদ্রাসা''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার সাহেরখালী ইউনিয়নে মাদ্রাসাটি অবস্থিত। == ইতিহাস == মাদ্রাসাটি ১৯৮১ সালে প্রতিষ্ঠিত হয়। বিশিষ্ট ধর্মীয় ব্যক্তিত্ব মুফতি মাওলানা মুজাফফর আহামদ, মাওলানা হাবিবুল্লাহ ছাহেব (রহ.) মাওলানা শাহ সুফি আব্দুল গণি সাহেব (রহ.) তিন ব্যক্তিত্বের নামের শেষাংশ প্রথমাংশ নিয়ে দ্বীনি প্রতিষ্ঠানের নামকরণ করা হয়। == পরিচালনা ব্যবস্থা == == == মাদ্রাসার অধ্যক্ষ জনাব আনোয়ারুল্যাহ আল মামুন। বর্তমানে দুই শতাধিক শিক্ষার্থী মাদ্রাসায় অধ্যয়নরত আছে। == অবকাঠামো == == পর্যায় == মাদ্রাসায় দাখিল পর্যায় পর্যন্ত পাঠদান করা হয়। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯৯%। == কৃতী শিক্ষার্থী == == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
