'''বিচারপতি সৈয়দ মোহাম্মদ হোসেন''' (১৯২৮ এপ্রিল ১৯৮৬) হলেন বাংলাদেশের একজন প্রখ্যাত আইনজীবী। পেশাগত জীবনে অনন্য সাধারণ অবদানের জন্য ১৯৯৭ সালে তাকে “সমাজসেবায় স্বাধীনতা পুরস্কার” প্রদান করা হয়। == জন্ম পারিবারিক পরিচিতি == == শিক্ষাজীবন == == কর্মজীবন == তিনি বাংলাদেশ সুপ্রিম কোর্টের "বিচারপতি" হিসাবে কর্মরত ছিলেন। == মৃত্যু == == রচনাবলী == "বাংলার কথা" শিরোনামে তার একটি গ্রন্থ রয়েছে। == পুরস্কার সম্মাননা == == পুরস্কার সম্মাননা == আইনের শাসন প্রতিষ্ঠায় অসাধারণ অবদানের জন্য ১৯৯৭ সালে বাংলাদেশের “সর্বোচ্চ বেসামরিক পুরস্কার” হিসাবে পরিচিত “স্বাধীনতা পুরস্কার” প্রদান করা হয় তাকে। == আরও দেখুন == সমাজসেবায় স্বাধীনতা পুরস্কার বিজয়ীদের তালিকা। == তথ্যসূত্র == == বহিঃসংযোগ == স্বাধীনতা পুরস্কার বিজয়ী
