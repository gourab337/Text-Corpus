'''মহিচাইল''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত চান্দিনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == চান্দিনা উপজেলার উত্তর-মধ্যাংশে মহিচাইল ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে বাতাঘাসী ইউনিয়ন, দক্ষিণ-পশ্চিমে গল্লাই ইউনিয়ন, দক্ষিণে দোল্লাই নবাবপুর ইউনিয়ন, দক্ষিণ-পূর্বে মাইজখার ইউনিয়ন, পূর্বে বাড়েরা ইউনিয়ন এবং উত্তরে মাধাইয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মহিচাইল ইউনিয়ন চান্দিনা উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চান্দিনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৫নং নির্বাচনী এলাকা কুমিল্লা-৭ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == চান্দিনা উপজেলা চান্দিনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
