'''দুর্গাপুর''' বাংলাদেশের জেলার অন্তর্গত আশুগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == দুর্গাপুর ইউনিয়নের আয়তন ৩০৩৮ একর (১২.২৯ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে দুর্গাপুর ইউনিয়নের মোট জনসংখ্যা ৩৪,৭৪৮ জন। মোট পরিবার ৬,০২৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ২৮২৬ জন। == ইতিহাস == দুর্গাপুর ইউনিয়ন পূর্বে সরাইল উপজেলাধীন ৬নং পানিশ্বর দক্ষিণ ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == আশুগঞ্জ উপজেলার সর্ব-উত্তরে দুর্গাপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পশ্চিমে আশুগঞ্জ সদর ইউনিয়ন; দক্ষিণে তালশহর পশ্চিম ইউনিয়ন; পূর্বে সদর উপজেলার তালশহর পূর্ব ইউনিয়ন সরাইল উপজেলার পানিশ্বর ইউনিয়ন; উত্তরে সরাইল উপজেলার পানিশ্বর ইউনিয়ন এবং উত্তর-পশ্চিমে মেঘনা নদী, কিশোরগঞ্জ জেলার ভৈরব উপজেলার আগানগর ইউনিয়ন ভৈরব পৌরসভা অবস্থিত। == প্রশাসনিক কাঠামো == দুর্গাপুর ইউনিয়ন আশুগঞ্জ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম আশুগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৪নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == আশুগঞ্জ উপজেলা আশুগঞ্জ থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
