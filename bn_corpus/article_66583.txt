'''আমড়াতলী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা আদর্শ সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == কুমিল্লা আদর্শ সদর উপজেলার সর্ব-উত্তরে আমড়াতলী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে গোমতী নদী, দুর্গাপুর উত্তর ইউনিয়ন কুমিল্লা সিটি কর্পোরেশন; পূর্বে পাঁচথুবী ইউনিয়ন; উত্তরে বুড়িচং উপজেলার বাকশীমূল ইউনিয়ন এবং পশ্চিমে বুড়িচং উপজেলার ষোলনল ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == আমড়াতলী ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার আওতাধীন ৪নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা কোতোয়ালী থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৪নং নির্বাচনী এলাকা কুমিল্লা-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == গোমতী কপি হাউস রসুলপুর বধ্য ভূমি == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: কাজী মোজাম্মেল == আরও দেখুন == কুমিল্লা আদর্শ সদর উপজেলা কোতোয়ালী থানা, কুমিল্লা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == আদর্শ সদর উপজেলার ইউনিয়ন
