'''পাথৈর''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত কচুয়া উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == কচুয়া উপজেলার পাথৈর ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে সাচার ইউনিয়ন, দক্ষিণে বিতারা ইউনিয়ন, পশ্চিমে মতলব দক্ষিণ উপজেলার নায়েরগাঁও উত্তর ইউনিয়ন কুমিল্লা জেলার দাউদকান্দি উপজেলার মারুকা ইউনিয়ন এবং উত্তরে কুমিল্লা জেলার দাউদকান্দি উপজেলার মোহাম্মদপুর পশ্চিম ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাথৈর ইউনিয়ন কচুয়া উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কচুয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৬০নং নির্বাচনী এলাকা চাঁদপুর-১ এর অংশ। == শিক্ষা প্রতিষ্ঠান == উচ্চ বিদ্যালয়২টি ১/বারৈয়ারা উচ্চ বিদ্যালয় ২/পাথৈর উচ্চ বিদ্যালয় প্রাথমিক বিদ্যালয় সংখ্যা ৮টি দাখিল মাদ্রাসা ২টি ১/বড়দৈল দাখিল মাদ্রাসা ২/গভরখোলা দাখিল মাদ্রাসা এবং একটি বিখ্যাত কওমী মাদ্রাসা আছে নাম ফতেবাপুর কাওমীমাদ্রাসা == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == এই ইউনিয়নের ছোট-বড় বাজার সংখ্যা মোট ৫টি যথাক্রমেঃঃ ১/মধুপুর বাজার ২/বারৈয়ারা বাজার ৩/তারিনি বাজার ৫/পাথৈর বাজার। == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == ১/ প্রয়াত রফিকুল ইসলাম রনি(সাবেক সংসদ সদস্য) ২/মোহাম্মদ মোশাররফ ৩/রাসেল মিয়াজি ৪/মিজানুর রহমান (সাবেক সচিব) ৫/ইঞ্জিনিয়ার জসীম == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান: মোস্তাফিজুর রহমান জুয়েল == আরও দেখুন == কচুয়া উপজেলা, চাঁদপুর কচুয়া থানা, চাঁদপুর চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == (চাঁদপুর) উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
