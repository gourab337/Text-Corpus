'''মালিগাঁও''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == মালিগাঁও ইউনিয়ন পূর্বে মোহাম্মদপুর পূর্ব ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == দাউদকান্দি উপজেলার পূর্বাংশে মালিগাঁও ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে মোহাম্মদপুর পশ্চিম ইউনিয়ন, উত্তর-পশ্চিমে জিংলাতলী ইউনিয়ন, উত্তরে ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন, পূর্বে ইলিয়টগঞ্জ দক্ষিণ ইউনিয়ন চান্দিনা উপজেলার সুহিলপুর ইউনিয়ন এবং দক্ষিণে চাঁদপুর জেলার কচুয়া উপজেলার সাচার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মালিগাঁও ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
