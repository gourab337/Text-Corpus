'''ফটিকছড়ি বালিকা পাইলট উচ্চ বিদ্যালয়''' বাংলাদেশের চট্টগ্রাম জেলার একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == প্রতিষ্ঠানটি চট্টগ্রাম জেলার ফটিকছড়ি পৌরসভার বিবিরহাট এলাকায় অবস্থিত। == ইতিহাস == ১৯৫৮ সালে এটি প্রতিষ্ঠিত হয়। == ব্যবস্থাপনা == বিদ্যালয় পরিচালনার জন্য জনাব আলহাজ্ব মোহাম্মদ নাসির উদ্দিনকে সভাপতি করে ১০ সদস্যের একটি পরিচালনা পরিষদ রয়েছে। == শিক্ষকবৃন্দ == বিদ্যালয়ের প্রধান শিক্ষক সৈয়দ মোহাম্মদ ইউসুফ। এছাড়া আরো ২৮ জন শিক্ষক-শিক্ষিকা প্রতিষ্ঠানে কর্মরত আছেন। == অবকাঠামো == বিদ্যালয় মূল ভবনটি একটি দ্বিতল ভবন। == শিক্ষা কার্যক্রম == এটি একটি বালিকা মাধ্যমিক বিদ্যালয়। বর্তমানে প্রায় দেড় হাজার শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == ফলাফল কৃতিত্ব == বিগত বছরের পাশের হার ৯৮%। == কৃতী শিক্ষার্থী == == আরো দেখুন == ফটিকছড়ি উপজেলার শিক্ষা প্রতিষ্ঠানের তালিকা *ফটিকছড়ি মডেল সরকারি প্রাথমিক বিদ্যালয় == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
