বাংলাদেশ এবং জিম্বাবুয়ে ক্রিকেট দল একে-অপরের বিরুদ্ধে ১৯৯৭ সাল থেকে ওডিআই ম্যাচ খেলে আসছে। দল দুইটি ২৬ অক্টোবর, ২০১৮ পর্যন্ত ৭২ বার মুখোমুখি হয়েছে তন্মধ্যে বাংলাদেশ জিতেছে ৪৪টি ম্যাচ এবং জিম্বাবুয়ে জিতেছে ২৮টি ম্যাচ। == ফলাফলের সংক্ষিপ্ত বিবরণ == মোট বাংলাদেশে জিম্বাবুয়েতে নিরপেক্ষ বাংলাদেশ জিতেছে ৪৪ ৩০ ১৩ জিম্বাবুয়ে জিতেছে ২৮ ১১ ১৫ টাই/ফলাফল হয়নি ম্যাচ খেলেছে ৭২ ৪১ ২৮ == ম্যাচের তালিকা == === বাংলাদেশে === ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- ---- === জিম্বাবুয়েতে === ---- === নিরপেক্ষ ভেন্যুতে === ---- ---- == তথ্যসূত্র == == আরও দেখুন == বাংলাদেশ জিম্বাবুয়ের মধ্যকার টেস্ট ম্যাচের তালিকা আন্তর্জাতিক ক্রিকেটের রেকর্ডসমূহ তালিকা ক্রিকেটে বাংলাদেশ ক্রিকেটে জিম্বাবুয়ে একদিনের আন্তর্জাতিক ক্রিকেট প্রতিপক্ষ ক্রিকেট তালিকা ক্রিকেট তালিকা বিভিন্ন বিষয়ভিত্তিক তালিকা
