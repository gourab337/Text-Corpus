'''পাঁচথুবী''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত কুমিল্লা আদর্শ সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == কুমিল্লা আদর্শ সদর উপজেলার পূর্বাংশে পাঁচথুবী ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে জগন্নাথপুর ইউনিয়ন কুমিল্লা সিটি কর্পোরেশন, পশ্চিমে কুমিল্লা সিটি কর্পোরেশন আমড়াতলী ইউনিয়ন, উত্তরে বুড়িচং উপজেলার বাকশীমূল ইউনিয়ন ভারতের ত্রিপুরা প্রদেশ এবং পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রশাসনিক কাঠামো == পাঁচথুবী ইউনিয়ন কুমিল্লা আদর্শ সদর উপজেলার আওতাধীন ৫নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম কুমিল্লা কোতোয়ালী থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৪নং নির্বাচনী এলাকা কুমিল্লা-৬ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == কুমিল্লা আদর্শ সদর উপজেলা কোতোয়ালী থানা, কুমিল্লা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == আদর্শ সদর উপজেলার ইউনিয়ন
