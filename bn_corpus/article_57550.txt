'''বুড়িরচর''' বাংলাদেশের নোয়াখালী জেলার অন্তর্গত হাতিয়া উপজেলার একটি ইউনিয়ন। == আয়তন == বুড়িরচর ইউনিয়নের আয়তন ৮৫ বর্গ কিলোমিটার, যা হাতিয়া উপজেলার মধ্যে তৃতীয় বৃহত্তম। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুযায়ী বুড়িরচর ইউনিয়নের জনসংখ্যা ৯৫,৬০০ জন। == অবস্থান সীমানা == হাতিয়া উপজেলার পূর্বাংশে বুড়িরচর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে চর ঈশ্বর ইউনিয়ন, উত্তর-পশ্চিমে তমরদ্দি ইউনিয়ন, পশ্চিমে সোনাদিয়া ইউনিয়ন, দক্ষিণে জাহাজমারা ইউনিয়ন এবং পূর্বে বঙ্গোপসাগর অবস্থিত। == প্রশাসনিক কাঠামো == বুড়িরচর ইউনিয়ন হাতিয়া উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হাতিয়া থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৩নং নির্বাচনী এলাকা নোয়াখালী-৬ এর অংশ। == শিক্ষা ব্যবস্থা == বুড়িরচর ইউনিয়নে ১টি কলেজ, ২টি মাধ্যমিক বিদ্যালয়, ২টি মাদ্রাসা ১১টি প্রাথমিক বিদ্যালয় রয়েছে। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == উপজেলা সদর থেকে মেইন রোডে বুড়িরচরের উদ্দেশ্যে বাস,টেক্সী,মোটর সাইকেল রিক্সায় যাতায়াত করা যায়। অন্যদিকে বয়ারচর চেয়ারম্যান ঘাট থেকে সী-ট্রাক থেকে এসে বাস যোগে সাগরিয়া যাতায়াত করা যায়। এছাড়াও জাহাজমারা থেকেও একই ভাবে যাতায়াত করা যায়। == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == রহমত বাজার আলাদী গ্রাম সূর্যমুখী == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান:- জিয়া আলী মোবারক কল্লোল == আরও দেখুন == হাতিয়া উপজেলা হাতিয়া থানা নোয়াখালী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
