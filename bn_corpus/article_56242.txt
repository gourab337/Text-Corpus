'''পরশুরাম পৌরসভা''' বাংলাদেশের ফেনী জেলার অন্তর্গত একটি পৌরসভা। == আয়তন == পরশুরাম পৌরসভার আয়তন ২২.৪০ বর্গ কিলোমিটার। == অবস্থান সীমানা == পরশুরাম উপজেলার মধ্য-পূর্বাংশে পরশুরাম পৌরসভার অবস্থান। পৌরসভার দক্ষিণে বক্স মাহমুদ ইউনিয়ন চিথলিয়া ইউনিয়ন, পশ্চিমে চিথলিয়া ইউনিয়ন মির্জানগর ইউনিয়ন এবং উত্তরে পূর্বে ভারতের ত্রিপুরা প্রদেশ অবস্থিত। == প্রতিষ্ঠাকাল == ২০০১ সালের ২২ এপ্রিল পরশুরাম পৌরসভা প্রতিষ্ঠিত হয়। == প্রশাসনিক এলাকা == পরশুরাম পৌরসভায় ৯টি ওয়ার্ড রয়েছে। পৌরসভার প্রশাসনিক কার্যক্রম পরশুরাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৫নং নির্বাচনী এলাকা ফেনী-১ এর অংশ। == যোগাযোগ ব্যবস্থা == পরশুরাম পৌরসভায় যোগাযোগের প্রধান সড়ক ফেনী-পরশুরাম সড়ক। সব ধরনের যানবাহনে যোগাযোগ করা যায়। == জনসংখ্যার উপাত্ত == ২০১১ সালের আদমশুমারি অনুযায়ী পরশুরাম পৌরসভার মোট জনসংখ্যা ২৯,৬৯১ জন। == জনপ্রতিনিধি == বর্তমান পৌর মেয়র: নিজাম উদ্দিন আহম্মদ চৌধুরী == আরও দেখুন == পরশুরাম উপজেলা পরশুরাম থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == জেলার পৌরসভা উপজেলা
