'''ঘাগুটিয়া''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত হোমনা উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == ঘাগুটিয়া ইউনিয়ন পূর্বে ঘাগুটিয়া পশ্চিম ইউনিয়ন নামে পরিচিত ছিল। == অবস্থান সীমানা == হোমনা উপজেলার ঘাগুটিয়া ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে হোমনা পৌরসভা নিলখী ইউনিয়ন, পূর্বে দুলালপুর ইউনিয়ন এবং উত্তরে পশ্চিমে তিতাস নদী জেলার বাঞ্ছারামপুর উপজেলার উজানচর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ঘাগুটিয়া ইউনিয়ন হোমনা উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম হোমনা থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == হোমনা উপজেলা হোমনা থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
