'''দিঘলী''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == দিঘলী ইউনিয়নের আয়তন ৪,২৫৮ একর (১৭.২৩ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ আদমশুমারী অনুসারে দিঘলী ইউনিয়নের জনসংখ্যা ২৪,৮৩৯ জন। এর মধ্যে পুরুষ ১১,৩৪৭ জন এবং নারী ১৩,৪৯২ জন। == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার পূর্বাংশে দিঘলী ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে হাজিরপাড়া ইউনিয়ন, পশ্চিমে মান্দারী ইউনিয়ন, দক্ষিণে কুশাখালী ইউনিয়ন এবং পূর্বে চর শাহী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দিঘলী ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ১৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম চন্দ্রগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৬নং নির্বাচনী এলাকা লক্ষ্মীপুর-৩ এর অংশ। == শিক্ষা ব্যবস্থা == দিঘলী ইউনিয়নের সাক্ষরতার হার ৪৯.৭০%। == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
