বাংলাদেশের কুমিল্লা জেলার অন্তর্গত তিতাস উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == তিতাস উপজেলার নারান্দিয়া ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে ভিটিকান্দি ইউনিয়ন, উত্তরে বলরামপুর ইউনিয়ন, পশ্চিমে কড়িকান্দি ইউনিয়ন জিয়ারকান্দি ইউনিয়ন এবং দক্ষিণে দাউদকান্দি উপজেলার জিংলাতলী ইউনিয়ন ইলিয়টগঞ্জ উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নারান্দিয়া ইউনিয়ন তিতাস উপজেলার আওতাধীন ৭নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম তিতাস থানার আওতাধীন। এটি জাতীয় সংসদের ২৫০নং নির্বাচনী এলাকা কুমিল্লা-২ এর অংশ। == শিক্ষা ব্যবস্থা == এই ইউনিয়নে মোট পাঁচটি সরকারি প্রাথমিক বিদ্যালয়, আটটি কিন্ডার গার্টেন এবং একটি উচ্চ বিদ্যালয় রয়েছে। নারান্দিয়া কলিমিয়া উচ্চ বিদ্যালয় অত্র ইউনিয়নের একমাত্র উচ্চ বিদ্যালয়। উচ্চ বিদ্যালয়টি অনেক প্রাচীন একটি উচ্চ বিদ্যালয় যা 1913 সালে প্রতিষ্ঠিত হয়। নারান্দিয়া কলিমিয়া উচ্চ বিদ্যালয়ের প্রতিষ্ঠাতা ডঃ মজিবুর রহমান খান। উচ্চ বিদ্যালয়টি অত্র উপজেলায় প্রতি বছর অনেক ভাল ফলাফল করে থাকে। == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == তিতাস উপজেলা তিতাস থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
