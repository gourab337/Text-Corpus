'''নাটাই দক্ষিণ''' বাংলাদেশের জেলার অন্তর্গত সদর উপজেলার একটি ইউনিয়ন। == আয়তন == নাটাই দক্ষিণ ইউনিয়নের আয়তন ২৬৮৩ একর (১০.৮৬ বর্গ কিলোমিটার)। == জনসংখ্যা == ২০১১ সালের আদমশুমারি অনুসারে নাটাই দক্ষিণ ইউনিয়নের মোট জনসংখ্যা ২৯,৭৯৬ জন। মোট পরিবার ৫,২৫৩টি। জনসংখ্যার ঘনত্ব প্রতি বর্গ কিলোমিটারে ২৭৪৪ জন। == ইতিহাস == == অবস্থান সীমানা == সদর উপজেলার পশ্চিমাংশে নাটাই দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে পৌরসভা নাটাই উত্তর ইউনিয়ন, উত্তরে নাটাই উত্তর ইউনিয়ন আশুগঞ্জ উপজেলার আশুগঞ্জ সদর ইউনিয়ন, পশ্চিমে সাদেকপুর ইউনিয়ন এবং দক্ষিণে তিতাস নদী নবীনগর উপজেলার নাটঘর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == নাটাই দক্ষিণ ইউনিয়ন সদর উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৫নং নির্বাচনী এলাকা এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতি ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == সদর উপজেলা সদর থানা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
