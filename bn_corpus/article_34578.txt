'''দুর্গাপুর পৌরসভা''' বাংলাদেশের ময়মনসিংহ বিভাগের নেত্রকোনা জেলার অন্তর্গত দুর্গাপুর উপজেলার অধীন একটি পৌরসভা। == আয়তন == এই পৌরসভার আয়তন ৫৯৬২ একর (২৪.১৩ বর্গ কিলোমিটার) দুর্গাপুর উপজেলা শহরের কাছাকাছি ৯টি ওয়ার্ড মিলে পৌরসভাটি গঠিত। == অবস্থান == নেত্রকোনা জেলা শহর থেকে ৫০কি.মি উত্তরে দুর্গাপুর পৌরসভা অবস্থিত। পৌরসভাটি সোমেশ্বরী নদীর তীরে অবস্থিত অপার সৌন্দর্যের লীলাভূমি এটি == প্রশাসনিক এলাকা == বাগিচাপাড়ায় গোরস্তান রোডের পাশে এটির প্রশাসনিক ভবন অবস্থিত। ভবনটি দোতলা বিশিষ্ট। == জনসংখ্যার উপাত্ত == দুর্গাপুর পৌরসভার সর্বমোট জনসংখ্যা- ২২৬৬১। পৌরসভায় লোক সংখ্যা ঘনত্ব প্রতি বর্গ কিলোমিটারে ৯১০ জন। == শিক্ষা == পৌরসভায় স্বাক্ষরতার হার ৫২.৩%। == অর্থনীতি == == কৃতী ব্যক্তিত্ব == কমরেড মণিসিংহ— বামপন্থী রাজনীতিবিদ, বাংলাদেশ কমিউনিস্ট পার্টি জালাল উদ্দীন তালুকদার প্রাক্তন সংসদ সদস্য বাংলাদেশ আওয়ামী লীগের রাজনীতিবিদ। == বিবিধ == == আরও দেখুন == == তথ্যসূত্র == দূর্গাপুর উপজেলা == বহিঃসংযোগ == বাংলাদেশের পৌরসভাসমূহের তালিকা জেলার পৌরসভা
