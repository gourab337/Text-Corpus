'''মারুকা''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত দাউদকান্দি উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == ইতিহাস == == অবস্থান সীমানা == দাউদকান্দি উপজেলার দক্ষিণাংশে মারুকা ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে দৌলতপুর ইউনিয়ন, উত্তরে বিটেশ্বর ইউনিয়ন, পূর্বে মোহাম্মদপুর পশ্চিম ইউনিয়ন, দক্ষিণ-পূর্বে চাঁদপুর জেলার কচুয়া উপজেলার পাথৈর ইউনিয়ন এবং দক্ষিণ-পশ্চিমে চাঁদপুর জেলার মতলব দক্ষিণ উপজেলার নায়েরগাঁও উত্তর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মারুকা ইউনিয়ন দাউদকান্দি উপজেলার আওতাধীন একটি ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম দাউদকান্দি থানার আওতাধীন। এটি জাতীয় সংসদের ২৪৯নং নির্বাচনী এলাকা কুমিল্লা-১ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == দাউদকান্দি উপজেলা দাউদকান্দি থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
