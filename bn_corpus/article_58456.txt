'''দালাল বাজার''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত লক্ষ্মীপুর সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লক্ষ্মীপুর সদর উপজেলার পশ্চিমাংশে দালাল বাজার ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে চর রুহিতা ইউনিয়ন; পূর্বে লক্ষ্মীপুর পৌরসভা দক্ষিণ হামছাদী ইউনিয়ন; উত্তরে দক্ষিণ হামছাদী ইউনিয়ন, রায়পুর উপজেলার বামনী ইউনিয়ন সোনাপুর ইউনিয়ন এবং পশ্চিমে রায়পুর উপজেলার চর মোহনা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == দালাল বাজার ইউনিয়ন লক্ষ্মীপুর সদর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লক্ষ্মীপুর সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == দালাল বাজার এন কে উচ্চ বিদ্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == দালাল বাজার ইউনিয়নের প্রধান হাট-বাজার হল দালাল বাজার। == দর্শনীয় স্থান == খোয়া সাগর দীঘি == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == লক্ষ্মীপুর সদর উপজেলা লক্ষ্মীপুর সদর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
