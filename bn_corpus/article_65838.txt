'''ডৌবাড়ী ইউনিয়ন''' বাংলাদেশের সিলেট জেলার গোয়াইনঘাট উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == == ইতিহাস == এই ইউনিয়নটি ২০১১ সালে নতুন ডৌবাড়ী ইউনিয়ন নামে পরিচিতি লাভ করে == প্রশাসনিক এলাকা == ডৌবাড়ী বলেশ্বর নিহাইন সাতকুড়ী কান্দি হাতির কান্দি তুড়ালী কান্দি কাঠাল কুড়ি কান্দি == আয়তন জনসংখ্যা == আয়তনঃ ৩৫ বর্গ কি: মিটার। জনসংখ্যাঃ ভোটার সংখ্যা ২৫৮৩০ জন। == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার :''' ২৫.৫ %। '''শিক্ষা প্রতিষ্ঠাণ''' রেজিষ্টার সরকারি প্রাথমিক বিদ্যালয় ৩টি বেসরকারী স: প্রা: বিদ্যালয় ৬টি মাধ্যমিক বিদ্যালয় টি মাদ্রাসা ৬টি উচ্চমাধ্যমিক বিদ্যালয় ২টি == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == '''বর্তমান চেয়ারম্যান-''' চেয়ারম্যানগণের তালিকা == আরও দেখুন == ১নং রুস্তমপুর ইউনিয়ন ২নং পশ্চিম জাফলং ইউনিয়ন ৩নং পুর্ব জাফলং ইউনিয়ন == তথ্যসূত্র == == বহিঃসংযোগ == গোয়াইনঘাট উপজেলা -বাংলাপিডিয়া ডৌবাড়ী ইউনিয়ন -বাংলাদেশ জাতীয় তথ্য বাতায়ন। উপজেলার ইউনিয়ন
