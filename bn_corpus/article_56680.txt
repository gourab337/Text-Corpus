'''চর মজলিশপুর''' বাংলাদেশের ফেনী জেলার অন্তর্গত সোনাগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাগাজী উপজেলার চর মজলিশপুর ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণ-পূর্বে বগাদানা ইউনিয়ন, পূর্বে ফেনী সদর উপজেলার ধলিয়া ইউনিয়ন, উত্তরে ফেনী সদর উপজেলার বালিগাঁও ইউনিয়ন, উত্তর-পশ্চিমে দাগনভূঁইয়া উপজেলার মাতুভূঁইয়া ইউনিয়ন, পশ্চিমে দাগনভূঁইয়া উপজেলার দাগনভূঁইয়া ইউনিয়ন এবং দক্ষিণ-পশ্চিমে ছোট ফেনী নদী নোয়াখালী জেলার কোম্পানীগঞ্জ উপজেলার চর পার্বতী ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর মজলিশপুর ইউনিয়ন সোনাগাজী উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাদেশের ইউনিয়ন নিবন্ধ উপজেলার ইউনিয়ন
