'''দশঘর ইউনিয়ন''' বাংলাদেশের সিলেট জেলার অন্তর্গত বিশ্বনাথ উপজেলার একটি ইউনিয়ন। == অবস্থান সীমানা == সিলেট জেলার বিশ্বনাথ উপজেলা সদর হতে ১০ কিঃমিঃ দক্ষিণ পশ্চিনে ৮নং দশঘর ইউনিয়ন পরিষদ অবস্হিত। == ইতিহাস == == গ্রাম সমূহ == == আয়তন জনসংখ্যা == == শিক্ষার হার শিক্ষা প্রতিষ্ঠাণ == '''শিক্ষার হার-''' ৩৮.৯৮ == দর্শনীয় স্থান == শাহ চান্দ শাহ কালুর মাজার == কৃতী ব্যক্তিত্ব == স্বাদক শাহ সুফী সৈয়দ আশুশাহ -মরমী কবি শাহ মোঃ আব্দুল মুকিম শাহ -মরমী দরবেশ মোঃ মখলিছুর রহমান -ভাষা সাহিত্যিক == জনপ্রতিনিধি == বর্তমান চেয়ারম্যান- ''মোহাম্মদ সফিক উদ্দিন আহমদ'' তালিকা '''ক্রম নং''' নাম''' '''সময়কাল''' মোঃ আমিন উদ্দিন (প্রয়াত) মোঃ রাশীদ আলী মৌলভী (প্রয়াত) মোঃ শফিকুর রহমান (প্রয়াত) মোঃ আবুল কালাম রফু মিয়া ''মোহাম্মদ সফিক উদ্দিন আহমদ'' -বর্তমান == আরও দেখুন == বিশ্বনাথ উপজেলা সিলেট জেলা সিলেট বিভাগ == তথ্যসূত্র == উপজেলার ইউনিয়ন
