'''জনার্দ্দনপুর উচ্চ বিদ্যালয়''' বাংলাদেশের একটি শিক্ষা প্রতিষ্ঠান। == অবস্থান == বাংলাদেশের চট্টগ্রাম জেলার অন্তর্গত মীরসরাই উপজেলার দুর্গাপুর ইউনিয়নের জনার্দ্দনপুর গ্রামে বিদ্যালয়টি অবস্থিত। == ইতিহাস == ১৯৭৪ সালে বিদ্যালয়টি প্রতিষ্ঠিত হয়। == পরিচালনা ব্যবস্থা == == == বিদ্যালয়ের প্রধান শিক্ষক জনাব মোহাম্মদ সাহাব উদ্দীন। বর্তমানে সাত শতাধিক শিক্ষার্থী বিদ্যালয়ে অধ্যয়নরত আছে। == অবকাঠামো == বিদ্যালয় ভবনটি একটি দ্বিতল ভবন। == কার্যক্রম == বিদ্যালয়ে সহ-শিক্ষা ব্যবস্থা রয়েছে। == কৃতিত্ব ফলাফল == বিগত বছরের পাশের হার ৯৯%। == আরও দেখুন == মীরসরাই উপজেলার শিক্ষা প্রতিষ্ঠান == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার শিক্ষা প্রতিষ্ঠান
