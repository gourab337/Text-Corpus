'''বাকই উত্তর''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লালমাই উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লালমাই উপজেলার সর্ব-পশ্চিমে বাকই উত্তর ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পূর্বে বাগমারা দক্ষিণ ইউনিয়ন, পূর্বে পেরুল উত্তর ইউনিয়ন, দক্ষিণে লাকসাম উপজেলার বাকই ইউনিয়ন, উত্তর-পশ্চিমে বরুড়া উপজেলার গালিমপুর ইউনিয়ন এবং উত্তরে বরুড়া উপজেলার শিলমুড়ি দক্ষিণ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == বাকই উত্তর ইউনিয়ন লালমাই উপজেলার আওতাধীন ৯নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লালমাই থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৮নং নির্বাচনী এলাকা কুমিল্লা-১০ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == নূরপুর বাজার ভাবক পাড়া বাজার কাপসতলা বাজার। == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লালমাই উপজেলা লালমাই থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
