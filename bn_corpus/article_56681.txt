বাংলাদেশের ফেনী জেলার অন্তর্গত সোনাগাজী উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == সোনাগাজী উপজেলার উত্তরাংশে মঙ্গলকান্দি ইউনিয়নের অবস্থান। ইউনিয়নের উত্তর-পশ্চিমে বগাদানা ইউনিয়ন, পশ্চিমে চর দরবেশ ইউনিয়ন, দক্ষিণে মতিগঞ্জ ইউনিয়ন, পূর্বে আমিরাবাদ ইউনিয়ন এবং উত্তরে ফেনী সদর উপজেলার ধলিয়া ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মঙ্গলকান্দি ইউনিয়ন সোনাগাজী উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম সোনাগাজী থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৭নং নির্বাচনী এলাকা ফেনী-৩ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == সোনাগাজী উপজেলা সোনাগাজী থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == বাংলাদেশের ইউনিয়ন নিবন্ধ উপজেলার ইউনিয়ন
