'''রামগড়''' বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত রামগড় উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রামগড় উপজেলার সর্ব-পশ্চিমে রামগড় ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে উত্তরে পাতাছড়া ইউনিয়ন; পশ্চিমে চট্টগ্রাম জেলার ফটিকছড়ি উপজেলার বাগানবাজার ইউনিয়ন, রামগড় পৌরসভা ভারতের ত্রিপুরা প্রদেশ এবং উত্তরে রামগড় পৌরসভা মাটিরাঙ্গা উপজেলার মাটিরাঙ্গা ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == রামগড় ইউনিয়ন রামগড় উপজেলার আওতাধীন ১নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রামগড় থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == রামগড় উপজেলা রামগড় থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
