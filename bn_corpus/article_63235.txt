'''সুবিদপুর পূর্ব''' বাংলাদেশের চাঁদপুর জেলার অন্তর্গত ফরিদগঞ্জ উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফরিদগঞ্জ উপজেলার উত্তর-পূর্বাংশে সুবিদপুর পূর্ব ইউনিয়নের অবস্থান। ইউনিয়নের পশ্চিমে সুবিদপুর পশ্চিম ইউনিয়ন, দক্ষিণে গুপ্টি পূর্ব ইউনিয়ন, পূর্বে হাজীগঞ্জ উপজেলার গন্ধর্ব্যপুর দক্ষিণ ইউনিয়ন বড়কুল পূর্ব ইউনিয়ন, উত্তরে হাজীগঞ্জ উপজেলার বড়কুল পশ্চিম ইউনিয়ন এবং উত্তর-পশ্চিমে হাজীগঞ্জ উপজেলার হাজীগঞ্জ সদর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == সুবিদপুর পূর্ব ইউনিয়ন ফরিদগঞ্জ উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফরিদগঞ্জ থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৩নং নির্বাচনী এলাকা চাঁদপুর-৪ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == ফরিদগঞ্জ উপজেলা ফরিদগঞ্জ থানা চাঁদপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
