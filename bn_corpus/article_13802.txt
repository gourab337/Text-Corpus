বাংলাদেশের খাগড়াছড়ি জেলার অন্তর্গত মহালছড়ি উপজেলার একটি ইউনিয়ন। == আয়তন == ক্যায়াংঘাট ইউনিয়নের আয়তন ১২,৮০০ একর (৫৭.৮০ বর্গ কিলোমিটার)। == জনসংখ্যা == == অবস্থান সীমানা == মহালছড়ি উপজেলার উত্তর-পূর্বাংশে ক্যায়াংঘাট ইউনিয়নের অবস্থান। ইউনিয়নের দক্ষিণে মুবাছড়ি ইউনিয়ন মহালছড়ি ইউনিয়ন, পশ্চিমে মাইসছড়ি ইউনিয়ন, উত্তরে খাগড়াছড়ি সদর উপজেলার কমলছড়ি ইউনিয়ন এবং পূর্বে রাঙ্গামাটি জেলার লংগদু উপজেলার আটারকছড়া ইউনিয়ন মাইনীমুখ ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == ক্যায়াংঘাট ইউনিয়ন মহালছড়ি উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম মহালছড়ি থানার আওতাধীন। এটি জাতীয় সংসদের ২৯৮নং নির্বাচনী এলাকা পার্বত্য খাগড়াছড়ি এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == মহালছড়ি উপজেলা মহালছড়ি থানা খাগড়াছড়ি জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
