'''মুদাফফরগঞ্জ দক্ষিণ''' বাংলাদেশের কুমিল্লা জেলার অন্তর্গত লাকসাম উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == লাকসাম উপজেলার মুদাফফরগঞ্জ দক্ষিণ ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে মুদাফফরগঞ্জ ইউনিয়ন, পূর্বে কান্দিরপাড় ইউনিয়ন, দক্ষিণ-পূর্বে মনোহরগঞ্জ উপজেলার মৈশাতুয়া ইউনিয়ন, দক্ষিণে মনোহরগঞ্জ উপজেলার ঝলম উত্তর ইউনিয়ন, পশ্চিমে চাঁদপুর জেলার শাহরাস্তি উপজেলার রায়শ্রী দক্ষিণ ইউনিয়ন রায়শ্রী উত্তর ইউনিয়ন এবং উত্তর-পশ্চিমে বরুড়া উপজেলার লক্ষ্মীপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == মুদাফফরগঞ্জ দক্ষিণ ইউনিয়ন লাকসাম উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম লাকসাম থানার আওতাধীন। এটি জাতীয় সংসদের ২৫৭নং নির্বাচনী এলাকা কুমিল্লা-৯ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == জনপ্রতিনিধি == == আরও দেখুন == লাকসাম উপজেলা লাকসাম থানা কুমিল্লা জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন
