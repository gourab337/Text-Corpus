'''চর মোহনা''' বাংলাদেশের লক্ষ্মীপুর জেলার অন্তর্গত রায়পুর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == রায়পুর উপজেলার মধ্যাংশে চর মোহনা ইউনিয়নের অবস্থান। ইউনিয়নের পূর্বে সোনাপুর ইউনিয়ন রায়পুর পৌরসভা, উত্তরে রায়পুর ইউনিয়ন, পশ্চিমে দক্ষিণ চর আবাবিল ইউনিয়ন উত্তর চর বংশী ইউনিয়ন, দক্ষিণ-পশ্চিমে দক্ষিণ চর বংশী ইউনিয়ন এবং দক্ষিণ-পূর্বে লক্ষ্মীপুর সদর উপজেলার চর রুহিতা ইউনিয়ন দালাল বাজার ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == চর মোহনা ইউনিয়ন রায়পুর উপজেলার আওতাধীন ৩নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম রায়পুর থানার আওতাধীন। এটি জাতীয় সংসদের ২৭৫নং নির্বাচনী এলাকা লক্ষ্মীপুর-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == == জনপ্রতিনিধি == == আরও দেখুন == রায়পুর উপজেলা রায়পুর থানা লক্ষ্মীপুর জেলা == তথ্যসূত্র == == বহিঃসংযোগ == উপজেলার ইউনিয়ন বাংলাদেশের ইউনিয়ন নিবন্ধ
