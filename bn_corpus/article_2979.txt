FTP(File Transfer Protocol) FTP এর পুরো abbreviation হল ফাইল ট্রান্সফার প্রোটোকল (File Transfer Protocol)। ফাইল ট্রান্সফার প্রোটোকল (File Transfer Protocol) বা এফ টি পি নেটওয়ার্কে ফাইল আদান প্রদানের একটি জনপ্রিয় বহুল প্রচলিত পদ্ধতি। এতে কমপক্ষে দুটি কম্পিউটারের অংশগ্রহণ প্রয়োজন হয়, একটি অন্যটি কমন FTP প্রোগ্রাম হল "Fetch" যা Mac অপারেটিং সিস্টেমে কাজ করে এবং WS_FTP বা Serv-U যেটি Windows কাজ করে। যাই হোক আপনি চাইলে ওয়েব ব্রাউজার বা উইন্ডোজ এক্সপ্লোরার ব্যবহার করে FTP server প্রবেশ করতে পারেন। তবে জনপ্রিয় পদ্ধতি হল FTP Client যেমন FileZilla ব্যবহার করা। বেশিরভাগ FTP server কোন ইউজার নেম বা পাসওয়ারড লাগে না লগ ইন করার জন্য। অন্য সার্ভারগুলোতে আপনাকে নির্দিষ্ট ইউজার নেম পাসওয়ার্ড ব্যবহার করে প্রবেশ করতে হবে। and data transfer=== Illustration of starting passive connection using port 21 FTP may run in ''active'' or ''passive'' mode, which determines how the data connection is established. বিজ্ঞান
