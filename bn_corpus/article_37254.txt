বাংলাদেশের ফেনী জেলার অন্তর্গত ফেনী সদর উপজেলার একটি ইউনিয়ন। == আয়তন == == জনসংখ্যা == == অবস্থান সীমানা == ফেনী সদর উপজেলার পাঁচগাছিয়া ইউনিয়নের অবস্থান। ইউনিয়নের উত্তরে শর্শদি ইউনিয়ন, পূর্বে ফেনী পৌরসভা, দক্ষিণ-পূর্বে বালিগাঁও ইউনিয়ন, দক্ষিণে দাগনভূঁইয়া উপজেলার জায়লস্কর ইউনিয়ন, দক্ষিণ-পশ্চিমে দাগনভূঁইয়া উপজেলার রাজাপুর ইউনিয়ন এবং পশ্চিমে দাগনভূঁইয়া উপজেলার সিন্দুরপুর ইউনিয়ন অবস্থিত। == প্রশাসনিক কাঠামো == পাঁচগাছিয়া ইউনিয়ন ফেনী সদর উপজেলার আওতাধীন ২নং ইউনিয়ন পরিষদ। ইউনিয়নের প্রশাসনিক কার্যক্রম ফেনী সদর থানার আওতাধীন। এটি জাতীয় সংসদের ২৬৬নং নির্বাচনী এলাকা ফেনী-২ এর অংশ। == শিক্ষা ব্যবস্থা == == শিক্ষা প্রতিষ্ঠান == পশ্চিম বিজয় সিংহ উচ্চ বিদ্যালয় পাঁচগাছিয়া জেড খান উচ্চ বিদ্যালয় পাঁচগাছিয়া বালিকা উচ্চ বিদ্যালয় মাদার কেয়ার ইন্টারন‍্যাশনাল স্কুল পাঁচগাছিয়া সরকারি প্রাথমিক বিদ্যালয় এলাহীগঞ্জ সরকারি প্রাথমিক বিদ‍্যালয় মাথিয়ারা সরকারি প্রাথমিক বিদ‍্যালয় == যোগাযোগ ব্যবস্থা == == খাল নদী == == হাট-বাজার == == দর্শনীয় স্থান == == কৃতী ব্যক্তিত্ব == সফিউল্লাহ, সাবেক রাষ্ট্রদূত বদিউর রহমান, সাবেক সচিব মোছলেউদ্দিন, চেয়ারম্যান, পাটকল অধিদপ্তর কাজী কিবরিয়া, চিত্রশিল্পী সুমাইয়া আন্দালিব কাজী == জনপ্রতিনিধি == == আরও দেখুন == ফেনী সদর উপজেলা ফেনী সদর থানা ফেনী জেলা == তথ্যসূত্র == == বহিঃসংযোগ == সদর উপজেলার ইউনিয়ন
